L'ultima versione di Editeur puo essere trovata sul Worldwide Web, a l'indirizzo : http://www.studioware.com

Per definire il comportamento di Editeur al lancio, scelga "Opzioni" "Opzioni", poi imposti le opzioni volute nel gruppo "Al lancio".

Editeur permette di lavorare con file Windows, Dos, Mac, o Unix. Scelga il formato nel dialogo "Apri", o lasci Editeur determinare il formato scegliando "Auto".

Editeur permette di lavorare non soltanto con caratteri di larghezza fissa, ma anche con caratteri di larghezza proporzionale. Scelga i caratteri in "Opzioni" "Caratteri schermo".

Il dialogo "Apri" permette l'appertura di file molteplici nella directory corrente.

Si può cambiare i filtri di file per difetto utilizzando "Opzioni" "Opzioni", poi scegliando "Filtri di file".

I dialoghi "Apri" e "Salva con nome" permettono di creare, dare un nuovo nome o sopprimere directories e file, come nella Gestione risorse.

Il caricamento dei file può essere interotto ad ogni momento premendo il tasto Esc o il pulsante Annulla.

Si può registrare tutti i file in una sola volta scegliendo "File" "Salva tutto". Soli i file modificati saranno riscritti.

Si può selezionare colonne o rettangoli scegliando l'opzione appropriata con il pulsante destro del mouse.

Per aggiungere il testo selezionato agli Appunti, premere i tasti Ctrl+A o il pulsante "Aggiungi".

Per spostare testo utilizzando il OLE Drag and Drop, selezionarlo, fare clic dentro mantenendo il pulsante sinistro premuto, tirare il mouse fino al posto dove vuole spostare il testo, poi rilasciare il pulsante del mouse.

Per tirare e lasciare il testo senza sopprimere l'originale (copy Drag and Drop), premere il tasto Ctrl prima di fare clic nel testo e mantenere il tasto premuto tirando.

Nei dialoghi Ricerca, Rimpiazza, e Ricerca file, può specificare una tabulazione utilizzando $t, un null utilizzando $0, e un $ utilizzando $$.

Editeur permette di ricercare un testo in una directory e tutte le sue directory, aprendo tutti i file contenenti questo testo.

Dizionari moteplici possono essere utilizzati in una sola verifica dell'ortografia. Entri semplicemente i nomi degli dizionari nell'elenco e scelga i dizionari da utilizzare. Dizionari sono disponibili per l'Inglese US e Britannico, il Tedesco, il Francese e l'Italiano.

Si può creare dizionari personali per la verifica d'ortografia : dentro Editeur, crei un file testo vuoto, digiti una parola per linea, registri il file con l'estenzione .dic, poi entri il suo nome nell'elenco dei dizionari personali utilizzando il comando "Verifica l'ortografia".

Utilizzando "Opzioni" "Strumenti", si può specificare strumenti esterni che potrà lanciare dal menu "Strumenti".

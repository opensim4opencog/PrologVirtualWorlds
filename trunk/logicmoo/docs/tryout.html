Player Name:

Create a user name and password.  You will be asked to create a MUD/MOO character.  You can change character's race and some other attributes at a latter time.

 

 

Identify your character and you will be asked again for your password (the same one from the MOO).   You web-theme setting is based on your character's Job.  (The 'Mage' theme is the only one being tested now)   The web interface primary function is to make things that would not be easily done over telnet simpler.  Such as exploring  the object model and room layouts.  The rooms and objects present themselves on the web as S-Expressions and minor bits of editing are possible.  This will increase as time goes on and feature requests are fulfilled.

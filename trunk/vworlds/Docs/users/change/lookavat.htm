Viewing Another Avatar's Profile

Each avatar in a virtual world is represented by a graphical image and a profile. The profile is seen when you click on that avatar for more details.

The information seen in another avatar's profile includes a description and detail information. How much of the detail information is seen is determined by the settings set by that avatar. This information includes:

  • Where from
  • Home page
  • Gender
  • Real name
  • Email alias
  • Age
This information is set to public, private, or available only to other avatars on the Friends list.

To see the profile of another avatar:

  1. In the graphical view of the world, right-click the avatar, then click (Name's) Profile. The Profile dialog box displays a description of the avatar.

    - OR -

    Left-click on the name of the avatar in the Avatars pane. Click (Name's) Profile.

  2. In the Profile dialog box, click Details. The profile information for the avatar displays the information that you are permitted to see. This information is controlled by that avatar and whether you are on their Friends list, or if they set the information to Private.

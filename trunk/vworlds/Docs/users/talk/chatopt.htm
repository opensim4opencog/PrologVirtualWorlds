Changing the Information Displayed in the Chat Window

Conversations are not the only subject matter displayed in the Chat window of the Microsoft Virtual Worlds client. The Chat window shows emotes from other avatars, gestures, Virtual World events, and Virtual World system messages.

Virtual World events are actions that occur within the virtual world environment that are broadcast to the room at large. For example, avatars connecting, disconnecting, dropping or taking objects are all events.

Virtual World system messages are information including, but not limited to, error messages and security violations on objects. For example, if you tried to modify an object that was created by a wizard or an owner, and you are rated resident, you are going to see a security violation system message in your Chat window.

To turn off the gestures in the Chat window:

  1. In the Virtual Worlds client, click Options on the menu bar. The Options dialog box appears.
  2. Click Show gestures.

To turn off displaying events in the Chat window:

  1. In the Virtual Worlds client, click Options on the menu bar. The Options dialog box appears.
  2. Click Show events.

To turn off dislaying system messages in the Chat window:

  1. In the Virtual Worlds client, click Options on the menu bar. The Options dialog box appears.
  2. Click Show system messages.

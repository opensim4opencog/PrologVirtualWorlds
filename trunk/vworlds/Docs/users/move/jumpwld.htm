Moving Between Worlds

Travel is simple in the virtual world environment. You can switch from one world to another at the click of a mouse. When you leave a world, you disappear, and after the room loads, you appear in the new one.

In Microsoft Virtual Worlds, you can "teleport" to different virtual worlds with the Directory dialog box. Worlds are listed by the server and world name, number of users, and any informational websites that may be related. You can sort the columns by clicking the title of the column. This is useful for locating the world that contains the most people, or the fewest.

The web site can list any protocols, history, download size, or other information about the world.

Remember, it is usually good manners to announce your intent to leave before departing a conversation.

To move to a new world:

  1. In Virtual Worlds, click Directory from the menu bar. The Directory dialog box appears.
  2. Click the name of the world that you wish to enter.

- OR -

  1. In Virtual Worlds, click Log Off from the menu bar. The Start screen appears.
  2. Click Directory. The Directory dialog box appears.
  3. Click the name of the world that you wish to enter.

You can also click on the web site for that world, if one is listed, to see more information on that world.

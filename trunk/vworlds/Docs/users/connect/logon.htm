Logging Into and Logging Off Virtual Worlds

Microsoft Virtual Worlds uses two programs to display a virtual world: the Virtual Worlds Server application, which stores all the worlds, and the Virtual Worlds Client application, where you can select worlds to visit. Before you can log on to any virtual world, the Virtual Worlds Server application must be running, either on your computer or a computer on a network. The Server application contains the worlds you can visit.

To log into a virtual world:

  1. On the Start menu, choose Progams, then click Microsoft Virtual Worlds.
  2. Select Virtual Worlds.

    The Microsoft Virtual Worlds Start screen appears.

  3. If this is your first visit, enter an avatar nickname. Avatar names can be 3 to 12 alphanumeric characters. You can add spaces by using the underscore character.
  4. Enter a destination, server name first, then a / character, then the world name. Click Go.

    - OR -

    You can also choose a world from the Directory dialog box. Click Directory to see the list of worlds online at the moment, and click on the world you want to visit.

The Virtual Worlds client is loaded and you are logged into a virtual world. You can talk to other people, manipulate objects, move around the space, or change your avatar's profile.

You can run multiple instances of the Virtual Worlds Client application to run multiple avatars, but you can put only one specific avatar into a specific world at any time. Your avatars are specific to your computer.

To log off of a virtual world:

  1. Click Log Off from the Virtual Worlds menu bar.
  2. Click OK to navigate away from the web page.

    The Virtual Worlds Start screen appears, and you can choose a different avatar or world, or close the Start screen.

If you are in a virtual world and decide to change worlds, you can click Directory to select a different world and teleport there. If you select a new world, you are automatically logged off from the old world and logged on to the new world.

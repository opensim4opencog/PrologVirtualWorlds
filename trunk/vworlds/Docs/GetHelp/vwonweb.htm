Virtual Worlds on the Web

The Microsoft Virtual Worlds web site is the place to check for files to download, information posted on the newsgroup, and news about the Virtual Worlds platform. The web site is located at http://vworlds.research.microsoft.com.

Download Files

You can install the various components of Virtual Worlds from the Virtual Worlds web site (including the SDK, client, server, and help files).

The Virtual Worlds Newsgroup

Virtual Worlds has a newsgroup that you can access from the web site to discuss the platform with other people interested in Virtual Worlds.

The Virtual Worlds newsgroup is a mutually-supporting group of developers. Once you join the newsgroup you can:

  • Exchange ideas and information with other developers, world builders, and Virtual Worlds fans in a threaded discussion area.
  • Ask questions of the general audience and the Virtual Worlds user assistance team.
  • Download code samples and example worlds.
  • Obtain news, updates, and bug reports about Virtual Worlds.

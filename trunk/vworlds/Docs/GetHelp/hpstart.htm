Getting Additional Help

Microsoft Virtual Worlds complements the architecture platform with a full set of documentation, including:

  • The User's Guide Start Page provides information you need to use Virtual Worlds: information about starting, talking, moving, and so on.

  • The User Interface Start Page contains all the context-sensitive help; for every dialog box and major window in the Virtual Worlds interface, there is a corresponding topic. Most of these items have a help button that opens a window for information about that item, while you are in Virtual Worlds.

  • The Programmer's Guide Start Page is our beginning/intermediate programmer's kit that explains the basics of programming script in Virtual Worlds. You can use the interface for a large portion of the tasks, but a passing knowledge of Visual Basic Script is very useful.

  • The API Start Page is the full reference of our object model and architecture. You can use this reference to customize your own world and objects.

  • The Server Reference Start Page helps you set up and use the Virtual Worlds Server to run worlds.

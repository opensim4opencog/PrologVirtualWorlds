Installation Start Page

Purpose

Microsoft Virtual Worlds is separated into two main programs, the client program and the server program. The Virtual Worlds client is used to access the worlds created, and to modify the contents in the worlds. The Virtual Worlds server application, which does not have to reside on the same computer, runs the worlds, controlling access to the worlds (online or offline, private or public), and creating, saving or deleting those worlds.

You can place the client installation on each computer that you want to access the worlds. Generally, you need only one server, unless you want to build worlds on the same computer as the client, and move the world later to the public server.

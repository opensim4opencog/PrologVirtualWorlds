Client Hardware and Software Requirements

Applications built on the Microsoft Virtual Worlds platform consist of client and server components. Both can run on the same machine, although this is not recommended for performance reasons. Multiple clients can run on the same machine. Each world runs on a single server, and each server can host multiple worlds.

Hardware requirements do vary greatly with the type of application and number of concurrent users so utilize these as recommended starting points.

Hardware Requirements, Preferred

Microsoft Virtual Worlds runs best with these preferred hardware requirements (or greater):

  • A Pentium II-266MHz (or greater) CPU
  • 64 MB of RAM
  • A 16-bit color, 1024 X 768 display on your monitor.
  • Peripherals: Mouse, hardware accelerated video card, sound card.
  • 30 MB of free disk space for the Client-Only installation.
  • 10 MB of free disk space, minimum, for the Server-Only installation.
  • 60 MB of free disk space, minimum, for the Complete installation.

Please note that 3D graphics performance on hardware accelerated video cards varies from manufacturer to manufacturer. Refer to the support newsgroup for more information about recommended cards and chipsets.

Hardware Requirements, Minimum

Microsoft Virtual Worlds can run with these minimum hardware requirements:

  • A Pentium 200MHz (or greater) CPU
  • 32 MB of RAM
  • A 256 color, 800 X 600 display on your monitor.
  • Peripherals: Mouse.
  • 30 MB of free disk space for the Client-Only installation.
  • 10 MB of free disk space, minimum, for the Server-Only installation.
  • 60 MB of free disk space, minimum, for the Complete installation.

Minimum Supporting Software Requirements

  • Microsoft Windows 95, Windows 98, or Windows 2000.
  • Internet Explorer 5.0 (or greater) Full Installation (which installs Web Publishing Wizard, DirectX 6.1 (or greater)) - http://www.microsoft.com/ie/

All of these software prerequisites must be installed before installing Microsoft Virtual Worlds.

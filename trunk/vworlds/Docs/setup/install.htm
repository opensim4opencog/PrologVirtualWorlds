Installing Microsoft Virtual Worlds

Applications built on the Microsoft Virtual Worlds platform consist of client and server components. Both can run on the same machine, although this is not recommended for performance reasons. Multiple clients can run on the same machine. Each world runs on a single server, and each server can host multiple worlds.

There are two choices for installation options with Virtual Worlds:

  • Install the Virtual Worlds Server-Only (for running worlds on a Windows NT 4.0 network server computer).
  • Install the Virtual Worlds SDK (everything), the authoring client and tools, the basic sample client and other sample applications, and the server application.

Note: We no longer supply a client-only installation as applications written using the Virtual Worlds platform will have their own custom clients. We do supply a "basic client" sample as part of our SDK installation.

Installing the Virtual Worlds Server-Only installation

This installation only includes the essential components to run the Virtual Worlds Server and is for system administrators running NT4.0 servers which do not support the Virtual Worlds graphical clients (because appropriate versions of DirectX are not supported), or for adminstrators trying to minimize disk footprints for server applications on other operating systems (Windows 95, Windows 98, and Windows 2000).

Installing the Virtual Worlds SDK (complete)

The Virtual Worlds SDK installation includes the authoring client and tools, the basic sample client and other sample applications, the server application, online help, and sample avatars. This installation also includes the Avatar Construction Kit, a tool that helps you customize your avatar graphics.

You can use this installation to create your own worlds, run a Virtual Worlds server, author your own objects and modify the world, and socialize with others.

Downloading Virtual Worlds

Before you can download the Virtual Worlds program, you must be a member of the developer program. More details are available at http://vworlds.research.microsoft.com. Once you are a member, you can use your username and password to download the installation of your choice from the above URL.

You should download and install the supporting software before downloading Virtual Worlds. Please refer to the hardware and software requirement links below for each installation before downloading.

To download the appropriate Virtual Worlds installation:

  1. Navigate to http://vworlds.research.microsoft.com.
  2. Click the "Download" link in the left hand column.
  3. Enter your username and password to access the download page.
  4. Choose the installation you want to download:
    • Virtual Worlds Server-Only - vwserver.exe
    • Virtual Worlds Complete - sdkfull.exe
  5. Run the appropriate setup file and follow the installation instructions. (This will be one of the files listed above with the .exe extension.) You may need to restart your computer after the installation is finished.

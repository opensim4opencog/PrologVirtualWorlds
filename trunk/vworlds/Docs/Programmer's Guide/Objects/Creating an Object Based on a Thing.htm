Creating an Object

You can populate your world with objects based on the Thing exemplar, and on exemplars that inherit from Thing. These exemplars, which are called the core exemplars, include Artifact, Avatar, Host, Interworld Portal, Room, Room Portal, StateMachine, Thing, and WebPage.

You can create an object based on the Thing exemplar in either of two ways:

  • by using the Object Wizard
  • using script commands in the Command Window

To create an object using the Object Wizard:

  1. In the Object Explorer, right-click the current room or any object in the room.
  2. From the shortcut menu that displays, click Create New Thing....
  3. When the Object Wizard displays, select the type of object you wish to create. For more information, see Object Wizard.
  4. Click Create.

To create an object using script commands in the Command Window:

  1. Determine which exemplar provides the best match for the kind of instance you want to create.
  2. Use the CreateInstance method of the World core object to create the instance.
    CreateInstance takes three arguments: the name of the object, the exemplar to use in creating the object, and (optionally) the room or container in which the object is to be created. CreateInstance returns a reference to the created object. In both examples, the name of the created object is Toy, and it is an instance of the exemplar. In example 1, the object is created in the user's inventory, the default location if no other container is specified for the third argument. In example 2, the object is created in the same room as the user - the Container holding User.
' Example 1
Set NewThing = World.CreateInstance("Toy", "Artifact")
' Example 2
Set NewThing = World.CreateInstance("Toy", "Artifact", User.Container)

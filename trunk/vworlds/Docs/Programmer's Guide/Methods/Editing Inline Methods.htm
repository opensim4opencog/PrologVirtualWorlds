Editing Inline Methods

You can edit an inline method to change its name, its parameters, or its code. You can also change its client/server setting, specifying whether it runs on the client, or on the server, or on both. You can specify that a menu item should be added to the object that has this method.

Methods are listed along with properties in the Object Explorer.

To edit an inline method:

  1. In the Objects pane of the Object Explorer, select the object whose method you want to edit.
  2. In the Methods pane, select the method you want to edit. If the Module field contains something other than Inline Module, you will not be able to edit the method.
  3. Click the Expand Dialog button on the right of this line.
  4. When the Inline Method Editor opens, make any changes you want to make.
    You can:
    • change the method parameters
    • change the code in the method body
    • add a menu item to the object, so that the method is invoked by clicking a menu item on its shortcut menu
    • change the Client/Server setting in the Execute on selection box
  5. Click OK.

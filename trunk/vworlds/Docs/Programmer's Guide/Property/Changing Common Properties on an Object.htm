Changing Common Properties on an Object

You can use user interface tools provided in Virtual Worlds to change many properties on an object, using either the Property Pages for the object or the Object Explorer. The object's Property Pages offer access to a small number of properties, organized in groups. The Object Explorer provides access to all properties of an object.

If you are the owner of an object, you can change the values of its properties. You will not be able to change properties for objects that you do not own.

To change an object's Name property using its Property Pages:

  1. Right-click an object in the world.
  2. On the shortcut menu that appears, click Properties....
  3. When the Object Properties dialog box for the object opens, type a new name for the object (for example, Cedric) in the Name text box.
  4. Click OK.
  5. In the graphical window, right click the object, and then click More about Cedric.... The name you entered in step 3 appears on this menu option.

To change an object's Description property using its Property Pages:

  1. Right-click an object in the world.
  2. On the shortcut menu that appears, click Properties....
  3. When the Object Properties dialog box for the object opens, type a new description for the object in the Description text area. This is the text that displays when you click More about for the object.
  4. Click OK.

To change an object's Name property using the Object Explorer:

  1. In the Object Explorer, select an object.
  2. In the Properties pane to the right, scroll down until you see the Name property.
  3. Select the Name property.
  4. Click the Expand Dialog button that appears on the right end of the selected line.
  5. In the Advanced Text Editor that opens, type a new name for the object.
  6. Click OK. The object now has a new name.

To change an object's Scale property using the Object Explorer:

  1. In the Object Explorer, select an object.
  2. Scroll down the Properties pane until you see the object's Scale property.
  3. Select the Scale property.
  4. Click the Expand Dialog button that appears on the right side of the selected line.
  5. Type 2 into each of the input boxes (they are marked X, Y, and Z). This doubles the size of the object in each of the three dimensions.
  6. Click OK. The object now appears in the graphical window twice as large in each scale direction.

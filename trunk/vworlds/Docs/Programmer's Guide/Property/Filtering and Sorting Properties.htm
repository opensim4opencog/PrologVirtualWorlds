Filtering and Sorting Properties

When you open the Object Explorer, and you select an object in the current room, the Object Explorer displays all properties (and methods) on that object. Some of the displayed properties are inherited from the object's exemplar, while the rest are instance properties that are specific to that object. By filtering the properties, you can specify that only exemplar properties should be displayed, or only instance properties, or both types.

You can also sort properties by name, by type, or by value.

Property Filtering Options

When you filter properties, you are not deleting any of them; you are telling the Object Explorer not to display the ones that are filtered out. You can set filtering options separately for properties and methods.

Filtering options persist when you change the selected object and view another object's properties. For example, if you have changed the filtering so that only instance properties are visible, you must change the filtering back in order to see the exemplar properties in any other object in your virtual world. By default, all properties (exemplar and instance) are displayed.

To display only instance properties for an object:

  1. Right-click anywhere in the object's Properties pane.
  2. If a check mark is displayed on Show Exemplar Properties, click this menu item to turn this option off. (Note: if no check mark is displayed, you are already seeing only instance properties.)

To display only exemplar properties for an object:

  1. Right-click anywhere in the object's Properties pane.
  2. If a check mark is displayed on Show Instance Properties, click this menu item to turn this option off. (Note: if no check mark is displayed, you are already seeing only exemplar properties.)

To display all properties for an object:

  1. Right-click anywhere in the object's Properties pane.
  2. If either the Show Instance properties menu item or the Show Exemplar Properties menu item is unchecked, click that menu item to turn that option on. (Note: if neither check mark is displayed, you will need to carry out these steps twice, once for each menu item, to turn on both options.)

Property Sorting Options

You can sort an object's properties in any of three ways:

  • by name
  • by type
  • by value

The first and second sorting options are done alphabetically. The third sorts values with blanks first, then numbers in numerical order, and then strings in alphabetic order. Sorting options persist. When you select another object, its properties are sorted using the option you selected previously. By default, properties are sorted by name.

To sort properties by name:

  1. Select the object whose properties you want to sort.
  2. In the Properties pane for the object, click the Name tab.

To sort properties by module:

  1. Select the object whose properties you want to sort.
  2. In the Properties pane for the object, click the Type tab.

To sort properties by subroutine name:

  1. Select the object whose properties you want to sort.
  2. In the Properties pane for the object, click the Value tab.

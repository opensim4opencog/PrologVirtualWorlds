Programmer's Guide Start Page

Purpose

The Virtual Worlds Programmer's Guide covers various topics related to creating and manipulating objects and programming behaviors in a virtual world.  The Technical Overviews attempt to provide a solid understanding of basic system concepts, and the Programming Objects section provides a more detailed look at object creation and manipulation, both through the user interface and the object API.

The topics in this guide, when combined with the information in the API Reference will enable the world developer to create custom worlds rich with complex behaviors.

AcceptEnter

 Event Handler

An event-handling method that is called on a Thing when it is about to enter a container using MoveInto. AcceptEnter is a member of Thing.  

Syntax

Sub AcceptEnter(pThing)
    statements
End Sub

The AcceptEnter method syntax has these parts:
Part Type Use Description
pThing Thing Required The container to which the object will move.

Return Value

This method returns a Boolean.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object is about to change containers using MoveInto. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Function AcceptEnter(NewContainer)
    Dim MoveOK
    '(Calls the parent's AcceptEnter method)
    MoveOK = This.InvokeSuperMethod(NewContainer)
    AcceptEnter = MoveOK 
End Function

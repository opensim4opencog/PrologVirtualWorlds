OnDeactivate

 Event Handler

An event-handling method that is called in response to the Deactivate event, which occurs when the last Avatar disconnects from a room.

Syntax

Sub OnDeactivate(pRoom)
    statements
End Sub

The OnDeactivate method syntax has these parts:
Part Type Use Description
pRoom Thing Required A reference to the room that the avatar is leaving.

The Connect event supports these additional handlers:
Event Handler Description
OnPeerDeactivate The handler for other objects in the same Container.
OnContainerDeactivate The handler for the object's Contents.
OnContentDeactivate The handler for the Container of the object.

Return Value

This method does not return a value.

Remarks

This method responds to an internal Deactivate event that occurs in Virtual Worlds when the last Avatar disconnects from a room. This method is not normally called by the user.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

This example is a server-side method that displays a message on the server when a room is deactivated. 

Sub OnDeactivate(theRoom)
    Trace "Deactivating " & theRoom.Name
    This.InvokeSuperMethod(theRoom)
End Sub

OnPortalArrive

 Event Handler

An event-handling method that is called in response to the PortalArrive event, which occurs when an object arrives in a region through a Portal.

Syntax

Sub OnPortalArrive(ArrivingObj, ThePortal)
    statements
End Sub

The OnPortalArrive method syntax has these parts:
Part Type Use Description
ArrivingObj Thing Required A reference to the object arriving through the portal.
ThePortal Portal Required The portal through which the object is arriving.

The PortalArrive event supports these additional handlers:
Event Handler Description
OnPeerPortalArrive The handler for other objects in the same Container.
OnContainerPortalArrive The handler for the object's Contents.
OnContentPortalArrive The handler for the Container of the object.

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when a file is dropped from the desktop into the world.

It is sometimes useful to override this method in order to display a welcome message or triggering other behaviors when a user arrives.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnPortalArrive(ArrivingObj, ThePortal)
    Trace ArrivingObj.Name & " just arrived."
    This.InvokeSuperMethod(Array(ArrivingObj, ThePortal))

End Sub

OnWorn

 Event Handler

An event-handling method that is called in response to the Worn event, which occurs when the Wear method of an Avatar is invoked.

Syntax

Sub OnWorn()
    statements
End Sub

The Connect event supports these additional handlers:
Event Handler Description
OnPeerWorn The handler for other objects on the Contents list of the Avatar that is wearing the object.
OnContainerWorn The handler for the Contents of the object being worn. For example, if object "A" contains objects named "A1" and "A2", this handler is fired on "A1" and "A2".
OnContentWorn The handler for the avatar that is wearing the object. Since the object that is being worn must already be in the container's Contents list, this handler is for the container (the Avatar) of the object that is being worn.

Return Value

This method does not return a value.

Remarks

This method responds to an internal Worn event that occurs in Virtual Worlds when the Wear method of an Avatar is invoked.

For an object to be worn, the object must already be added to the Contents list of an avatar. For more information on wearing objects, see the Wear topic.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnWorn()
    Trace "The Wear method has been invoked."
    This.InvokeSuperMethod
End Sub

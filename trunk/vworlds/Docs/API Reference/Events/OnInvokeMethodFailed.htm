OnInvokeMethodFailed

 Event Handler

An event-handling method called in response to the InvokeMethodFailed event, which occurs when InvokeMethod fails.

Syntax

Sub OnInvokeMethodFailed(bstrMethodName)
    statements
End Sub

The OnTell method syntax has these parts:
Part Type Use Description
bstrMethodName String Required The name of the method that has failed to fire.

Return Value

This method does not return a value.

Remarks

The most common reason for an invocation failure is that the method does not exist.

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an InvokeMethod call fails. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Sub OnInvokeMethodFailed(strMethodName)
    World.Trace CStr(strMethodName) & " method failed."
    This.InvokeSuperMethod(strMethodName)
End Sub

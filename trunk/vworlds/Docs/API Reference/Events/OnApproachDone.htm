OnApproachDone

 Event Handler

An event-handling method that is called in response to the ApproachDone event, which occurs when the Approach method has finished.

Syntax

Sub OnApproachDone()
    statements
End Sub

The OnApproachDone method syntax has these parts:
Part Type Use Description

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when the Approach method has finished. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

See the property PositionDuration for more information about how this method is used.

Example

Sub OnApproachDone()

'(Calls the parent's OnApproachDone method) 

This.InvokeSuperMethod

End Sub

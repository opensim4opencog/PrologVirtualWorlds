OnIntersecter

 Event Handler

An event-handling method that is called on an object when it collides with another object's boundary.

Syntax

Sub OnIntersecter(pIntersectee, fX, fY, fZ, iBoundaryIndex, iEdgeIndex, bTransition)
    statements
End Sub

The OnIntersecter method syntax has these parts:
Part Type Use Description
pIntersectee Thing Required The object whose boundary has been collided with.
fX Float Required The x-coordinate of the position of the intersection.
fY Float Required The y-coordinate of the position of the intersection.
fZ Float Required The Z-coordinate of the position of the intersection.
iBoundaryIndex Integer Required The index of the boundary with the cell that is intersected.
iEdgeIndex Integer Required The index of the edge within the boundary.
bTransition Boolean Required A flag describing the type of cell intersection.

Return Value

This method does not return a value.

Remarks

The x, y, and z coordinates are relative to the frame containing the object (typically the room's frame).

A Transition value of 1 indicates the object attempted to cross a boundary from the outside; a value of 0 indicates that a boundary was exited. See the Boundary core object for more information.

If the IsInverse property on the boundary is True, the boundary marks a hole. A False value for this property denotes an ordinary boundary.

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds on collision by an avatar with the boundary around a Thing. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

This example displays the name of the object that the calling object has collided with. It also displays the system time of the collision.

Sub OnIntersecter (Object2, x, y, z, BoundaryIndex, EdgeIndex, Transition)
    World.Trace "The object has intersected with: " & Object2.Name & " At " & Time
    This.InvokeSuperMethod (Array(Object2, x, y, z, BoundaryIndex, EdgeIndex, Transition))
End Sub

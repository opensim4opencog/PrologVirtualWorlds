OnLoadGeometry

 Event Handler

An event-handling method that is called in response to the LoadGeometry event, which occurs whenever the geometry for an object is loaded on a client.

Syntax

Sub OnLoadGeometry(pNewFrame)
    statements
End Sub

The OnLoadGeometry method syntax has these parts:
Part Type Use Description
pNewFrame Thing Required Child frame containing geometry for the object.

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when the geometry of an object has changed, including when it is deleted. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

The geometry for an object will be loaded when that object enters a room the user is in, when the object is made visible (from being invisible), or when the geometry for that object is changed.

The geometry frame for the object (not the transform frame) is passed to OnLoadGeometry as a parameter.

Example

Sub OnLoadGeometry(newFrame)
    World.Trace newFrame.Name & " has been loaded."
    This.InvokeSuperMethod(newFrame)
End Sub

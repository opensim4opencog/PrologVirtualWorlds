AllowTransport

 Event Handler

An event-handling method that is called on a portal through which a Thing is attempting to leave.

Syntax

Sub AllowTransport(pThing)
    statements
End Sub

The AllowTransport method syntax has these parts:
Part Type Use Description
pThing Thing Required The object attempting to move through the portal.

Return Value

This method returns a Boolean.

Remarks

Returns a Boolean flag indicating whether a source portal will transport a given object. True will be returned if the portal allows transport; False otherwise.

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object tries to leave through a source portal.  This method and AcceptTransport can be overridden in script so that only users meeting specified conditions can transport through the portal. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Function AllowTransport(TheThing)
    Dim MoveOK
    MoveOK = This.InvokeSuperMethod(TheThing)
    AllowTransport = MoveOK 
End Function

AcceptTransport

 Event Handler

An event-handling method that is called on a Thing attempting to transport through a destination portal.

Syntax

Sub AcceptTransport(pThing)
    statements
End Sub

The AcceptTransport method syntax has these parts:
Part Type Use Description
pThing Thing Required The object attempting to move through the portal.

Return Value

This method returns a Boolean.

Remarks

Returns a Boolean flag indicating whether a destination portal will accept transport of a given object. True will be returned if the portal allows transport; False otherwise.

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object tries to arrive through a destination portal. This method and AllowTransport can be overridden in script so that only users meeting specified conditions can transport through the portal. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Function AcceptTransport(TheThing)
    Dim MoveOK
    '(Calls the parent's AcceptTransport method)
    MoveOK = This.InvokeSuperMethod(TheThing)   
    AcceptTransport = MoveOK 
End Function

OnTell

 Event Handler

An event-handling method that is called in response to the Tell event, which occurs when a Thing object has said something to another Thing object, using either of the communication methods Say or Emote.

Syntax

Function OnTell(Speaker, Hearers, Text, Mode)
    statements
End Function

The OnTell method syntax has these parts:
Part Type Use Description
Speaker Thing Required A reference to the Thing that is speaking.
Hearers PropertyList Required The list of Things being spoken to.
Text String Required The text to be said or emoted.
Mode Long Required The mode of communication (See Remarks.)

The Tell event supports these additional handlers:
Event Handler Description
OnPeerTell The handler for other objects in the same Container.
OnContainerTell The handler for the Contents of the object.
OnContentTell The handler for the object's container.

Return Value

Returns the (possibly modified) text that has been communicated, with a String.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds whenever someone whispers, says, or emotes a message.

The Mode parameter can have two values: 0 to Say something, and 1 to Emote something. For either of these modes, OnPeerTell is sent only to other objects within the same container as the invoking object.

There is a suite of On*Tell methods, which includes OnTell, OnContainerTell, OnContentTell, and OnPeerTell. OnTell, OnContentTell, and OnContainerTell can all return a modified version of the string being "told." OnPeerTell cannot. For more information on event handlers, see the Event Handlers overview.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Function OnTell(Speaker, Hearers, Text, Mode)
    Trace Speaker.Name & " said " CStr(Text)
    This.InvokeSuperMethod(Array(Speaker, Hearers, Text, Mode))

End Function

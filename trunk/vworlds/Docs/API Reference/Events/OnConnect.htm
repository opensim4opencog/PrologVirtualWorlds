OnConnect

 Event Handler

An event-handling method that is called in response to the Connect event, which occurs when an Avatar connects to a world.

Syntax

Sub OnConnect(pAvatar)
    statements
End Sub

The OnConnect method syntax has these parts:
Part Type Use Description
pAvatar Thing Required A reference to the avatar who is connecting.

The Connect event supports these additional handlers:
Event Handler Description
OnPeerConnect The handler for other objects in the same Container.
OnContainerConnect The handler for the object's Contents.
OnContentConnect The handler for the Container of the object.

Return Value

This method does not return a value.

Remarks

This method responds to an internal Connect event that occurs in Virtual Worlds when an Avatar connects to the world. This method is not normally called by the user.

In addition, an overriding OnConnect method can run on either the client or the server; however, due to timing issues, it should be specified as a server-side method.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

  

Sub OnConnect(theAvatar)
    Trace theAvatar.Name & " logged on at  " & Now()
    This.InvokeSuperMethod theAvatar
End Sub

OnEnter

 Event Handler

An event-handling method that is called in response to an Enter event, which occurs when an object has entered a new container.

Syntax

Sub OnEnter(Where, What)
    statements
End Sub

The OnEnter method syntax has these parts:
Part Type Use Description
Where Thing Required A reference to the container that has been entered.
What Thing Required A reference to The object that has entered the container.

The Enter event supports these additional handlers:
Event Handler Description
OnPeerEnter The handler for other objects in the same Container.
OnContainerEnter The handler for the object's Contents.
OnContentEnter The handler for the Container of the object.

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object has entered a container.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnEnter(Where, What)
    Trace What.Name & " has entered " & Where.Name
    This.InvokeSuperMethod (Array(Where, What))
End Sub

OnCreate

 Event Handler

An event-handling method that is called in response to a Create event.

Syntax

Sub OnCreate(pThing)
    statements
End Sub

The OnCreate method syntax has these parts:
Part Type Use Description
pThing Thing Required The object being created.

Return Value

This method does not return a value.

Remarks

The user does not usually call the OnCreate method; it responds to an internal event that occurs in Virtual Worlds when you create an object. If you create an exemplar using the OnCreate method, the method is called every time an instance of the exemplar is created. You must run this method on the server, not the client.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnCreate(TheThing)
    World.Trace TheThing.Name & " has been created."
    This.InvokeSuperMethod(TheThing)
End Sub

OnPositionDone

 Event Handler

An event-handling method that is called in response to the PositionDone event, which occurs when interpolation of the Position property has finished.

Syntax

Sub OnPositionDone()
    statements
End Sub

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when interpolation of the Scale property has finished. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Sub OnPositionDone()
    World.Trace " OnPositionDone has been fired."
    This.InvokeSuperMethod 
End Sub

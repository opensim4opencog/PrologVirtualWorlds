OnLeave

 Event Handler

An event-handling method that is called in response to the Leave event, which occurs immediately before an object exits its container.

Syntax

Sub OnLeave(OldContainer, LeavingObject)
    statements
End Sub

The OnLeave method syntax has these parts:
Part Type Use Description
OldContainer Thing Required A reference to the container the object is leaving.
LeavingObject Thing Required A reference to the object that is leaving.

The Leave event supports these additional handlers:
Event Handler Description
OnPeerLeave The handler for other objects in the same Container.
OnContainerLeave The handler for the leaving object's Contents.
OnContentLeave The handler for the Container of the object.

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds immediately before an object exits its container.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnLeave(OldRoom, LeavingAvatar)
    Trace LeavingObject.Name & " is leaving " & OldContainer.Name
    This.InvokeSuperMethod(Array(OldRoom, LeavingAvatar))
End Sub

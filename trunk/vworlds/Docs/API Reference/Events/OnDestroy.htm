OnDestroy

 Event Handler

An event-handling method that is called in response to a Destroy event.

Syntax

Sub OnDestroy(pThing)
    statements
End Sub

The OnDestroy method syntax has these parts:
Part Type Use Description
pThing Thing Required The object being destroyed.

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object is destroyed. This method can be overridden in script if you want to extend the behavior of the OnDestroy event handler. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

An overriding version of this event handler works best if it created as a server-side method, but it can be created as a client-side method, if necessary.

Example

Sub OnDestroy(TheThing)
    World.Trace TheThing.Name & " has been destroyed."
    This.InvokeSuperMethod(TheThing)
End Sub

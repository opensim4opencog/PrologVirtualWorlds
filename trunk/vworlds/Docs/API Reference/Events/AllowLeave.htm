AllowLeave

 Event Handler

An event-handling method that is called on a container when an object is about to change containers using the MoveInto method. AllowLeave is a member of Thing.  

Syntax

Sub AllowLeave(pThing)
    statements
End Sub

The AllowLeave method syntax has these parts:
Part Type Use Description
object Thing Required The container of the Thing attempting to leave.
pThing Thing Required The object that is attempting to leave.

Return Value

This method returns a Boolean, where True indicates the object has permission to leave the container.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object is about to change containers using MoveInto. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Function AllowLeave(OldThing )
    Dim MoveOK
    MoveOK = This.InvokeSuperMethod(OldThing)
    AllowLeave = MoveOK 
End Function

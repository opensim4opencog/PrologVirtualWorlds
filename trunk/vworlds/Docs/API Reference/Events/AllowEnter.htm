AllowEnter

 Event Handler

An event-handling method that is called on a container when an object is about to enter a container with MoveInto.

Syntax

Sub AllowEnter(pThing)
    statements
End Sub

The AllowEnter method syntax has these parts:
Part Type Use Description
pThing Thing Required The object seeking permission to enter the container.

Return Value

This method returns a Boolean.

Remarks

Returns a Boolean flag indicating whether to allow entry of an object into the calling object's contents. True will be returned if the object is allowed to enter, and False otherwise.

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when NewThing attempts to move into the contents of a new container with MoveInto.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Function AllowEnter(NewThing)
    Dim MoveOK
    MoveOK = This.InvokeSuperMethod(NewThing)
    AllowEnter = MoveOK 
End Function

OnUpDone

 Event Handler

An event-handling method that is called in response to the UpDone event, which occurs when interpolation of the Up property has finished.

Syntax

Sub OnUpDone()
    statements
End Sub

Return Value

This method does not return a value.

Remarks

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when interpolation of the Up property has finished.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnUpDone()
    World.Trace " OnUpDone has been fired."
    This.InvokeSuperMethod 
End Sub

OnRButtonDown

 Method

Method that is called when the right mouse button has been pressed. This method returns a Boolean, but use of the return value is optional.

Syntax

Sub OnRButtonDown(hWnd, x, y)
statements
End Sub

The OnRButtonDown method syntax has these parts:
Part Type Use Description
hWnd Integer Required The window in which the mouse event occurred.
x Integer Required The x-coordinate of the mouse pointer.
y Integer Required The y-coordinate of the mouse pointer.

Return Value

This event-handling method returns a Boolean.

Remarks

Returns a Boolean flag that indicates whether the right mouse button has been pressed. If it has been, the value is True; if not, it is False. The OnLButtonDown, OnLButtonUp, and OnRButtonUp methods are also available methods that can be overridden using the same technique.

The most common use of this method is to override it in a function or subroutine to fire additional methods when the right mouse button is pressed. For example, to detect when an object has been clicked, you can add an overidden version of this method to that object. Each time that object is right-clicked, this method is invoked. Use of the return value is optional.

Note:   To ensure that the internal procedures that are tied to this method are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

This example overrides the OnRButtonDown method and displays the mouse-pointer location when the method was fired. To use this example, add this subroutine to an object.

Sub OnRButtonDown(Window, x, y)
    World.Trace "Clicked at: " & CStr(x) & ", " & CStr(y)
    This.InvokeSuperMethod(Window, x, y)
End Sub

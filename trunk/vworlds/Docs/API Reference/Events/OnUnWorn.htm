OnUnWorn

 Event Handler

An event-handling method that is called in response to the UnWorn event, which occurs when the UnWear method of an Avatar is invoked.

Syntax

Sub OnUnWorn()
    statements
End Sub

The Connect event supports these additional handlers:
Event Handler Description
OnPeerUnWorn The handler for other objects on the Contents list of the Avatar that is unwearing the object.
OnContainerUnWorn The handler for the Contents of the object being unworn. For example, if object "A" contains objects named "A1" and "A2", this handler is fired on "A1" and "A2".
OnContentUnWorn The handler for the avatar that is unwearing the object. Since the object that is being unworn must already be in the container's Contents list, this handler is for the container (the Avatar) of the object that is being worn.

Return Value

This method does not return a value.

Remarks

This method responds to an internal UnWorn event that occurs in Virtual Worlds when the UnWear method of an Avatar is invoked. For more information on wearing objects, see the UnWear topic.

Note:   To ensure that the internal procedures that are tied to each event handler are invoked, you must call InvokeSuperMethod within the overridden subroutine. For more information on using this method, see the InvokeSuperMethod topic.

Example

Sub OnUnWorn()
    Trace "The UnWear method has been invoked."
    This.InvokeSuperMethod
End Sub

AcceptLeave

 Event Handler

An event-handling method that is called on a Thing when it is about to exit a container. AcceptLeave is a member of Thing.  

Syntax

Sub AcceptLeave()
    statements
End Sub

Return Value

This method returns a Boolean.

Remarks

Returns a Boolean flag indicating whether it is OK to leave an old Container. True will be returned if the object will permit its own exit, and False is returned otherwise.

This method is not normally called by the user; it responds to an internal event that occurs in Virtual Worlds when an object is about to leave its container via MoveInto. If you find it necessary to override this event-handling method, InvokeSuperMethod should also be called to maintain the existing functionality of the overridden method.

Example

Function AcceptLeave()
    Dim MoveOK
    '(Calls the parent's AcceptEnter method)
    MoveOK = This.InvokeSuperMethod
    AcceptLeave = MoveOK 
End Function

ModuleName

 Property of the Module Object

Returns the name of the specified Module object. ModuleName is a read-only String.  

Syntax

object.ModuleName

The ModuleName property syntax has these parts:
Part Type Use Description
object Module Required An object reference to a Module object.

Remarks

You can use this property when you want to find out the name of a particular module.

A module receives its name when it is created (using the CreateScriptModule method). After the module is created, its name can act as one of the parameters used in the CreateMethod method.  

Example

Dim myModule

Set myModule = World.Module("myScript")
myModuleName = myModule.ModuleName

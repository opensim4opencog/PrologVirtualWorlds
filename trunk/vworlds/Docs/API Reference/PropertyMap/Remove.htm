Remove

 Method of PropertyMap

Removes a collection item from a property map. Remove is a member of the PropertyMap collection. 

Syntax

object.Remove bstrName

The Remove method syntax has these parts:
Part Type Use Description
object PropertyMap Required An object reference to a PropertyMap collection.
bstrName String Required

The name of the item that you want to remove from the PropertyMap collection.

Return Value

This method does not return a value.

Remarks

With the Remove method, you can remove an item from the middle of a list by specifying the name of the key that you want to remove. 

Reshuffling the Collection After a Remove Action

The following process takes place within the Remove method: The Count property is adjusted to fill in the vacant point in the collection. For example, if a property map initially has four properties, and the third property is removed, then the property at the fourth position will be shifted down to the third position.

The Count property is adjusted to match the total count of properties in the property map. Therefore, you may want to avoid using the Count property as an iteration counter. If you include an increment or decrement method (such as the Add method) inside a loop, the Count property will be changed within your loop and an iteration counter depending on the value may not be accurate. 

    
              
 



Examples



 
   This example removes a property from a property map.


Sub OceanViewInnFeatures()
Dim myHotTub
Dim myStonehenge
Dim myBridge
Dim myOceanViewInnFeatures
Dim i
Dim myCount




'Establish the link to the object in the virtual world.

Set myHotTub = CreateInstance("Spa", "Artifact", User.Container)
Set myStonehenge = _
  CreateInstance("Stonehenge", "Artifact",  User.Container)
Set myBridge = _
  CreateInstance("Bridge", "Artifact",  User.Container)




'Create the collection.

Set myOceanViewInnFeatures = CreatePropertyMap
 
'Establish the link for the object to the collection. 
myOceanViewInnFeatures.Property("myHotTub") = "100 degrees"
myOceanViewInnFeatures.Property("myStonehenge") = True
myOceanViewInnFeatures.Property("myBridge") = "Golden Gate"

For Each i In myOceanViewInnFeatures 
    Trace i & " " & myOceanViewInnFeatures.Property(i)
Next


'Get the count of items in the collection.

myCount = myOceanViewInnFeatures.Count



'Remove the bridge from the collection. 
myOceanViewInnFeatures.Remove "myBridge"



'Get the new count of items in the collection.

myCount = myOceanViewInnFeatures.Count



'Display the collection again.

For i = 0 To myCount - 1
    Trace i & " " & myOceanViewInnFeatures.Property(i).Name
Next



'Display the new count of items in the collection.

Trace myOceanViewInnFeatures.Count
End Sub

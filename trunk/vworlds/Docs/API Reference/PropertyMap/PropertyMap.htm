PropertyMap

Collection

A PropertyMap object contains an unordered list of paired keys and values, known as key/value pairs. Once you have inserted a key/value pair (element) into a map, you can efficiently retrieve or delete the pair using the key to access it. You can also iterate over all the elements in the map.

A key/value pair maps unique keys to values. The key is a string that represents the name of the property in the property map. The value is the contents of the property. A property map uses the string that makes up the key instead of an index number to access the value of each property.

Using the Object

You can create an unordered list of pairs with the PropertyMap object by using the CreatePropertyMap method. 

Dim mySpanishNumbers
Set mySpanishNumbers = World.CreatePropertyMap

After you create a property map, you can add key/value pairs to it using the Property property. In the following example, the key/value pairs are English-Spanish word pairs for the first three numbers. For example, the key "three" is paired with the value "tres".

mySpanishNumbers.Property("one") = "uno"
mySpanishNumbers.Property("two") = "dos"
mySpanishNumbers.Property("three") = "tres"

To see the number of items in a property map, you can use the Count property. 

Total = mySpanishNumbers.Count

You can use the following code to iterate through the values of all key/value pairs stored in the property map. 

Dim myKey
For Each myKey In mySpanishNumbers
    Print myKey
Next

You can remove items from a property map using the Remove or RemoveAll methods.  

mySpanishNumbers.RemoveAll

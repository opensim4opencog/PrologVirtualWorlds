HeightUpper

 Property of Boundary

Sets or returns the vertical position of the upper edge of a Boundary in relation to the object to which the the boundary is assigned. HeightUpper is a read/write Float.

Syntax

object.HeightUpper = floatValue

The HeightUpper property syntax has these parts:
Parts Type Use Description
object Boundary Required An object reference that accesses a Boundary.
floatValue Float Required Any valid expression that evaluates to a Float value.

Remarks

The HeightUpper property establishes the y-axis location of the top of a Boundary. If the IsPassable property of a Boundary is set to False (default), the Boundary behaves as a solid object, allowing other objects to be placed on the top surface. Avatars, for example, can step onto a Boundary, however avatars are limited to stepping onto have a HeightUpper value of one unit or less.

When a Boundary is assigned to an object, the HeightUpper property is set to the upper edge of that object, as opposed to the x-z plane. The value of this property is relative to the center point of the object to which the boundary is assigned.

The HeightUpper property applies to all Boundary points as a whole. Individual points of a Boundary do not have separate HeightUpper properties.

Important:   After a Boundary is applied to an object, you must use the following syntax to set the value of the HeightUpper property: 

object.BoundaryList.Property(indexNumber).HeightUpper = floatValue

In the previous statement, object refers to the object that the Boundary is applied to, indexNumber corresponds to the index number of the Boundary, and floatValue refers to any valid expression that evaluates to a Float.

Note:   The value of the HeightUpper property must be greater than the value of the HeightLower property.

Examples

The first example creates a Boundary around an existing object and changes the HeightUpper property. The argument "myObject" is the name of the existing object being passed to the subroutine. 

Sub myBoundary(myObject)

Dim myFireHydrant
Dim myOuterEdge

'Obtain a reference to an existing object. 

Set myFireHydrant = User.Container.FindWithin(myObject)

'Create a four-sided outer boundary. 

Set myOuterEdge = World.CreateBoundary(Array(1,1, 1,-1, -1,-1, -1,1))

'Change HeightUpper property. 

myOuterEdge.HeightUpper = 4.32

'Apply the boundary to the existing object. 

myFireHydrant.BoundaryList.Add myOuterEdge

End Sub

Example 2

This example changes the HeightUpper property after the boundary has been assigned to an object. "myObject" is the name of an existing object. To use this example, substitute the name of an existing object for "myObject". 

Dim myOuterEdge

'Obtain a reference to the third boundary

'in an existing object's boundary list.
Set myOuterEdge = myObject.BoundaryList.Property(2)

'Change the HeightUpper property. 

myOuterEdge.HeightUpper = 7.65

Example 3

This example changes the HeightUpper property and then confirms the change in the Command Window. To use this example, substitute the name of an existing object for "myObject". 

myObject.BoundaryList.Property(2).HeightUpper = 7.543
Trace myObject.BoundaryList.Property(2).HeightUpper

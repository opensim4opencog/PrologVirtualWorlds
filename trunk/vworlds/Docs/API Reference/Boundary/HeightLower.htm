HeightLower

 Property of Boundary

Sets or returns the vertical position of the lower edge of a Boundary in relation to the object to which the the boundary is assigned. HeightLower is a read/write Float.

Syntax

object.HeightLower = floatValue

The HeightLower property syntax has these parts:
Parts Type Use Description
object Boundary Required An object reference that accesses a Boundary.
floatValue Float Required Any valid expression that evaluates to a Float value.

Remarks

When a Boundary is assigned to an object, the HeightLower property is set to the lower edge of that object, as opposed to the x-z plane. The value of this property is relative to the center point of the object to which the boundary is assigned.

The HeightLower property applies to all Boundary points as a whole. Individual points of a Boundary do not have separate HeightLower properties.

Important:   After a Boundary is applied to an object, you must use the following syntax to set the value of the HeightLower property: 

object.BoundaryList.Property(indexNumber).HeightLower = floatValue

In the previous statement, the term object refers to the object that the Boundary is applied to, indexNumber corresponds to the index number of the Boundary, and floatValue" refers to any valid expression that evaluates to a Float.

Note:   The value of the HeightLower property must be less than the value of the HeightUpper property.

Examples

The first example creates a Boundary around an existing object and changes the HeightLower property. The argument "myObject" is the name of the existing object being passed to the subroutine. 

Sub myBoundary(myObject)

Dim myFireHydrant
Dim myOuterEdge

'Obtain a reference to an existing object. 

Set myFireHydrant = User.Container.FindWithin(myObject)

'Create a four-sided outer boundary. 

Set myOuterEdge = World.CreateBoundary(Array(1,1, 1,-1, -1,-1, -1,1))

'Change the HeightLower property. 

myOuterEdge.HeightLower = -1.37

'Apply the boundary to the existing object. 

myFireHydrant.BoundaryList.Add myOuterEdge

End Sub

Example 2

This example changes the HeightLower property after the boundary has been assigned to an object. "myObject" is the name of an existing object. To use this example in the Command Window, substitute the name of an existing object for "myObject". 

Dim myOuterEdge

'Obtain a reference to the third boundary.

'in an existing object's boundary list.
Set myOuterEdge = myObject.BoundaryList.Property(2)

'Change the HeightLower property. 

myOuterEdge.HeightLower = -1.32

Example 3

This example changes the HeightLower property and then confirms the change in the Command Window. To use this example, substitute the name of an existing object for "myObject". 

myObject.BoundaryList.Property(2).HeightLower = -1.62

Rotate

 Method of Boundary

Rotates a Boundary around the center point of an object. Rotate is a member of the Boundary object.

Syntax

object.Rotate fAngle

The Rotate method syntax has these parts:
Parts Type Use Description
object Boundary Required An object reference that accesses a Boundary.
fAngle Float Required An expression that evaluates to a Float. The angle is measured in radians.

Return Value

This method does not return a value.

Remarks

The Rotate method uses radians to measure the degree of rotation. The maximum useful rotation range is 2 pi radians (about 360 degrees), making the usable float range from 0 through 6.28315.

This method bases the start point for rotation from the first point in the boundary array. If a boundary is created using the AutoGenerate method, the start point is 0.0 on the y-axis of the boundary coordinate system. If the boundary is custom generated using an array, the start point is the first point listed in the array. Each invocation of the Rotate method calculates the degree of rotation from the latest position of the first Vertex, so if a boundary is rotated once, the new start point is based on the current location of the first vertex. For more information on the boundary coordinate system, see the Boundary topic.

Examples

The first example creates a Boundary around an existing object and invokes the Rotate method to change the position of the boundary. The argument "myObject" is the name of the existing object being passed to the subroutine. 

Sub myBoundary(myObject)

Dim myFireHydrant
Dim myOuterEdge

'Obtain a reference to an existing object. 

Set myFireHydrant = User.Container.FindWithin(myObject)

'Create a four-sided outer boundary. 

Set myOuterEdge = World.CreateBoundary(Array(1,1, 1,-1, -1,-1, -1,1))

'Rotate the boundary. 

myOuterEdge.Rotate 1.2

'Apply the boundary to the existing object. 

myFireHydrant.BoundaryList.Add myOuterEdge

End Sub

Example 2

This example invokes the Rotate method on the third boundary, after that boundary has been assigned to an object. "myObject" is the name of an existing object reference. To use this example, substitute the name of an existing object for "myObject". 

Dim myOuterEdge

'Obtain a reference to the third boundary

'in an existing object's boundary list.
Set myOuterEdge = myObject.BoundaryList.Property(2)

'Rotate the boundary. 

myOuterEdge.Rotate 2.34

Example 3

This example rotates the third boundary without assigning the boundary to a variable. To use this example, substitute the name of an existing object for "myObject". 

myObject.BoundaryList.Property(2).Rotate 1.5

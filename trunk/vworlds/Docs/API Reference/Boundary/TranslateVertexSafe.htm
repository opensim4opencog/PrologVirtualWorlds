TranslateVertexSafe

 Method of Boundary

Moves an existing Vertex to a new location. TranslateVertexSafe is a member of the Boundary object.

Syntax

object.TranslateVertexSafe(nIndex, fX, fY)

The TranslateVertexSafe method syntax has these parts:
Parts Type Use Description
object Boundary Required An object reference that accesses a Boundary.
nIndex Long Required An expression that evaluates to a Long. Corresponds to the index number of a Vertex on a Boundary.
fX Float Required The new position of the vertex on the boundary x-axis.
fY Float Required The new position of the vertex on the boundary y-axis.

Return Value

This method returns a Boolean value.

Remarks

The TranslateVertexSafe method translates (moves) a Vertex only if the translation results in the boundary maintaining a simple polygon, as opposed to a complex polygon (See Figure 1). A Boundary must be a simple polygon to function properly. If the Vertex is moved to a position that results in the boundary maintaining a simple polygon, the TranslateVertexSafe method returns True. Otherwise, the method returns False, and the relocation is not executed.


Figure 1.

The new coordinates are relative to the point of origin of the object the boundary is assigned to. The values supplied only relate to the two-dimensional boundary coordinate system, not the 3-D world coordinate system. For more information on the coordinate system of boundaries, see the Boundary topic.

When a Vertex is moved using the TranslateVertexSafe method, the rendering and index order of the remaining points stays the same.

Examples

This example creates a Boundary around an existing object and invokes the TranslateVertexSafe method to move a specific Vertex. The argument "myObject" is the name of the existing object being passed to the subroutine. 

Sub myBoundary(myObject)

Dim myFireHydrant
Dim myOuterEdge
Dim myResult

'Obtain a reference to an existing object. 

Set myFireHydrant = User.Container.FindWithin(myObject)

'Create a boundary without a specified size. 

Set myOuterEdge = World.CreateBoundary

'Generate an eight-sided boundary. 

myOuterEdge.AutoGenerate myFireHydrant, 8

'Apply the boundary to the existing object. 

myFireHydrant.BoundaryList.Add myOuterEdge

'Move the third vertex. 

myResult = myFireHydrant.BoundaryList.Property(0).TranslateVertexSafe(2, 3, 3)

'Verify that the vertex was moved. 

Trace myResult

End Sub

Scale

 Method of Boundary

Sizes a Boundary around the center point of an object. Scale is a member of the Boundary object.

Syntax

object.Scale fX, fY

The Scale method syntax has these parts:
Parts Type Use Description
object Boundary Required An object reference that accesses a Boundary.
fX Float Required An expression that evaluates to a Float. Corresponds to the x-axis of the boundary coordinate system.
fY Float Required An expression that evaluates to a Float. Corresponds to the y-axis of the boundary coordinate system.

Return Value

This method does not return a value.

Remarks

The Scale method changes the size of an entire Boundary on the x-y plane of the boundary coordinate system. The values supplied in the parameters are percentages that are expressed as floats, where a value of 1.0 equals 100 percent (the current size of the boundary). For more information on the boundary coordinate system, see the Boundary topic.

The location of each point on a boundary is incremented or decremented by the values supplied, based on the center point of the boundary. You cannot scale the points individually using the Scale method. To move an individual point on a boundary, use the TranslateVertexSafe method.

Tip:   Although both parameters are required, there may be instances where you want to change only one parameter. In these cases, multiplying the parameter that you want to remain unchanged by a value of 1.0 (100 percent) will leave the value unchanged.

Examples

This example creates a Boundary around an existing object and invokes the Scale method to change the size of the boundary. The argument "myObject" is the name of the existing object being passed to the subroutine. 

Sub myBoundary(myObject)

Dim myFireHydrant
Dim myOuterEdge

'Obtain a reference to an existing object. 

Set myFireHydrant = User.Container.FindWithin(myObject)

'Create a four-sided outer boundary. 

Set myOuterEdge = World.CreateBoundary(Array(1,1, 1,-1, -1,-1, -1,1))

'Scale the boundary. 

myOuterEdge.Scale 2.0, 3.0

'Apply the boundary to the existing object. 

myFireHydrant.BoundaryList.Add myOuterEdge

End Sub

Example 2

This example invokes the Scale method after the boundary has been assigned to an object. "myObject" is the name of an existing object. To use this example, substitute the name of an existing object for "myObject". 

Dim myOuterEdge

'Obtain a reference to the third boundary

'in an existing object's boundary list.
Set myOuterEdge = myObject.BoundaryList.Property(2)

'Scale the x-axis of the boundary, leaving the y-axis unchanged. 

myOuterEdge.Scale 2.34, 1.0

Example 3

This example resizes the third boundary without assigning the boundary to a variable. To use this example, substitute the name of an existing object for "myObject". 

myObject.BoundaryList.Property(2).Scale 1.5, 2.0

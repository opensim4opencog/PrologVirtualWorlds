Translate

 Method of Boundary

Moves a Boundary in a specified direction relative to the current location. Translate is a member of the Boundary object.

Syntax

object.Translate fX, fY

The Translate method syntax has these parts:
Parts Type Use Description
object Boundary Required An object reference that accesses a Boundary.
fX Float Required An expression that evaluates to a Float. Corresponds to the x-axis of the boundary coordinate system.
fY Float Required An expression that evaluates to a Float. Corresponds to the y-axis of the boundary coordinate system.

Return Value

This method does not return a value.

Remarks

The Translate method moves an entire Boundary on the x-y plane of the boundary coordinate system.

The Translate method moves a Boundary relative to its current location. When a boundary is assigned to an object, the boundary is located relative to the center point of the object. So if a boundary is translated (moved) to a different location, it is still relative to the object that it is assigned to. If that object changes location, the boundary changes location as well, maintaining its relative location to the object.

For more information on the boundary coordinate system, see the Boundary topic.

Examples

This example creates a Boundary around an existing object and invokes the Translate method to move the entire boundary. The argument "myObject" is the name of the existing object being passed to the subroutine. 

Sub myBoundary(myObject)

Dim myFireHydrant
Dim myOuterEdge

'Obtain a reference to an existing object. 

Set myFireHydrant = User.Container.FindWithin(myObject)

'Create a four-sided outer boundary. 

Set myOuterEdge = World.CreateBoundary(Array(1,1, 1,-1, -1,-1, -1,1))

'Translate the boundary. 

myOuterEdge.Translate 2.0, 3.0

'Apply the boundary to the existing object. 

myFireHydrant.BoundaryList.Add myOuterEdge

End Sub

Example 2

This example demonstrates calling the Translate method after the boundary has been assigned to an object. "myObject" is the name of an existing object. To use this example, substitute the name of an existing object for "myObject". 

Dim myOuterEdge

'obtain a reference to the third boundary

'in an existing object's boundary list.
Set myOuterEdge = myObject.BoundaryList.Property(0)

'Translate the boundary. 

myOuterEdge.Translate 2.34, 1.0

Example 3

This example moves the boundary without assigning the boundary to a variable. To use this example, substitute the name of an existing object reference for "myObject". 

myObject.BoundaryList.Property(0).Translate 1.5, 2.0

Trace

 Method of World

Displays a string of numerals or text to the Command Window. Trace is a member of the World object.  

Syntax

object.Trace bstr

The Trace method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstr String Required The expression to be displayed.

Return Value

This method does not return a value.

Remarks

Even though the bstr parameter is of type String, the Trace method will evaluate numeric and other expressions in its operation. It can evaluate numeric expressions using any of the arithmetic or logical operators of VBScript, as well the mathematical functions. Trace can display strings concatenated by '+' or '&'. To lessen ambiguity when you use addition expessions, however, use '&' for this purpose.

Trace displays True and False for logical expressions, which is different from the behavior of Print.  

Example

World.Trace "3 + 5 = " & 3 + 5     'Displays 3 + 5 = 8
World.Trace 1 OR 2	           'Displays 3
World.Trace Hex(255)	           'Displays FF
World.Trace 3 > 2	           'Displays True 
World.Trace "My name is " & User.Name	'Displays My name is <your name>

Print

 Method of World

Displays a string of text or numerals to the Command Window. Print is a member of the World object.  

Syntax

object.Print varAnything

The Print method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
varAnything Variant Required The text or numeric expression to display.

Return Value

This method does not return a value.

Remarks

The Print method can be used to display text (character strings) or numeric expressions. In addition, an object reference can be supplied as the varAnything parameter, resulting in the display of the contents of the ToAscii property of that object.

Print can evaluate numeric expressions using any of the arithmetic or logical operators of VBScript, as well the mathematical functions.

Print can display strings concatenated by '+' or '&'. To lessen ambiguity with addition expessions, however, it is recommended that you use '&' for this purpose.  

Example

World.Print "3 + 5 = " & 3 + 5 	   'Displays 3 + 5 = 8
World.Print 1 OR 2		   'Displays 3
World.Print Hex(255)		   'Displays FF
World.Print 3 > 2		   'Displays -1  (indicates True - 0 is used for False
World.Print "My name is " & User.Name	  'Displays My name is <your name>

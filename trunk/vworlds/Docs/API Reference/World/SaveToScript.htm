SaveToScript

 Method of World

Saves the contents of a world to a text file. SaveToScript is a member of the World object.  

Syntax

object.SaveToScript bstrFilename

The SaveToScript method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstrFilename String Required The name of the file that will hold the data for the world.

Return Value

This method does not return a value.

Remarks

SaveToScript is provided mainly as a way to enable compatibility between different versions of Virtual Worlds. While it is possible to edit the script that is saved, it is not recommended that you do this.

To invoke this method you must be a Wizard or the owner of the world.  

Example

World.SaveToScript "myworld.vbs"

ShowWelcomeMessage

 Method of World

Displays a greeting to every user who enters the world for the first time. ShowWelcomeMessage is a member of the World object.  

Syntax

object.ShowWelcomeMessage 

The ShowWelcomeMessage method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Return Value

This method does not return a value.

Remarks

The ShowWelcomeMessage method displays the HTML page specified in the WelcomeURL property of the World object.

After this method acts, the FirstTimeUser key is preserved until the avatar leaves the world, so other methods within the world can check for it as well.

ShowWelcomeMessage is a client-side method.  

Example

World.ShowWelcomeMessage

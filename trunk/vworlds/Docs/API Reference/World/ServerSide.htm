ServerSide

 Property of World

A Boolean flag that indicates whether the current method is running on the server. ServerSide is a read/write Boolean.  

Syntax

To set this property:
object.ServerSide = BooleanValue

To return this property:
object.ServerSide

The ServerSide property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
BooleanValue Boolean Required True is the method is executing on the server.

Remarks

The server hosts a World object, as does each of the connected clients. On the server, the ServerSide property is initialized to True, but on each client it is initialized to False. For the currently executing method, ServerSide will be True if the method is executing on the server, and will be False if the method is executing on a client. This property is most useful when used inside methods that can be called from either the client or the server. By including code that determines where the method is running, the user can cause the method to take one course of action if it is running on the server, and another if it is running on a client.  

Example

Result = World.ServerSide

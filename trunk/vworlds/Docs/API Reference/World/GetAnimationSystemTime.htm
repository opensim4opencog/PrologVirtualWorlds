GetAnimationSystemTime

 Method of World

Returns the number of milliseconds that have elapsed since the world has been active. GetAnimationSystemTime is a member of the World object.  

Syntax

object.GetAnimationSystemTime

The AnimationSystemTime property syntax has these parts:
Part Type Use Description
object World Required An object that references a World object.

Return Value

This method returns a Long that represents the length of time that the world has been running.

Remarks

The time value stored by this property is marked each time the world becomes active. The passage of time is not recorded when the world is inactive.

You can use this property in conjunction with the GetAnimationStartTime method to determine the exact keyframe of an animation during run-time.

Example

This example determines how long an animation has been running and puts the number of milliseconds into the "myPositionClock" variable. 

Dim myPositionClock

myPositionClock = myBaseball.GetAnimationSystemTime - _
  myBaseball.GetAnimationStartTime("Position")

DefaultAvatarExemplar

 Property of World

Provides a basic template for custom default avatars. The DefaultAvatarExemplar is the starting point for all other avatars in the world. DefaultAvatarExemplar is a read/write Avatar.  

Syntax

object.DefaultAvatarExemplar = AvatarValue

The DefaultAvatarExemplar property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
AvatarValue Avatar Required The default exemplar for the avatar.

Remarks

All avatars in a world are based on the DefaultAvatarExemplar for the world. When you log on to a world, the avatar you create for that world is based on the DefaultAvatarExemplar. Virtual Worlds provides a basic DefaultAvatarExemplar that the Owner of a new world can use "as is" or use as a base for a custom DefaultAvatarExemplar.

The Owner of the world can create a custom DefaultAvatarExemplar that derives its standard attributes from the DefaultAvatarExemplar, but has special properties, methods, or characteristics that are unique to the custom avatar. Those special characteristics will be inherited by every avatar that logs on to the world.

It is important to note that only world owners can create a new DefaultAvatarExemplar that is derived from the original one. If you do create a custom DefaultAvatarExemplar, the following restrictions apply:

  • As the world owner you cannot set your avatar to the DefaultAvatarExemplar. Instead, add the all the properties or methods that you want for your custom avatar to your DefaultAvatarExemplar.

    • DefaultAvatarExemplar.AddProperty "IsInterworldTraveller", False
DefaultAvatarExemplar.AddProperty "IsInterworldPassportValid", False
DefaultAvatarExemplar.AddProperty "InterworldPassport"
  • The custom avatar cannot become a Host.
 

Example

This example creates a custom DefaultAvatarExemplar.

Sub CreateCustomDefaultAvatar()
Dim MyCustomDefaultAvatarExemplar
Dim CustomDefaultAvatar

'Create a new DefaultAvatarExemplar.
Set MyCustomDefaultAvatarExemplar = World.CreateExemplar("NewAvatar", _
	"Avatar")


'Add unique properties.
MyCustomDefaultAvatarExemplar.AddProperty "IsInterworldTraveller", False
MyCustomDefaultAvatarExemplar.AddProperty "IsInterworldPassportValid", False
MyCustomDefaultAvatarExemplar.AddProperty "InterworldPassport"

'Set the custom default avatar that you just created as the 'DefaultAvatarExemplar.
World.DefaultAvatarExemplar = MyCustomDefaultAvatarExemplar

'Verify that the new avatar is the default.
Trace World.DefaultAvatarExemplar.Name
End Sub

DestroyTimer

 Method of World

Destroys a previously-created timer. DestroyTimer is a member of the World object.  

Syntax

object.DestroyTimer pTimer

The DestroyTimer method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
pTimer Object Required The timer to be destroyed.

Return Value

This method does not return a value.

Remarks

This method stops the timer before destroying it.  

Example

World.DestroyTimer CountdownTimer

WorldName

 Property of World

A String that holds the name of the world. WorldName is a read-only String.  

Syntax

object.WorldName

The WorldName property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

WorldName is a system constant that cannot be changed by anyone.  

Example

To get the WorldName property:

   NameOfWorld = World.WorldName

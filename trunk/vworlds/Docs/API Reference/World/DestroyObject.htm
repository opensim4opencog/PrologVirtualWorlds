DestroyObject

 Method of World

Permanently removes an object from the world. DestroyObject is a member of the World object.  

Syntax

object.DestroyObject pthing

The DestroyObject method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
pthing Thing Required The object to destroy.

Return Value

This method does not return a value.

Remarks

This method will fail on attempts to delete the User, the user's Container, other avatars, and rooms. Be very careful when using this method, because any object for which this method succeeds will be permanently deleted from the world.

Note:   Invoking the DestroyObject method will change the object reference to a stub, changing the IsStub property of the Thing object to True.  

Example

The example code that follows destroys a previously-created object (named "Goner") that is in the user's room (User.Container). An object reference, MarkedObject, is obtained by using the Thing method FindWithin.

Dim MarkedObject
Set MarkedObject = User.Container.FindWithin("Goner")
World.DestroyObject MarkedObject

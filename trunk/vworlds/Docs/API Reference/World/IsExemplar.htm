IsExemplar

 Method of World

Determines whether or not an object is an exemplar. IsExemplar is a member of the World object.  

Syntax

object.IsExemplar(Object)

The IsExemplar method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
Object Thing Required The object to inspect.

Return Value

This method returns a Boolean.

Remarks

IsExemplar returns True if the object is an exemplar, and False if the object is an instance of an exemplar.  

Example

IsAnExemplar = World.IsExemplar (RaceCar)

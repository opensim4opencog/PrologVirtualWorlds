ShowURL

 Method of World

Displays the content of the specified URL. ShowURL is a member of the World object.  

Syntax

object.ShowURL bstrTitle, bstrURL [, cx] [, cy]

The ShowURL method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstrTitle String Required The title of the window.
bstrURL String Required The URL to display.
cx Integer Optional The width of the window in pixels. The default is 0.
cy Integer Optional The height of the window in pixels. The default is 0.

Return Value

This method does not return a value.

Remarks

This method displays the content at the specified URL. The upper left corner of the HTML window will be at the point (0, 0). If the optional cx and cy parameters are included, they specify the width and height of the window.

TheShowURL method displays only one window at a time. If the ShowURL method is called while a window created by a prior call to this method is still being displayed, it refreshes the window title (if different in the later call) and the window content, but ignores any changes in the window's position and size. To display more than one HTML window, use ShowURLEx.  

Example

World.ShowURL "A Window", "http://research.microsoft.com/vwg", 650, 400

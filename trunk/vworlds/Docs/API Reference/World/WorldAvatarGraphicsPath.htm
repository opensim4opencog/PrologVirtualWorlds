WorldAvatarGraphicsPath

 Property of World

A String that holds the path to the directory in which avatar graphics files are located. This path is world-specific. WorldAvatarGraphicsPath is a read-only String.  

Syntax

object.WorldAvatarGraphicsPath

The WorldAvatarGraphicsPath property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

Avatar graphics include sprite files (with .SPR file extensions) and sprite image files (with .GIF file extensions).

To find avatar graphics for a specific world, Virtual Worlds starts its search from the Local Content directory. The default path to the world avatar graphics directory is:

C:\Program Files\Microsoft Virtual Worlds\Local Content\Worlds\<World Name>\Avatar Graphics\.

This property is initialized when Virtual Worlds starts, as a combination of the contents of AvatarGraphicsPath and WorldName.  

Example

This example displays the WorldAvatarGraphicsPath property in the Command Window. 

Trace World.WorldAvatarGraphicsPath

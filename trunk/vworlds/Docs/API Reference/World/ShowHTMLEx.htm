ShowHTMLEx

 Method of World

Shows HTML in a pop-up window. ShowHTMLEx is a member of the World object.  

Syntax

object.ShowHTML bstrWinName, bstrTitle, bstrHTML, dispArg [, x] [, y] [, cx] [, cy]

The ShowHTML method syntax has these parts:
Part Type Use Description
object World Required An object reference to a World object.
bstrWinName String Required The name of the window.
bstrTitle String Required The title of the window.
bstrHTML String Required The HTML text to display.
dispArg Variant Required A DHTML object, such as that defined in an <Object> tag. Or, this can also be a Virtual Worlds object, such as World.User.
[x] Variant Optional The x-coordinate of the upper left corner of the window. The default value is 0.
[y] Variant Optional ] The y-coordinate of the upper left corner of the window. The default value is 0.
[cx] Variant Optional The width of the window in pixels. The default value is 0.
[cy] Variant Optional The height of the window in pixels. The default value is 0.

Return Value

This method does not return a value.

Remarks

This method displays HTML content in a modeless pop-up window, with the given window title and name. If the optional x, y, cx, or cy parameters are included, they specify the coordinates and width and height of the window. The default value for each of these parameters is 0, which means that the upper left corner of the window will be at the point (0, 0), and the window's width and height will be 0.

The DispArg parameter can be a DHTML object (such as an object defined in an <OBJECT> tag on the page), or it may be a Virtual Worlds object (such as World.User). Any Automation object matches the type of this parameter; the Helper object on the popped-up web page gives you access to that variable. If you do not need to pass an argument for DispArg, use Nothing.

If ShowHTMLEx is called more than once for the same window, it refreshes the window, but ignores any changes in the window's position and size.

The "Ex" in this method's name stands for "extended."  

Example

Dim HTMLStr
HTMLStr = " ... some HTML text ..."
World.ShowHTMLEx "Window1", "Edit", HTMLStr, Nothing, 20, 10, 150, 100

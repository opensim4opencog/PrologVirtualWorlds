CreateFrame

 Method of World

Creates a Virtual Worlds 3D frame object. CreateFrame is a member of the World object.  

Syntax

object.CreateFrame

The CreateFrame method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Return Value

This method returns a Frame.

Remarks

To create and work with frames in Microsoft Virtual Worlds, you need to understand the background information on frames in Virtual Worlds. In HTML, frames are used in a two-dimensional pane to divide a page on the screen. In Virtual Worlds, frames are used to define aspects of the world.

Defining Frame Relationships

The term frame is derived from an object's physical frame of reference. You can place objects in a scene by stating their spatial relationship to a relevant reference frame; they are not simply placed in world space.

A frame is used to position objects in a scene, and visuals (such as meshes) take their positions and orientation from frames. A scene is defined by a frame that has no parent frame; that is, a frame at the top of the hierarchy of frames. This frame is also sometimes called a scene frame or master frame. The scene defines the frame of reference for all of the other objects. In Virtual Worlds, frames are represented by the VWFrame object. The relationship between Virtual Worlds objects and their frames is described in the documentation for the GetFrame and GetGeometryFrame methods of the Thing exemplar.

The frames in a scene are arranged in a tree structure. Frames can have a parent frame and child frames. Child frames have positions and orientations relative to their parent frames. If the parent frame moves, the child frames also move. You can also remove frames from one parent frame and add them to another at any time by using the AddChild method of the VWFrame object. To remove a child frame entirely, use the DeleteChild method. To retrieve a frame's parent and children frames, use the GetChildren and GetParent methods. The MoveInto method of the Thing exemplar will also indirectly affect the frame hierarchy.  

Example

Set newFrame = World.CreateFrame

IsAvatar

 Method of World

Determines whether or not an object is an Avatar. IsAvatar is a member of the World object.  

Syntax

object.IsAvatar(pthing)

The IsAvatar method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
pthing Thing Required The Thing object reference to inspect.

Return Value

This method returns a Boolean.

Remarks

Use IsAvatar to differentiate between users (who have avatar profiles and connections) and other objects.

IsAvatar will return True for instances of subexemplars of Avatar (for example, Host).  

Example

In the following example, an object named Runner in the same room as the user (in User.Container) is being inspected to determine whether or not it is an Avatar. A True value for the IsAnAvatar variable means that Runner is an Avatar, while a False value means that it is not.

Dim ObjectRef
Set ObjectRef = User.Container.FindWithin("Runner")
IsAnAvatar = World.IsAvatar( ObjectRef )

WorldPath

 Property of World

A String that holds the path to the client's directory in which world checkpoint files are located. This path is world-specific. WorldPath is a read-only String.  

Syntax

object.WorldPath

The WorldPath property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

A checkpoint file (extension .VWC) uses a Virtual Worlds-specific binary file format to store all objects in a world, including their methods and properties. A checkpoint file is not directly readable.

This property is initialized with the value in the WorldPath entry of the Windows Registry (the subkey is: \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\V-Worlds\Paths).  

Example

To get the WorldPath property:

   ThePath = World.WorldPath

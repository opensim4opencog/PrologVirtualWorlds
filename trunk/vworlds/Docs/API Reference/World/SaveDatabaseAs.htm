SaveDatabaseAs

 Method of World

Saves checkpoint files. SaveDatabaseAs is a member of the World object.  

Syntax

object.SaveDatabaseAs bstrNewName 

The SaveDatabaseAs method syntax has these parts:
Part Type Use Description
object World Required An object reference to a World object.
bstrNewName String Required The new name of the database.

Return Value

This method does not return a value.

Remarks

To invoke this method, you must be the owner of the world, or a wizard. The checkpoint file (.vwc) is saved to your \Microsoft Virtual Worlds\Worlds directory.

Note:   Do not enter an extension for the filename. The extension .vwc is automatically appended to your filename.

Example

World.SaveDatabaseAs "Art Gallery"

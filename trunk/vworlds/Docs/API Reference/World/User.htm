User

 Property of World

Returns a reference to the Thing that represents the currently connected avatar. User is a read-only Thing.  

Syntax

object.User

The User property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

A common usage of the User property is to find an object that is in the same room as the User. For example, to obtain a reference to the object Rock5 (assumed to be in the Room with the User), type the following into the Command Window:

Set RockRef = World.User.Container.FindWithin("Rock5")

After this VBScript executes, RockRef refers to the object that is returned.

Example

The following example displays the name of your avatar in the Command Window.

Set Result = World.User
Trace Result.Name

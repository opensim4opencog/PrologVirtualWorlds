DefaultRoom

 Property of World

The default Room in a world. DefaultRoom is a read/write Room.  

Syntax

To set this property:
object.DefaultRoom = RoomValue

To return this property:
object.DefaultRoom

The DefaultRoom property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
RoomValue Room Required The default Room.

Remarks

This property specifies the room into which the user will be placed upon first entering the world.

If there is a value specified for the DefaultPortal property, new users are sent through it; otherwise, they are sent to the DefaultRoom.

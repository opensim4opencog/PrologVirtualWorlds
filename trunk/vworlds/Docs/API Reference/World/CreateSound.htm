CreateSound

 Method of World

Creates a new Sound object. CreateSound is a member of the World object.

Syntax

object.CreateSound(SoundURL, bIs3D, bIsLooping)

The CreateSound method syntax has these parts:
Parameters Type Use Description
object Sound Required An object reference that accesses a Sound object.
SoundURL String Required A valid URL that references a sound file.
bIs3D Boolean Required Set this parameter to True when the sound file contains 3D properties and False when it contains 2D properties.
bIsLooping Boolean Required Set this parameter to True when you want the sound file to continually play and False when you want to play the file once.

Return Value

This method returns a Sound object.

Remarks

You can play a Sound object by associating it with a Thing object. This object can be a speaker, tree, or another avatar. any artifact, avatar, state machine, or other object that is of type Thing can play a sound through the use of its Sound property.

You do this by setting the Sound property of the Thing object to your new Sound object as shown in the following statements.  

Dim myTree
Dim myBirdSound

myBirdSound = "bird.wav"
Set myTree = User.Container.FindWithin("Oak Tree")

myTree.Sound = myBirdSound

You can stop the sound by setting the IsLooping property of the Sound object to False. 

Dim mySong

mySong = myTree.Sound
mySong.IsLooping = False

When you set the bIsLooping and bIs3D parameters for the CreateSound, you determine the state of the IsLooping and bIs3D properties for the Sound object, such as the "mySong" Sound object in the previous statements.

Note:   Rooms can contain default Sound objects. The object that you create plays at the same time as the default sound. You can turn off a sound by setting the sound file to an empty string. 

User.Container.Sound = ""

Example

This example creates a Sound object and associates it with an object in the world.

To run this example: You must have a sound file named bird.wav and a tree named Oak Tree with which to associate it in your world. Or, you may substitute a file and object of your choice. 


Sub AddBirdSong()
Dim myTree
Dim mySong
Dim mySound

mySound = "file://C:/Program Files/Microsoft Virtual
Worlds/Local Content/SDK/Library/Sounds/bird.wav"

'Create a 3D, continually looping sound object. 

Set mySong = CreateSound(mySound, True, True)
Set myTree = User.Container.FindWithin("Oak Tree")

myTree.Sound = mySong

End Sub

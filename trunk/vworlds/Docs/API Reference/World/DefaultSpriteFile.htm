DefaultSpriteFile

 Property of World

A String that holds the name of the default sprite file. DefaultSpriteFile is a read/write String.  

Syntax

To set this property:
object.DefaultSpriteFile = StringValue

To return this property:
object.DefaultSpriteFile

The DefaultSpriteFile property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
StringValue String Required The path to the default .spr file for an avatar.

Remarks

When Microsoft Virtual Worlds starts, this property is initialized to "default.spr". The World Owner can change this property to default to any valid sprite file. Any user entering a world will be represented by the contents of this file unless she chooses to use a different sprite image.

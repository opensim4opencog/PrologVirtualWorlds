RemoveModule

 Method of World

Permanently removes a specific module, by name, from the world. RemoveModule is a member of the World object.  

Syntax

object.RemoveModule bstrName

The RemoveModule method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstrName String Required The name of the module to remove.

Return Value

This method does not return a value.

Remarks

Only a Wizard or the Owner of the world can call this method.

You can use the CreateScriptModule method of World to create script modules. 

Example

World.RemoveModule "myModule"

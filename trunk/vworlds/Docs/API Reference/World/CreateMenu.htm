CreateMenu

 Method of World

Creates a shortcut menu for an object. CreateMenu is a member of the World object.  

Syntax

object.CreateMenu()

The CreateMenu method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Return Value

This method returns a Menu, which is a specific type of PropertyList.

Remarks

Every Thing object has a Menu property, which contains a list of menu items. Each of these menu items can perform an action when clicked.

To extend the exemplar menu of a Thing object, use the following steps:

  1. Create the new menu, using CreateMenu.
  2. Get the exemplar for the object to which the new menu will belong, using the Exemplar property of World.
  3. Copy the exemplar menu items to the new menu, using the CopyTo method of PropertyList.
  4. Add your new menu items.
  5. Assign the newly created menu to the object's Menu property.
 

Example

The example that follows shows each of the steps in the foregoing list. If you want to add custom menu items, you can do this after the menu has been added to the object (after the fourth line of code). See the CreateMenuItem method of World.

Sub CreateVaseMenu()
Dim myVase
Dim myVaseMenu
Dim myArtifact
Dim myArtifactMenu
Dim myMenuItem
Dim myCount
Dim myMenuItem, myMenuItem2

Set myVase = CreateInstance("Vase", "Artifact", User.Container)
Set myVaseMenu = World.CreateMenu

'Get a reference to the artifact exemplar. 

Set myArtifact = Exemplar("Artifact")

'Copy the standard artifact menu. 

myArtifact.Menu.CopyTo myVaseMenu

'Add items to the menu.

Set myMenuItem = World.CreateMenuItem("MyItemText", MI_CHECKED, User, "", "MyMethod")
Set MyMenuItem2 = World.CreateMenuItem("MyItemText2", MI_DEFAULT, User, "", "MyMethod2")
myVaseMenu.Add myMenuItem
myVaseMenu.Add myMenuItem2

myVase.Menu = myVaseMenu
myCount = myVase.Menu.Count - 1

For myMenuItem = 0 To myCount
    Trace myVase.Menu.Property(myMenuItem)
Next

End Sub

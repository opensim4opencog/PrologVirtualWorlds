Synchronize

 Method of World

Synchronizes the client with the server. Synchronize is a member of the World object.  

Syntax

object.Synchronize 

The Synchronize method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Return Value

This method does not return a value.

Remarks

This method synchronizes the client and server by sending a message to the server and waiting for confirmation that the message has been processed. When the confirmation is returned, the server is synchronized with the changes you have made prior to the call.  

Example

World.Synchronize

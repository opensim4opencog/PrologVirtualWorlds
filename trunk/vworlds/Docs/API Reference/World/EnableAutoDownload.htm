EnableAutoDownload

 Property of World

A Boolean flag that specifies whether new avatar sprite files will be downloaded automatically to all clients when an avatar enters a Room. EnableAutoDownload is a read-only Boolean.  

Syntax

To set this property:
object.EnableAutoDownload = BooleanValue

To return this property:
object.EnableAutoDownload

The EnableAutoDownload property syntax has these parts:
Part Type Use Description
object World Required An object reference that evaluates to a World object.
BooleanValue Boolean Required A flag that indicates whether the avatar sprite files of an avatar entering a room are downloaded automatically to all clients in that room. The default value is True.

Remarks

This property is local; each client sets the value for itself only.

A True value for this property indicates that the client will download sprite files that are not on the local computer. A False value indicates that the client will not download unknown sprites, but will download the default avatar to display in their place.

This property can be set from script, but only by the user.

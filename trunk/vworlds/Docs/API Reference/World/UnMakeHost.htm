UnMakeHost

 Method of World

Changes an avatar's exemplar from Host to Avatar. UnMakeHost is a member of the World object.  

Syntax

object.UnMakeHost pTarget

The UnMakeHost method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
pTarget Thing Required The Avatar whose exemplar will be changed from a Host to an Avatar.

Return Value

This method does not return a value.

Remarks

A host is an avatar that can log off other avatars. 

Only a wizard can call this method. Once it is called, Target ceases to be a Host.  

Example

World.UnMakeHost MCEmCee

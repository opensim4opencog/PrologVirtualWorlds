MakeHost

 Method of World

Changes an avatar's exemplar from Avatar to Host. MakeHost is a member of the World object.  

Syntax

object.MakeHost pTarget

The MakeHost method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
pTarget Avatar Required The avatar whose exemplar will be changed.

Return Value

This method does not return a value.

Remarks

A host is an avatar that can log off other avatars. 

Only a Wizard can call this method.

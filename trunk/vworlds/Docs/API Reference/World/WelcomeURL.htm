WelcomeURL

 Property of World

A String that holds the URL of the web page that greets a user who enters a world for the first time. WelcomeURL is a read/write String.  

Syntax

To set this property:
object.WelcomeURL = StringValue

To return this property:
object.WelcomeURL

The WelcomeURL property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
StringValue String Required An expression that evaluates to a String, which is the URL of a Web page.

Remarks

The ShowWelcomeMessage method of World refers to the WelcomeURL property.

The default value for this property is "client\welcome.htm", but its value can be set individually for different worlds. If the value is set to an empty string, no welcome message will be shown.

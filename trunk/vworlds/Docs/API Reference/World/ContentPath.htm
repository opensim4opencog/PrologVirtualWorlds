ContentPath

 Property of World

A String that holds the path of the directory that contains local content. ContentPath is a read-only String.  

Syntax

object.ContentPath

The ContentPath property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

This property is read-only; it cannot be set by anyone.

Local content includes geometry files, texture map files, scripts, and other basic building blocks of the world.

This property is initialized with the value in the ContentPath entry of the Windows Registry (the subkey is: \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\V-Worlds\Paths).  

The default path to this directory is:

C:\Program Files\Microsoft Virtual Worlds\Local Content\.

Example

To get the ContentPath value:

   Result = World.ContentPath

AvatarGraphicsPath

 Property of World

A String holding the path of the directory that contains avatar graphics. AvatarGraphicsPath is a read-only String.  

Syntax

object.AvatarGraphicsPath

The AvatarGraphicsPath property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

Avatar graphics include sprite files (whose filenames end in .SPR) and sprite image files (whose filenames end in .GIF).

This property is initialized with the value in the AvatarGraphicsPath entry of the Windows Registry (the subkey is: \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\V-Worlds\Paths).  

The default path to this directory is:

C:\Program Files\Microsoft Virtual Worlds\Local Content\Avatar Graphics\.

Example

To get the AvatarGraphicsPath value:

   Result = World.AvatarGraphicsPath

IsValidExemplar

 Method of World

Returns a flag indicating whether the parameter provided is a reference to an Exemplar. IsValidExemplar is a member of the World object.  

Syntax

object.IsValidExemplar(bstrType)

The IsValidExemplar method syntax has these parts:
Part Type Use Description
object World Required An object reference that evaluates to a World object.
bstrType String Required A string value that is the name of an Exemplar. The name of an exemplar is contained in the Name property of exemplars based on the Thing object.

Return Value

This method returns a Boolean indicating whether the value corresponds to the Name property of an existing exemplar.

Remarks

The IsValidExemplar method is useful for verifying whether the String provided in the bstrType parameter is the name of an exemplar. The CreateInstance method of the World object, for example, requires a string reference to an exemplar (the name of the exemplar) as a parameter. If that parameter is based on a built-in exemplar, such as an Artifact, it is usually not necessary to verify that the reference is an exemplar. However, when a custom exemplar is created, you can use the IsValidExemplar method as a safety measure to ensure that the custom exemplar has already been instantiated before invoking a method that requires a reference to an exemplar.

For more information on creating a custom exemplar, see the CreateExemplar topic.

Example

The following example uses the IsValidExemplar method to verify whether a new exemplar has been created. The example first creates a new exemplar named "myLightExemplar" based on the existing Light exemplar, verifies that it was successfully created by using the IsValidExemplar method, and if it was, creates an instance of that exemplar. 

Dim myNewExemplar
Set myNewExemplar = World.CreateExemplar("myLightExemplar", "Light")

'Verify that the new exemplar has been created in the Command Window. 

Trace World.IsValidExemplar("myLightExemplar")

'If the new exemplar has been created, create an instance of it. 

If World.IsValidExemplar("myLightExemplar") = True Then
World.CreateInstance("myLightInstance", "myLightExemplar", User.Container)
End If

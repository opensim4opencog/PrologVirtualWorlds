Connect

 Method of World

Connects a user to a server as an Avatar. Connect is a member of the World object.  

Syntax

object.Connect(bstrName, bstrPassword)

The Connect method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstrName String Required The name of the Avatar to be connected.
bstrPassword String Required The password to use for logging on.

Return Value

This method returns a Avatar.

Remarks

The first time you connect using a new name, an Avatar will be created for you. When you connect again normally, you must give the password that you gave when the Avatar was created. When you are connected, you will be able to change the password in the Object Explorer or Command Window.  

Note:   If an incorrect password is supplied in the bstrPassword parameter, the server returns an error and prompts for a valid password.

Example

Set SomeAvatar = World.Connect("Cruiser", "XYZ123")

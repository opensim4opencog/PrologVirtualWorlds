Avatars

 Property of World

Returns the Avatar objects that exist in a world. Avatars is a read-only PropertyList.  

Syntax

object.Avatars

The Avatars property syntax has these parts:
Part Type Use Description
object World Required An object of type World. The default is the current world.

Remarks

Because this property is a property list, you can find out specific information about each avatar that exists in your world. See the Example section for more details.

You can also use the Avatars property if you're trying to track a custom key for all of the avatars for a game scenario, as described in the AvatarProfile object.

Example

This example displays the image file that is associated with each avatar listed in the Avatars property. The subroutine also verifies that the avatar listed is not a stub by checking the value of the IsStub property of each avatar. 

Sub FindHosts()
Dim myIsHost
Dim myAvatar

For Each myAvatar In Avatars
  If not Avatars.Property(myAvatar).IsStub Then
    Print Avatars.Property(myAvatar).Profile.KeyValue("SpriteGeometry")
  Else 
    Print "IsStub"
  End If
Next

End Sub

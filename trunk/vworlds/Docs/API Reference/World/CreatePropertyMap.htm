CreatePropertyMap

 Method of World

Creates an unordered collection of custom-defined items. CreatePropertyMap is a member of the World object.

Syntax

object.CreatePropertyMap([varOrphan])

The CreatePropertyMap method syntax has these parts:
Parameters Type Use Description
object PropertyMap Required An object reference that accesses a PropertyMap collection.
[varOrphan] Variant Optional A flag that indicates whether the property list is created within a virtual world or is an orphaned, nested, data structure. The value is True if the property list is orphaned. The default is False.

Return Value

This method returns a new PropertyMap collection.

Remarks

A property map is an unordered list of keys and values. You access data in a property map by using its key. You can add String, Boolean, Float, Integer, or objects as properties to the property map. However, Microsoft Virtual Worlds does not support arrays. For more details on property maps, refer to the topic PropertyMap.

You can create a nested property map in Virtual Worlds. However, such a property map cannot be attached to a world, it must be orphaned.  To create a property map that does not reference a world, set the varOrphan parameter to True, as shown in the following code sample. The varOrphan parameter is set to False by default.

Set myPropertyMap = CreatePropertyMap(True)

Once you have orphaned a property map, you can never attach it to a world. You might want to orphan a property map for use as a generic nested data structure.

Example

This example creates a PropertyMap collection that is not attached to a world.

Sub CreateInterworldPassportForUser()
Dim myInterworldPassports
Dim myProps
Dim myID
Dim myName
Dim myCountry
Dim myLastVisit
Dim myDateOfLastVisit
Dim myDateOfEntry
Dim myPointOfEntry
Dim myIsResident

'Create interworld passport collection. 

Set myInterworldPassports = CreatePropertyMap(True)

'Create properties. 

myInterworldPassports.Property "myPassportID" 
myInterworldPassports.Property "myName"
myInterworldPassports.Property "myCountryOfOrigin"
myInterworldPassports.Property "myLastCountryVisited"
myInterworldPassports.Property "myDateOfLastVisit"
myInterworldPassports.Property "myDateOfEntry"
myInterworldPassports.Property "myPointOfEntry"
myInterworldPassports.Property "myIsCurrentResident"

Set myProps = User.Contents

'Add property values. 

myID = myProps.Find("myPassportID")
myID = 3000927843
myName = myProps.Find("myName")
myName = "Tiki"

myCountry = myProps.Find("myCountryOfOrigin")
myCountry = "Galatic Center"
myLastVisit = myProps.Find("myLastCountryVisited")
myLastVisit = "USA"
myDateOfLastVisit = myProps.Find("myDateOfLastVisit")
myDateOfLastVisit = "11/09/1999"

myDateOfEntry = myProp.Find("myDateOfEntry")
myDateOfEntry = "12/2/1999"

myPointOfEntry = myProp.Find("myPointOfEntry")
myPointOfEntry = "Interworld Portal 2206"
myIsCurrentResident = myProp.Find("myIsCurrentResident")
myIsCurrentResident = True

End Sub

DefaultPortal

 Property of World

The default Portal of a World. DefaultPortal is a read/write Portal.  

Syntax

To set this property:
object.DefaultPortal = PortalValue

To return this property:
object.DefaultPortal

The DefaultPortal property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
PortalValue Portal Required The default portal.

Remarks

This value specifies which portal users enter through when coming into the world. If there is a default portal, new users are sent through it; otherwise, they are sent to the default room, specified by the DefaultRoom property.

When Virtual Worlds starts, this property is not initialized.

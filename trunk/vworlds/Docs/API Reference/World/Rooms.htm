Rooms

 Property of World

A PropertyList containing a list of rooms that have been added to a World. Rooms is a read-only PropertyList.  

Syntax

World.Rooms

The Rooms property syntax has these parts:
Part Type Use Description
World World Required An instance of the World object.

Remarks

The Rooms property is a zero-based property list that contains the list of rooms that have been added to a world. The Rooms property is automatically updated when a room is added. Rooms are added to the list in the order that they are created.

Note:   This property list can contain a stub, which can lead to an error. For more information on preventing errors related to stubs, see the IsStub topic.

Example

This example uses the Rooms property to display the value of the Name property of the first room in the list.

Trace World.Rooms.Property(0).Name

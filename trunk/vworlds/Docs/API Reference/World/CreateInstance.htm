CreateInstance

 Method of World

Creates an instance of a particular object in a specified container. CreateInstance is a member of the World object.  

Syntax

object.CreateInstance(bstrName, varExemplar [, varContainer])

The CreateInstance method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstrName String Required The name of the instance.
varExemplar Variant Required The exemplar for the instance to be created.
[varContainer] Variant Optional The container in which the instance will be created.

Return Value

This method returns a Thing.

Remarks

The Exemplar parameter can be either a reference to an exemplar (type Thing), or the name of the exemplar (type String).

If the Container parameter is omitted, the object will usually be created in the Contents of the User who created the object, unless the object created is a Room, in which case, the World will be its container.

Objects of type Avatar cannot be created with this method. You will receive an error if you try to do so.  

Example

The first example invokes the CreateInstance method and supplies the String name of an exemplar for the varExemplar parameter.>.

Set NewThing = World.CreateInstance("Toy", "Artifact", User.Container)

Example 2.

This example obtains a reference to an existing object named "myObject", and supplies the object reference as the varExemplar parameter.

Dim myInstance
Dim myInstanceExemplar

Set myObject1 = User.Container.Findwithin("myObject")
Set myInstanceExemplar = myObject1.Exemplar

Set NewThing = World.CreateInstance("Toy", myInstanceExemplar, User.Container)

Example 3.

The third example uses the CreateInstance method to add the instance to the contents of the User.

Set NewThing = World.CreateInstance("Toy", "Artifact")

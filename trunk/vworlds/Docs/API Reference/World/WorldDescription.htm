WorldDescription

 Property of World

A String that describes the world. WorldDescription is a read/write String.  

Syntax

To set this property:
object.WorldDescription = StringValue

To return this property:
object.WorldDescription

The WorldDescription property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
StringValue String Required An expression that evaluates to a String.

Remarks

When Virtual Worlds starts, this property is uninitialized. If you are the Owner of the world, you can set this property.

The best use of this property is to provide users of the world with a brief description of the world.

This

 Property of World

A Thing that holds a reference to the object associated with the method that is currently executing. This is a read-only Thing.  

Syntax

object.This

The This property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
ThingValue Thing Required The object associated with the method that is currently running.

Remarks

When a method on an object is executing, the This property of the World object contains a reference to that object.  This is useful both as a shortcut notation and when writing script code that might be executed as a method on a number of different objects.  This has a meaningful value only if it is accessed from within the body of an executing method.

To determine the object that is calling the method (as opposed to the object on which the method is executing), see the Caller property of the World object.  

Example

The following piece of code could be added as a method to any object derived from the Thing exemplar and cause the object to change it's geometry to the default sprite file. 

 
   Sub
  SetGeometryToDefault World.This.GeometryName =  
     "default.spr"
End Sub

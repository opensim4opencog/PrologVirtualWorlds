Module

 Property of World

An object reference (or pointer) to the named Module. Module is a read-only String.  

Syntax

object.Module

The Module property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

Returns an object reference to the module for which the user is searching.

Example

In the following example, the Module named "myModule" is presumed to have been created previously, probably by a call to the CreateScriptModule method of World. 

Set Result = World.Module("myModule")

IsValidModule

 Method of World

Returns a Boolean flag indicating whether a module has been added to a world. IsValidModule is a member of the World object.  

Syntax

World.IsValidModule(bstrName)

The IsValidModule method syntax has these parts:
Part Type Use Description
World World Required An object reference that accesses a World object.
bstrName String Required The name of a module expressed as a String.

Return Value

This method returns a Boolean.

Remarks

The IsValidModule method verifies whether a module has been added to a world. If the method returns True, the module has been added, and the members of that module can be accessed. If the method returns False, the module has not been successfully added to the world.

For more information on adding modules, see the CreateScriptModule topic. For more information on removing modules, see the RemoveModule topic.

Example

This example uses a subroutine to first verify that a module exists, and if so, then remove the module and display a confirmation in the Command Window. The parameter "myStrModuleName" is a String. 

Sub mySafeRemoveModule(myStrModuleName)
Dim myFlag

myFlag = World.IsValidModule(myStrModuleName)
If myFlag = True Then
    World.RemoveModule myStrModuleName
    Trace myStrModuleName & " has been removed."
Else
    Trace myStrModuleName & " is not a valid module name."
End If

End Sub

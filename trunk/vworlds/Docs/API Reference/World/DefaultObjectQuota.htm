DefaultObjectQuota

 Property of World

Sets or returns the default number of items that an avatar can create in a world. DefaultObjectQuota property is a read/write Long. 

Syntax

object.DefaultObjectQuota = longValue

The DefaultObjectQuota property syntax has these parts:
Parameters Type Use Description
object World Required A reference to the World object.
longValue Long Required The default upper limit of objects an avatar can create in a world.

Remarks

This property determines the default for the ObjectQuota value given to all new avatars. The standard DefaultObjectQuota value is 10. Changing this value affects only avatars that you create after you change the quota.

A value of -1 for this property indicates that an avatar can create an infinite number of objects.

Example

This example changes the default quota. 

 
Sub CheckDefaultQuota()
Dim myDefaultQuota

Set myDefaultQuota = World.DefaultObjectQuota

If myDefaultQuota = 10 Then myDefaultQuota = 20

End Sub

WorldContentPath

 Property of World

A String that holds the path to a directory in which local content files are located for a particular world. This path is world-specific. WorldContentPath is a read-only String.  

Syntax

object.WorldContentPath

The WorldContentPath property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

This property is read-only; it cannot be set by anyone.

To find content for a specific world, Virtual Worlds starts its search in the Local Content directory. The default path to world-specific content is:

C:\Program Files\Microsoft Virtual Worlds\Local Content\Worlds\<World Name>\

When Virtual Worlds starts, WorldContentPath is initialized with a combination of the contents of ContentPath and WorldName.  

Example

To get the WorldContentPath property:

   ThePath = World.WorldContentPath

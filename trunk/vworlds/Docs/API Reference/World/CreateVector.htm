CreateVector

 Method of World

Creates a Vector object from the x, y, and z parameter values. CreateVector is a member of the World object. 

Syntax

object.CreateVector(x, y, z)

The CreateVector method syntax has these parts:
Parameters Type Use Description
object World Required An object reference to a World object.
x Float Required Represents the x value of the vector's x-coordinate.
y Float Required Represents the y value of the vector's y-coordinate.
z Float Required Represents the z value of the vector's z-coordinate.

Return Value

This method returns a new vector based on the coordinate values of the x, y, and z parameters.

Remarks

A vector in Microsoft Virtual Worlds is a set of 3 floats. You can adjust how your object appears in a Virtual Worlds space by using the following objects with the x, y, and z coordinates.

Direction

 Position
EntryOrientation RotationAxis
EntryPoint Scale
LightAttenuation Up
LightColor

After you create a vector, you must assign it to an object.  

Example

This example creates a new vector and assigns it to an existing object in Virtual Worlds.

Dim myVector

'Create a new vector.
Set myVector = CreateVector(5, 0, 10)

'Assign the vector to an existing object.
Set myHouse.Position = myVector

AbortEvent

 Method of World

Cancels an event. AbortEvent is a member of the World object.  

Syntax

object.AbortEvent 

The AbortEvent method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Return Value

This method does not return a value.

Remarks

An event is only aborted if the method that calls for the abort is in an event handler on the server.  

Example

World.AbortEvent

Owner

 Property of World

A Thing that represents the owner of the World. Owner is a read-only Thing.  

Syntax

object.Owner

The Owner property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

Every object in the World, including the world itself, has an Owner. The Owner is the Avatar who created the object. Once an object is created, its ownership never changes.  

Example

To find the name of the Owner of the world:

   Set myOwnerReference = World.Owner
   Trace myOwnerReference.Name

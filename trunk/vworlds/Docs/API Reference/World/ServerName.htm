ServerName

 Property of World

A String that contains the name of the Virtual Worlds server the world resides on. ServerName is a read-only String.  

Syntax

object.ServerName

The ServerName property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Remarks

For a world running locally, the ServerName property contains "localhost". For a world hosted by a machine other than the one running the client, ServerName contains the name (or I.P. address) of that computer.  

Example

Result = World.ServerName

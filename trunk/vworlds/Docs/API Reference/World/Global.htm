Global

 Property of World

Provides a way to add members to the World object. Global is a read/write Thing.  

Syntax

World.Global

The Global property syntax has these parts:
Part Type Use Description
World World Required An instance of the World object.

Remarks

The Global property of the World object allows only a world owner to add a method or property to the World object. The Global property is a reference to a Thing that allows the use of AddMethod and AddProperty to add members to the World object. After a member has been added using the Global property, the new member is accessible to all users of a world as a member of the World object. Note, however, that any objects created in a Global method are owned by the user, so Global won't have permission to modify them, even in the same method.

The Global property can also be used to remove an added method or property from the World object, using the RemoveMethod and RemoveProperty methods of the Thing object.

Example

To add a property to the World object:

World.Global.AddProperty "myNewProperty"

To access the added property after it has been added to the World object:

World.myNewProperty

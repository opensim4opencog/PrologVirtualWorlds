ShowHTML

 Method of World

Creates an HTML pop-up window. ShowHTML is a member of the World object.  

Syntax

object.ShowHTML bstrTitle, bstrHTML [, cx] [, cy]

The ShowHTML method syntax has these parts:
Part Type Use Description
object World Required An object reference to a World object.
bstrTitle String Required The title of the window.
bstrHTML String Required The HTML text to display.
[cx] Variant Optional The width of the window in pixels. The default value is 0.
[cy] Variant Optional The height of the window in pixels. The default value is 0.

Return Value

This method does not return a value.

Remarks

This method shows HTML content in a browser window with the given title. The upper left corner of the HTML window is at the point (0, 0). If the optional cx and cy parameters are included, they specify the window's width and height. The default value for cx and cy parameters is 0.

ShowHTML displays only one window at a time. If ShowHTML is called while a window created by a prior call to this method is still being displayed, it refreshes the window title (if different in the later call) and the window content, but ignores any changes in the window's position and size. To display more than one HTML window, use ShowHTMLEx.

This method works only on the client.  

Example

Dim HTMLStr
HTMLStr = " ... some HTML text ..."
World.ShowHTML "Edit Artifact", HTMLStr, 50, 80

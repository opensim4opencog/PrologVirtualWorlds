Disconnect

 Method of World

Disconnects the user from a world. Disconnect is a member of the World object.  

Syntax

object.Disconnect 

The Disconnect method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.

Return Value

This method does not return a value.

Remarks

Disconnect sets the User property to NULL.  

Example

World.Disconnect

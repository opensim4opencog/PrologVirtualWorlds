RootURL

 Property of World

A String that holds the path for the default location of the world's content. RootURL is a read/write String.  

Syntax

To set the property:
object.Property = stringValue

To return the property:
object.Property

The RootURL property syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
stringValue String Required The path of the default location of the content of the world.

Remarks

The RootURL property consists of a collection of directory paths, separated by semicolons. The directories to which the paths refer should contain world content.

With the RootURL property, you can extend the default search locations to include other remote or local URLs, but only the world Owner and Wizards can set the RootURL property.

Other Directories Used for Searches

Microsoft Virtual Worlds will automatically search the following directories by default (the string Worldname is replaced by the name of the relevant world in actual usage):

\Program Files\Microsoft Virtual Worlds\Local Content
\Program Files\Microsoft Virtual Worlds\Avatar Graphics
\Program Files\Microsoft Virtual Worlds\Worlds\Worldname
\Program Files\Microsoft Virtual Worlds\Worldname\Avatar Graphics   

These other directories do not appear by default in the RootURL property, but are a part of Virtual Worlds searching mechanism. The RootURL property provides you with a way to add to this default search by including directories on a local or remote site that are not included in the searching mechanism by default.

Example

To get the RootURL value:

   Result = World.RootURL

To set the RootURL value:

   World.RootURL = "http://vworlds.microsoft.com/myworld/content/"

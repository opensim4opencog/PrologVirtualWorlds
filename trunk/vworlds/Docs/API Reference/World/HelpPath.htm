HelpPath

 Property of World

A String that holds the path of the directory containing help documentation. HelpPath is a read-only String.  

Syntax

object.HelpPath

The HelpPath property syntax has these parts:
Part Type Use Description
object World Required An expression that evaluates to a World object.

Remarks

This property is initialized with the value in the HelpPath entry of the Windows Registry (the subkey is: \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\V-Worlds\Paths).

To find help documentation, Virtual Worlds looks in the directory indicated by the HelpPath. The default path to this directory is:

C:\Program Files\Microsoft Virtual Worlds\Docs\.

 

Example

To get the HelpPath value:

Result = World.HelpPath

ShowURLEx

 Method of World

Displays the content of the specified URL in a modeless pop-up window. ShowURLEx is a member of the World object.  

Syntax

object.ShowURLEx bstrWinName, bstrTitle, bstrURL, dispArg [, x] [, y] [, cx] [, cy]

The ShowURLEx method syntax has these parts:
Part Type Use Description
object World Required An object reference that accesses a World object.
bstrWinName String Required The name of the window.
bstrTitle Integer Required The title of the window.
bstrURL String Required The URL of the page to display.
dispArg Object Required A DHTML object, such as that defined in an <Object> tag. Or, this can also be a Virtual Worlds object, such as World.User.
[x] Variant Optional The x-coordinate of the upper left corner of the window in pixels. The default value is 0.
[y] Variant Optional The y-coordinate of the upper left corner of the window in pixels. The default value is 0.
[cx] Variant Optional The width of the window in pixels. The default value is 0.
[cy] Variant Optional The height of the window in pixels. The default value is 0.

Return Value

This method does not return a value.

Remarks

This method displays the page content at a URL in a modeless pop-up window, with the specified name and title. If the optional x, y, cx, or cy parameters are included, they specify the position of the upper left corner of the window, and the window's width and height. The default value for each of these parameters is 0.

The dispArg parameter can be a DHTML object (such as an object defined in an <OBJECT> tag on the page), or it may be a Virtual Worlds object (such as World.User). Or, it can be any Automation object that matches the type of this parameter; the Helper object on the popped-up web page gives you access to that variable. If you do not need to pass a value for dispArg, use Nothing.

If ShowURLEx is called more than once for the same window, it refreshes the window, but ignores any changes in the window's position and size.

The "Ex" in this method's name stands for "Extended."  

Example

World.ShowURLEx "Window1", "Edit", "http://research.microsoft.com/vwg", Nothing, 20, 100, 550, 400

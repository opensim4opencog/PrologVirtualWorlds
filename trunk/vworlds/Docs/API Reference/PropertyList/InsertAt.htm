InsertAt

 Method of PropertyList

Adds a new item at a specified position in an existing collection. InsertAt is a member of the PropertyList collection.

Syntax

object.InsertAt index, var

The InsertAt method syntax has these parts:
Parameters Type Use Description
object PropertyList Required An object reference that accesses a PropertyList collection.
index Long Required The index number of the position where the item will be added in the PropertyList collection.
var Variant Required

The name of the item to add to the PropertyList collection.

Return Value

This method does not return a value.

Remarks

The number of items in a PropertyList collection is maintained by the Count property. Valid indices are 0 through Count - 1. You can add an item to an existing collection at any position from the beginning of the collection (index = 0) to one element past the end of the collection (index = Count). If a property is inserted anywhere except at the first element past the end, all existing property list elements from the point of insertion to the end are shifted one position higher in the property list. For example, if you have a property list with four elements (indexes 0 = red, 1 = blue, 2 = green, and 3 = yellow), and you insert a property at index 2, the properties that were formerly at indexes 2 and 3 will be shifted to indexes 3 and 4 (0 = red, 1 = blue, 2 = purple, 3 = green, and 4 = yellow).

Even though you can insert an item at an index greater than Count (that is, two or more positions beyond the end of the property list), you should not do this. The item itself will be valid and can be accessed, but there will be one or more items in the property list between that one and the last valid item in the list that are inaccessible. A script error results if you attempt to iterate through a property list that contains invalid items.

When you add a property or method to an object, you are actually adding a clone of that property or method. If you want to modify the property or method for the object, you must be careful to refer to it explicitly; otherwise, you will be modifying only your local copy of the property. 

Note:   You can use the Add method to add items at the end of a collection.  

Example

This example adds a new item to an existing collection. This example assumes that collections named "myLibrary" and "myNumberList" exist and contain more than three items.

'This statement adds a book title to the "myLibrary" collection. 

myLibrary.InsertAt 3, "A Tale of Two Cities"

'This statement adds an integer item named "2" to the "myNumberList" collection. 

myNumberList.InsertAt 3, 2

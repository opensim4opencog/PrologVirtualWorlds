IsEmpty

 Property of PropertyList

Returns True if the specified collection is empty, False if at least one item exists in the collection. IsEmpty is a read-only Boolean.

Syntax

object.IsEmpty

The IsEmpty property syntax has these parts:
Parameters Type Use Description
object PropertyList PropertyMap Required An expression that returns a PropertyList or PropertyMap collection.

Remarks

You can use the IsEmpty property to test existing PropertyList or PropertyMap collections for their contents.

Examples

These examples test for an existing collection. If the collection exists, the code continues by adding new items to the collection. If the collection is empty, the code creates a new collection under a different name. The collection in the first example is a property list. 


Dim Classics

Classics = "Silas Marner"

If myLibrary.IsEmpty Then
    Set myBookCollection = CreatePropertyList
    myBookCollection.Add "Classics"
Else
    myLibrary.Add Classics
End If

Example 2

The second example uses the IsEmpty property with a property map. 


If myLibrary.IsEmpty Then
    Set myBookCollection = CreatePropertyMap
    myBookCollection.Property("Classics") = "Silas Marner"
Else
    myLibrary.Property("Classics") = "Silas Marner"
End If

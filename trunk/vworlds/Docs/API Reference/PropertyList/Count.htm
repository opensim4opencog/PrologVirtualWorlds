Count

 Property of PropertyList

Returns the number of items in the specified PropertyList or PropertyMap collection. Count is a read-only Long.

Syntax

object.Count

The Count property syntax has these parts:
Parts Type Use Description
object PropertyList PropertyMap Required An expression that returns a PropertyList or PropertyMap collection object.

Remarks

Reports the number of items in a collection.

Important:   You may want to avoid using the Count property as an iteration counter. If you include an increment or decrement method (such as the Add method) inside a loop, the Count property will be changed within your loop. As a result, an iteration counter may not be accurate, depending on the number of items added or removed within the loop.

Example

This example iterates through a collection using the Count property to determine the number of items in the collection.

'These statements assume you have created the property list, "myBagOfTricks". 

Dim i
Dim myCount

myCount = myBagOfTricks.Count - 1 

For i = 0 To myCount
    Trace myBagOfTricks.Property(i)
Next

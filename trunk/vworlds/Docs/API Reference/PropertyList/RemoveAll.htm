RemoveAll

 Method of PropertyList

Removes all items from a collection. RemoveAll is a member of the PropertyList and PropertyMap collections.

Syntax

object.RemoveAll

The RemoveAll method syntax has these parts:
Parameters Type Use Description
object

PropertyList PropertyMap

 Required An object reference that accesses a PropertyList or PropertyMap collection.

Return Value

This method does not return a value.

Example

This example uses the IsEmpty property to check if a collection contains data and then removes all of the items from the collection. This example assumes that you have an existing collection named "myLibrary" that contains a collection of book titles. 


Sub ClearCollection()

If myLibrary.IsEmpty Then 
    Trace "Done"
Else
    myLibrary.RemoveAll
    Trace "The collection is empty."
End If

End Sub

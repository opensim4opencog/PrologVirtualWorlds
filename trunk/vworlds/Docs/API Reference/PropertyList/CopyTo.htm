CopyTo

 Method of PropertyList

Copies the specified collection to the destination collection. CopyTo is a member of the PropertyList and PropertyMap collections.

Syntax

object.CopyTo ppropertylist
-or-
object.CopyTo ppropertymap

The CopyTo method syntax has these parts:
Parameters Type Use Description
object

PropertyList PropertyMap

 Required An object reference to a PropertyList or PropertyMap collection.
ppropertylist PropertyList
Required
The name of the destination collection of type PropertyList or PropertyMap.

-or-
ppropertymap

PropertyMap

Return Value

This method does not return a value.

Remarks

When you add a property to an object, you are adding a copy of that property. The CopyTo method recreates each property of the source collection in the destination collection. If the destination property list or property map already contains properties, the CopyTo method appends the copied properties to the end of the destination property list or property map. A destination property list or property map must already exist before you call this method. The CopyTo method does not automatically create a new list or a new map.

Example

This example copies a collection of ceramic items from "myCeramicCollection" to a new collection named "myPots".

Sub PrintCeramicCollection()
Dim myCeramicCollection 
Dim myCount
Dim myTotalCeramics
Dim myHighfire1
Dim myHighfire2
Dim myHighfire3

'Declare the new collection name for the copy of the collection. 

Dim myPots

Set myCeramicCollection = CreatePropertyList
Set myPots = CreatePropertyList

myHighfire1 = "Vase"
myHighfire2 = "Salad Bowl"
myHighfire3 = "Teapot"

myCeramicCollection.Add myHighfire1
myCeramicCollection.Add myHighfire2
myCeramicCollection.Add myHighfire3

myTotalCeramics = myCeramicCollection.Count - 1

'Use the following statement to create a copy of the ceramic collection. 

myCeramicCollection.CopyTo myPots

'Iterate through the collection and print the names of the properties. 

For myCount = 0 myTotalCeramics
    Trace myPots.Property(myCount)
Next End
Sub

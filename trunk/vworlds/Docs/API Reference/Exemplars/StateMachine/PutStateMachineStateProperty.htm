PutStateMachineStateProperty

 Method of the StateMachine Exemplar

Sets the value of a property on a particular state machine when the machine is in a specified state. PutStateMachineStateProperty is a member of the StateMachine exemplar.  

Syntax

object.PutStateMachineStateProperty bstrStateMachineName, bstrState, bstrPropertyName, var

The PutStateMachineStateProperty method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The state machine name.
bstrState String Required The state machine state.
bstrPropertyName String Required The state machine property.
var Variant Required The value of the property.

Return Value

This method does not return a value.

Remarks

Although this method can be used to set the value of any property of a particular state machine in any state, its primary purpose is to set the length of time a state will be active. Such a property is often called TimeOut.

This method runs on either the client or the server.  

Example

In the example that follows, the first line will cause the StateMachine object, MyStMach, to display a message when the Waiting state of the State1 internal state machine is entered. The second line (shown in bold) sets a property (named TimeOut) of the Waiting state to 10000 milliseconds, or 10 seconds. The third line indicates that, if the Waiting state times out, the state should be set to the Finish state and a message should be displayed.

MyStMach.PutStateMachineInlineStateMethod METHOD_CLIENT, "State1", "Waiting", "OnEnterState",
   "this.Say ""Entered State1"" "
MyStMach.PutStateMachineStateProperty "State1", "Waiting", "TimeOut", 10000
MyStMach.PutStateMachineInlineStateMethod METHOD_CLIENT, "State1", "Waiting", "OnTimeOut",
   "this.SetState ""Finish"" : this.Say ""Timed out!"" "

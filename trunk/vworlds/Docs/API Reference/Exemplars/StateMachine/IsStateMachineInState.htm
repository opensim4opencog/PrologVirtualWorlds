IsStateMachineInState

 Method of the StateMachine Exemplar

Find whether the specified state machine is in the specified state. IsStateMachineInState is a member of the StateMachine exemplar.  

Syntax

object.IsStateMachineInState (bstrStateMachineName, bstrState)

The IsStateMachineInState method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The name of the state machine.
bstrState String Required The state of the state machine.

Return Value

This method returns a Boolean.

Remarks

This method executes on either the client or the server.

To find a state machine's current state, use the Trace method of World, and pass it the name of the state machine object and the internal state machine for which you want to find the state.

A related method of StateMachine, IsInState, is intended for use on the Default state machine.  

Example

inState = MyStateMachine.IsStateMachineInState("State1", "Waiting")

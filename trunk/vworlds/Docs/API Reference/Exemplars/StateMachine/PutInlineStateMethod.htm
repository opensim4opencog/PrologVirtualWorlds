PutInlineStateMethod

 Method of the StateMachine Exemplar

Adds an in-line method associated with a specified state of the default state machine. PutInlineStateMethod is a member of the StateMachine exemplar.  

Syntax

object.PutInlineStateMethod lFlags, bstrState, bstrMethodName, bstrCode [, varParams]

The PutInlineStateMethod method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
lFlags Long Required A flag that specifies whether the inline method will run on the client or the server or both.
bstrState String Required The state machine state.
bstrMethodName String Required The name of the inline method.
bstrCode String Required The VBScript code to execute.
[varParams] Variant Optional The list of one or more parameters for the inline method.

Return Value

This method does not return a value.

Remarks

PutInlineStateMethod runs on the server.

The lFlags parameter uses the METHOD_CLIENT and METHOD_SERVER properties of World.

When a state machine is in a particular state, it will invoke any method associated with that state. Two methods that a state machine will invoke on itself are OnEnterState and OnTimeOut. Use OnEnterState for the MethodName parameter to cause your state machine to perform some action when it enters the given state. Use OnTimeOut for the MethodName parameter to cause it to take an action after the period of time specified has elapsed. Neither OnEnterState nor OnTimeOut takes parameters. (See the PutStateProperty and PutStateMachineStateProperty methods of StateMachine for information about setting a TimeOut value.)

Two other methods that may be useful are OnLButtonUp and OnPeerEnter (one of several methods in the OnEnter event handler group). Use OnLButtonUp for the MethodName parameter to cause your state machine to respond to a click of the left mouse button. Use OnPeerEnter for the MethodName parameter to cause your state machine to respond to another object entering the same room (or other container) as your state machine.

The Params parameter is used when the inline method takes parameters. These should be listed in a pair of quotation marks, and comma-separated if there is more than one parameter. The default value of Params is the empty string (""), so you can omit this parameter if there are no parameters to pass to the inline method.  

Example

Note that if the inline code contains an embedded String, the beginning and ending double-quotes must appear twice each, as in the following example. Also note that the OnLButtonUp event handler requires three arguments to be passed to it. Your inline method does not need to use these arguments. These arguments are shown in the last parameter.

MyStateMachine.PutInlineStateMethod World.METHOD_CLIENT, "Waiting", "OnLButtonUp",_
   "this.Say ""I been clicked!"" ", "win, x, y"

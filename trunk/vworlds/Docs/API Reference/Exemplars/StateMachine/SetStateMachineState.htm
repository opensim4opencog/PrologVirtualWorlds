SetStateMachineState

 Method of the StateMachine Exemplar

Sets a specified state machine to a particular state. SetStateMachineState is a member of the StateMachine exemplar.  

Syntax

object.SetStateMachineState bstrStateMachineName, bstrState

The SetStateMachineState method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The state machine name.
bstrState String Required The state machine state.

Return Value

This method does not return a value.

Remarks

This method sets the state machine (Name) to a state (State); it can be run from either the client or the server. To determine a state machine's current state, use the Trace method of World, and pass it both the name of the state machine object and the internal state machine for which you want to determine the state. To set the default internal state machine (Default) to a new state, use the SetState method of StateMachine.  

Example

MyStateMachine.SetStateMachineState "State", "TurnLeft"

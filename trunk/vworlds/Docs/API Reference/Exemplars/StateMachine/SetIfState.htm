SetIfState

 Method of the StateMachine Exemplar

Set the state of the default state machine (Default) to a new state, but only if it is in a given state. SetIfState is a member of the StateMachine exemplar.  

Syntax

object.SetIfState bstrOldState, bstrNewState

The SetIfState method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrOldState String Required The current state.
bstrNewState String Required The state to be changed to.

Return Value

This method does not return a value.

Remarks

This method sets the state of the default state machine (Default) to a new state (bstrNewState), but only if it is in the given state (CurrentState); it can be run from the client or the server. For internal state machines other than Default, use the SetStateMachineIfState method of StateMachine. 

Example

MyStateMachine.SetIfState "Waiting", "TurnLeft"

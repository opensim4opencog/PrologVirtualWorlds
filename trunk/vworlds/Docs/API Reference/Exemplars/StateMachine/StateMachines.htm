StateMachines

 Property of the StateMachine Exemplar

A PropertyList containing the names of internal state machines (including Default) associated with a state machine. StateMachines is a read/write String.  

Syntax

object.StateMachines = PropertyListValue

The StateMachines property syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
PropertyListValue String Required A property list of the names of the internal state machines.

Remarks

The StateMachines PropertyList contains the names (each of which is a String) of the default state machine (Default) and any other state machines that are part of the invoking state machine. The first name on this list will always be Default. Additional names are added automatically when other internal state machines are added (using the AddStateMachine method of StateMachine).  

Example

To get the StateMachines property:

   Set Properties = MyStateMachine.StateMachines

To set the StateMachines property:

   MyStateMachine.StateMachines = Properties

Note: The variable Properties in the second example must have already been created as a PropertyList (by the CreatePropertyList method of World) and initialized to the proper values.

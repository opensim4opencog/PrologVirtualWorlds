PutStateMachineInlineStateMethod

 Method of the StateMachine Exemplar

Adds an in-line method for a state-specific event on a specified state machine. PutStateMachineInlineStateMethod is a member of the StateMachine exemplar.  

Syntax

object.PutStateMachineInlineStateMethod lFlags, bstrStateMachineName, bstrState, bstrMethodName, bstrCode [, varParams]

The PutStateMachineInlineStateMethod method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
lFlags Long Required A flag that specifies whether the inline method will run on the client or server or both.
bstrStateMachineName String Required The state machine name.
bstrState String Required The state machine state.
bstrMethodName String Required The name of the inline method.
bstrCode String Required The VBScript code to execute.
[varParams] Variant Optional The list of one or more parameters.

Return Value

This method does not return a value.

Remarks

This method adds an inline method (on the client or server) to be invoked by a state machine when it is in a particular state. PutStateMachineInlineStateMethod runs on either the client or the server.

The lFlags parameter uses the METHOD_CLIENT and METHOD_SERVER properties of World.

A state machine will invoke a method associated with a particular state when that it is in that state. Two methods that a state machine can invoke are OnEnterState and OnTimeOut. Use OnEnterState for the MethodName parameter to cause a state machine to perform some action when the given state is entered. Use OnTimeOut for the MethodName parameter to cause a state machine to take some action after the specified period of time has elapsed. (Neither OnEnterState nor OnTimeOut takes parameters. See the PutStateProperty and PutStateMachineStateProperty methods of StateMachine for information about setting a TimeOut value.)

The Params parameter is used when the inline method takes parameters. Parameters should be listed in a pair of quotation marks, and separated by commas if there is more than one. The default value of Params is the empty string (""); you can omit Params if there are no parameters to pass to the inline method.

Two other useful methods are OnLButtonUp and OnPeerEnter (one of several methods in the OnEnter event handler suite). Use OnLButtonUp for the MethodName parameter to cause a state machine to respond to a click of the left mouse button. Use OnPeerEnter for the MethodName parameter to cause a state machine to respond to another object entering the same room (or other container).  

Example

Note that if the inline code contains an embedded String, the beginning and ending double-quotes must appear twice each, as in the following example. Also note that the OnLButtonUp event handler requires three arguments to be passed to it. Your inline method does not need to use these arguments. These arguments are shown in the last parameter.

MyStateMachine.PutStateMachineInlineStateMethod World.METHOD_CLIENT, "State1",_
   "Waiting", "OnLButtonUp", "this.Say ""I been clicked!"" ", "win, x, y"

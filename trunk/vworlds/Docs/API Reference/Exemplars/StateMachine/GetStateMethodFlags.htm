GetStateMethodFlags

 Method of the StateMachine Exemplar

Gets the client/server flag for a specific state machine inline method. GetStateMethodFlags is a member of the StateMachine exemplar.  

Syntax

object.GetStateMethodFlags (bstrStateMachineName, bstrState, bstrMethodName)

The GetStateMethodFlags method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The name of the state machine.
bstrState String Required The name of the state.
bstrMethodName String Required The name of the method (as it is known to the StateMachine instance).

Return Value

This method returns a Long.

Remarks

GetStateMethodFlags is used to determine whether an inline event handler method (Handler) associated with a StateMachine with a given name and state runs on the client or the server. The values returned will be one or both of the constants METHOD_CLIENT or METHOD_SERVER.

This method will run on either the client or the server.  

Example

Flags = MyStateMachine.GetStateMethodFlags("State1", "Waiting", "OnLButtonUp")

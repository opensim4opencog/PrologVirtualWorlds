GetStateProperty

 Method of the StateMachine Exemplar

Gets a state-specific property of a state machine. GetStateProperty is a member of the StateMachine exemplar.  

Syntax

object.GetStateProperty (bstrStateMachineName, bstrState, bstrPropertyName)

The GetStateProperty method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The specific state machine name.
bstrState String Required The state of the state machine.
bstrPropertyName String Required The name of a property of the state machine.

Return Value

This method returns a Variant.

Remarks

This method returns the value of a property associated with a particular state machine, state, and property name. The type of the value that is returned will be the same as the type used in either of the StateMachine methods PutStateProperty (used for Default) or PutStateMachineStateProperty (used for other internal state machines). This method will run on either the client or the server.

This method's principal use is to get any state machine state-specific property. TimeOut is an example of such a property; its purpose is to control the length of time a state will be active.  

Example

Value = MyStateMachine.GetStateProperty("Default", "Waiting", "TimeOut")

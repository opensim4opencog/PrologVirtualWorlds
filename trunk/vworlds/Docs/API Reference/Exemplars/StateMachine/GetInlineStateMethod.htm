GetInlineStateMethod

 Method of the StateMachine Exemplar

Gets a string containing the inline method code for a specific state machine, state, and event. GetInlineStateMethod is a member of the StateMachine exemplar. 

Syntax

object.GetInlineStateMethod (bstrStateMachineName, bstrState, bstrMethodName)

The GetInlineStateMethod method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The name of the state machine.
bstrState String Required The name of the state.
bstrMethodName String Required The name of the method (as it is known to the StateMachine instance).

Return Value

This method returns a String.

Remarks

This method executes on the server. 

Example

Code = MyStateMachine.GetInlineStateMethod("StateMachine1", "Waiting", "OnLButtonUp")

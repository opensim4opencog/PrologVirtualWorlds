SetState

 Method of the StateMachine Exemplar

Set the state of the default state machine (Default) to a particular state. SetState is a member of the StateMachine exemplar.  

Syntax

object.SetState bstrState

The SetState method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrState String Required The name of the state.

Return Value

This method does not return a value.

Remarks

This method sets the default state machine (Default) to a different state (bstrState); it can be run from either the server or the client. It can be used to set the initial state of a state machine, and is often used to set a new state in a user-written inline event handler such as OnTimeOut or OnEnterState.

To set the state of an internal state machine other than Default, use the SetStateMachineState method of StateMachine.

To find an internal state machine's current state, use the Trace method of World and pass it the name of the state machine object and the internal state machine for which you want to find the state. 

Example

MyStateMachine.SetState "TurnLeft"

SetStateMachineIfState

 Method of the StateMachine Exemplar

Set the state of a specified state machine to a new state, but only if it is in a given state. SetStateMachineIfState is a member of the StateMachine exemplar.  

Syntax

object.SetStateMachineIfState bstrStateMachineName, bstrOldState, bstrNewState

The SetStateMachineIfState method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrStateMachineName String Required The state machine name.
bstrOldState String Required The current state.
bstrNewState String Required The state to be changed to.

Return Value

This method does not return a value.

Remarks

This method sets the state of the specified state machine (bstrStateMachineName) to a new state (bstrNewState), but only if it is in the given state (CurrentState); it should be run from the server. The related SetIfState method of StateMachine can be used to perform the same action on the default internal state machine, Default.  

Example

MyStateMachine.SetStateMachineIfState "StateMachine1", "Waiting", "TurnLeft"

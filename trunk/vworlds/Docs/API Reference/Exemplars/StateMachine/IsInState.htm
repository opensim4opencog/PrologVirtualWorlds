IsInState

 Method of the StateMachine Exemplar

Determines whether the default state machine is in the specified state. IsInState is a member of the StateMachine exemplar.  

Syntax

object.IsInState (bstrState)

The IsInState method syntax has these parts:
Part Type Use Description
object StateMachine Required An object reference that accesses a StateMachine object.
bstrState String Required The state of the state machine.

Return Value

This method returns a Boolean.

Remarks

This method is a special case of the IsStateMachineInState method of StateMachine. It executes on either the client or the server.

To find a state machine's current state, the Trace method of World, by passing it the name of the state machine object and the internal state machine for which you want to find the state. For example, to find out the state of the Default internal state machine in the SM state machine object, execute this code in the Command Window:

Trace SM.Default
 

Example

inState = MyStateMachine.IsInState("Waiting")

The foregoing example is equivalent to 

inState = MyStateMachine.IsStateMachineInState("Default", "Waiting")

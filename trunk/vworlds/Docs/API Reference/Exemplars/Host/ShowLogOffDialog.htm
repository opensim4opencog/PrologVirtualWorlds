ShowLogOffDialog

 Method of the Host Exemplar

Brings up a logoff dialog box that asks the host why the user is being logged off. ShowLogOffDialog is a member of the Host exemplar.  

Syntax

object.ShowLogOffDialog pTarget

The ShowLogOffDialog method syntax has these parts:
Part Type Use Description
object Host Required An object reference that accesses a Host object.
pTarget Thing Required The avatar to log off.

Return Value

This method does not return a value.

Remarks

The Target user cannot be a wizard; if a wizard is set as the Target, the method will fail. 

Example

HostNancy.ShowLogOffDialog RudeGwen

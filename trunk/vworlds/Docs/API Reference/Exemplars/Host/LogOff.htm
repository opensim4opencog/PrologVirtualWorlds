LogOff

 Method of the Host Exemplar

Disconnects the specified user from Microsoft Virtual Worlds. LogOff is a member of the Host exemplar.  

Syntax

object.LogOff pTarget, bstrReason

The LogOff method syntax has these parts:
Part Type Use Description
object Host Required An object reference to a Host object.
pTarget Thing Required The object that you want to log off, usually a user.
bstrReason String Required A reason for logging the user off of the system.

Return Value

This method does not return a value.

Remarks

As a host, you can log a user off of Virtual Worlds. Whether used for system maintenance or just to log off users that do not follow system rules, you can use this method to terminate their connection.

Example

This example logs off a user that attempted unauthorized use of system resources.

Dim userJohn
Set userJohn = User.Container.FindWithin("John")

myHost.LogOff userJohn, "Unauthorized use of the system."

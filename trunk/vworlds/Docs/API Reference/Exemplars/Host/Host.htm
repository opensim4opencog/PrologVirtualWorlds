Host

Exemplar

A host is an avatar that can log off other avatars. For example, a Host has the ability to log off an avatar that is not observing established guidelines of conduct in a world.

Using the Host Exemplar

Only wizards can make avatars into hosts or hosts into ordinary avatars. They can do this in one of the following two ways:

  • They can use the MakeHost or UnMakeHost methods of the World.
  • They can right-click an avatar, choose Properties..., choose Wizard, and then select the check box for Host.

A host sees an additional menu item, Log Off, when she clicks on other avatars that are not wizards. Clicking on this item gives her a dialog in which she enters a reason for logging off the user, and then allows her to log off the user. All avatars logged on see a message that the host is logging off the user for the given reason. A host cannot log off a wizard, but can log off another host. Logging off an avatar disconnects him from the world, but he can then log back on.

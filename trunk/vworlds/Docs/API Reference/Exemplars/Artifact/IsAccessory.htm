IsAccessory

 Property of the Artifact Exemplar

Set this property to True if you want the Artifact to be an avatar graphic. IsAccessory is a read/write Boolean.  

Syntax

object.IsAccessory = BooleanValue

The IsAccessory property syntax has these parts:
Part Type Use Description
object Artifact Required An object reference that accesses an Artifact object.
BooleanValue Boolean Required True if the artifact is an avatar graphic.

Remarks

An avatar graphic (or accessory) is an artifact with a two-dimensional representation in a virtual world. While you are wearing it, it is displayed on your avatar, both in your self-view pane and in the graphical windows of others in the room.

An avatar graphic can act like clothing, jewelry, a mask, a wig, a badge or button, a pet, or anything else that might appear attached to an avatar. You can place an avatar graphic anywhere on your avatar.

To make an Artifact an avatar graphic, set its IsAccessory property to True. You should also make sure that its AccessoryGraphic property is set to point to a small graphics file containing a 2-D image.

If an artifact has its IsAccessory property set to True, Microsoft Virtual Worlds automatically adds the Artifact to the right-click menu of your avatar.

Example

This example creates an artifact and adds an avatar graphic to the artifact. You must either create a RainyDayHat.gif and modify the URL to match its location or substitute an image of your choice.

Public Sub CreateHat()
Dim myRainyDayHat

'Create an artifact. 

Set myRainyDayHat = _
    CreateInstance("Rainy Day Hat", "Artifact", User.Container)

'Make the artifact an accessory. 

myRainyDayHat.IsAccessory = True

'Get the image. 

myRainyDayHat.AccessoryGraphic = _
    "http://Webs/Images/RainyDayHat.gif"

'Take the artifact. 

User.Take myRainyDayHat

'Wear the artifact. 

User.Wear myRainyDayHat

End Sub

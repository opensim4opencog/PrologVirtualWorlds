AccessoryGraphic

 Property of Artifact Exemplar

Returns or sets the URL for the image file of the accessory graphic. AccessoryGraphic is a read/write String.

Syntax

To set the property:
object.AccessoryGraphic = stringValue

To return the property:
object.AccessoryGraphic

The AccessoryGraphic property syntax has these parts:
Parameters Type Use Description
object Artifact Required An object reference to an Artifact object.
stringValue String Required The URL of the graphic.

Remarks

In a virtual world, an accessory graphic appears as an object that can be worn or held by an avatar. The AccessoryGraphic property supplies the location of the image as a URL. The URL assigned to this property must point to a graphics file that contains a two-dimensional image. The image-file format can be any one of the following: .bmp, .gif, .jpg, or .ppm.

If you add an accessory graphic to an Artifact exemplar, the value of the IsAccessory property must be True.

Coordinating Palettes for Accessories

It is possible to use different palettes for your avatars than the ones that are installed with Microsoft Virtual Worlds. If you do so, we recommend that you use the same palette for your accessories.

Example

This example creates an artifact and adds an accessory graphic to the artifact. To run this example: You must either create a StrawHat.gif and modify the URL to match its location, or substitute an image of your choice.

Public Sub CreateHat()
Dim myStrawHat
Dim myImageForHat

'Create an artifact. 

Set myStrawHat = _
    CreateInstance("Straw Hat", "Artifact", User.Container)

'Make sure the artifact is an accessory. 

StrawHat.IsAccessory = True

'Get the artifact. 

Set StrawHat = _
    User.Container.FindWithin ("Straw Hat")

'Get the image. 

StrawHat.AccessoryGraphic = _
    "http://Webs/Scripts/Images/StrawHat.gif"


'Wear the artifact.

User.Wear StrawHat

End Sub

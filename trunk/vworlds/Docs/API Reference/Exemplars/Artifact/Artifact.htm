Artifact

Exemplar

An artifact can be any object in a Room, such as a table, chair, picture, tree, house, and so on. Artifacts can be classified into two types: A regular artifact, such as a chair, or an artifact that can be worn as an accessory.

Creating Artifacts

If you want to create an artifact that is a chair, you use the CreateInstance method as shown in the following code. 

Set myChair = World.CreateInstance("Chair", "Artifact", _
  User.Container)

After you have created the chair, you must specify the graphic you want to display for that artifact.

Note:   The following URL for "myGraphic" may wrap, depending on the settings for your browser, but it is one continuous line of code.

Dim myGraphic

myGraphic = _
"file://C:\Program Files\Microsoft Virtual Worlds\Local 
Content\SDK\Library\Models\Furniture\Chair1.x"

myChair.GeometryName = myGraphic
myChair.Position = 1, 0, 0

 

Virtual Worlds stores the models for the graphics in the 
Microsoft Virtual Worlds\Local Content\SDK\Library\Models folder. However, you 
can use any graphic as long as you specify the complete URL.


Creating Accessories from Artifacts


      
 The second type of      
 artifact available in Virtual      
 Worlds is an accessory, which is an artifact that can be worn and has its 
IsAccessory property set to True. You can create articles of clothing, such as a hat, for 
your avatar by making your artifact an accessory. The following code statement assumes that you have created an artifact named "StrawHat".



'After this statement executes, "StrawHat" will be an accessory.

StrawHat.IsAccessory = True


After you have created an accessory item, you can use the Wear and UnWear 
methods of the 
avatar to add or remove accessories from your avatar, as shown in the following 
statement.



 'This statement adds the accessory to an avatar.

User.Wear StrawHat




 'This statement removes the accessory from an avatar.

User.UnWear StrawHat


To display a specific image for your accessory, you can 
use the AccessoryGraphic property as shown in the following 
statement. 



  'Display the specified image as the accessory.

StrawHat.AccessoryGraphic = _
    "http://Webs/Scripts/Images/StrawHat.gif"

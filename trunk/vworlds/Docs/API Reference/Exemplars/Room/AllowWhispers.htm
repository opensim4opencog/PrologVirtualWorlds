AllowWhispers

 Property of the Room Exemplar

A Boolean flag indicating whether whispers are allowed in the Room. AllowWhispers is a read/write Boolean.  

Syntax

object.AllowWhispers = BooleanValue

The AllowWhispers property syntax has these parts:
Part Type Use Description
object Room Required An instance of the Room exemplar.
BooleanValue Boolean Required An expression that evaluates to a Boolean. The default value is True.

Remarks

By setting this property to False, content creators prevent whispering in a room. Avatars will no longer see the "Whisper To..." menu item on other avatars, or the "Whisper..." menu item on themselves.

Example

To get this property:

   IsAllowed = MyRoom.AllowWhispers

To set this property:

   MyRoom.AllowWhispers = True

FogColor

 Property of the Room Exemplar

Sets or returns the numeric value of the fog color for the specified Room object. FogColor is a read/write Long.  

Syntax

To set a property:
object.FogColor = longValue

To return a property:
object.FogColor

The FogColor property syntax has these parts:
Part Type Use Description
object Object Required An object reference that accesses a Room object.
longValue Long Required An expression that evaluates to a valid color value. For possible values, refer to the table in the Remarks section.

Remarks

You can use the FogColor property to change the effect fog has on a Room object. The following table shows the full color values for red, green, blue, white, and black.

Color Decimal value RGB value Hexadecimal value
Red 16,711,680 (255 * 256 * 256) RGB(0, 0, 255) &HFF0000
Green 65,280 (255 * 256) RGB(0, 255, 0) Do not use.
Blue 255 RGB(255, 0, 0) &H0000FF, same as &HFF
White 16,777,215 RGB(255, 255, 255) &HFFFFFF
Black 0 RGB(0, 0, 0) &H000000

Coding with Color Constants

Microsoft Visual Basic provides color constants, such as vbRed, vbCyan, and so on. The 3-D view in Microsoft Virtual Worlds is an HTML page, which uses different constants than the constants used by Visual Basic. For example, the constant vbBlue displays as red in Virtual Worlds.

Example

This example changes the fog color using hexidecimal and RGB color values.

The following statement changes the fog color to light blue. 

User.Container.FogColor = &HDDDDDDD

'The following statement sets the fog color to green. 

User.Container.FogColor = RGB(0, 255, 0)

Portals

 Property of the Room Exemplar

A PropertyList of portals in the room. Portals is a read-only PropertyList.  

Syntax

object.Portals

The Portals property syntax has these parts:
Part Type Use Description
object Portals Required An object reference that accesses a Room object.

Remarks

The Portals property contains a PropertyList that represents every portal within the current Room, including both room portals and interworld portals.  

Example

This example displays the name of each portal in the Command Window. 

Set Doorways = User.Container.Portals

For Each Portal in Doorways
	Trace Portal.Name

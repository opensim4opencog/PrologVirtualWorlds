FogEnd

 Property of the Room Exemplar

Sets or returns the ending position of a fog effect for the specified Room object. FogEnd is a read/write Short.  

Syntax

To set the property:
object.FogEnd = shortValue

To return the property:
object.FogEnd

The FogEnd property syntax has these parts:
Part Type Use Description
object Object Required An object reference that accesses a Room object.
shortValue Short Required An expression that evaluates to a Short.

Remarks

You set the location and size of the fog effect in the 3-D view of a room by using the FogStart and FogEnd properties.  The end value is relative to the FogStart position and together they represent the extent of the fog effect.

Settings for various sizes of fog effects are shown in the following table.

Size of fog area FogStart value FogEnd value
Small

1

5

Medium

1

15

Large

-1000

1000

Fog effects can vary in size with starting and ending values from -1000 through 1000.

Example

This example sets the ending location of a fog area to 90.

User.Container.FogEnd = 90

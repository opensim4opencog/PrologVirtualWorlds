Fog

 Property of the Room Exemplar

Sets or returns True if fog is enabled for the specified Room object. Fog is a read/write Boolean.  

Syntax

To set a property:
object.Fog = BooleanValue

To return a property:
object.Fog

The Fog property syntax has these parts:
Part Type Use Description
object Object Required An object reference that accesses a Room object.
BooleanValue Boolean Required An expression that evaluates to True or False.

Remarks

You enable fog in a room by setting the Fog property to True. To establish the extent to which the fog effect applies to the 3-D view, use the FogStart and FogEnd properties. You can also vary the color that the fog displays by varying the value of the FogColor property.

Creating Realistic Fog

There are multiple factors that are involved in creating a scene with realistic-looking fog. The factors are the viewing angle or camera position, the near clipping plane, the far clipping pane, and the polygonal complexity of the model. The following diagram shows a typical viewing area of fog.

Camera Position Affects Fog Visibility

An area of fog is set relative to the camera position. The fog effect can be only applied across an entire polygon, according to its distance from the camera in the 3-D view. The fog effect does not fad realistically over an are.

Near and Far Clipping Planes

Near and far clipping planes encapsulate the area of fog display in your world. The near plane is set with the FogStart property and can be set to any positive or negative number. If you set the FogStart property with a negative number in the current view, objects up close will already have a faded effect. The fog will fade towards the color that you set using the FogColor property. 

myRoom.FogStart = -10

For a fog effect to appear heavier around farther objects than nearer ones, the FogEnd property must be set to a number greater than the FogStart property. If you set both properties to the same number, the fog will not be visible.

Polygonal Complexity Affects Realism

The fog effect can be only applied across an entire polygon and does not realistically fade over a few polygons. To increase the realism of fog effects, user a ground plane that is subdivided into many polygons.

Example

This example creates a bank of fog in a virtual world.

Sub MakeFog()
Dim myRoom

Set myRoom = User.Container

'Establish the characteristics of the fog bank. 

myRoom.FogStart = 20
myRoom.FogEnd = 150
myRoom.FogColor = &HDDDDDDD

'Start the fog. 

myRoom.Fog = True

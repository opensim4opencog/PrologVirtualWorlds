Avatars (Room)

 Property of the Room Exemplar

A PropertyList of every Avatar who is currently, or has previously been, in the Room, whether connected or not. Avatars is a read-only PropertyList.  

Syntax

object.Avatars

The Avatars property syntax has these parts:
Part Type Use Description
object Room Required An object reference that accesses a Room object.

Remarks

The Avatars property stores a PropertyList of every Avatar that has entered the specified Room, regardless of whether they are logged on or not.

Example

    
Set Visitors = User.Container.Avatars

DefaultAvatarPosition

 Property of the Room Exemplar

Sets or returns the default entry point of a Room. DefaultAvatarPosition is a read/write Vector.  

Syntax

To set this property:
object.DefaultAvatarPosition = vectorValue

To retrieve this property:
object.DefaultAvatarPosition

The DefaultAvatarPosition property syntax has these parts:
Part Type Use Description
object Room Required An object reference that accesses a Room object.
vectorValue Vector Required A Vector that specifies the default entry point of a Room. The default value is (0, 0, 0).

Remarks

The DefaultAvatarPosition property stores the default entry point of a Room. When an Avatar enters the default room of a World for the first time, the Position property of that avatar is set to the DefaultAvatarPosition value. After the first log in, each time the avatar logs out, the avatar's Position property is set to the last position recorded before logging out of that room. The updated Position property establishes the entry point for that avatar each subsequent time the avatar logs in.

If an avatar or object moves from one room to another using a Portal, the destination room uses the EntryPoint property of the portal to determine the Position property of the avatar or object.

If an avatar teleports to a room (moves from one room to another without using a portal), Microsoft Virtual Worlds first looks for the default portal of a room to establish the avatar's position. If a default portal has not been set, Virtual Worlds then looks for any portal in that room, and sets the property to the first portal listed in that room. If there is no portal in that room, the avatar's position is set to the DefaultAvatarPosition property of that room.

Example

This example displays the DefaultAvatarPosition property in the Command Window, changes the value, and then displays the new value. Since the value is a Vector, the ToAscii method is used to display the result. "User.Container" is a reference to the room the user is currently visiting.

Trace "Old Position: " & User.Container.DefaultAvatarPosition.ToAscii
User.Container.DefaultAvatarPosition = World.CreateVector(4, 0, 7)
Trace "New Position: " & User.Container.DefaultAvatarPosition.ToAscii

GeometryName

 Property of the Thing Exemplar

Sets or returns the geometry name of the specified Thing object. GeometryName is a read/write String property.

Syntax

To set the property:
object.GeometryName = stringValue

To return the property:
object.GeometryName

The GeometryName property syntax has these parts:
Parameters Type Use Description
object Thing Required An object reference to a Thing object.
stringValue String Required A String that represents the name of a sprite file. This can either be a partial path, the filename, or a valid URL.

Remarks

Use this property to set the geometry image for an object from script. Changing this property immediately affects the on-screen appearance of an object in a virtual world.

Note:   The geometry of an object disappears when you set the geometry name of an object to an empty string ("").

Example

This example illustrates how to change the image of your avatar in script.

Note:   The default folder for the sprite images is \Program Files\Microsoft Virtual Worlds\Local Content\Avatar Graphics. 


Sub ChangeAvatarLook()

'Print the current image of your avatar to the Command Window. 

Trace User.GeometryName

If User.GeometryName <> "gracie.spr" Then
    'Set a new image for your avatar.
    User.GeometryName = "gracie.spr"
End If

End Sub

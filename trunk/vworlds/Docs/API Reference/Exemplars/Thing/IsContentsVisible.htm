IsContentsVisible

 Property of the Thing Exemplar

A Boolean flag that indicates whether an object's contents can be seen graphically in the world. IsContentsVisible is a read/write Boolean.  

Syntax

object.IsContentsVisible = booleanValue

The IsContentsVisible property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
booleanValue Boolean Required Indicates whether the contents of the object can be seen. The default is False.

Remarks

To ensure that the objects inside the container are visible, set this property to True; to ensure that the objects are not visible, set it to False.

To assure that all of the objects in a container (including the geometry) ar available to every user in the room (and will thereby be displayed in each client renderer), you must first set the container's IsOpen property to True. For more information, see the IsOpen topic.

Example

To get the value:

Result = PeerAvatar.IsContentsVisible

To set the value:

PeerAvatar.Isopen = True
PeerAvatar.IsContentsVisible = True

Emote

 Method of the Thing Exemplar

Displays the specified gesture and text from the list of available gestures in Microsoft Virtual Worlds. Emote is a member of the Thing exemplar.

Syntax

object.Emote varGesture [, varText]

The Emote method syntax has these parts:
Parameters Type Use Description
object Thing Required An object reference that accesses a Thing object.
varGesture Variant Required A string that references a valid Virtual Worlds gesture.
[varText] Variant Optional A variant that specifies the text that is displayed in the chat-history window.

Return Value

This method does not return a value.

Remarks

When you invoke the Emote method, it broadcasts a tell event (see the OnTell event handler) to all of the objects within speaking range.

The value of the parameters affects the behavior of the Emote method in the following ways:

  • If the value of the varGesture parameter is a String, the Emote method executes the named gesture.
  • If you omit the optional varText parameter, the Emote method displays the default text for the specified gesture.

Although the varText parameter is a variant, the chat-history window does not display integers or values for other variant types. If you add an integer or float number as the value for the varText parameter, the chat-history window displays "Georgina smile" instead of "Georgina smiles" and ignores the integer or float value.

Example

This example demonstrates various ways to use the Emote method.

You can call the Emote method with just a string as the only parameter. When this executes, the chat-history window displays the default gesture name. The following statement executes the smile gesture for your avatar. If your avatar's name is Georgina, other users will see the text "Georgina smiles." in the chat-history window. 

User.Emote "smile"

You can call the Emote method and supply a string that displays text to the chat-history window. The following executes the smile gesture for your avatar and returns the string "greets everyone!" to the chat-history window. If your avatar's name is Georgina, other users will see the text "Georgina greets everyone!". 

User.Emote "smile", "greets everyone!"

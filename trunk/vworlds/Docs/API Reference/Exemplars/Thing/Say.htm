Say

 Method of the Thing Exemplar

Causes a string to be said. Say is a member of the Thing exemplar.  

Syntax

object.Say strText

The Say method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
strText String Required The text to say.

Return Value

This method does not return a value.

Remarks

Running this method causes the object to speak the string; this sends a Tell event.  

Example

AThing.Say "Blurgle..."

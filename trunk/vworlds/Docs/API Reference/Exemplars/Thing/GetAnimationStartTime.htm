GetAnimationStartTime

 Method of the Thing Exemplar

Returns the global time (in milliseconds) at which the animation specified by the bstrPropertyName parameter was started for the current Thing object. GetAnimationStartTime is a member of the Thing exemplar.  

Syntax

object.GetAnimationStartTime(bstrPropertyName)

The GetAnimationStartTime method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrPropertyName String Required A String value that represents one of the valid property names corresponding to the animation action you want to perform.

Valid Values for the bstrPropertyName Parameter

You can use the following property names as values for the bstrPropertyName parameter: Direction, LightAttenuation, LightColor, Position, Scale, Up, And File .

Return Value

This method returns a time value in milliseconds as a Long. This value is the time that the specified animation started running, or zero if the specified property is not animating on the object.

Remarks

You can use this method in conjunction with the GetAnimationSystemTime property to determine the current keyframe of the animation that is currently running.

The GetAnimationStartTime method returns 0 for the following two conditions:

  • If the current animation is running only on the current client.
  • If the specified animation does not exist or was stopped.

Example

This example determines how long an animation has been running and puts the number of milliseconds into the "myPositionClock" variable. 

Dim myPositionClock

myPositionClock = myBaseball.AnimationSystemTime - _
  myBaseball.GetAnimationStartTime("Position")

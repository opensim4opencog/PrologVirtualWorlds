DoNamedGesture

 Method of the Thing Exemplar

Returns the collection of the available gestures in Microsoft Virtual Worlds. DoNamedGesture is a member of the Thing exemplar.

Syntax

object.DoNamedGesture(bstrGestureName)

The DoNamedGesture method syntax has these parts:
Parameters Type Use Description
object

Thing

 Required An object reference that accesses a Thing object.
bstrGestureName String Required The name of a gesture.

Return Value

Returns a string that represents a gesture for each item in the property list.

Remarks

When you use the DoNamedGesture method, you do not have to use the gesture in combination with the Emote method to programmatically articulate emotions for your avatar. By default, Virtual Worlds has 13 gestures available for use by your avatar.

You can also use the DoGesture method to articulate a specific gesture. The difference between these two methods is that you must provide the integer value for the gesture with the DoGesture method, whereas you provide the name of the gesture with the DoNamedGesture method.

If you want to accompany a gesture with text that your avatar speaks, you must use the Emote method. The DoGesture and DoNamedGesture methods do not send text to the chat-history window.

Example

The following example prints a list of the gestures to the Command Window and then applies the smile and turn gestures to the avatar. 


Sub AvatarDoGesture()
Dim myGestures
Dim myGesture

Set myGestures = User.GetNamedGestures

'Iterate through the gestures and display their names and index numbers. 

For myGesture = 0 To 12
    Trace myGestures.Property(myGesture) & " - " & myGesture
Next

'Execute the wave and sleep gestures. 

User.DoNamedGesture "Smile"
User.DoNamedGesture "Turn"
End Sub

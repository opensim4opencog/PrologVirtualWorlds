Whisper

 Method of the Thing Exemplar

Sends a whisper message to an object or list of objects. Whisper is a member of the Thing exemplar.  

Syntax

object.Whisper strText [, userOrList]

The Whisper method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
strText String Required The text to whisper.
[userOrList] Variant Optional The target or targets to whisper to.

Return Value

This method does not return a value.

Remarks

You can set an individual target by assigning a Thing to TheTarget; to set more than one target, use a PropertyList.

Other objects in the room will be notified that the current object is whispering, but will not hear what is said.  

Example

Set MyThing = World.CreateInstance("MyThing", "Thing", User.Container)
MyThing.Whisper "TheText", User

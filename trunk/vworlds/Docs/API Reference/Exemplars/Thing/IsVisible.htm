IsVisible

 Property of the Thing Exemplar

A Boolean flag that indicates whether or not an object can be seen by the user. IsVisible is a read/write Boolean.  

Syntax

object.IsVisible = booleanValue

The IsVisible property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
booleanValue Boolean Required True is the object can be seen.

Remarks

If this property is set to False, the object cannot be seen. The default value for this property is True.  

Example

To get the IsVisible value:

   Result = AThing.IsVisible

To set the IsVisible value:

   AThing.IsVisible = False

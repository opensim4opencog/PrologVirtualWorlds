DirectionDuration

 Property of the Thing Exemplar

A Long that contains the time over which a change in the Direction property should be interpolated. DirectionDuration is a read/write Long.  

Syntax

object.DirectionDuration = LongValue

The DirectionDuration property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
LongValue Long Required The time, in milliseconds, for interpolating a change in direction.

Remarks

If you set DirectionDuration on an object to 5000, and then set the same object's Direction property to a new value, the object's Direction property appears to change smoothly from its previous value to its new value over a period of 5 seconds.

When this interpolation is complete, the method OnDirectionDone is invoked on the object.  

Example

To get this property:

   TheDuration = MyThing.DirectionDuration

To set this property:

   MyThing.DirectionDuration = 5000

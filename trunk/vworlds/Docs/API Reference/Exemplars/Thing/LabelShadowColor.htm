LabelShadowColor

 Property of the Thing Exemplar

Sets or returns the shadow color of the Label of a Thing object. LabelShadowColor is a read/write Vector.

Syntax

object.LabelShadowColor = vectorValue

The LabelShadowColor property syntax has these parts:
Parts Type Use Description
object Thing Required An instance of the Thing exemplar.
vectorValue Vector Required A Vector value that corresponds to red, green, and blue color components. The default color setting black (0, 0, 0).

Remarks

The x, y, and z values of the Vector data type correspond, respectively, to the red, green, and blue color values. The normalized range of each vector component for this property is from 0 to 1, where a value of 0 is equal to a color intensity level of 0%, and a value of 1 is equal to a color intensity level of 100%.

The LabelShadowColor property controls the color value of the shadow behind a Label of a Thing object. It does not affect the color of the actual Label. To change the color of the Label, adjust the LabelFontColor property.

To remove the shadow from the Label, set the LabelShadowColor property to Nothing.

Examples

This example changes the Label and LabelShadowColor property of a Thing object. In this example, "myThing" is a reference to an instance of the Thing object.

'Provide a label for the object. 

myThing.Label = "Bobs Label"

'Change the font color to red. 

myThing.LabelShadowColor = World.CreateVector(1, 0, 0)

'Confirm that the color was changed to the new Vector value. 

Trace "The new color value is: " & myThing.LabelShadowColor.ToAscii

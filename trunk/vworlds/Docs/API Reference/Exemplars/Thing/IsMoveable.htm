IsMoveable

 Property of the Thing Exemplar

A Boolean flag indicating whether the Position, Direction, and Scale properties of an object can be changed. IsMoveable is a read/write Boolean.  

Syntax

object.IsMoveable = booleanValue

The IsMoveable property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
booleanValue Boolean Required A setting of True means the object can be moved. The default value is True.

Remarks

The default value for IsMoveable is True. Normally, when this property on an object is False, the Position, Direction, and Scale properties of an object, within its container, cannot be changed. However, an avatar who is a Wizard or the owner of the object can still move the object, as well as change its orientation, and size.

Note:   The Container of an object and a Portal object can always change the Position, Direction, and Scale, regardless of the IsMoveable setting.  

Example

To get this property:

   Result = MyThing.IsMoveable

To set this property:

   MyThing.IsMoveable = True

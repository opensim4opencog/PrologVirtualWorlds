GetFrame

 Method of the Thing Exemplar

Retrieves the root frame for the object. GetFrame is a member of the Thing exemplar.  

Syntax

object.GetFrame()

The GetFrame method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.

Return Value

This method returns a reference to the root frame.

Remarks

This method should only be used to retrieve geometry that is already loaded or composed.

It is important to distinguish between two types of frames in Virtual Worlds. The first kind is the root frame of an object, referred to as the transform frame; this frame is returned by the GetFrame method of the Thing exemplar. The second is a child frame containing the geometry that has been built from sprites, meshes, textures and other information in the geometry file. This arrangement allows you to change the geometry of a transform frame without having to disturb the transform hierarchy. This is referred to as the geometry frame, and is returned by the GetGeometryFrame method.  

Example

Set TheFrame = MyThing.GetFrame

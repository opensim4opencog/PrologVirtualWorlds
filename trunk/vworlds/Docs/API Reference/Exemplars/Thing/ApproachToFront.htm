ApproachToFront

 Property of the Thing Exemplar

A Boolean flag indicating whether an object should be approached from its front. ApproachToFront is a read/write Boolean.  

Syntax

object.ApproachToFront = BooleanValue

The ApproachToFront property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
BooleanValue Boolean Required An expression that evaluates to a Boolean. The default value for this property is True.

Remarks

This property is used by the Approach method of Thing. Its value determines whether an approaching object stops directly in front of the destination object, or at some other point.

If the destination object's ApproachToFront property is True, the approaching object stops at a randomly chosen point in front of the destination. More precisely, the approaching object stops at a point inside a circle whose center is directly in front of the destination object.

If the destination object's ApproachToFront property is False, the approaching object stops at an randomly chosen point near the destination object. See the Approach topic for more information.

 

Example

To get this property:

   ToFront = TheObject.ApproachToFront

To set this property:

   TheObject.ApproachToFront = False

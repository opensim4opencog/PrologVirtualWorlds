IsTakeable

 Property of the Thing Exemplar

A Boolean flag that determines whether the object can be taken by another object.IsTakeable is a read/write Boolean.  

Syntax

object.IsTakeable = booleanValue

The IsTakeable property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
booleanValue Boolean Required True is the object can be taken.

Remarks

An object's IsTakeable property determines who has permission to invoke Take on the object.

The values of the IsTakeable and CanChangeContainer properties on a Thing object can potentially affect two different operations on the object: a MoveInto operation, and a Take operation. The following table summarizes the effects of all possible values of CanChangeContainer and IsTakeable.

CanChangeContainer IsTakeable Results
True True Take menu item appears on object's shortcut menu. Object can be taken or moved by anyone.
True False Take menu item does not appear, but object can be moved by anyone who invokes MoveInto on the object.
False True Take menu item does not appear. Object can not be taken or moved unless by its owner or a wizard.
False False Take menu item does not appear. Object can not be taken or moved unless by its owner or a wizard.

The General Property Page for an object sets both CanChangeContainer and IsTakeable to True if the Can be taken checkbox is checked. If this the box is unchecked, only IsTakeable is set to False.

The default value for this property is False.  

Example

To get the IsTakeable value:

   Result = AThing.IsTakeable

To set the IsTakeable value: 

   AThing.IsTakeable = False

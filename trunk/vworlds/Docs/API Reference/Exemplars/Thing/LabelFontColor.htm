LabelFontColor

 Property of the Thing Exemplar

Sets or returns the color of the Label of a Thing object. LabelFontColor is a read/write Vector.

Syntax

object.LabelFontColor = vectorValue

The LabelFontColor property syntax has these parts:
Parts Type Use Description
object Thing Required An instance of the Thing exemplar.
vectorValue Vector Required a Vector value that corresponds to red, green, and blue color components. The default setting is white (1, 1, 1).

Remarks

The x, y, and z values of the Vector data type correspond, respectively, to the red, green, and blue color values. The normalized range of each vector component for this property is from 0 to 1, where a value of 0 is equal to a color intensity level of 0%, and a value of 1 is equal to a color intensity level of 100%.

The LabelFontColor property controls the color value of a Label of a Thing object, but it does not affect the color of the shadow behind it. The color of this shadow can be changed by adjusting the LabelShadowColor property.

Example

This example changes the Label and LabelFontColor property of a Thing object. In this example, "myThing" is a reference to an instance of the Thing object.

'Provide a label for the object. 

myThing.Label = "Bob's Label"

'Change the font color to red. 

myThing.LabelFontColor = World.CreateVector(1, 0, 0)

'Confirm that the color was changed to red. 

Trace "The new color value is: " & myThing.LabelFontColor.ToAscii

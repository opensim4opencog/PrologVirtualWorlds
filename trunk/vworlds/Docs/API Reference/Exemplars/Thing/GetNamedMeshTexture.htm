GetNamedMeshTexture

 Method of the Thing Exemplar

Returns the URL of a given mesh's texture. GetNamedMeshTexture is a member of the Thing exemplar.  

Syntax

object.GetNamedMeshTexture(bstrMeshName)

The GetNamedMeshTexture method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrMeshName String Required The name of the mesh.

Return Value

This method returns a String.

Remarks

This method returns the texture from the first mesh of the given name on the object.  

Example

     LegMeshURL = Table.GetNamedMeshTexture ("Leg1")

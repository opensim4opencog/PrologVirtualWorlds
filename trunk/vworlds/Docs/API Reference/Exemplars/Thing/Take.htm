Take

 Method of the Thing Exemplar

Causes an object to be taken. Take is a member of the Thing exemplar.  

Syntax

object.Take pthing

The Take method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
pthing Thing Required The object to take.

Return Value

This method does not return a value.

Remarks

This method takes TakeObject, which must be in the same container as the Thing attempting to take it, and adds it to the contents of the calling Thing. For Take to succeed, both the CanChangeContainer property and the IsTakeable property of TakeObject must be True.  

Example

AThing.Take myObject

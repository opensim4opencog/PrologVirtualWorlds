LabelPlacement

 Property of the Thing Exemplar

Sets or returns the position of the Label for a Thing object. LabelPlacement is a read/write Long.

Syntax

object.LabelPlacement = longValue

The LabelPlacement property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing exemplar.
longValue Long Required An expression that evaluates to one of the enumerated types in the following table. Combinations of these values are not valid.

Enumerated Types For the longValue Parameter

Enumerated type Value Description
LABEL_CENTER 0 The label is centered on the face of the object.
LABEL_ABOVE 1 The label is centered above the object. The default setting.
LABEL_BELOW

2

 The label is centered below the object.
LABEL_RIGHT

4

 The label is centered to the right of the object.
LABEL_LEFT

8

 The label is centered to the left of the object.
 

Remarks

The LabelPlacement property contains the position of the Label of a Thing object relative to the bounding box of the object. A bounding box is the smallest possible area that all of the polygons of an object will fit within. The Label is always placed beyond the outer edges of an object, regardless of its scale. For more information on the boundary coordinate system and bounding boxes, see the Boundary topic.

A Label assigned any valid position (enumerated in the previous table) is visible unless it is obstructed by an adjacent object. For example, if you set the LabelPlacement property to LABEL_BELOW, and the base of that object rests on a floor with an opaque geometry, then the Label would not be visible-the floor would obstruct your view.

Note:   Unlike many enumerated types, expressions that combine settings (for example, "LABEL_ABOVE + LABEL_RIGHT") are not valid. Combinations of LabelPlacement values use a standard error position by default.

Example

The following example gives an object a Label and then changes the position of the Label. In this example, "strMyName" equals "Bob", "strMyLabel" equals "Bob's Label", and "IntMyLabelPosition" equals "LABEL_RIGHT".

Sub LabelPosition(strMyName, strMyLabel, lntMyLabelPosition)
Dim myVehicle
Dim myOldLabelPosition

'Acquire a reference to the object named "Bob". 

Set myVehicle = User.Container.FindWithin(strMyName)

'Provide a label for the object that says, "Bob's Label". 

myVehicle.Label = strMyLabel

'Print the previous location to the Command Window. 

Set myOldLabelPosition = myVehicle.LabelPlacement
Print "The previous LabelPlacement was: " & myOldLabelPosition

'Change the label position to "LABEL_RIGHT". 

myVehicle.LabelPlacement = strMyLabelPosition

'Print the new position in the Command Window. 

Print "The new position is: " & myVehicle.LabelPlacement

End Sub

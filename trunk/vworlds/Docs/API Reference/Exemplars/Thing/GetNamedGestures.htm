GetNamedGestures

 Method of the Thing Exemplar

Returns the collection of the available gestures for a Thing object in Microsoft Virtual Worlds. GetNamedGestures is a member of the Thing exemplar.

Syntax

object.GetNamedGestures()

The GetNamedGestures property syntax has these parts:
Parts Type Use Description
object

Thing

 Required An object reference to a Thing object.

Return Value

Returns a property list of gesture names.

Remarks

You can use the gestures in the property list in combination with the Emote method of the Thing object to programmatically articulate emotions for your avatar. By default, Virtual Worlds has 13 gestures available for use by your avatar.

Gestures Available in Virtual Worlds

Gesture nameDescription
Idle The default stationary pose for the avatar.
Wave Causes the avatar to wave.
Smile Causes the avatar to smile.
Flirt Causes the avatar to flirt.
Shrug Causes the avatar to shrug.
Trick 1 Causes the avatar to perform an act.
Trick 2 Causes the avatar to perform an act.
Turn Causes the avatar to turn.
Laugh Causes the avatar to laugh.
Blush Causes the avatar to blush.
Angry Causes the avatar to angry.
Trick 3 Causes the avatar to pose for a picture.
Sleep Causes the avatar to sleep.

You can also use the DoGesture or the DoNamedGesture  method to articulate a specific gesture. The difference between these two methods is that with the DoGesture method you must provide the integer value for the gesture, whereas with the DoNamedGesture method, you provide the name of the gesture.

If you want to accompany a gesture with text that your avatar speaks, you must use the Emote method. The DoGesture method does not send text to the chat-history window.

Example

This example prints a list of the gestures to the Command Window and then applies the wave and smile gestures to the avatar. 


Sub MakeAvatarWaveSmile()
Dim myGestures
Dim myGesture

Set myGestures = User.GetNamedGestures

'Iterate through the gestures and display their names and index numbers. 

For myGesture = 0 To 12
    Trace myGestures.Property(myGesture)
Next

'Execute the wave and smile gestures with text echoed to the chat-history window. 

User.Emote "Wave"
User.Emote "Smile", "Hi Everyone!"
End Sub

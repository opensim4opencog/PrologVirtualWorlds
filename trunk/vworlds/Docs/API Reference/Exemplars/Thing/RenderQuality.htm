RenderQuality

 Property of the Thing Exemplar

A Short that specifies the rendering quality. RenderQuality is a read/write Long.  

Syntax

object.RenderQuality = longValue

The RenderQuality property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
longValue Short Required An expression that evaluates to a valid Long value. See the table in the Remarks section for valid values.

Remarks

This property applies to all meshes in the frame of the object. It takes a value from 0 to 4, as follows:

Value Meaning Comment
0 Wireframe No shading
1 Solid shading Not affected by lights
2 Solid shading Affected by lights; each triangle will be a solid color
3 Gouraud shading Affected by lights; smoothly shaded
4 Phong shading Like Gouraud shading, but better quality
 

Example

To get this property:

   TheQuality = MyThing.RenderQuality

To set this property:

   MyThing.RenderQuality = 4

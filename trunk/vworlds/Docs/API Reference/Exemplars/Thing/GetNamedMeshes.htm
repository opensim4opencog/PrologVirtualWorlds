GetNamedMeshes

 Method of Thing Exemplar

Returns a PropertyList containing all unique names of the meshes on the object. GetNamedMeshes is a member of the Thing exemplar.  

Syntax

object.GetNamedMeshes()

The GetNamedMeshes method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.

Return Value

This method returns a PropertyList.

Remarks

There may be several meshes of the same name on the object. This method will return all unique names; that is, it will return exactly one of each mesh name that appears on the object.  

Example

Dim myMeshNames
Set myMeshNames = Table.GetNamedMeshes

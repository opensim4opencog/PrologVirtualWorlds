Report

 Method of the Thing Exemplar

Causes a string to be reported. Report is a member of the Thing exemplar.  

Syntax

object.Report bstrReport

The Report method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrReport String Required The text to report.

Return Value

This method does not return a value.

Remarks

This method reports the string to every object in the calling object's container with an OnTell event. Reports are typically used to provide low-priority user interface feedback. Note that Report runs only on the client; only the user of the client on which it is called will ever see reported text when this method runs.  

Example

MyThing.Report "Error!"

CanChangeContainer

 Property of the Thing Exemplar

Flag that determines whether the object can moved into another Thing object using the MoveInto or Take method. CanChangeContainer is a read/write Boolean.  

Syntax

object.CanChangeContainer = booleanValue

The CanChangeContainer property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
booleanValue Boolean Required True is the object can be moved.

Remarks

The values of the CanChangeContainer and IsTakeable properties on a Thing object can potentially affect two different operations on the object: a MoveInto operation, and a Take operation. The following table summarizes the effects of all possible values of CanChangeContainer and IsTakeable.

CanChangeContainer IsTakeable Results
True True Take menu item appears on object's shortcut menu. Object can be taken or moved by anyone.
True False Take menu item does not appear, but object can be moved by anyone who invokes MoveInto on the object.
False True Take menu item does not appear. Object can not be taken or moved unless by its owner or a wizard.
False False Take menu item does not appear. Object can not be taken or moved unless by its owner or a wizard.

The General property page for an object sets both CanChangeContainer and IsTakeable to True if the Can be taken checkbox is checked. If this the box is uncheck, only IsTakeable is set to False.

The default value of CanChangeContainer is True.  

Example

To get CanChangeContainer: 

   Set IsOKToMove = AThing.CanChangeContainer

To set CanChangeContainer: 

   AThing.CanChangeContainer = False

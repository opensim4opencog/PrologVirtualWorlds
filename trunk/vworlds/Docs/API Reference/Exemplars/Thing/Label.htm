Label

 Property of the Thing Exemplar

Places a label above a Thing object. Label is a read/write String.

Syntax

object.Label = stringValue

The Label property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing exemplar.
stringValue String Required Any valid expression that evaluates to a string.

Remarks

The Label property provides a means of placing a name tag on a Thing object. When a label is applied to a Thing object, the following defaults are applied.

Property Default setting
LabelFontColor White (1,1,1)
LabelFontName Arial
LabelFontSize 24pt
LabelPlacement Above (1)
LabelShadowColor Black (0,0,0)
LabelStyle None (0)

When an object is visible in the 3-D view, the label faces you at all times.

Examples

This example takes several label elements as arguments to quickly create a label and customize the formatting. In this example, "strName" equals "Bob", "strLabel" equals "Bob's Label", "strFont" equals "Helvetica", "intSize" equals "16", and "vecLabelColor" equals "(1,0,0)", which corresponds to the color red.

Sub MyLabelFormat(strName, strLabel, strFont, intSize, vecLabelColor)

'Acquire a reference to the object named "Bob". 

Dim MyThing
Set MyThing = User.Container.FindWithin(strName)

'Set the label to "Bob's Label". 

MyThing.Label = strLabel

'Change the font to Helvetica. 

MyThing.LabelFontName = strFont

'Change the size of the text to 16. 

MyThing.LabelFontSize = intSize

'Change the color of the text to red. 

MyThing.LabelColor = vecLabelColor

End Sub

Example 2

This example sets the Label caption, retrieves the current setting, and prints the results to the Command Window. 

Dim myThing
Dim myConfirmation

'Acquire a reference to an existing object named "MyThing" and change its label. 

Set myThing = User.Container.FindWithin("MyThing")
myThing.Label = "This is a Label"

'Retrieve the value of the Label property and print the results in the Command Window 

Set myConfirmation = myThing.Label
Print myConfirmation

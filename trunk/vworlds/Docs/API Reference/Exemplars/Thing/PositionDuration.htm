PositionDuration

 Property of the Thing Exemplar

A Long that contains the time over which a change in the Position property should be interpolated. PositionDuration is a read/write Long.  

Syntax

object.PositionDuration = longValue

The PositionDuration property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
longValue Long Required A Long that corresponds to the time value in milliseconds.

Remarks

If you set PositionDuration on an object to 5000, and then set the same object's Position property to a new value, the object's Position property appears to change smoothly from its previous value to its new value over a period of 5 seconds.

When this interpolation is complete, the method OnPositionDone is invoked on the object.  

Example

To get this property:

   TheDuration = MyThing.PositionDuration

To set this property:

   MyThing.PositionDuration = 5000

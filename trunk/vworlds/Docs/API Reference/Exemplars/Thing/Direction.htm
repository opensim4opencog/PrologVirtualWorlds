Direction

 Property of the Thing Exemplar

Specifies the orientation of the object. The orientation is the direction that the object faces. Direction is a read/write Vector.  

Syntax

To set the property:
object.Direction = VectorValue

To return the property:
object.Direction

The Direction property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
VectorValue Vector Required The orientation of the object.

Remarks

The Direction property consists of a vector that describes the orientation of the front of the object. The vector originates at the center of the object and points through the front of the object. You can set the direction of an object in two ways: You can create a vector and assign it, or you can use the x, y, and z properties of the existing vector.

To set the direction of an object with a vector, use the following code. 

Dim myPlane

Set myPlane = World.CreateVector(0.0, 0.0,  -9.0)< /P  >< /P  >< /P >< /P >

The previous code points the front of the plane counter-clockwise from its original position. To set an individual coordinate, use the following code.

myPlane.Direction.x = 9.0

The Direction and Up properties together determine the rotation of the object.

The Vector topic has detailed information about the way vectors work in Microsoft Virtual Worlds.

Important:   Do not set the Up vector to 0, 0, 0. This causes D3D problems. Also, never set the Direction and Up vectors to the same values.  

Example

This example demonstrates the use of the Direction property.

Dim myVector

'Create a vector to establish coordinates for x, y, and z.
'Retrieve the y-coordinate. 

Set myVector = CreateVector(10, 5, 3)
Trace myVector.y

'Establish the y-coordinates independently of a vector object. 

Dim myChair

Set myChair = User.Container.FindWithin("Chair")

myChair.Direction.y = 2
myChair.Position.y = 4
myChair.RotationAxis.y = 1
myChair.Up.y = 0

'This next code block retrieves the values for the
'x-coordinate independently of a vector object. 

Dim myChairY

Trace myChair.Direction.y
Trace myChair.Up.y
myChairY = myChair.Position.y

IsCameraAnchor

 Property of the Thing Exemplar

A Boolean flag that indicates whether the object will show up in the client's list of potential camera viewpoints. IsCameraAnchor is a read/write Boolean.  

Syntax

object.IsCameraAnchor = BooleanValue

The IsCameraAnchor property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
BooleanValue Boolean Required An expression that evaluates to a Boolean.

Remarks

The default value for this property is False.  

Example

To get this property:

   Result = MyThing.IsCameraAnchor

To set this property:

   MyThing.IsCameraAnchor = True

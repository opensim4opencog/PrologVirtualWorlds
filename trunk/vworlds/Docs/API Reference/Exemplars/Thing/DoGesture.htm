DoGesture

 Method of the Thing Exemplar

Returns the specified gesture from the list of available gestures in Microsoft Virtual Worlds. DoGesture is a member of the Thing exemplar.

Syntax

object.DoGesture iGesture

The DoGesture property syntax has these parts:
Parts Type Use Description
object Thing Required An object reference that accesses a Thing object.
iGesture Integer Required Represents a valid Virtual Worlds gesture.

Return Value

This method does not return a value.

Remarks

When you use the DoGesture method, you do not have to use the gesture in combination with the Emote method of the Thing object. With the DoGesture method, you programmatically articulate emotions for your avatar by specifying the integer that references the gesture that you want. Virtual Worlds has a default of 13 gestures available for use by your avatar.

Note:   When you execute the DoGesture method, there is no accompanying text sent to the chat-history window. You must use the Emote method in order to add text to the chat-history window.

Example

The following example prints a list of the gestures to the Command Window and then applies the wave and sleep gestures to the avatar. 


Sub AvatarDoGesture()
Dim myGesture

Set myGestures = User.GetNamedGestures

'Iterate through the gestures and display their names and index numbers.
For myGesture = 0 To 12
    Trace User.GetNamedGestures.Property(myGesture) & " - " & myGesture
Next

'Execute the wave and sleep gestures.
User.DoGesture 1
User.DoGesture 12
End Sub

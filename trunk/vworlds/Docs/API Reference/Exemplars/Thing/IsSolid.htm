IsSolid

 Property of the Thing Exemplar

A Boolean flag indicating whether the object has an impassable boundary. IsSolid is a read/write Boolean.  

Syntax

object.Property = booleanValue

The IsSolid property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
booleanValue Boolean Required True is the object has a boundary.

Remarks

Setting this property to True triggers the OnIsSolidChanged event handler, thereby creating a simple, hexagonal, impassable Boundary around the Thing based on the size of its geometry. Setting this property to False triggers the same event; in this case, it removes the boundary that was previously created.  

Example

To get this property:

   ImpassableFlag = MyThing.IsSolid

To set this property:

   MyThing.IsSolid = True

Position

 Property of the Thing Exemplar

A Vector that specifies an object's position in three-dimensional space. Position is a read/write Vector.  

Syntax

To set the property:
object.Position = VectorValue

To return the property:
object.Position

The Position property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
VectorValue Vector Required The position of the object in three-dimensional space.

Remarks

The coordinates supplied in the Position property are relative to the center point of the world.

This property is a read-only Vector when the calling object is the avatar of a connected client.  

Example

This example creates an instance of the Artifact exemplar, changes the Position property, and then displays the new value in the Command Window.

Dim myChair

Set myChair = World.CreateInstance("myChairObject", "Artifact", User.Container)
myChair.Position = World.CreateVector(5, 2, 7)
Trace myChair.Position.ToAscii

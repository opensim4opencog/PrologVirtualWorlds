LabelFontName

 Property of the Thing Exemplar

Sets or returns the font for the Label of a Thing object. LabelFontName is a read/write String.

Syntax

object.LabelFontName = stringValue

The LabelFontName property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing exemplar.
stringValue String Required Any valid font expressed as a string.

Remarks

The default setting of the LabelFontName property is Arial.

The LabelFontName property derives a list of valid fonts from the list of fonts installed in your Windows folder. To add a font to this list, follow the instructions in the Windows 98 Help system under the heading, "To add a new font to your computer."

If an invalid font is assigned to the LabelFontName property, the font displayed in the 3-D view is undefined, but the name in the LabelFontName property is listed as the last string assigned to it. For example, if you wanted to change the font to Courier and you misspelled it "Dourier," the font displayed in the 3-D view would default an undefined font. However, if you used the Trace or Print method to display the current LabelFontName property setting, it would return "Dourier" in the Command Window.

Example

This example gives an object a Label and then changes the font. In this example, "strName" equals "Bob", "strLabel" equals "Bob's Label", and "strFont" equals "Helvetica". 

Sub MyLabelFont(strName, strLabel, strFont)
Dim MyLamp
Dim OldFont

'Acquire a reference to the object named "Bob". 

Set MyLamp = User.Container.FindWithin(strName)

'Provide a label for the object that says, "Bob's Label". 

MyLamp.Label = strLabel

'Print the name of the previous font to the Command Window. 

Set OldFont = MyLamp.LabelFontName
Print "The previous font was: " & OldFont

'Change the font. 

MyLamp.LabelFontName = strFont

'Print the new font in the Command Window. 

Print "The new font is: " & MyLamp.LabelFontName

End Sub

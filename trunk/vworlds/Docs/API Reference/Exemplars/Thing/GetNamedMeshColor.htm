GetNamedMeshColor

 Method of the Thing Exemplar

Returns the color of a given mesh. GetNamedMeshColor is a member of the Thing exemplar.  

Syntax

object.GetNamedMeshColor(bstrMeshName)

The GetNamedMeshColor method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrMeshName String Required The name of the mesh.

Return Value

This method returns a Long.

Remarks

This method returns the color of the first mesh of the given name on the object.

To understand the value returned by GetNamedMeshColor, it is useful to know the internal hexadecimal (base 16) format this method uses. In this format, a color is represented by three pairs of hexadecimal digits in the form &HBBGGRR. Each pair of digits can hold any of 256 values, from &H0 to &HFF (0 to 255). The left-most pair of digits represents the amount of blue in the color, the middle pair represents the amount of green, and the right-most pair represents the amount of red. The entire number can represent any of 16,777,215 distinct colors.

The following table lists the internal hexadecimal color representation and the value returned for three colors.

Color Hexadecimal Value Method Return Value
Pure Red &H0000FF 255
Pure Green &H00FF00 65280
Pure Blue &HFF0000 16711680
 

Example

LegMeshColor = Table.GetNamedMeshColor ("Leg1")

IsNoticeable

 Property of the Thing Exemplar

Sets or returns a Boolean value of True if the specified Thing object is marshalled to the client computers. IsNoticeable is a read/write Boolean.  

Syntax

To set the property:
object.IsNoticeable = BooleanValue

To return the property:
object.IsNoticeable

The IsNoticeable property syntax has these parts:
Part Type Use Description
object Thing Required An object reference to a Thing object.
BooleanValue Boolean Required Set to True when you want to marshall content of a Thing object to client computers. The default is False.

Remarks

When set to True, this property marshalls content for an individual Thing object in a container from the server to client computers. 

Use the IsOpen method to marshall the content for a container.

Example

This example creates a character and a hot tub, places the character in the hot tub, and then marshalls the change to the client computers.

Sub MarshallObject()
Dim Spa
Dim Hugh

Set Spa = CreateInstance("Spa", "Artifact", User.Container)
Set Hugh = CreateInstance("Hugh Mann", "State Machine", User.Container)

Spa.GeometryName = "SDK\Library\Models\Other\Hottub.x"
Hugh.GeometryName = "SDK\Library\Models\Character\Man1.x"

Spa.Take User.Container.FindWithin("Hugh Mann")

Hugh.IsNoticeable = True

End Sub

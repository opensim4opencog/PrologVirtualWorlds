SetNamedMeshTexture

 Method of the Thing Exemplar

Sets the texture of the given mesh to the given texture. SetNamedMeshTexture is a member of the Thing exemplar.  

Syntax

object.SetNamedMeshTexture bstrMeshName, bstrTextureURL

The SetNamedMeshTexture method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrMeshName String Required The name of the mesh.
bstrTextureURL String Required The URL of the texture to use.

Return Value

This method does not return a value.

Remarks

When you use this method to set the texture of a named mesh, the change applies to every mesh on the object with that name.  

Example

Table.SetNamedMeshTexture "Leg1", "http://www.microsoft.com/vworlds/woodgrn.gif"

SetAnimationIndex

 Method of the Thing Exemplar

Changes the starting or ending position for an animation while it is running. This method essentially provides a random-access, fast-forward/rewind feature. SetAnimationIndex is a member of the Thing exemplar exemplar.

Syntax

object.SetAnimationIndex bstrPropertyName, varIndex [, varOptions]

The SetAnimationIndex method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrPropertyName String Required A String value that represents one of the valid property names corresponding to the values provided for the StartAnimation method.
varIndex Float Required A Float that specifies the keyframe where the animation executes.

If the keyframe is set before the start or after the end of the animation path, the keyframe will be snapped to the front or the end of the animation, respectively.

If the animation is executing and set to run once, it will be stopped if the keyframe is set to the end of the animation.

[varOptions] Variant Optional

A bitflag with component values that determine the overall behavior of the animation. The bitflag values are added together to compose the resulting flag. 

Default value for this flag A_LINEAR + A_ONCE.

Valid Values for the bstrPropertyName Parameter

You can use the following property names as values for the bstrPropertyName parameter.

Property Description
Direction Set the index of an ongoing "Direction" animation.
LightAttenuation Set the index of an ongoing "LightAttenuation" animation.
LightColor Set the index of an ongoing "LightColor" animation.
Position Set the index of an ongoing "Position" animation.
Scale Set the index of an ongoing "Scale" animation.
Up Set the index of an ongoing "Up" animation.
File Set the index of an ongoing "File" animation.

Return Value

This method does not return a value. 

Remarks

This method positions an animation at any keyframe while the animation is running. To affect the starting position of an animation before it runs, use the varIndex parameter of the StartAnimation method. 

Note:   Objects with animations imbedded in their geometry can set their index even if the animation is not running.

Example

This example changes the index position, direction, and file.

Sub ReIndex()
Dim myTigerHawk

Set myTigerHawk = User.Container.FindWithin("TigerHawk")

myTigerHawk.SetAnimationIndex "Position", 12000, 71
myTigerHawk.SetAnimationIndex "Direction", 32000, 71
myTigerHawk.SetAnimationIndex "File", 15, 31

End Sub

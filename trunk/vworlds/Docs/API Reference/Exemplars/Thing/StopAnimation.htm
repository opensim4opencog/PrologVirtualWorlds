StopAnimation

 Method of the Thing Exemplar

Stops an animation that is currently executing for the specified object. StopAnimation is a member of the Thing exemplar. 

Syntax

object.StopAnimation bstrPropertyName [, varOptions]

The StopAnimation method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrPropertyName String Required A String value that represents one of the valid property names corresponding to the animation action you want to perform.
[varOptions] Variant Optional Enumerated types that specify the type of animation that is running.

Valid Values for the bstrPropertyName Parameter

You can use the following property names as values for the bstrPropertyName parameter.

Property Description
Direction Interpolate the Direction property of an object over the path provided in the varPathMap parameter of the StartAnimation method.
LightAttenuation Interpolate the LightAttenuation property of an object over the path provided in the varPathMap parameter of the StartAnimation method.
LightColor Interpolate the LightColor property of an object over the path provided in the varPathMap parameter of the StartAnimation method.
Position Interpolate the Position property of an object over the path provided in the varPathMap parameter of the StartAnimation method.
Scale Interpolate the Scale property of an object over the path provided in the varPathMap parameter of the StartAnimation method.
Up Interpolate the Up property of an object over the path provided in the varPathMap parameter of the StartAnimation method.
Xfile Run the animation embedded in the geometry file.

Return Value

This method does not return a value.

Remarks

The StopAnimation method stops an animation that was previously begun with the StartAnimation method. If you use the varCallBack parameter of the StartAnimation method, the StopAnimation method does not affect the firing of the method specified by the varCallBack parameter.

Example

The following statements assume that you have previously created a PropertyMap object named "mapPosition" that contains the vectors for the movement of the baseball. 

Dim myBaseball

Set myBaseball = CreateInstance("Baseball", "Artifact", User.Container)

'The number 2 represents a continually looping animation. 

myBaseball.StartAnimation "Position", myBaseball.mapPosition, , 2

'The animation loops until stopped with the StopAnimation method. 

myBaseball.StopAnimation

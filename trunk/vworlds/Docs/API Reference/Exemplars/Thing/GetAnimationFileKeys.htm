GetAnimationFileKeys

 Method of the Thing Exemplar

Returns a property list of animation keys contained in the file. GetAnimationFileKeys is a member of the Thing exemplar.  

Syntax

object.GetAnimationFileKeys

The GetAnimationFileKeys method syntax has these parts:
Part Type Use Description
object Thing Required An object reference to a Thing object.

Return Value

This method does not return a value.

Remarks

This method looks at the current geometry of the object for animation data. If the method fails to find geometry data, it returns an empty property list.

By checking the Count property of the returned list you can determine whether the file can be animated with the StartAnimation method. 

Example

This example fills in the key list for the baseball animations from the "Baseball.x" animation file.

Set myPropertyList = myFileObject.GetAnimationFileKeys

If myPropertyList.Count > 0 Then
    myBaseballFile.StartAnimation "File", , , , .001
Else
    Print "The file does not contain animation information in it."
End If

Drop

 Method of the Thing Exemplar

Causes an object to be dropped from the contents of the Thing object. Drop is a member of the Thing exemplar.  

Syntax

object.Drop pthing

The Drop method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
pthing Thing Required The object to drop.

Return Value

This method does not return a value.

Remarks

The Drop method enables its caller to transfer an object in the caller's inventory to the room (or other container) the caller is in.  

Example

TheThing.Drop ObjectToDrop

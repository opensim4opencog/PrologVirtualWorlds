Sound

 Property of the Thing Exemplar

A property that contains sound data for the current Thing. Sound is a read/write String.  

Syntax

object.Sound = stringValue

The Sound property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
stringValue String Required The URL of the sound file.

Remarks

To create a Sound object, use the CreateSound method of World. To begin playing a Sound object, set the Sound property on a Thing to refer to the Sound object you want to play. In this version of Virtual Worlds, there is no other way to play a sound. See the Example section of this topic to see how this is done.  

Example

To get the Sound property:

   Set Result = MyThing.Sound

To set the Sound property:

   MyThing.Sound = SoundObject

Here is an example of how to create a sound, and attach it to a Thing. The file path is for illustration purposes only.

Set BirdSound = World.CreateSound( "File://c:\eagle2.wav", False, False )   ' Line 1
Set BotRef = User.Container.FindWithin( "Bot" )			  ' Line 2
BotRef.Sound = BirdSound					  ' Line 3

In line 1 of this example, CreateSound uses the file EAGLE2.WAV to create a Sound object named BirdSound. The second parameter indicates that this is a 2D sound, rather than a 3D sound. The third parameter specifies that the sound should be played once only.

In line 2, a reference (BotRef) to a Thing object named Bot is obtained. Bot is assumed to be in the same Container (that is, room) as the User.

In line 3, the Sound property on BotRef is set to refer to BirdSound, the Sound object created in line 1. As soon as this line is executed, the sound will begin playing. To stop the sound from playing, set the object's Sound property to Nothing.

RotationAxis

 Property of the Thing Exemplar

A Vector that specifies the axis of rotation of the object. RotationAxis is a read/write Vector.  

Syntax

object.RotationAxis = VectorValue

The RotationAxis property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
VectorValue Vector Required The axis of the rotation of the object. The default value is Null.

Remarks

The RotationAxis property species a direction vector with the origin set at the center of the object.

This property has no effect if the object is an avatar.

 

Example

To get this property:

   Set TheVec = MyThing.RotationAxis

To set this property:

   MyThing.RotationAxis = TheVec

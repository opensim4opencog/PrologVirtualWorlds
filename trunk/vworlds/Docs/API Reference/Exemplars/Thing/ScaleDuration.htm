ScaleDuration

 Property of the Thing Exemplar

A Long that contains the time over which a change in the Scale property should be interpolated. ScaleDuration is a read/write Long.  

Syntax

object.ScaleDuration = longValue

The ScaleDuration property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing exemplar.
longValue Long Required The time, in milliseconds, over which a change in the Scale property should be interpolated. The default value is 0.

Remarks

If you set ScaleDuration on an object to 5000, and then set the same object's Scale property to a new value, the object's Scale property appears to change smoothly from its previous value to its new value over a period of 5 seconds.

When this interpolation is complete, the method OnScaleDone is invoked on the object.  

Example

To get this property:

   TheDuration = MyThing.ScaleDuration

To set this property:

   MyThing.ScaleDuration = 5000

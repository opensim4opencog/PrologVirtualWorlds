Scale

 Property of the Thing Exemplar

A Vector that specifies the scaling factor for an object in three dimensions. Scale is a read/write Vector.  

Syntax

object.Scale = VectorValue

The Scale property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
VectorValue Vector Required The scaling factor for an object in three dimensions.

Remarks

If you set any component of an object's Scale property to 0, the object's Direction and Up properties are then reset to new values. For this reason, it is not recommended that you set any component of this property to 0.  

Example

This example demonstrates the use of the Scale property.

Dim myVector

'Create a vector to establish coordinates for x, y, and z.
'Retrieve the y-coordinate. 

Set myVector = CreateVector(10, 5, 3)
Trace myVector.y

'Establish the y-coordinates independently of a vector object. 

Dim myChair

Set myChair = User.Container.FindWithin("Chair")

myChair.Direction.y = 2
myChair.Scale.y = 4
myChair.RotationAxis.y = 1
myChair.Up.y = 0.15

'This next code block retrieves the values for the
'x-coordinate independently of a vector object. 

Dim myChairY

Trace myChair.Scale.y
Trace myChair.Up.y
myChairY = myChair.Position.y

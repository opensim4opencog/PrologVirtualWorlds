RotationRate

 Property of the Thing Exemplar

A Float that specifies the rate of rotation of the object. RotationRate is a read/write Float.  

Syntax

object.RotationRate = floatValue

The RotationRate property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
floatValue Float Required A Float value that determines the rotation speed of an object.

Remarks

This property represents the number of radians that the object should be rotated each time the scene is rendered. (There are 2 * Pi radians in 360 degrees.) If the scene is being rendered at twenty frames per second and you want the object to make one revolution per second, you should set RotationRate to (2 * Pi / 20), or approximately 0.3.

A positive value for RotationRate causes the object to revolve clockwise (assuming the object is rotating about its vertical axis and is viewed from above). A negative value causes the object to revolve in the opposite direction.

This property has no effect if the object is an avatar.

The default value for this property is 0.  

Example

To get this property:

   RotRate = MyThing.RotationRate

To set this property:

   MyThing.RotationRate = 0.2

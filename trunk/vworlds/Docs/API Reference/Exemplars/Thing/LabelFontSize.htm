LabelFontSize

 Property of the Thing Exemplar

Sets or returns the font size of the Label of a Thing object. LabelFontSize is a read/write Long.

Syntax

object.LabelFontSize = longValue

The LabelFontSize property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing exemplar.
longValue Long Required Any valid expression that evaluates to a Long. The font size is measured in points.

Remarks

The default setting of the LabelFontSize property is 24 point. Points are the only available unit of measurement for this property.

Example

This example gives an object a Label and then changes the font size. In this example, "strName" equals "Bob", "strLabel" equals "Bob's Label", and "intSize" equals "16". 

Sub MyLabelSize(strName, strLabel, lntSize)
Dim MyWorker
Dim OldSize

'Acquire a reference to the object named "Bob". 

Set MyWorker = User.Container.FindWithin(strName)

'Provide a label for the object that says, "Bob's Label". 

MyWorker.Label = strLabel

'Print the previous font size to the Command Window. 

Set OldSize = MyWorker.LabelFontSize
Print "The previous font size was: " & OldSize

'Change the font size to 16. 

MyWorker.LabelFontSize = OldSize

'Print the new font size in the Command Window. 

Print "The new font size is: " & MyWorker.LabelFontSize

End Sub

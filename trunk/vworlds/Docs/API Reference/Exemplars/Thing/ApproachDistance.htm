ApproachDistance

 Property of the Thing Exemplar

A Float that specifies the approximate distance between an approaching object and its destination object at the end of an approach. ApproachDistance is a read/write Float.  

Syntax

object.ApproachDistance = floatValue

The ApproachDistance property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
floatValue Float Required An expression that evaluates to a Float.

Remarks

A destination object's ApproachDistance property defines a circle centered on the object, and with a radius equal to this property's value. An approaching object ends up at some point in a second circle whose center lies on the circle determined by ApproachDistance.

The default value for this property is 4.0 units.

The Approach method of Thing uses three properties to determine the final position of an object when it approaches a destination object: ApproachDistance, ApproachToFront, and ApproachRadius. For a complete discussion of all three properties, see Approach.  

Example

To get this property:

   Dist = AThing.ApproachDistance

To set this property:

   AThing.ApproachDistance = 3.0

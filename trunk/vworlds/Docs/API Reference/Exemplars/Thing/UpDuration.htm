UpDuration

 Property of the Thing Exemplar

A Long that contains the time over which a change in the Up property should be interpolated. UpDuration is a read/write Long.  

Syntax

object.UpDuration = longValue

The UpDuration property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing exemplar.
longValue Long Required The time, in milliseconds, over which a change in the Up value is interpolated. The default value is 0.

Remarks

If you set UpDuration on an object to 5000, and then set the same object's Up property to a new value, the object's Up property appears to change smoothly from its previous value to its new value over a period of 5 seconds.

When this interpolation is complete, the method OnUpDone is invoked on the object.  

Example

To get this property:

   TheDuration = MyThing.UpDuration

To set this property:

   MyThing.UpDuration = 5000

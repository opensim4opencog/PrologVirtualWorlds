IsUIVisible

 Property of the Thing Exemplar

A Boolean flag that is used by the sample clients to indicate whether to display the object in a room or inventory contents list. IsUIVisible is a read/write Boolean.  

Syntax

object.IsUIVisible = booleanValue

The IsUIVisible property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing object.
booleanValue Boolean Required True is the object can be displayed.

Example

To get this property:

   ShowFlag = MyThing.IsUIVisible

To set this property:

   MyThing.IsUIVisible = True

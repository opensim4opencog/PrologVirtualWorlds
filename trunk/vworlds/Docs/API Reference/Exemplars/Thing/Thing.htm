Thing

Exemplar

The base exemplar of all objects in the virtual world. 

Using the Object

The Thing exemplar is the root of the core exemplars, which include Artifact, Avatar, Host, Portal, Room, StateMachine, Thing, and WebPage. Every visible object in the world inherits, directly or indirectly, from Thing. As such, these objects inherit all of the Thing methods and properties.

A Thing is an abstract object that is not normally used to create instances. While you can create instances from this exemplar, you will probably want to use the more specialized core exemplars for this purpose. A Thing has geometry, position, orientation, collision events, and other physical characteristics. A Thing can move from place to place, communicate, take and drop other objects, display emotions, and perform other actions.

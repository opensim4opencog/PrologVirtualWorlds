ApproachRadius

 Property of the Thing Exemplar

A Float that specifies the radius of a circle used to position an object that is approaching another object. ApproachRadius is a read/write Float.  

Syntax

object.ApproachRadius = floatValue

The ApproachRadius property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
floatValue Float Required An expression that evaluates to a Float.

Remarks

Each of the ApproachDistance and ApproachRadius properties defines a circle that is used to position an approaching object. ApproachDistance defines a circle centered on the destination object, with a radius equal to the value stored in this property. ApproachRadius defines a second circle, centered at some point on the first circle, with a radius equal to the value stored in ApproachRadius. The final position of an approaching object is a randomly chosen point within this second circle.

The default value for this property is 1.0 unit.

The Approach method of Thing uses three properties to determine the final position of an object when it approaches a destination object: ApproachDistance, ApproachToFront, and ApproachRadius. For a complete discussion of all three properties, see Approach.  

Example

To get this property:

   Radius = TheObject.ApproachRadius

To set this property:

   TheObject.ApproachRadius = 2.0

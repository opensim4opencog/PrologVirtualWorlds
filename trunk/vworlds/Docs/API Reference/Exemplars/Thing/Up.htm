Up

 Property of the Thing Exemplar

Specifies the direction that represents the object's subjective "up." From the current orientation in the world, "up" represents a positive y-coordinate value. Up is a read/write Vector.  

Syntax

To set the property:
object.Up = VectorValue

To return the property:
object.Up

The Up property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
VectorValue Vector Required The vector that represents the "up" direction (0, 1 0).

Remarks

The Up property consists of a vector that describes the orientation of the top of the object. The vector originates at the center of the object and points through the top of the object. The default vector for the Up property is (0, 1, 0), which orients an object in an upright position in the world. The vector for the Up property is projected onto a plane perpendicular to the vector for the Direction property, effectively limiting the vector for the Up property to two degrees: up or down.

The Direction and Up properties together determine the rotation of the object.

The Vector topic has detailed information about the way vectors work in Microsoft Virtual Worlds.

Important:   Do not set the Up vector to 0, 0, 0. This causes D3D problems. Also, never set the Direction and Up vectors to the same values.

Example

This example demonstrates the use of the Up property.

Dim myVector

'Create a vector to establish coordinates for x, y, and z.
'Retrieve the y-coordinate. 

Set myVector = CreateVector(10, 5, 3)
Trace myVector.y

'Establish the y-coordinates independently of a vector object. 

Dim myChair

Set myChair = User.Container.FindWithin("Chair")

myChair.Direction.y = 2
myChair.Position.y = 4
myChair.RotationAxis.y = 1
myChair.Up.y = 0.15

'This next code block retrieves the values for the
'x-coordinate independently of a vector object. 

Dim myChairY

Trace myChair.Scale.y
Trace myChair.Up.y
myChairY = myChair.Position.y

Shout

 Method of the Thing Exemplar

Shouts a string. Shout is a member of the Thing exemplar.  

Syntax

object.Shout strText

The Shout method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
strText String Required The text to shout.

Return Value

This method does not return a value.

Remarks

Shouts the string, causing a Tell event to be sent. The PeerTell event will be sent to everyone in the same container.

Note that in the current client, there is no effective difference between shouting something and simply saying something.  

Example

AThing.Shout "Hey, everybody!"

LightColor

 Property

Sets or returns the color that an instance of the Light exemplar projects. This property is a read-write Vector data type.

Syntax

Object.LightColor = vectorValue

The LightColor property syntax has the following parts:
Parts Type Use Description
Object Light Required An instance of the Light exemplar.
vectorValue Vector Required Any valid expression that evaluates to a Vector data type. The default value is (1, 1, 1).

Remarks

The LightColor property provides full control over the hue and intensity of a Light object. The three elements of the Vector data type, x, y, and z, correspond, respectively, to the red, green, and blue values of light. These three values combine to create the color of the light the object emits.

The default value of an instance of the Light exemplar is (1, 1, 1), which is the equivelent of 100 percent brightness of each of the components of the Vector data type.

You can use negative values for the components of the LightColor property to produce "dark light." On surfaces that share light effects from multiple sources, negative values combine mathematically with the values of other Light objects, reducing the rendered effect of the other light. This technique creates interesting details and complex lighting effects that can make a scene appear more dramatic.

To view the value of the LightColor property in the Command Window, use the ToAscii method of Vector.

Note:   To change the value of the LightColor property, use the CreateVector method of the World object. Although you can retrieve each vector value through the X, Y, and Z properties of Vector, changes to these values are implemented only when an entire vector value is assigned.

Example

This example creates an instance of the Light exemplar and then adjusts the value of the LightColor property. The parameter "myName" is the name of the new instance. The parameters "myRed", "myGreen", and "myBlue", are the color values. 

Sub CreateMyLight(myName, myRed, myGreen, myBlue)
Dim NewLight
Set NewLight = World.CreateInstance(myName, "Light", User.Container)

'Change the color value to the values supplied. 

NewLight.LightColor = World.CreateVector(myRed, myGreen, myBlue)
Display the change in the Command Window.

Trace NewLight.LightColor.ToAscii

End Sub

LightAttenuation

 Property of the Light Exemplar

Sets or returns the behavior over distance of light emitted from specific types of Light objects. LightAttenuation is a read/write Vector.

Syntax

object.LightAttenuation = vectorValue

The LightAttenuation property syntax has these parts:
Parts Type Use Description
object Light Required An instance of the Light exemplar.
vectorValue Vector Required Any valid expression that evaluates to a Vector of float values. The default value is (1.0, 0.0, 0.0).

Remarks

The LightAttenuation property defines the behavior of light over distance. The value of the LightColor property is combined with the attenuation to produce the color intensity for each vertex within the attenuated area.

The three values of the LightAttenuation vector correspond to three types of attenuation: constant, linear, and quadratic. All three values are normalized at 1.0, but each value has a different effect on the behavior of light over distance. These members contain floating point values typically ranging from 0.0 to 1.0, controlling a light's constant, linear, and quadratic attenuation. Many applications set the LightAttenuation.y member to 1.0 and the others to 0.0 (as in (0.0, 1.0, 0.0)), resulting in light intensity that attenuates evenly over distance-from maximum intensity at the source, to zero intensity at the light's maximum range. You can combine attenuation values to get more complex attenuation effects. Or, you might set them to values outside of the normal range to create even stranger attenuation effects; negative attenuation values make a light that gets brighter over distance.

Directional, parallel, and ambient lights are not affected by the LightAttenuation property.

Example

This example sets the value of the LightAttenuation property to have a Linear behavior.

myLightInstance.LightAttenuation = World.CreateVector(0.0, 1.0, 0.0)

LightType

 Property of the Light Exemplar

Sets or returns the type of a Light object. LightType is a read/write Long.

Syntax

object.LightType = longValue

The LightType property syntax has these parts:
Parts Type Use Description
object Light Required An instance of the Light exemplar.
longValue Long Required See the following table for a list of valid values. The default value is 2.

LightType values
Value Type Description
0 Ambient A light source that illuminates everything in a scene, regardless of the orientation, position, and surface characteristics of the objects in the scene. Because this type of light illuminates a scene with equal strength everywhere, the position and orientation of the light is inconsequential. Ambient lights are only affected by color and intensity.
1 Point A light source that radiates equally in all directions from its origin. Point light sources require the system to calculate a new lighting vector for every facet or normal they illuminate, and so are computationally more expensive than a parallel point light source. They produce a more faithful lighting effect than parallel point light sources, however.
2 Spot A light source that emits a cone of light. Only objects within the cone are illuminated. The cone produces light of two degrees of intensity; the central, brightly lit section (the umbra) acts as a point source, and an outer, dimly lit section (the penumbra) merges with the surrounding deep shadow. You can specify the angles of each of these two sections by modifying the other properties (LightUmbra and LightPenumbra), which determine the shape of the cone it produces.
3 Directional This light appears to illuminate all objects with equal intensity, as if it were at an infinite distance from the objects. Directional light has orientation but no position, and it is commonly used to simulate distant light sources, such as the sun. It is the best choice of light to use for maximum rendering speed.
4 Parallel point A light source that illuminates objects with parallel light, but the orientation of the light is taken from the position of the light source. For example, two meshes on either side of a parallel point source are lit on the side that faces the position of the source. The parallel point light source offers similar rendering-speed performance to the directional light.

Remarks

The LightType values can be categorized into two distinct types: ambient light and direct light. Ambient light can be thought of as the general lighting level of an entire scene, as opposed to a light source affecting each vertex in a world from a specific position. Ambient light illuminates all objects on all sides of each object, regardless of the position of the light. Direct lights, including point, spot, directional, and parallel lights, project light from a specific location. Since direct lights are based on their position, each vertex of an object is rendered depending on where the light is located.

Example

This example creates an instance of the Light exemplar and then changes the LightType property.

Dim myLightInstance
Set myLightInstance = World.CreateInstance("myLight1", "Light", User.Container)
myLightInstance.LightType = 4

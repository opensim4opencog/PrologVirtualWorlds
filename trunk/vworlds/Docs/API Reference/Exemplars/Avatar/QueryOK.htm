QueryOK

 Property of the Avatar Exemplar

Returns True if the user clicked the OK button on a query dialog box. QueryOK is a read-only Boolean.

Syntax

object.QueryOK

The QueryOK property syntax has these parts:
Parameters Type Use Description
object Avatar Required An object reference to an Avatar object.

Remarks

The Alert, Confirm, and Prompt methods each open a dialog box containing an OK button. If the user clicks OK in any of these dialog boxes, the QueryOK property for that avatar is set to True. If the user closes the dialog box or clicks Cancel (for the Confirm and Prompt methods), the value for the QueryOK property is set to False. By default, the value is False.

Example

This example checks the QueryOK property of Eve's avatar. If its value is False, then the program calls the SendAlert procedure.

Sub VerifySendAlertCall()
Dim myAvatarEve

Set myAvatarEve = User.Container.FindWithin("Eve")

If myAvatarEve.QueryOK = False Then 
    SendAlert
End If


Sub SendAlert()
Dim myPhoneMap
Dim myAvatarEve

'Set a reference to the avatar. 

Set myAvatarEve = User.Container.FindWithin("Eve")

'Create a property map. 

Set myPhoneMap = CreatePropertyMap

'Add properties to the property map. 

PhoneMap.Property("Erik") = "555-1234"
PhoneMap.Property("Samantha") = "555-2468"

'Display an Alert dialog box to Eve.
'When Eve closes the dialog box, the "myPhoneMap" property map
'is transferred to the QueryParam property of Eve's avatar. 

myAvatarEve.Alert "Phone Numbers", "Transfer?", PhoneKiosk, _
    "TransferMethod", myPhoneMap

End Sub

Wear

 Method of the Avatar Exemplar

Displays the specified accessory on the image of an avatar. In a virtual world, an accessory graphic appears as an object that can be worn or held by an avatar. You use the IsAccessory property of the Artifact object to create an accessory. Wear is a member of the Avatar exemplar.

Syntax

object.Wear pThing

The Wear method syntax has these parts:
Part Type Use Description
object Avatar Required An object reference that accesses an Avatar object.
pThing Thing Required An expression that evaluates to a Thing object.

Return Value

This method does not return a value. 

Remarks

An avatar can "wear" an accessory only if the following three conditions apply: 

  • The object referenced by the pthing parameter exists in the avatar's inventory.
  • The object referenced by the pthing parameter's IsAccessory property is set to True.
  • The object referenced by the pthing parameter is not currently being worn.

The major differences between an artifact and an accessory are that the object intended to be the accessory must have the IsAccessory property set to True and the AccessoryGraphic property must be set to the image file for the accessory.

Example

This example shows you how to create an artifact and add an accessory graphic to the artifact. You must either create a StrawHat.gif and modify the URL to match its location, or substitute an image of your choice.

Sub CreateHat()
Dim StrawHat
Dim ImageForHat

'Create an artifact. 

Set StrawHat = CreateInstance("Straw Hat", "Artifact", User.Container)
StrawHat.IsAccessory = True

'Get the image. 

StrawHat.AccessoryGraphic = _
    "http://Webs/Scripts/Images/StrawHat.gif"

'Wear the artifact. 

User.Wear StrawHat

End Sub

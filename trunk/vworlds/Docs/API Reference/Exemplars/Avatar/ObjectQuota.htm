ObjectQuota

 Property of the Avatar Exemplar

Returns the number of items that an avatar can create in a world. ObjectQuota is a read-only Long. 

Syntax

object.ObjectQuota

The ObjectQuota property syntax has these parts:
Parameters Type Use Description
object Avatar Required An object reference to an Avatar object.

Remarks

Each time an avatar creates an object, the number stored in the avatar's object quota decreases by one. Conversely, when an object is destroyed, the object quota increases by one. If the object quota reaches zero, the avatar can no longer create objects.

Some avatars can have an infinite object quota, represented by a value of -1. This value neither increases nor decreases, allowing the avatar to create an infinite number of objects.

The default object quota for all avatars in a world is specified by the DefaultObjectQuota property of the World object. 

Example

By using the following statement, you can determine the number of items your avatar can create. 

 
Print User.ObjectQuota

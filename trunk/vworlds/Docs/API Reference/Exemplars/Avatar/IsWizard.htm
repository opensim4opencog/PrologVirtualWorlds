IsWizard

 Property of the Avatar Exemplar

Returns True if the avatar is a wizard. IsWizard is Boolean.

Syntax

This property is read/write for avatars that are wizards:
object.IsWizard = BooleanValue

This property is read-only for all other avatars:
object.IsWizard

The IsWizard property syntax has these parts:
Parameters Type Use Description
object Avatar Required An object reference to an Avatar object.
BooleanValue Boolean Required True the avatar is a wizard. The default for this property is False.

Remarks

The first avatar to exist in your world is given wizard and world owner status. As such, you can change the world's name and set up its security. The IsWizard property for a world owner's avatar cannot be set to False.

Any avatar that is created later or visits your world through a portal does not have wizard status unless another wizard in the world sets the wizard permissions for them in the Object Properties dialog box (also known as the Wizard Property Page). Only a wizard can change the value of the IsWizard property for another avatar.

If an avatar is a wizard, the Authoring Tools tab is visible in the right pane of the Microsoft Virtual Worlds environment. An avatar who is a wizard has the ability to modify almost all properties for any object in the world and has the ability to grant wizard, author, or host status to other avatars.

Example

The following procedure can be called by a program running in the background that monitors new avatars or avatars visiting the world through a portal.

The example first checks to see if a new avatar is a wizard. If not, the procedure sends a Prompt message to the new avatar asking whether it wants to be a wizard. If the avatar clicks Cancel, an Alert message is sent to a designated wizard. If the avatar clicks OK, the avatar's QueryOK property is set to True and an Alert message is sent to the designated wizard to review the request. This example assumes that Gracie is a wizard in the world and that the participant is the avatar that wants wizard status. 


Sub NotifyWizard()
Dim myWizard
Dim myNewAvatar
Dim myMsg

Set myNewAvatar = User
Set myWizard = User.Container.FindWithin("Gracie") 

myMsg = "Do you want to be a wizard? 
    I will send a message to the resident wizard...
    Click OK if you want to be a wizard,
    otherwise, click Cancel."

'Check if the IsWizard property is False.
'If so, send a message to the avatar. 

If myNewAvatar.IsWizard <> True Then
    myAvatar.Prompt "Message to Wizard", myMsg, _
        User.Name & " wants to be a wizard", myWizard

'Check the status of the QueryOK property and send the
'appropriate message to the wizard. 

    If myNewAvatar.QueryOK <> False Then
        myWizard.Alert "Warning no Authoring", _
            User.Name & " decided not to be a wizard." 
    Else
        myWizard.Alert "Need New Author Permission", _
            User.Name & " wants to be a wizard."
    End If
End If
End Sub

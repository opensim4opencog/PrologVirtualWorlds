RemoveFromUserProfileList

 Method of the Avatar Exemplar

Removes the specified property from a key in the profile list of an avatar. RemoveFromUserProfileList is a member of the Avatar exemplar. 

Syntax

object.RemoveFromUserProfileList pthing, bstrKey

The RemoveFromUserProfileList method syntax has these parts:
Part Type Use Description
object Avatar Required An object reference that accesses an Avatar object.
pthing Thing Required An object reference that accesses a Thing object.
bstrKey String Required A string that represents a valid key in the key list of the profile of the avatar.

Return Value

This method does not return a value.

Remarks

The RemoveFromUserProfileList method is used only for removing properties from keys in avatar profiles. For example, if you have a property list that was added to a key in a user's profile, you can use the RemoveFromUserProfileList method to remove an individual property from that property list. To remove a key/value pair from an avatar's profile, use the RemoveKey method of AvatarProfile. 

The AvatarProfile object for an avatar stores data in key lists, which are composed of key/value pairs. A key/value pair is a property name paired with its value. A key is a string that represents the name of a property in the property map. The value is the contents of that property. A property map uses the string that makes up the key instead of an index number to access the value of each property.

Key/value pairs are distinguished by the type of value in the pair. Types of keys include numeric keys (whose corresponding value is a number), string keys (whose corresponding value is a string), and property keys lists (whose corresponding value is a property list).

Example

This example checks the profile list for the specified key. If the key is found, it removes the "PassportID" property from the key.

Sub RemovePassportIDProperty()
Dim myKey
Dim myKeyList
Dim myKeyCount

Set myKeyList = User.Profile.Keys
myKeyCount = myKeyList.Count - 1

For myKey = 0 To myKeyCount

  'Check each key for a passport ID. 

  If myKeyList.Property(myKey) = "PassportID" Then

    'Remove the property from the key. 

    User.RemoveFromUserProfileList Passport, "PassportID"
  End If
Next
End Sub

AddToUserProfileList

 Method of the Avatar Exemplar

Adds an object to a key in the profile list of an Avatar object. AddToUserProfileList is a member of the Avatar exemplar.

Syntax

object.AddToUserProfileList pthing, bstrKey

The AddToUserProfileList method syntax has these parts:
Part Type Use Description
object Avatar Required An object reference to an Avatar object.
pthing Thing Required An object reference to a Thing object.
bstrKey String RequiredA string that represents a valid key in the keylist of the profile of the avatar.

Return Value

This method does not return a value. 

Remarks

The AddToUserProfileList method is used only for adding properties to keys in avatar profiles. To add a simple key/value pair to an avatar's profile, use the AddKey method of AvatarProfile.

The AvatarProfile object for an avatar stores data in key lists, which are composed of key/value pairs . A key/value pair is a property name paired with its value. The key is a string that represents the name of a property in the property map. The value is the contents of that property. A property map uses the string that makes up the key instead of an index number to access the value of each property.  

Example

This example creates a passport object, adds a "PassportID" key to the user's profile with reference to the passport object, and then removes the key. 

Sub AddPassportKey()
Dim myNewPassport
Dim myPass
Dim myKeys
Dim myCount
Dim myDoesExist

Set myKeys = User.Profile.Keys
Set myCount = myKeys.Count - 1

'Create the passport object. 

Set myNewPassport = _
  CreateInstance("Passport", "Artifact", User.Container)

'Get a reference to the new object. 

Set myPass = User.Container.FindWithin("Passport")

'Add a new key to the user's profile. 

User.AddToUserProfileList myPass, "PassportID"

'Verify that the new key exists in the user's profile. 

For i = 0 To myCount
    Trace myKeys.Property(i)
Next

'Another way to verify that the new key exists is
'to user the IsInUserProfileList method. 

myDoesExist = User.IsInUserProfileList(myPass, "PassportID")
Trace myDoesExist

'To remove the new key. 

User.RemoveFromUserProfileList myPass, "PassportID"

End Sub

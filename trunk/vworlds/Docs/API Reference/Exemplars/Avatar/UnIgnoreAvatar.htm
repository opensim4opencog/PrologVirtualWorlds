UnIgnoreAvatar

 Method of the Avatar Exemplar

Removes the name of the specified Thing object from the Ignoring property list in the specified avatar's profile. UnIgnoreAvatar is a member of the Avatar exemplar. 

Syntax

object.UnIgnoreAvatar ThingValue

Parameters

Parameters Type Use Description
object Avatar Required An object reference to an Avatar object.
ThingValue Thing Required An expression that evaluates to a Thing object, such as an artifact or avatar.

Return Value

This method does not return a value. 

Remarks

This method works hand in hand with a default key, named "Ignoring" in the AvatarProfile object. If you use the UnIgnoreAvatar method to recognize an avatar that was previously ignored, the formerly ignored avatar's name is removed from the Ignoring property list in your profile.

If you remove another avatar or object from your ignored list, you receive messages from the object again. A message is displayed in the chat-history window and in the Command Window that you are no longer ignoring the object. The object can now access the Approach menu item (or any other menu item, for that matter) for your avatar. The image for the avatar or object is restored in your Microsoft Virtual Worlds environment from the little monster locked inside a stoppered bottle to the original image.

Example

This example shows you how to ignore and unignore an object and an avatar, and then how to print a list of ignored items in the Command Window.

To run this example: You must have another avatar and object present in your world. You can substitute an avatar and object of your choice for Gracie's avatar or the "Red Baron's Plane".

Note:   In most cases, the Ignoring property is Property(0). However, you can check this by running the program in the example for the KeyValue method. 


 
Sub IgnoreStuff()
Dim myIgnoredAvatar
Dim myIgnoredObject
Dim myIgnorePropertyList
Dim myCount
Dim i

Set myIgnoredAvatar = User.Container.FindWithin("Gracie")
Set myIgnoredObject = User.Container.FindWithin("Red Baron's Plane")
Set myIgnorePropertyList = _
    User.Profile.KeyValue(User.Profile.Keys.Property(0))

'Add the avatar and object to the Ignoring property list in your profile. 

User.IgnoreAvatar myIgnoredAvatar
User.IgnoreObject myIgnoredObject
myCount = myIgnorePropertyList.Count - 1

'Print the ignored items to the Command Window. 

For i = 0 To myCount
    Print myIgnorePropertyList.Property(i)
Next

'Remove the ignored name from the property list. 

User.UnIgnore myIgnoredAvatar

'Print the ignored items to the Command Window again to
'verify that the avatar is not ignored. 

For i = 0 To myCount 
    Print myIgnorePropertyList.Property(i)
Next

End Sub

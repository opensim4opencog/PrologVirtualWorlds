IsAuthor

 Property of the Avatar Exemplar

Returns True if the avatar is an author. IsAuthor is a Boolean.

Syntax

This property is read/write for avatars that are wizards:
object.IsAuthor = BooleanValue

This property is read-only for all other avatars:
object.IsAuthor

The IsAuthor property syntax has these parts:
Parameters Type Use Description
object Avatar Required An object reference to an Avatar object.
BooleanValue Boolean Required True the avatar is an author. The default for this property is False .

Remarks

An author is an avatar who can change and create objects (provided they don't exceed their quota) in a world. A wizard is the creator and designer of objects in a world. An owner and wizard can create and design a world, along with all of the objects in that world.

The first avatar to exist in your world is given authoring and wizard status and is considered the owner of the world. Any avatar that is created later or visits your world through a portal does not have authoring properties unless another wizard in the world sets the authoring permissions for them in the Object Properties dialog box (also known as the Wizard Property Page ). A wizard can also set this property using script.

If an avatar is an author, the Authoring Tools tab is visible in the right pane of the Microsoft Virtual Worlds environment.

Example

The following procedure could be called by a program running in the background that monitors new avatars or avatars visiting the world through a portal.

The example first checks to see if a new avatar is an author. If not, the procedure sends a Prompt message to the new avatar asking whether it wants to be an author. If the avatar clicks Cancel, an Alert message is sent to a designated wizard. If the avatar clicks OK, the avatar's QueryOK property is set to True. This example assumes that Gracie is a wizard in the world and that the user is the avatar that wants authoring permission. 


Sub NotifyWizard()
Dim myWizard
Dim myNewAvatar
Dim myMsg

Set myNewAvatar = User
Set myWizard = User.Container.FindWithin("Gracie") 

myMsg = "Do you want to be an author? 
    I will send a message to the resident wizard...
    Click OK if you want to be an author,
    otherwise, click Cancel."

'Check if the IsAuthor property is False.
'If so, send a message to the avatar. 

If myNewAvatar.IsAuthor <> True Then
    myAvatar.Prompt "Message to Wizard", myMsg, _
        User.Name & " wants to be an author", myWizard

'Check the status of the QueryOK property and send the
'appropriate message to the wizard. 

    If myNewAvatar.QueryOK <> False Then
        myWizard.Alert "Warning no Authoring", _
            User.Name & " decided not to be an author." 
    Else
        myWizard.Alert "Need New Author Permission", _
            User.Name & " wants to be an author."
    End If
End If
End Sub

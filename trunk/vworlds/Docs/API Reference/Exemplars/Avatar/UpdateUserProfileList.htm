UpdateUserProfileList

 Method of the Avatar Exemplar

Adds a check mark next to the specified menu item if the specified avatar exists in the profile of the user. UpdateUserProfileList is a member of the Avatar exemplar.

Syntax

object.UpdateUserProfileList(pmi)

The UpdateUserProfileList method syntax has these parts:
Parameters Type Use Description
object AvatarProfile Required An object reference that accesses an AvatarProfile object.
pmi MenuItem Required A reference to a menu item.

Return Value

This method does not return a value. 

Remarks

This method is used to manage check marks on shortcut menus in Microsoft Virtual Worlds. For example, if the owner of a menu item is in the user's AvatarProfile property list, then the menu item appears on the shortcut menu with a check mark to the left of it. However, if the owner of the menu item is not in the user's profile, the UpdateUserProfileList method generates an error when you attempt to update the profile.

When you create the menu item by using the CreateMenuItem method, you specify the owner of the menu item. If the value of this parameter exists in the user's profile, the menu item is selected. Otherwise, the menu item is not selected.

Example

This example uses the UpdateUserProfileList method to indicate when an avatar is a member of a particular group. In this example, a group named "CoWorkers" is associated with a menu item of the same name. When the user right-clicks an avatar that is a member of the CoWorkers group, a check mark is placed next to the "CoWorkers" menu item on the avatar's shortcut menu. 


Sub CoWorkerSetup
Dim myNewMenuItem

User.Profile.AddKey "CoWorkers", CreatePropertyList

Set myNewMenuItem = CreateMenuItem("Add to CoWorker list", 0, _
  User, "UpdateUserProfileList", "MenuUserProfileList", "CoWorkers")
End Sub

If you added more keys later, you would use the following statement to update the avatar's profile.

User.UpdateUserProfileList myNewMenuItem

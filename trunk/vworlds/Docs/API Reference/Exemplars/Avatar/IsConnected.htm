IsConnected

 Property of the Avatar Exemplar

Returns the state of a connection to a virtual world. Returns True if the specified avatar is connected. IsConnected is a read-only Boolean. 

Syntax

object.IsConnected

The IsConnected property syntax has these parts:
Parameters Type Use Description
object Object Required An object reference to an Avatar object.

Remarks

If the invoking Avatar object is currently connected to the world, this property is True. If the invoking Avatar object is not currently connnected, this property does not exist for that Avatar object. Before querying this property for other avatars in your world, check first to see whether the property exists by using the IsValidProperty method of the Thing object.

Examples

The first example checks for the existence of the IsConnected property for the specified avatar. If the property exists, then the avatar is connected to the world; if not, the property does not exist.

Sub IsUserConnected()
Dim myAvatar

Set myAvatar = User.Container.FindWithin("Eve")

If myAvatar.IsValidProperty("IsConnected") Then
   Print myAvatar.IsConnected
Else
   Print myAvatar.Name & " is not connected."
End If

Example 2

You can also use the following statement to return the connected state of an avatar.

Print User.IsConnected

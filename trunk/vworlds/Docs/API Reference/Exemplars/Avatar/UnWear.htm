UnWear

 Method of the Avatar Exemplar

Removes the specified accessory from the image of an avatar. UnWear is a member of the Avatar exemplar. 

Syntax

object.UnWear pThing

The UnWear method syntax has these parts:
Part Type Use Description
object Avatar Required An object reference that accesses a Avatar object.
pThing Thing Required An expression that evaluates to a Thing object.

Return Value

This method does not return a value. 

Remarks

An avatar can "remove" an accessory only if the following three conditions apply: 

  • The object referenced by the pthing parameter exists in the avatar's inventory.
  • The object referenced by the pthing parameter's IsAccessory property is set to True.
  • The object referenced by the pthing parameter is currently being worn.

The major difference between an artifact and an accessory is that the object intended to be the accessory must have the IsAccessory property set to True.

Example

This example shows you how to create an artifact and remove an accessory graphic from the artifact. You must either create a RainyDay.gif and modify the URL to match its location, or substitute an image of your choice. This example assumes that the specified avatar is already wearing an artifact named "Sunny Day Hat".

Public Sub CreateHat()
Dim RainyDayHat
Dim SunnyDayHat
Dim ImageForHat

'Get the current accessory. 

Set SunnyDayHat = User.Container.FindWithin("Sunny Day Hat")

'Create an artifact. 

Set RainyDayHat = _
    CreateInstance("Rainy Day Hat", "Artifact", User.Container)

'Make sure the artifact is an accessory. 

RainyDayHat.IsAccessory = True

'Get the image for the new accessory. 

RainyDayHat.AccessoryGraphic = _
    "http://Webs/Scripts/Images/RainyDay.gif"

'Remove the current artifact. 

User.UnWear SunnyDayHat

'Wear the artifact. 

User.Wear RainyDayHat

End Sub

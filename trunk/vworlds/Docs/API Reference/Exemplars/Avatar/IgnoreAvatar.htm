IgnoreAvatar

 Method of the Avatar Exemplar

Adds the name of the specified Thing object to the Ignoring property list in the specified avatar's profile. IgnoreAvatar is a member of the Avatar exemplar. 

Syntax

object.IgnoreAvatar ThingValue

The IgnoreAvatar method syntax has these parts:
Parameters Type Use Description
object Avatar Required An object reference to an Avatar object.
ThingValue Thing Required An expression that evaluates to a Thing object, such as an artifact or avatar.

Return Value

This method does not return a value. 

Remarks

This method works hand in hand with a default key, named "Ignoring", in the AvatarProfile object. If you use the IgnoreAvatar method to ignore another avatar or other object, the ignored object's name is added to the Ignoring property list in your profile.

If you ignore an object, you do not receive messages from that object. A message is displayed in the chat-history window and in the Command Window that you are ignoring the object. An ignored object does not have access to the Approach menu item (or any other menu item, for that matter) on the ignoring avatar. This means that the ignored object cannot approach the ignoring avatar (your avatar). The object that is ignored appears in the Microsoft Virtual World environment as a little monster locked inside a stoppered bottle. The original image of the object or avatar does not appear in the world until you use the UnIgnoreAvatar method to stop ignoring the object.

Note:  You cannot ignore your own avatar, and another avatar cannot add avatars and objects to the Ignoring property list in your profile.  

Example

This example shows you how to ignore an object and an avatar, and then how to print a list of ignored items in the Command Window.

To run this example: You must have another avatar and object present in your world. You can substitute an avatar and object of your choice for Gracie's avatar or the "Red Baron's Plane".

Note:   In most cases, the Ignoring property is Property(0). However, you can check this by running the program in the example for the KeyValue method. 


 
Sub IgnoreStuff()
Dim myIgnoredAvatar
Dim myIgnoredObject
Dim myIgnorePropertyList
Dim myCount
Dim i

Set myIgnoredAvatar = User.Container.FindWithin("Gracie")
Set myIgnoredObject = User.Container.FindWithin("Red Baron's Plane")
Set myIgnorePropertyList = _
    User.Profile.KeyValue(User.Profile.Keys.Property(0))

'Add the avatar and object to the Ignoring property list in your profile. 

User.IgnoreAvatar myIgnoredAvatar
User.IgnoreAvatar myIgnoredObject
myCount = myIgnorePropertyList.Count - 1

'Print the ignored items to the Command Window. 

For i = 0 To myCount
    Print myIgnorePropertyList.Property(i)
Next

'Remove the ignored name from the property list. 

User.UnIgnore myIgnoredAvatar

'Print the ignored items to the Command Window again to
'verify that the avatar is not ignored. 

For i = 0 To myCount 
    Print myIgnorePropertyList.Property(i)
Next

End Sub

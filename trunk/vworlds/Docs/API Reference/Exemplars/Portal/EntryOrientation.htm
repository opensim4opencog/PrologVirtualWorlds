EntryOrientation

 Property of the Portal Exemplar

A Vector specifying the orientation of an object when it transports through a portal. EntryOrientation is a read/write Vector.  

Syntax

object.EntryOrientation = VectorValue

The EntryOrientation property syntax has these parts:
Part Type Use Description
object Portal Required An object reference that accesses a Portal object.
VectorValue Vector Required The orientation vector of a new arrival in a room.

Remarks

When an object arrives in a room, its position and orientation are set by two Room properties: EntryPoint and EntryOrientation. EntryPoint represents the object's location in the room, and EntryOrientation represents the direction it is facing. This property's default value is the vector (0, 0, 1). 

The Direction property is set to the EntryOrientation value when the object enters the destination room.

Example

To get the EntryOrientation:

   Set EntryDir = MyPortal.EntryOrientation

To set the EntryOrientation:

   MyPortal.EntryOrientation = World.CreateVector(2,2,2)

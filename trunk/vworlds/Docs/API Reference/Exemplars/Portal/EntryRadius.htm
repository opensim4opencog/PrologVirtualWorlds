EntryRadius

 Property of the Portal Exemplar

A Float specifying the radius of a semi-circle around an EntryPoint into which new arrivals to a Room are placed. EntryRadius is a read/write Float.  

Syntax

To set this property:
object.EntryRadius = floatValue

To return this property:
object.EntryRadius

The EntryRadius property syntax has these parts:
Part Type Use Description
Object Portal Required An object reference that evaluates to a Portal.
floatValue Float Required An expression that evaluates to a Float.

Remarks

New arrivals to a Room are placed at random positions in a semi-circular area around the EntryPoint, within a radius of EntryRadius meters. This prevents new arrivals from stacking up at the same point. The default value for EntryRadius is 3.0.

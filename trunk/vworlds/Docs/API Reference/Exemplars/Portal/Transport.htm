Transport

 Method of the Portal Exemplar

Moves a Thing object through a source Portal to a destination Portal. Transport is a member of the Portal exemplar.  

Syntax

object.Transport pthing [, iIndex]

The Transport method syntax has these parts:
Part Type Use Description
object Portal Required An object reference that accesses a Portal object.
pthing Thing Required The object to send through the portal.
[iIndex] Integer Optional The index of the destination to send the SentObject.

Return Value

This method does not return a value. 

Remarks

The Thing represented by SentObject is moved to the portal numbered by iIndex in the portal's Destinations property list. iIndex is optional; if omitted, SentObject will be sent to the first portal in the list.

If you find that the object you are trying to transport is not actually being transported, it may be that Destinations is failing during server processing. In such circumstances, it is useful to check the server trace window. For more information on server trace messages, see the Virtual Worlds Server Application topic. 

Example

MyPortal.Transport ThingObject

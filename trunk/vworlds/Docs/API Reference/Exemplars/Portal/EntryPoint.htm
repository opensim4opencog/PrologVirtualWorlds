EntryPoint

 Property of the Portal Exemplar

A Vector specifying the location at which new arrivals appear in a room when traveling through a specified portal. EntryPoint is a read/write Vector.  

Syntax

To set this property:
object.EntryPoint = VectorValue

To return this property:
object.EntryPoint

The EntryPoint property syntax has these parts:
Part Type Use Description
object Portal Required An object reference that accesses a RoomPortal object.
VectorValue Vector Required The vector where new arrivals appear in the room.

Remarks

This property's default value is the vector (0, 0, 0). 

The Position property is set to the EntryPoint value when the object enters the destination room.

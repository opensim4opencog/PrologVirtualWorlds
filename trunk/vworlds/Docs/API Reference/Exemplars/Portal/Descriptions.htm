Descriptions

 Property of the Portal Exemplar

A PropertyList containing descriptions of portal destinations. Descriptions is a read/write PropertyList. 

Syntax

To set this property:
object.Descriptions = PropertyListValue

To return this property:
object.Descriptions

The Descriptions property syntax has these parts:
Part Type Use Description
object Portal Required An object reference that accesses a Portal object.
PropertyListValue PropertyList Required A list containing descriptions of portal destinations.

Remarks

The Descriptions property list consists of a list of strings, each of which is a description of the corresponding destination in the Destinations property list. It is possible to get and set the strings of this property individually, just as it is possible to modify individual elements of any other PropertyList.

For a more detailed example of the use of this property, see the Destinations property page.

HTMLURL

 Property of the WebPage Exemplar

A String that contains the URL to be displayed. HTMLURL is a read/write String.  

Syntax

object.HTMLURL = stringValue

The HTMLURL property syntax has these parts:
Part Type Use Description
object WebPage Required An object reference to an instance of the WebPage exemplar.
stringValue String Required The URL that will be displayed when the WebPage object is activated.

Remarks

Contains the URL to the content that is displayed when a user activates a WebPage object by clicking on it with the mouse or invoking the Show method via script.

Example

To get the value of HTMLURL:

   Result = MyWindow.HTMLURL

To set the value of HTMLURL:

   MyWindow.HTMLURL = "http://www.microsoft.com"

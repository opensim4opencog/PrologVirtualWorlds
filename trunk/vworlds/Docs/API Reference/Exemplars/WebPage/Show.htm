Show

 Method of the WebPage Exemplar

Displays a browser window showing the content at a specified URL. Show is a member of the WebPage exemplar.  

Syntax

object.Show 

The Show method syntax has these parts:
Part Type Use Description
object WebPage Required An object reference that accesses a WebPage object.

Return Value

This method does not return a value.

Remarks

This method brings up a browser window and displays the content that is located at the URL contained in the HTMLURL property of the Web Page object.  

Example

This example will create a web page object that points to the Microsoft.com web site and then will show the web page there.

set theWebPage = createinstance("Web Page","WebPage")
theWebPage.HTMLURL = "http://www.microsoft.com"
theWebPage.Show

WebPage

Exemplar

An object that can open a browser window and display content. 

Using the Object

When activated, a WebPage object opens a browser window and displays some content located at an URL. The URL is stored as a property of the WebPage object.

WebPage objects can be used for a variety of purposes, depending on the geometry that represents the object in the world. They can be used to create notes, newspapers, instructions, newbie information, links to external web pages, and so forth.

Flags

 Property of MenuItem

Sets or returns a flag that determines the display setting for a MenuItem object. Flags is a read-write Long.  

Syntax

object.Flags = longValue

The Flags (MenuItem Object) property syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference that accesses a MenuItem object.
longValue Long Required A flag that determines the display setting for a menu item.

Remarks

When you create a menu item by using the CreateMenuItem method, you set flags that determine the display setting of the menu item that you are creating.

The flags used for a MenuItem object are made up of combinations of Long values based on the constants shown in the table below.  For convenience, each flag constant is exposed as a read-only property of the World object.

The flag indicates to the menu display routines in the user interface how the menu item should be processed and displayed. You can combine these flags to customize your menus by adding them together. For example, a flag value of 3 (MI_DISABLED + MI_GRAYED) indicates that the menu item is disabled and displays the item on the menu in light gray. These flags are analogous to the WIN32 MIxxx menu flags.

The flag names, values, and descriptions are shown in the following table.

Flag Value Description
MI_DEFAULT 0 Menu item is displayed in default format.
MI_GRAYED 1 Menu item is grayed.
MI_DISABLED 2 Menu item is disabled.

MI_INVISIBLE

 4 Menu item is not displayed.
MI_CHECKED 8 Menu item is checked.
MI_POPUP 16 Menu item displays a submenu.
MI_SEPARATOR 2048 Insert separator between menu items.

Example

To get the Flags property and check for a specific value:

Result = MyMenuItem.Flags

If (Result AND World.MI_CHECKED) = World.MI_CHECKED Then
   World.Trace "MenuItem is checked!"
End If

To set the Flags property:

MyMenuItem.Flags = World.MI_CHECKED +  World.MI_GRAYED

Arguments

 Property of MenuItem

Sets and returns a list of parameters that are passed to a method invoked by a MenuItem object. Arguments is a read/write Variant.  

Syntax

To set the property:

object.Arguments = Value

To return the property:

object.Arguments

The Arguments property syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference to a MenuItem.
Value Variant (Array or other) Required If a single parameter is required, Value contains that parameter.  If multiple parameters are required, Value will be an array that contains those parameters.

Remarks

When you create a menu item that calls a method, the Arguments property contains the arguments for that method. You use an array to store the parameters for the method.  Note that when you get the Arguments property from the menu item, it is returned as a PropertyList object, not an array.

The number of argument elements in the array must be the same as the number of parameters in the method. You must pass the arguments to the method as an array. You can use the Visual Basic Script Array function to create an initialized array, as in the second example below.  

Example

You can substitute the Method object for the MenuItem object in any of the following statements.

To set the arguments: 

myMenuItem.Arguments = Array(10, 20, 30)
myMenuItem.Arguments = Array(bstrAvatarName, 5)

To return the arguments: 

myParams = myMenuItem.Arguments

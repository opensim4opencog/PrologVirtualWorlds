MenuItem

Object

The MenuItem object holds a single menu item in an object's shortcut menu. 

Using the Object

Every object that is visible in Microsoft Virtual Worlds can have an attached shortcut menu, which is made up of one or more MenuItem objects. An object's menu (see the Menu property of Thing) is a PropertyList in which each element contains a menu item object. The object's menu can be modified to provide additional behaviors for that object.

To create a menu item, use the CreateMenuItem method of the World object.  

Dim myMenuItem

Set myMenuItem = _
  World.CreateMenuItem("Friends", _
  MI_CHECKED, User, "", "ShowFriends")

You can check text on menu items by using the MenuText property as shown in the following procedure:

Sub CheckMenuItem()
Dim myPriceTag
Dim myContents
Dim myContentsCount
Dim myCount

Set myRoomItems = User.Container.Contents
myRoomItemsCount = User.Container.Contents.Count - 1
myMenuItemsCount = myRoomItems.Property(myRoomItem).Menu.Count-1

For myRoomItem = 0 To myRoomItemsCount
  For myMenuItem = 0 to myMenuItemsCount
    If myRoomItems.Property(myRoomItem).Menu.Property(myMenuItem).MenuText _
         = "More about " Then
      Print myRoomItems.Property(myRoomItem)
      Trace "Menu okay."
      myIsMenu = True
      Exit For
    End If
  Next
  If Not myIsMenu Then
    Print myRoomItems.Property(myRoomItem)
    Trace "Needs menu"
  End If
Next

End Sub

Execute

 Method of MenuItem

Runs the method associated with the specified menu item. Execute is a member of the MenuItem object.  

Syntax

object.Execute

The Execute method syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference to a MenuItem object.

Return Value

This method does not return a value.

Remarks

The Execute method performs the action specified by the MenuItem object. 

Example

This example prints the properties of a user's menu items to the Command Window, finds the Whisper menu item, and executes the method associated with it.

Sub ExecuteWhisper()
Dim myMenu
Dim myMenuItem
Dim mnuText

Set myMenu = User.Menu
mnuText = "Menu Text: Whisper to..., Method: WhisperTo"

For Each myMenuItem In myMenu
    Print myMenuItem
    If myMenuItem.ToAscii = mnuText Then
        myMenuItem.Execute
    End If
Next
End Sub

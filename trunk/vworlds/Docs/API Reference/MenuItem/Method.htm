Method

 Property of MenuItem

A String that holds the name of the method that a menu item represents. Method is a read-only String.  

Syntax

expression.Method

The Method property syntax has these parts:
Part Type Use Description
expression MenuItem Required An expression that evaluates to a String.

Remarks

This property represents the name of the method to be called when a menu item is selected. For example, if the menu belongs to a Portal object named SomePortal, and the value of this property on the menu item is "DoSomething", then the method SomePortal.DoSomething will be called when that menu item is clicked.

Important Note: This property holds the name of a method (of type String), not the method itself.  

Example

To get the method name for a menu item:

   Result = MyMenuItem.Method

To set the method name for a menu item:

   MyMenuItem.Method = "Wave"

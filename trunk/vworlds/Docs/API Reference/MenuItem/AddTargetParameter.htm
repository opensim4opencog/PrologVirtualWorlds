AddTargetParameter

 Property of MenuItem

When this property is True, the target parameter is added as the first parameter for the method that is executed from a menu item. The target parameter is the first parameter for the method attached to a menu item. AddTargetParameter is a read/write Boolean.  

Syntax

To set the property:
object.AddTargetParameter = BooleanValue

To return the property:
object.AddTargetParameter

The AddTargetParameter property syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference to a MenuItem object.
BooleanValue Boolean Required Determines whether a parameter is added to the method or not.

Remarks

When set to True, you can use this property to add a reference to the shortcut menu of the current Thing object as the first parameter for the method invoked by the menu item. The method must be written to handle that parameter. 

Example

To set this property:

User.Menu.Property(7).AddTargetParameter = True

To return this property:

myIsAddParam = User.Menu.Property(7).AddTargetParameter

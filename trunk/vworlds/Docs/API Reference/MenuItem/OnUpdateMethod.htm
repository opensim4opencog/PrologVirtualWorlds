OnUpdateMethod

 Property of MenuItem

Sets or retrieves the name of an update method to be invoked by a MenuItem before teh menu item is displayed in a right-click menu for a Thing. OnUpdateMethod is a read/write String.  

Syntax

object.OnUpdateMethod = strValue

The OnUpdateMethod property syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference that accesses a MenuItem object.
strValue String Required The name of the method to be invoked.

Remarks

The OnUpdateMethod property stores the name of a method that is invoked on a menu item every time a menu is composited. Each time the menu is composited, Microsoft Virtual Worlds checks the OnUpdateMethod property to see if there is a specified method for this property, and if there is, it invokes that method. The method specified is the name of a method associated with the Thing object that the menu is attached to.

When a menu item is created using the CreateMenuItem method of the World object, the name of this method is specified in the bstrUpdateMethod parameter. The OnUpdateMethod property stores the name of that method. Having access to the value of this property means that you can change the specified method dynamically by changing the property value, instead of having to re-create the menu item each time a change takes place.

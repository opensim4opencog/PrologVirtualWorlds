Duplicate

 Method of MenuItem

Copies an existing menu item including submenus (if they exist). Duplicate is a member of the MenuItem object.

Syntax

object.Duplicate

The Duplicate method syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference that accesses a MenuItem object.

Return Value

This method returns a copy of the specified MenuItem object.

Remarks

The MenuItem object consists of references to other objects, such as a list of submenus (if present), the menu item's owner, and other data. 

Example

This example duplicates a menu item. 

Set myNewMenuItem = myMenuItem.Duplicate

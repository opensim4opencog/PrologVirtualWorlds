MenuText

 Property of MenuItem

Returns the text specified by the bstrMenuText parameter for the CreateMenuItem method. This text is displayed on the shortcut menu that appears when you right-click an object in the world. MenuText is a read/write String.  

Syntax

To set the property:
object.MenuText = stringValue

To return the property:
object.MenuText

The MenuText property syntax has these parts:
Part Type Use Description
object MenuItem Required An object reference that accesses a MenuItem object.
stringValue String Required An expression that evaluates to a String that represents the text for a menu item.

Remarks

You can use accelerator keys (shortcut key combinations) on shortcut menus in Microsoft Virtual Worlds by preceding the first character of the menu item with an ampersand. The following script enables users to use a key combination to invoke the method associated with the Laugh menu item.

   MyMenuItem.MenuText = "&Laugh"

This menu item appears in the object's menu with the first character underlined, "Laugh". You can invoke the menu item in either of two ways:

  • Clicking the menu item
  • Pressing SHIFT+L

You can use the MenuText property to ensure that all menus for each artifact in your world contain the same information. For example, if you create a shop where people can create objects and "sell" them, you can use a loop to check that all menus have a "price tag" menu item associated with each shortcut menu.

Example

This example checks each item in a room for specific text on a menu.

Sub CheckMenuItem()
Dim myPriceTag
Dim myContents
Dim myContentsCount
Dim myCount

Set myRoomItems = User.Container.Contents
myRoomItemsCount = User.Container.Contents.Count - 1
myMenuItemsCount = myRoomItems.Property(myRoomItem).Menu.Count-1

For myRoomItem = 0 To myRoomItemsCount
  For myMenuItem = 0 To myMenuItemsCount
    If myRoomItems.Property(myRoomItem).Menu.Property(myMenuItem).MenuText _
         = "More about " Then
      Print myRoomItems.Property(myRoomItem)
      Trace "Menu OK"
      myIsMenu = True
      Exit For
    End If
  Next
  If Not myIsMenu Then
    Print myRoomItems.Property(myRoomItem)
    Trace "Needs menu"
  End If
Next

End Sub

MethodName

 Property of the Method Object

Sets or returns the method name within the specified method's module. MethodName is a read/write String.  

Syntax

To set the property:
object.MethodName = stringValue

To return the property:
object.MethodName

The MethodName property syntax has these parts:
Part Type Use Description
object Method Required An object reference that accesses a Method object.
stringValue String Required An expression that evaluates to the name of the method that you want to set.

Remarks

You must set the method name before you attach the method to an object. 

Example

To get the method name: 

myDisplayMethodName = myMethod.MethodName
Trace myDisplayMethodName

To set the method name: 

myMethod.MethodName = "Display"
Trace myMethod.MethodName

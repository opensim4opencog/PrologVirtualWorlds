Code

 Property of the Method Object

A String that contains the code for an inline method. Code is a read-only String.  

Syntax

object.Code

The Code property syntax has these parts:
Part Type Use Description
object Method Required An object reference to a Method.

Remarks

If you want to use quotation marks within a string, you must include one extra quotation mark for each one within the string. Thus, the quotation-marked phrase is surrounded by another pair of quotation marks, and the whole statement is enclosed by a pair of quotation marks for a total of six (minimum) in the statement. Within the code, the desired quotation marks will be doubled. See the Examples section that follows for an example of a string embedded within another string.

If you need to put several script statements in the code string, separate each statement from the next with a colon ( : ). You can also use a carriage return to separate the lines by concatenating two quoted statements with CHR(13) between them (see examples). 

Note:   The Code method works only with inline methods, not with external modules.

Examples

To get the code for an inline method:

   Result = MyMethod.Code

To separate statements with a colon:

   MyMethod.Code = "Print ""Hello!"" : Print ""Hello again!"" "

To separate statements with a carriage return:

   MyMethod.Code = "Print ""Hello!"" " + CHR(13) + "Print ""Hello again!"" "

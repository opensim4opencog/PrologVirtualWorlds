Purpose

Welcome to the API Reference for Microsoft Virtual Worlds. The API Reference was created with the following structure:

Note that additional topics are stored in the Methods and Properties folders for each data type.

Exemplars are template objects used to create the objects that can be seen in a virtual world. An exemplar can be used to create identical copies of itself, which are called instances.

You can also use an exemplar as the basis for a new exemplar. When a new exemplar is created, it inherits all of the properties and methods of its parent exemplar, but you can modify aspects of the new exemplar to expand upon those of its parent.

All of the Exemplars provided in Virtual Worlds are created from the Thing exemplar.

The Other Data Types in Virtual Worlds are a collection of utility data types. The Other Data Types are independent of each other. Note that there are two methods (IsOfType and ToASCII) and two properties (Type and World) that are inherited by all other data types.

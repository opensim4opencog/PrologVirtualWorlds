KeyOwner

 Property of AvatarProfile

Returns the name of the avatar who owns the specified key within the AvatarProfile object. KeyOwner is a read-only AvatarProfile object.

Syntax

object.KeyOwner(bstrKey)

The KeyOwner property syntax has these parts:
Parts Type Use Description
object AvatarProfile Required An expression that evaluates to an AvatarProfile object.
bstrKey String Required The name of a key in your Avatar's profile.

Remarks

This property returns a reference to the owner of the key as an object. The KeyOwner property is set as one of the parameters during the creation of a new key using the AddKey method.

Example

In this example, you use the Avatar property together with the KeyOwner property to check whether the owner of the avatar owns all of the profile keys. If not, this example prints the names of the other key owners.

Sub GetOtherOwnerList()
Dim myKey
Dim myKeyCount
Dim myKeyOwner
Dim myProfile
Dim myProfileAvatarOwner
Dim myKeyList

Set myKeyList = User.Profile.Keys
Set myProfile = User.Profile

myKeyCount = myKeyList.Count - 1
myProfileAvatarOwner = User.Profile.Avatar.Name

For myKey = 0 To myKeyCount
    myKeyOwner = _
      myProfile.KeyOwner(myKeyList.Property(myKey)).Name

    'Check if the owner of the key is the same
    'as the owner of the avatar. 

    If myProfileAvatarOwner <> myKeyOwner Then

        'If the key owner is not the avatar owner,
        'send the key and key owner values to the
        'Command Window. 

        Trace myKeyList.Property(myKey)
        Print myProfileAvatarOwner
    End If
Next

End Sub

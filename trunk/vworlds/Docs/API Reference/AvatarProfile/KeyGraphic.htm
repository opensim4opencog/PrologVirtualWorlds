KeyGraphic

 Property of AvatarProfile

Sets or returns the URL for the specified key in the AvatarProfile object. KeyGraphic is a read/write String.   

Syntax

object.KeyGraphic(bstrKey) = stringValue

The KeyGraphic property syntax has these parts:
Parts Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrKey String Required An expression that evaluates to the name of a key in the profile list of an avatar.
stringValue String Required An expression that evaluates to a valid URL.

Remarks

Along with creating your own Keys for the profile of your avatar, you can also include an image, either by creating the key with the AddKey method or by using the KeyGraphic method to add an image later.

The image that you assign to the KeyGraphic property is not used in the Microsoft Virtual Worlds environment. However, it can be used in an environment that is customized for the display of it.  

Example

This example checks each key in an avatar's profile for an existing image. If no image is attached to the key, it attaches a default image.

Sub AttachDefaultGraphic()
Dim myKey
Dim myKeyList
Dim myKeyCount

Set myKeyList = User.Profile.Keys
myKeyCount = myKeyList.Count - 1

For myKey = 0 To myKeyCount

'Check each key for an image. 

  If _
    User.Profile.KeyGraphic(Keylist.Property(myKey)) = "" Then
    'Assign the default image if none exists.
    User.Profile.KeyGraphic(Keylist.Property(myKey)) = _
      "file://C:/Images/Default.bmp"
  Trace KeyList.Property(myKey) & " KeyGraphic has been set to Default.bmp"
  End If
Next
End Sub

RemoveKey

 Method of AvatarProfile

Removes a key/value pair from the profile for the specified avatar. RemoveKey is a member of the AvatarProfile object.

Syntax

object.RemoveKey bstrKey

The RemoveKey method syntax has these parts:
Parts Type Use Description
object AvatarProfile Required An object reference that accesses an AvatarProfile object.
bstrKey String Required The name of the key to remove.

Return Value

This method does not return a value. 

Remarks

This method is used to remove keys and key values from the profile of an avatar. To remove a key, you must be the owner of the key or the owner of the avatar with the key attached.

To remove an object from a key list, a list of the key/value pairs in an avatar's profile. use the RemoveFromUserProfileList method of Avatar.

An avatar's profile contains an unordered list of paired keys and values. Once you have inserted a key/value pair (element) into a map, you can efficiently retrieve or delete the pair using the key to access it. You can also iterate over all the elements in the map. 

A key/value pair maps unique keys to values. The key is a string that represents the name of the property in the property map. The value is the contents of the property. A property map uses the string that makes up the key instead of an index number to access the value of each property.

Example

This example checks each avatar in a room for a profile key named "PassportID". If the key is found, the program removes the key from the profile of the avatar. 

Sub NewProfileKey()
Dim myKeyOwner
Dim allAvatars
Dim myPassID

'Substitute the name of your avatar for "Eve" in the following statement. 

Set myKeyOwner = User.Container.FindWithin("Eve")
Set allAvatars = User.Container.Avatars

'Check each avatar in the room. 

For Each myUser In allAvatars
  'If a passport exists, remove the passport key.
  If myUser.Profile.IsValid("PassportID") Then
    myUser.Profile.RemoveKey "PassportID"
    Trace myUser.Name & "'s passport has been removed from the system."
  Else
    Trace myUser.Name & "'s passport is invalidated!"
  End If
Next
End Sub

UnbanObject

 Method of AvatarProfile

Removes the restriction previously created with the BanObject method, and allows the specified Thing object to add keys to an avatar's profile. UnbanObject is a member of the AvatarProfile object.

Syntax

object.UnbanObject pthing

The UnBanObject method syntax has these parts:
Parameters Type Use Description
object AvatarProfile Required An object reference to an AvatarProfile object.
pthing Thing Required An object reference to a Thing object, such as an avatar.

Return Value

This method does not return a value. 

Remarks

The UnbanObject method restores the ability of an object to add Keys to the profile of your avatar. This object can be a reference to a user or to another object, such as a game.

Example

This example prevents another user (Gracie) from adding keys to your profile.

Sub BanGracie
Dim myAvatarProfile
Dim myUserToUnban

'Get the avatar you want to protect. 

Set myAvatarProfile = User.Profile

'Get the avatar you want to unban. 

Set myUserToUnban = User.Container.FindWithin("Gracie")

'Remove the ban. 

myAvatarProfile.UnbanObject myUserToUnban 
End Sub

Keys

 Property of AvatarProfile

Returns a PropertyList collection from the profile of an avatar. Keys is a read-only AvatarProfile property.

Syntax

object.Keys

The Keys property syntax has these parts:
Parameters Type Use Description
object AvatarProfile Required An expression that returns an AvatarProfile object.

Remarks

Keys are a property list that contains settings for your avatar. The keys are stored in your avatar's profile. In Microsoft Virtual Worlds, you can update the default keys in the profile list for your avatar, or you can create your own keys by using the AddKey method. These keys contain information about you and your avatar. For example, the GestureText key contains either the default text for the gesture or the last phrase you used with the Emote method. The following table lists the keys that are available for your avatar's profile in Virtual Worlds.

Ignoring
0

1

Example

This example prints the value of a single key and iterates through the entire list of keys in the profile of an avatar. 

 
Sub ListKeys()
Dim myKey
Dim myKeyList
Dim myKeyCount

'Get the Keys collection. 

Set myKeyList = User.Profile.Keys
myKeyCount = myKeyList.Count - 1

'Trace each key to the Command Window. 

For myKey = 0 To myKeyCount
    Print myKeyList.Property(myKey)
    Trace Typename(User.Profile.KeyValue(myKeyList.Property(myKey)))
Next

'Now trace a single key. 

Trace User.Profile.KeyValue("SpriteGeometry")

End Sub

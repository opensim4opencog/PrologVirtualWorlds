BanObject

 Method of AvatarProfile

Prevents the specified object from adding items to an AvatarProfile list. BanObject is a member of the AvatarProfile object.

Syntax

object.BanObject pthing

The BanObject method syntax has these parts:
Parameters Type Use Description
object AvatarProfile Required An object reference to an AvatarProfile object.
pthing Thing Required An object reference to a Thing object, such as an avatar.

Return Value

This method does not return a value. 

Remarks

The BanObject method prevents an object, such as another avatar or program, from adding Keys to the profile of your avatar.  If you have already banned several objects from adding keys to your profile list, this method appends new banned objects to the current list.

Anyone can add a key to the profile of an avatar; you don't have to be a wizard in the world to add a key. In addition, any object can create and add a key to a profile. For example, if you ran a chess tournament in your world, the chess program could add a key that contained the final score of the game to the profile of each player. To prevent the chess program from adding a score to your own profile, use the BanObject method as shown in the following statements. 

Dim myAvatarProfile
Dim myChessGame

     'Get the profile for your avatar.
Set myAvatarProfile = User.Profile

'Get the object that you want to ban. 
'This assumes that the name of the object is "Chess."

Set myChessGame = User.Container.FindWithin("Chess")

'Ban the game object from adding keys.
myAvatarProfile.BanObject myChessGame

In some cases, you may find that a user is adding keys to your profile and you'd prefer not to have those extra keys attached to your profile. You can ban other users from adding keys to your profile as shown in the Example section.

Example

This example prevents another avatar object (Gracie) from adding keys to Eve's profile.

Sub BanGracie
Dim myAvatarProfile
Dim myObjectToBan

'Get a reference to the avatar you want to prevent from having keys added to its profile. 

Set myAvatarProfile = User.Profile

'Get the avatar you want to ban. 

Set myObjectToBan = User.Container.FindWithin("Gracie")

'Ban the object from adding keys. 

myAvatarProfile.BanObject myObjectToBan 
End Sub

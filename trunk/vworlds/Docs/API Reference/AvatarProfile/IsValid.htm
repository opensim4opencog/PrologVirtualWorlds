IsValid

 Method of AvatarProfile

Returns True if the string specified by the bstrKey parameter exists as a key in an AvatarProfile object. IsValid is a member of the AvatarProfile object.

Syntax

object.IsValid(bstrKey)

The IsValid method syntax has these parts:
Parameters Type Use Description
object AvatarProfile Required An object reference to an AvatarProfile object.
bstrKey String  Required The name of the key in an avatar's profile.

Return Value

This method returns True if the key exists in the avatar's profile, otherwise, it returns False.

Remarks

When you use the IsValid method, it searches the specified avatar profile. You can use the indexes of the Keys in the profile list of an avatar to gain access to each item in the list as shown in the following statements.  

For myCount = 0 To User.Profile.Keys.Count-1
    Trace User.Profile.Keys.Property(myCount) & " is a valid item " & _
      User.Profile.IsValid(User.Profile.Keys.Property(myCount))
Next

For security reasons, when using the IsValid method with the AvatarProfile object, this method does not distinguish between an absent key and a key that the user does not have permission to access. It will return False in both cases.

Example

This example returns the value of a single key and iterates through the entire list of keys in the profile of an avatar to check if the keys are valid.

Sub CheckIfValidKey()
Dim myKey
Dim myKeylist

'Get the Keys collection. 

Set myKeylist = User.Profile.Keys

'Trace each key and whether it's valid to the Command Window. 

For Each myKey In myKeylist
    Trace myKey & ":" & User.Profile.IsValid(myKey)
Next

'Now trace a single key. 

Trace User.Profile.IsValid(Friends)

End Sub

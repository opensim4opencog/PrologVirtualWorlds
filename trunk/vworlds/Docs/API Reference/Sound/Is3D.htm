Is3D

 Property of the Sound Object

Returns True when the specified Sound object is set to three-dimensional stereo when you first create the Sound object. Is3D is a read-only Boolean.

Syntax

object.Is3D

The Is3D property syntax has these parts:
Parameters Type Use Description
object Sound Required An object reference to a Sound object.

Remarks

You can set your sound file to play in two- or three-dimensional stereo with the CreateSound method. If you set the bIs3D parameter to True, the Sound object is set to 3-D.

Defining 2-D and 3-D Sound Objects

A 2-D sound is an ambient sound in a virtual world, such as the sound of crickets or birds. This type of sound is heard throughout the world. You can adjust the pan and volume of the sound, but these adjustments will remain at the same level throughout the world.

A 3-D sound has position and orientation. It uses a sound projection cone that has 3-D qualities as well as a spatial orientation. With a 3-D sound, as you move throughout the world, the sound changes. An example of this type of sound is the sound of a crowd in a room. As you approach the object that the sound is attached to, the volume increases.

After you have set the sound to either 2-D or 3-D, use the following methods to adjust various sound parameters. Use the Set2DData method to set the 2-D parameters, such as volume, frequency, and pan information. To set the 3-D parameters for the sound cone, such as the inside and outside sound cone angles, the outside volume, and the minimum and maximum distance, use the Set3DData method.

Examples

The first example creates a  Sound object, associates it with an object in the world, and sets the sound to 2-D.

Sub MakeThunder()
Dim myCloud
Dim myWavFile
Dim myThunderClap

'You can use the default "cloud" object in Virtual Worlds. 

Set myCloud = User.Container.FindWithin("cloud")

'Get the sound file. 

myWavFile = "file://C:/Sound Files/Thunder.wav"

'Create a 2-D Sound object. 

Set myThunderClap = CreateSound(myWavFile, False, True)

'Associate the sound with an object. 

myCloud.Sound = myThunderClap

'Set the 2-D data. 

 
myThunderClap.Set2DData -1000, 50, 500
End Sub

Example 2

To set the sound to 3-D and adjust the 3-D effects, use the following code.

Set myThunderClap = CreateSound(myWavFile, True,  True)

myThunderClap.Set3DData 20, 40, -1000, 5, 25

SoundURL

 Property of the Sound Object

Sets or returns the URL for a sound file. SoundURL is a read/write String property. 

Syntax

To set the property:
expression.SoundURL = stringValue

To return the property:
expression.SoundURL

The SoundURL property syntax has these parts:
Parameters Type Use Description
expression Sound Required An expression that evaluates to a Sound object.
stringValue String Required A valid URL that references a sound file.

Remarks

When you first create a Sound object, you set the SoundURL parameter for the CreateSound method to a valid URL. This determines the sound file for the SoundURL property of the Sound object. When you associate the Sound object with an object in the world, the value for the SoundURL property is carried over to the object.  

Resetting the SoundURL property of a Sound object will stop the sound, then restart it with the new sound.

Example

This example changes the sound file for a specific object.

To run this example: You must have a Sound object existing in your world. To create the objects shown in this example, refer to the CreateSound method. 


Sub ChangeToBirdSong()
Dim myTree
Dim mySong
Dim mySnd

'Get the Sound object and the object associated with it. 

Set myTree = User.Container.FindWithin("Oak Tree")
Set mySong = myTree.Sound
mySnd = "file://C:/Program Files/Microsoft Virtual
Worlds/Local Content/SDK/Library/Sounds/bird.wav"

If Not mySong.SoundURL = mySnd Then
    mySong.SoundURL = mySnd
End If

End Sub

Get2DData

 Method of the Sound Object

Returns the two-dimensional volume, frequency, and pan information for a Sound object and places the information in the pvarlVolume, pvarlFrequency, and pvarlPan parameters. Get2DData is a member of the Sound object.

Syntax

object.Get2DData pvarVolume, pvarlFrequency, pvarlPan

The Get2DData method syntax has these parts:
Part Type Use Description
object Sound Required An object reference to a Sound object.
pvarlVolume Long Required The volume of a sound, in hundredths of a decibel.
pvarlFrequency Long Required The frequency of a sound, in hertz.
pvarlPan Long Required The amount by which one channel is louder than the other, in hundredths of a decibel.

Return Value

This method does not return a value.   

Remarks

The amount of attenuation applied to the sound means that the volume is lowered by the specified amount of hundredths of decibels. The 2-D sound effects include volume control, frequency of the sound effect, and volume control for each speaker. 

To create 3-D sound effects, refer to the CreateSound, Sound, Get3DData, or Set3DData topics.

Effects of the pvarlVolume Parameter

The pvarlVolume parameter is specified in hundredths of decibels (dB). Allowable values are from 0 (no attenuation or weakening of the transmission signal) through -10,000 (silence). The value 0 represents the original, unadjusted volume of the stream. The value -10,000 indicates an audio volume attenuated by 100 dB, which, for all practical purposes, is silence.

Effects of the pvarlFrequencyParameter

The pvarlFrequency parameter represents the number of times the sound is sampled, and is specified in hertz (Hz). This parameter must be in the range from 100 through 100,000. You can also set the value to 0, which means that the sound is not sampled. Common frequency rates are 8,800 Hz, 14,400 Hz, and 22,000 Hz.

Effects of the pvarlPan Parameter

The value in the pvarlPan parameter is measured in hundredths of a decibel (dB), in the range from -10,000 through 10,000. This parameter sets the volume for each speaker, or channel. There are three possible settings for the two channels:

  • Set the right channel to silence: Use the value -10,000. The right channel is attenuated by 100 dB.
  • Set the left channel to silence: Use the value 10,000. The left channel is attenuated by 100 dB.
  • Set both channels to full volume: Use 0. Both channels are attenuated by 0 dB.

As you can see from the previous table, at any setting other than 0, one of the channels is at full volume and the other is attenuated. By setting the pvarPan parameter to -2,173, the left channel is automatically set to full volume, while the right channel is attenuated by 21.73 dB. Similarly, setting the pvarPan parameter to 870, attenuates the left channel by 8.7 dB, and automatically sets the right channel to full volume. 

The lPan parameter acts conjunctively with the lVolume parameter to adjust the effects of the sound.    

Example

  mySound.Get2DData nVolume, nFrequency, nPan

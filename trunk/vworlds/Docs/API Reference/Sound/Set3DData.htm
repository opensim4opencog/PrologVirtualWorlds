Set3DData

 Method of the Sound Object

Sets the three-dimensional angle and volume information for a 3-D Sound object. Set3DData is a member of the Sound object.

Syntax

object.Set3DData dwInsideConeAngle, dwOutsideConeAngle, lConeOutsideVolume, flMinDistance, flMaxDistance

The Set3DData method syntax has these parts:
Part Type Use Description
object Sound Required An object reference to a Sound object.
dwInsideConeAngle Long Required The inside angle, in degrees, of the sound projection cone.
dwOutsideConeAngle Long Required The outside angle, in degrees, of the sound projection cone.
lConeOutsideVolume Long Required The outside cone volume in hundredths of a decibel.
flMinDistance Float Required The approach distance, in meters, within which the volume will not increase.
flMaxDistance Float Required The distance, in meters, beyond which the sound will not get any quieter.

Return Value

This method does not return a value. 

Remarks

You can set your sound file to play in two- or three-dimensional stereo with the CreateSound method. If you set the bIs3D parameter to True, the Sound object is set to 3-D.

You can manipulate 3-D sounds by using the parameters for the Set3DData method to change the dimensions of the sound projection cones. These changes affect the attenuation of the sound, which means that the volume is lowered by the specified amount of hundredths of a decibel.

A 3-D sound has position as well as spatial orientation. In Microsoft Virtual Worlds, sound cones are made up of an inner cone and an outer cone. Within the inner cone, the volume of the sound is what it would be if there were no cone; in other words, there is no attenuation from the normal volume. Outside the outer cone, the normal volume is attenuated by a specified outside cone volume. The angle between the inner and outer cones is a zone of transition between the inside volume and the outside volume.

Effects of the dwInsideConeAngle and dwOutsideConeAngle Parameters

The dwInsideConeAngle and dwOutsideConeAngle parameters describe the scope of the Sound object. The outside cone angle must always be equal to or greater than the inside cone angle. To get the orientation of sound cones, use the Get3DData method.

Effects of the lConeOutsideVolume Parameter

The lConeOutsideVolume parameter is expressed in hundredths of a decibel and is a negative value because it represents decible attenuation from the default volume of 0. Allowable values are from 0 (no attenuation) through -10,000 (silence).

To create an omnidirectional 3-D sound source, set the lConeOutsideVolume parameter to 0 (no attenuation) and both the dwInsideConeAngle and dwOutsideConeAngle parameters to 360 degrees. The sound will have no apparent orientation. These are the default values when you create a sound, so unless you explicitly specify other values, all 3-D sounds start out as omnidirectional.

Changes to these parameters may be delayed by as much as one second.

Effects of the flMinDistance and flMaxDistance Parameters

As you approach a Sound object in the world, you reach a point where the sound does not increase in volume. This is the distance set by the flMinDistance parameter. The flMaxDistance is the point at which the sound does not decrease as you walk away from a Sound object. 

Example

mySound.Set3DData 20, 40, -1000, 5, 25

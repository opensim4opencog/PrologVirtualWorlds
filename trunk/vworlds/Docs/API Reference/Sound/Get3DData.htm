Get3DData

 Method of the Sound Object

Returns the three-dimensional angle and volume information for a 3-D Sound object. Get3DData is a member of the Sound object. 

Syntax

object.Get3DData dwInsideConeAngle, dwOutsideConeAngle, lConeOutsideVolume, flMinDistance, flMaxDistance

The Get3DData method syntax has these parts:
Part Type Use Description
object Sound Required An object reference to a Sound object.
dwInsideConeAngle Long Required The inside angle, in degrees, of the sound projection cone.
dwOutsideConeAngle Long Required The outside angle, in degrees, of the sound projection cone.
lConeOutsideVolume Long Required The outside cone volume in hundredths of a decibel.
flMinDistance Float Required The approach distance, in meters, within which the volume does not increase.
flMaxDistance Float Required The distance, in meters, beyond which the sound does not decrease.

Return Value

This method does not return a value.

Remarks

There are three basic ways to represent sound within a virtual world:

  • A sound with an orientation is a sound cone. The amplitude of the sound at a given distance is the same in all directions.
  • A sound with no orientation is a point source, or sound source.
  • A sound projection cone is a sound that has 3D qualities and a spatial orientation.

In Virtual Worlds, sound cones are made up of an inner cone and an outer cone. Within the inner cone, the volume of the sound is what it would be if there were no cone; in other words, there is no attenuation from the normal volume. Outside the outer cone, the normal volume is attenuated by a specified number of decibels. The angle between the inner and outer cones is a zone of transition between the inside volume and the outside volume.

Effects of the dwInsideConeAngle and dwOutsideConeAngle Parameters

The dwInsideConeAngle and dwOutsideConeAngle parameters describe the scope of the Sound object. The outside cone angle must always be equal to or greater than the inside cone angle. To set the orientation of sound cones, use the Set3DData method.

Effects of the lConeOutsideVolume Parameter

The lConeOutsideVolume parameter is expressed in hundredths of a decibel and is a negative value because it represents attenuation from the default volume of 0. Allowable values are from 0 (no attenuation) through -10,000 (silence).

Effects of the flMinDistance and flMaxDistance Parameters

As you approach a Sound object in the world, you reach a point where the sound does not increase in volume. This is the distance set by the flMinDistance parameter. The flMaxDistance is the point at which the sound does not decrease as you walk away from a Sound object.

Example

All five of the parameters of Get3DData are output parameters. This means that after this method has been called, each of these parameters will be reset to a new value.

mySound.Get3DData InsideConeAngle, _
    OutsideConeAngle, _
    ConeOutsideVolume, _
    MinDistance, _
    MaxDistance

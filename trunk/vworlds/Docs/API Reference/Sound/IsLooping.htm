IsLooping

 Property of the Sound Object

When set to True the Sound object is set to continually play. IsLooping is a read/write Boolean. 

Syntax

To set the property:
object.IsLooping = BooleanValue

To return the property:
object.IsLooping

The IsLooping property syntax has these parts:
Parameters Type Use Description
object Sound Required An object reference that accesses a Sound object.
BooleanValue Boolean Required Set this property to True to continually play the sound. Set it to False to play the sound one time.

Remarks

When you first create a Sound object, you set the bIsLooping parameter for the CreateSound method to True or False. This determines the state of the IsLooping property for the Sound object and whether the sound plays once or continuously. 

Resetting the IsLooping property will stop the sound, then restart it in the new mode. 

Example

This example gets a Sound object, determines if the sound is continually playing, and if it is, changes it to play only one time.

To run this example: You must have a Sound object set to continually play. To create the objects shown in this example, refer to the CreateSound method. 


Sub AddBirdSong()
Dim myTree
Dim mySong

'Get the Sound object and the object associated with it. 

Set myTree = User.Container.FindWithin("Oak Tree")
Set mySong = myTree.Sound

If mySong.IsLooping Then
    mySong.IsLooping = False
End If

End Sub

Length

 Method of Vector

Returns the length of the calling Vector. Length is a member of the Vector data type.  

Syntax

object.Length()

The Length method syntax has these parts:
Part Type Use Description
object Vector Required An object reference that accesses a Vector object.

Return Value

This method returns a Float.

Remarks

The Length method returns the distance between the origin, usually (0,0,0), and the value of the calling vector.

Example

This example creates a Vector and returns the length of the new vector by calling the Length method.   

Dim myNewVector
Set myNewVector = World.CreateVector(2, 3, 4)
Trace myNewVector.Length

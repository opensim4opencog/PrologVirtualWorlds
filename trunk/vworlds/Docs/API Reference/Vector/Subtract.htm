Subtract (Vector)

 Method of Vector

Subtracts a Vector value from the value of the calling vector, and assigns the result to a vector passed as the second parameter. Subtract is a member of the Vector data type.  

Syntax

object.Subtract pv, pvres

The Subtract method syntax has these parts:
Part Type Use Description
object Vector Required An object reference that accesses a Vector object.
pv Vector Required The Vector value to be subtracted.
pvres Vector Required A Vector variable that is assigned the result of the Subtract method.

Return Value

This method does not return a value.

Remarks

The Subtract method provides a means of subtracting vector values. The first parameter (pv) is the value to be subtracted from the value of the calling object. The second parameter (pvres) is a vector variable that is assigned the result of the subtraction. The Vector that invokes this method is not changed.

The pvres parameter must be instantiated as a Vector object before it is sent as a parameter.

Examples

The first example subtracts two vectors and assigns the result to a third vector variable. 

Dim myVector1
Dim myVector2
Dim myVecResult

'Create the vectors. 

Set myVector1 = World.CreateVector(2, 2, 2)
Set myVector2 = World.CreateVector(1, 1, 1)
Set myVecResult = World.CreateVector(0, 0, 0)

'Call the Subtract method and subtract the values. 

myVector1.Subtract myVector2, myVecResult

'Display the result in the Command Window. 

Trace myVecResult.ToAscii

Example 2

The second example subtracts a value from the invoking object by assigning the result of the subtraction to that object.

Dim myVector1
Dim myVector2

'Create the vectors. 

Set myVector1 = World.CreateVector(3, 3, 3)
Set myVector2 = World.CreateVector(1, 1, 1)

'Call the Subtract method and subtract the values,
'assigning the result to the calling vector. 

myVector1.Subtract myVector2, myVector1

'Display the result in the Command Window. 

Trace myVecResult.ToAscii

Y (Vector)

 Property of Vector

Sets or returns the y-coordinate value for a Vector data type. Y (Vector) is a read/write Float.

Syntax

To set the property:
object.y = VectorValue

To return the property:
object.y

The y property syntax has these parts:
Parts Type Use Description
object Vector Required An object reference to a Vector object.
VectorValue Float Required An expression that evaluates to a y-coordinate that is a Float .

Remarks

The y property returns the y-coordinates for the following properties that use vectors:

Direction

 Position
EntryOrientation RotationAxis
EntryPoint Scale
LightAttenuation Up
LightColor

The vector coordinates become the end point of a line which originates at the center of the object.

Important:   Setting the value for the y property does not remote, persist, or cause the property-changed handler to be invoked. If you want these changes to persist and remote, etc., use the Set method to change the vector values.

Example

This example demonstrates the use of the y property with various data types.

Dim myVector

'Create a vector to establish coordinates for x, y, and z.
'Retrieve the y-coordinate. 

Set myVector = CreateVector(10, 5, 3)
Trace myVector.y

'Establish the y-coordinates independently of a vector object. 

Dim myChair

Set myChair = World.CreateInstance("Chair", "Artifact", User.Container)

'The following changes will appear on your local computer. 

myChair.Direction.y = 2
myChair.Position.y = 4
myChair.RotationAxis.y = 1
myChair.Up.y = 0

'This next code block retrieves the values for the
'y-coordinate independently of a vector object. 

Dim myChairY

Trace myChair.Scale.y
Trace myChair.Up.y
myChairY = myChair.Position.y

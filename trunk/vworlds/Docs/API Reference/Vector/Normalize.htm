Normalize

 Method of Vector

Modifies a copy of the values of the calling Vector to create a unit-length vector with the same direction as the source. The result is assigned to the supplied parameter. Normalize is a member of the Vector data type.  

Syntax

object.Normalize pvres

The Normalize method syntax has these parts:
Part Type Use Description
object Vector Required An object reference that accesses a Vector object.
pvres Vector Required The Vector that is assigned the normalized value.

Return Value

This method does not return a value.

Remarks

The Normalize method creates a vector by modifying a copy of the values of the calling vector and assigning the result to the supplied parameter (pvres). The new vector is a unit-length vector with the same direction as the source.

Note:   The pvres parameter must be initialized as a vector object prior to being passed as a parameter. The initial values are then overwritten by the Normalize method.

Example

This example creates two vectors, and then normalizes a copy of the values of the invoking vector and assigns the result to the second vector. 

Dim myVector1
Dim myVectorResult

'Create the vectors. 

Set myVector1 = World.CreateVector(1, 1, 1)
Set myVectorResult = World.CreateVector(0, 0, 0)

'Call the Normalize method and normalize the invoking object,
'assigning the result to the second parameter. 

myVector1.Normalize myVectorResult

'Display the result in the Command Window. 

Trace myVectorResult.Length

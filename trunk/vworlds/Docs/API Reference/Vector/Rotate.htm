Rotate

Method of Vector

Rotates a Vector about the y-axis. Rotate is a member of the Vector data type.  

Syntax

object.Rotate fTheta, pvres

The Rotate method syntax has these parts:
Part Type Use Description
object Vector Required An object reference that accesses a Vector object.
fTheta Float Required The radian angle value represented as a Float by which the vector is rotated.
pvres Vector Required A Vector that is assigned the result of the Rotate method.

Return Value

This method does not return a value.

Remarks

The Vector that invokes this method is not changed. The Rotate method rotates a Vector around the y-axis by multiplying the value of the calling Vector by the value supplied in the first parameter (fTheta), and assigning the result to the second parameter. The fTheta parameter is a radian value represented as a Float.

The pvres parameter must be instantiated as a Vector object before it is sent as a parameter.

Examples

The first example creates two vectors, and then rotates a vector and assigns the result to the second parameter. 

Dim myVector1
Dim myVectorResult

'Create the vectors. 

Set myVector1 = World.CreateVector(1, 1, 1)
Set myVectorResult = World.CreateVector(0, 0, 0)

'Rotate the vector. 

myVector1.Rotate 1.2, myVectorResult

'Display the new value in the Command Window. 

Trace myVectorResult.ToAscii

Example 2

The second example rotates the invoking object by assigning the result of the Rotate method to that object.

Dim myVector1

'Create the vector. 

Set myVector1 = World.CreateVector(2, 2, 2)

'Call the Rotate method and multiply the values,
'assigning the result to the calling vector. 

myVector1.Rotate 1.1, myVector1

'Display the result in the Command Window. 

Trace myVecResult.ToAscii

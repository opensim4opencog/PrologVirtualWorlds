X (Vector)

 Property of Vector

Sets or returns the x-coordinate value for a Vector object. X (Vector) is a read/write Float.

Syntax

To set the property:
object.x = VectorValue

To return the property:
object.x

The x property syntax has these parts:
Parts Type Use Description
object Vector Required An object reference to a Vector data type.
VectorValue Float Required An expression that evaluates to an x-coordinate that is a Float .

Remarks

The x property returns the x-coordinates for the following properties that use vectors:

Direction

 Position
EntryOrientation RotationAxis
EntryPoint Scale
LightAttenuation Up
LightColor

The vector coordinates become the end point of a line which originates at the center of the object.

Important:   Setting the value for the x property does not remote, persist, or cause the property-changed handler to be invoked. If you want these changes to persist and remote, etc., use the Set method to change the vector values.

Example

This example demonstrates the use of the x property with various data types.

Dim myVector

'Create a vector to establish coordinates for x, y, and z.
'Retrieve the x-coordinate. 

Set myVector = CreateVector(10, 5, 3)
Trace myVector.x

'Establish the x-coordinates independently of a vector object. 

Dim myChair

Set myChair = World.CreateInstance("Chair", "Artifact", User.Container)

'The following changes will appear on your local computer. 

myChair.Direction.x = 2
myChair.Position.x = 4
myChair.RotationAxis.x = 1
myChair.Up.x = 0

'This next code block retrieves the values for the
'x-coordinate independently of a vector object. 

Dim myChairX

Trace myChair.Scale.x
Trace myChair.Up.x
myChairX = myChair.Position.x

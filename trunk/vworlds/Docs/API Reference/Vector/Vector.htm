Vector

Data Type

Represents the direction of a line in a three-dimensional space. This is a data type specific to Microsoft Virtual Worlds.

Description of a Vector Object

A Vector object usually represents a three-dimensional geometric vector, a directed line segment that extends from the point of origin to its endpoint. The properties of the Vector object in Virtual Worlds contain the values for the x, y, and z coordinates as Float values.

In Virtual Worlds, you can also use a vector to represent the attenuation of a Light object, or its color. 

Creating a Vector

Use the CreateVector method to create a vector. For example, use the following code to create a vector.

Dim myVector

'Create a new vector. 

Set myVector =  CreateVector(2.0, 0.0, 0.0)

This code creates a vector that begins at the point (0, 0, 0) and ends at the point (2, 0, 0). This vector is two units long and extends in the positive x-axis direction.

You can use the Set method to change the values of a vector after the vector has been created. The following code changes the scale of a lamp.

To run this code: Create an artifact in your world and name it Study Lamp. 

Dim myLamp

'Assign the variable to the study lamp. 

Set myLamp =  User.Container.FindWithin("Study Lamp")

'Change the direction of the lamp using the Scale method. 

myLamp.Direction.Set 2, 1, 0

The x, y, and z properties provide access to the individual coordinates of a vector and allow you to adjust a single coordinate on your local computer, as shown in the following code.

Dim myLamp


'Assign the variable to the study lamp.

Set myLamp =  User.Container.FindWithin("Study Lamp")



   'Position the lamp.

myLamp.Position.y = 4

Important:   Setting the value for these properties does not remote, persist, or cause the property-changed handler to be invoked. If you want these changes to persist and remote, etc., use the Set method to change the vector values.

RotateAboutAxis

 Method of Vector

Rotates a Vector around a given axis. RotateAboutAxis is a member of the Vector data type.  

Syntax

object.RotateAboutAxis fTheta, fAxisX, fAxisY, fAxisZ

The RotateAboutAxis method syntax has these parts:
Part Type Use Description
object Vector Required An object reference that accesses a Vector object.
fTheta Float Required The radian angle value represented as a Float by which the Vector is rotated.
fAxisX Float Required The x coordinate of the vector that acts as an axis around which the calling Vector is rotated.
fAxisY Float Required The y coordinate of the vector that acts as an axis around which the calling Vector is rotated.
fAxisZ Float Required The z coordinate of the vector that acts as an axis around which the calling Vector is rotated.

Return Value

This method does not return a value.

Remarks

The RotateAboutAxis method rotates the calling object around a supplied vector that acts as an axis. The axis is determined by the fAxisX, fAxisY, and fAxisZ parameters. The vector is rotated by multiplying the angle supplied in the first parameter (fTheta) by the supplied user-defined axis. The fTheta parameter is a radian value represented as a Float.

Example

This example rotates a vector by the given angle around the axis provided in the parameters. 

Dim myVector1

'Create the vector. 

Set myVector1 = World.CreateVector(1, 1, 1)

'Rotate the vector. 

myVector1.RotateAboutAxis 1.2, 3, 2, 4

'Display the new value in the Command Window. 

Trace myVector1.ToAscii

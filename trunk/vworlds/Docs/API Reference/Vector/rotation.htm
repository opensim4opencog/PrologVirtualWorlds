Rotation

 Property of Vector

Returns the rotation angle of a Vector about the y-axis in radians, relative to the vector (1, 0, 0). Rotation is a read-only Float.  

Syntax

object.Rotation

The Rotation property syntax has these parts:
Part Type Use Description
object Vector Required An object reference that accesses a Vector.

Remarks

The Rotation property contains the angle of rotation of a Vector relative to the starting position of (1, 0, 0). The Rotation value is stored as a radian value, where 2 pi radians is approximately 360 degrees. The rotation value is commonly used to establish placement and orientation of a vector within a scene. To change the Rotation value, use the Rotate method of the Vector data type.

A simple example to understand the rotation value is to draw a circle on a cartesian plane as if you were viewing the plane from directly above the y-axis. On the x-z plane, draw a circle with a radius of 10 units, with the center point aligning with the axis intersection point (this radius will give you whole number coordinates at 45-degree intervals.) Map out the vectors that intersect the circle at 45-degree angles from either the x- or the y-axis. Using the CreateVector method of the World object, iterate through the list of vectors, acquire the rotation value for each vector, and convert the result from radians to degrees using the formula: degrees = radianValue * 57.297. Although the Vector methods related to rotation all use radians, for many people degrees are easier to visualize.

Example

This example creates a subroutine that converts the value of the Rotation property to degrees, and then displays the result in the Command Window. In this example, the parameter "myVector" is a Vector object. 

Sub myRotationConversion(myVector)

Dim myRadian
Dim myDegreeValue

'Acquire the Rotation value in radians. 

myRadian = myVector.Rotation

'Convert the value and round it to the nearest whole degree. 

myDegreeValue = Round(myRadian * 57.297)

'Trace a brief description and the conversion to the Command Window. 

Trace myVector.ToAscii & " has a Rotation value of: " & myRadian & "."
Trace "The converted value is " & myDegreeValue & " Degrees."

End Sub

ToAscii

 Method

Returns the name of an object as an ASCII string. ToAscii is a member of all objects in Microsoft Virtual Worlds.

Syntax

object.ToAscii

The ToAscii method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that can access any object in Virtual Worlds.

Return Value

This method returns a string that corresponds to the name of the specified object. 

Remarks

When working with PropertyList and PropertyMap objects, you can use the ToAscii method to return the number of objects in the collection as a string. 

Dim myTeaList
Dim myTeaCount

Set myTeaList = CreatePropertyList

myTeaList.Add "Earl Grey"
myTeaList.Add "Orange Spice"
myTeaList.Add "English Breakfast"
myTeaCount = myTeaList.ToAscii

Print "myTeaList has " & myTeaCount & "."

The variable "myTeaCount" now contains the value "3 properties" and prints "myTeaList has 3 properties." to the Command Window.

If you just want the number of items in the collection and you don't care if the value is a Long, you can use the Count property instead of the ToAscii method.

Example

This example takes a vector (the direction of a portal), and uses the ToAscii method to return the direction as a string.

To run this example: You must have a portal existing in your world. Substitute the name of the portal in your world for the one in the example.

Sub DirectionToString()
Dim myPortal
Dim myDirection

Set myPortal = User.Container.FindWithin("Portal to Home")

myDirection = myPortal.Direction.ToAscii

Print myDirection

End Sub

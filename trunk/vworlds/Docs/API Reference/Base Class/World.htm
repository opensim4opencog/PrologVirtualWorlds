World

 Property

Returns the World object. World is a read-only World object. 

Syntax

object.World

The World property syntax has these parts:
Parameters Type Use Description
object Object Required An expression that evaluates to a Virtual Worlds object.

Remarks

This property contains a reference to the World object and exists on every object in Microsoft Virtual Worlds. The property provides access to the properties, methods, and events that exist for the World object.

Example

Set myArtifact = World.CreateInstance("myArt1", "Artifact", User.Container)

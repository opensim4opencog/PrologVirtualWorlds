Type

 Property

Returns the data type of the specified object. Type is a read-only String. 

Syntax

object.Type

The Type property syntax has these parts:
Parts Type Use Description
object Thing Required An expression that evaluates to any of the objects in Microsoft Virtual Worlds.

Remarks

Certain objects in Virtual Worlds are objects derived from the Thing exemplar object and certain objects are derived from other objects. Each of these derived objects inherits all of the properties and methods from its parent object. For example, the Thing exemplar directly inherits all of the characteristics of the Thing object, and the Avatar object inherits all of the characteristics of the Thing exemplar and Thing object. The following table shows the relationships between all of the exemplar objects in Virtual Worlds in hierarchial order and also shows other Virtual Worlds objects. Example

This example uses the keys in the property list from the AvatarProfile object to illustrate how to use the Type method to check for a specific data type.

Sub TestType()
Dim myIsType
Dim myKey
Dim myKeyList
Dim myKeyCount

'Initialize the type variable. 

myIsType = User.Profile.Keys.IsOfType("Property List")

'Get the Keys collection. 

Set myKeyList = User.Profile.Keys
myKeyCount = myKeyList.Count - 1

'Trace each key to the Command Window if myIsType = True. 

If myKeyList.Type = myIsType Then
  For myKey = 0 To myKeyCount
    Print myKeyList.Property(myKey)
    Trace Typename(User.Profile.KeyValue(myKeyList.Property(myKey)))
  Next

End Sub

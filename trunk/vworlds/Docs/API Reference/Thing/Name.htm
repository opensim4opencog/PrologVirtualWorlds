Name

 Property of Thing

A String that contains the name of the Thing. Name is a read/write String.  

Syntax

object.Name

The Name property syntax has these parts:
Part Type Use Description
object String Required A reference to an instance of the Thing object.

Remarks

An object is given a name by its creator when it is created. (See the CreateInstance method of World.) Except for an avatar's name, which cannot be changed by anyone, an object's name can be changed by its owner, a wizard, or the world owner.

The name of an object can be used to obtain a reference to that object. The object reference can then be used to invoke methods and query properties on the object. The third code sample in the next section shows how you can get a reference to an object whose name you know.  

Example

To get the Name property: 

   Result = AThing.Name

To set the Name property: 

   AThing.Name = "Droid"

If there is an object named Tree, and you are in the same room as Tree, you can use its Name property to obtain a reference to it in this way:

   Set TreeRef = User.Container.FindWithin("Tree")

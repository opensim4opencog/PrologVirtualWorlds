Exemplar

 Property of Thing

Obtains the Thing from which the current object is derived. Exemplar is a read-only Thing.  

Syntax

object.Exemplar

The Exemplar property syntax has these parts:
Part Type Use Description
object Thing Required An expression that evaluates to a Thing.

Remarks

Each object in a world that is in the Thing exemplar's inheritance chain has an Exemplar property. This property contains a reference to the exemplar used to create the object.  

Example

To get the Exemplar of a Thing: 

  Set Result = AThing.Exemplar

You can determine the exemplar for an object (named UnkObject in the following example) in the following way:

Set UnkObjRef = User.Container.FindWithin("UnkObject")
Set Exemp = UnkObjRef.Exemplar
Print Exemp.Type

If UnkObject is an instance of the Artifact exemplar, the third line of the preceding example will display Artifact.

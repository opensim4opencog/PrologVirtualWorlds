IsValidInstanceProperty

 Method of Thing

Determines whether the specified property exists on the object instance. IsValidInstanceProperty is a member of the Thing object.  

Syntax

object.IsValidInstanceProperty(bstrName)

The IsValidInstanceProperty method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrName String Required The name of the property on the instance.

Return Value

This method returns a Boolean indicating whether the property exists on an instance of an exemplar.

Remarks

This method checks to see whether the specified property exists on the object instance, rather than on the exemplar of the instance. For example, if this method is called on an Artifact named RaceCar, for a property named MagWheels, it returns True if MagWheels exists on RaceCar. It does not check to see whether this property exists on the Artifact exemplar.

The IsValidInstanceProperty method is especially useful when verifying whether a property of an instance has been changed from the default value. This method can be used in tandem with the IsValidProperty method of the Thing object to first verify that the property exists, and if it does, check whether the default value has been changed on the instance of the exemplar. If the property has not been changed, the IsValidInstanceProperty method returns False. If the property has been changed, the IsValidInstanceProperty method returns true.  

Example

This example verifies when a property is changed on an instance of an exemplar by invoking the IsValidInstanceProperty method. The example first creates a custom exemplar named "myArtifactEx", adding a new property ("Age") to that exemplar. The default return value of the IsValidInstanceProperty is displayed when a property is in its default state. Then, the property value is changed, and the change is reflected in the new return value for the IsValidInstanceProperty method.

'Create a custom exemplar. 

Sub CreateMyExemplar()
Dim myEx
Dim myInstance

Set myEx = World.CreateExemplar("myArtifactEx", "Artifact")
myEx.AddProperty "Age", 28, PSBIT_FRIENDLY

'Create an instance of the custom exemplar. 

Set myInstance = World.CreateInstance("myArt1", "myArtifactEx", User.Container)

'Display the defaults for both the IsValidProperty and IsValidInstanceProperty methods. 

If myInstance.IsValidProperty("Age") = True Then
    Trace "Property exists on the Exemplar."
    Trace "Return value of IsValidInstanceProperty is: " &_ 
    myInstance.IsValidInstanceProperty("Age")
Else
    Trace "Property does not exist on either the exemplar or the instance."
End If

'Change the value of the property. 

myInstance.Age = 20
Trace "Property is changed."

'Display the new value for the IsValidInstanceProperty 

Trace "The IsValidInstanceProperty value is now: " & myInstance.IsValidInstanceProperty("Age")

End Sub

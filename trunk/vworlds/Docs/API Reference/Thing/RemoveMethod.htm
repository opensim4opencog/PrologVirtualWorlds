RemoveMethod

 Method of Thing

Removes a method from a Thing object. RemoveMethod is a member of the Thing object.  

Syntax

object.RemoveMethod bstrName

The RemoveMethod method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrName String Required The name of the method to remove.

Return Value

This method does not return a value.

Remarks

When you add a method to an object, you are adding a copy of that method. If you want to modify or remove the copy of that method from the object, you must be careful to refer to it explicitly; otherwise, you will be modifying or removing the original method (instead of the copy). For more information on this topic, see Creating a Method from a Module in the Programmer's Guide.

To add a method, use the AddMethod method of Thing.  

Example

MyObject.RemoveMethod "List"

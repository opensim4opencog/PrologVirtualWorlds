Contents

 Property of Thing

A PropertyList that holds the objects contained by the calling object. Contents is a read/write PropertyList.  

Syntax

object.Contents = PropertyListValue

The Contents property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
PropertyListValue String Required A list that holds the objects inside the calling object.

Remarks

A Room is contained within a World, and is part of the world's Contents. In the same way, objects are containted in a Room, and are part of its Contents. In contrast, the Container property of Thing refers to the object that contains the calling object.

All objects in the inventory of an avatar will be listed in the avatar's Contents. (In fact, an avatar's inventory is synonymous with an avatar's contents.) Only those objects at the same level inside the calling object will show up on that object's Contents list. For example, if an avatar has a box object in his or her inventory, the box object will appear in the avatar's Contents, although any objects inside the box will not appear on the same list. The objects inside the box would appear on the box's Contents list; if the box contains a smaller box, any objects within the smaller box will not appear on the larger box's list, and so on.

This property will be empty if there are no objects inside the calling object.  

Example

Set Result = AThing.Contents

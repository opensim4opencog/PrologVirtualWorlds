RemoveProperty

 Method of Thing

Removes a property from a Thing object. RemoveProperty is a member of the Thing object.  

Syntax

object.RemoveProperty bstrName

The RemoveProperty method syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.
bstrName String Required The name of the property to remove.

Return Value

This method does not return a value.

Remarks

When you add a property to an object, you are adding a copy of that property. If you want to modify or remove the copy of that property from the object, you must be careful to refer to it explicitly; otherwise, you will be modifying or removing the original property (instead of the copy).

To add a property, use the AddProperty method of Thing.  

Example

AThing.RemoveProperty "Age"

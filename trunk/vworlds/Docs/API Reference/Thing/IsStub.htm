IsStub

 Property of Thing

Returns True when the specified Thing object has not had its properties and methods marshalled. IsStub is a read-only Boolean.  

Syntax

object.IsStub

The IsStub property syntax has these parts:
Part Type Use Description
object Thing Required An object reference to a Thing object.

Remarks

You can use the IsStub property to check whether you can access a Thing object's methods and properties. If IsStub is False, you can set or return properties and run methods. 

Example

This example checks if an object is a stub, if not, it sets the objects position in a room. If the object is a stub, it sends a message to the Command Window. 

If myChair.IsStub = False Then
    Set myChair.Position 4.789, 1.000, 7.000
Else
    Trace "Properties and methods are not marshalled yet."
End If

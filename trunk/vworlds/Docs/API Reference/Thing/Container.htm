Container

 Property of Thing

The Thing that contains the calling object. Container is a read-only Thing.  

Syntax

object.Container

The Container property syntax has these parts:
Part Type Use Description
object Thing Required An object reference that accesses a Thing object.

Remarks

The Container of a Room is the World in which it exists; an avatar's Container is the Room in which the avatar currently exists; an avatar is the Container for any objects in the avatar's inventory. Most objects in a World exist within some sort of containing object.

The Container property is set by the system each time an object changes to a new container. For example, when the MoveInto method of the Thing Exemplar is used to change the container of an object, the Container property is automatically updated. Likewise, when an object moves from one world or room to another, the Container property is automatically updated.

The objects within a Container make up its Contents.  

Example

This example displays, in the Command Window, the contents of the container that the user is currently within.

Dim myContentObject

For Each myContentObject in User.Container.Contents
    Trace myContentObject.Name

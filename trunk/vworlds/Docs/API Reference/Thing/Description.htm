Description

 Property of Thing

A String that characterizes a Thing. Description is a read/write String.  

Syntax

object.Description = stringValue

The Description property syntax has these parts:
Part Type Use Description
object Thing Required An instance of the Thing object.
stringValue String Required An expression that evaluates to a String.

Example

To get the Description: 

   Result = AThing.Description

To set the Description: 

   AThing.Description = "My hat"

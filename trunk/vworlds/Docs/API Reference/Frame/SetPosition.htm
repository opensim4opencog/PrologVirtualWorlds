SetPosition

 Method of Frame

Sets the position of the calling frame relative to the specified reference frame. SetPosition is a member of the Frame object.  

Syntax

object.SetPosition pvwfRef, fPosX, fPosY, fPosZ

The SetPosition method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwfRef Frame Required The frame for which you want to retrieve the position.
fPosX Float Required The x-coordinate offset of the new position of the frame.
fPosY Float Required The y-coordinate offset of the new position of the frame.
fPosZ Float Required The z-coordinate offset of the new position of the frame.

Return Value

This method does not return a value.

Remarks

Sets the Frame parameter to Nothing if you want the other input values in coordinates relative to the world's origin. Set the Frame parameter to the parent of the calling frame to set the other input values relative to the parent frame's origin.

The three coordinate arguments (fPosX, fPosY, and fPosZ) represent distances (or offsets) between the corresponding coordinates of the invoking frame and the reference frame.  

Example

MyFrame.SetPosition RefFrame, 0.0, 2.0, 0.0

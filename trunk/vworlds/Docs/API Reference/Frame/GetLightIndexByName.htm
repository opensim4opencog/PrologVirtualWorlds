GetLightIndexByName

 Method of Frame

Gets the index of a light by name. GetLightIndexByName is a member of the Frame object.  

Syntax

object.GetLightIndexByName(bstrName)

The GetLightIndexByName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
bstrName String Required The name of the light to find.

Return Value

This method returns a Long that corresponds to the index number of a light that has been added to a Frame object.

Example

This example adds two lights to a frame, and then uses the GetLightIndexByName method to retrieve the index number of the second light. The returned index number can then be used as a parameter to invoke other methods on that light, such as the GetLightColor method. In this example, "myFrame" is an existing instance of the Frame object.

Dim Idx
Dim fRed, fGreen, fBlue

myFrame.CreateLight "Light1"
myFrame.CreateLight "Light2"

'Acquire the index number of the second light. 

Idx = MyFrame.GetLightIndexByName("Light2")

'Use the return value as a parameter in another method. 

myFrame.GetLightColor Idx, fRed, fGreen, fBlue

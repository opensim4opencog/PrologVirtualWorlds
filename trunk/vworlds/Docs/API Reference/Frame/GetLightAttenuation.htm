GetLightAttenuation

 Method of Frame

Gets a light's attenuation (constant, linear, and quadratic). GetLightAttenuation is a member of the Frame object.  

Syntax

object.GetLightAttenuation nIndex, fConst, fLinear, fQuadratic

The GetLightAttenuation method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light that is associated with a Frame object.
fConst Variant Required A Variant that will receive the return value for the constant attenuation factor. The default return value is 1.0.
fLinear Variant Required A Variant that will receive the return value for the linear attenuation factor. The default return value is 0.0.
fQuadratic Variant Required A Variant that will receive the return value for the quadratic attenuation factor. The default return value is 0.0.

Return Value

This method does not return a value, but it populates the fConst, fLinear, and fQuadratic parameters with their respective values.

Remarks

Directional, parallel, and ambient lights are not affected by the light attenuation values.

Light attenuation works closely with the range and color values of a light object to define the behavior of light over distance. The range of a light defines the endpoint of the attenuation. The RGB color values combine with the attenuation to produce the color intensity for each vertex within the range.

The three values returned by the GetLightAttenuation method correspond to three types of attenuation: constant, linear, and quadratic. All three values are normalized at 1.0, but each value has a different effect on the behavior of light over distance (See Figure 1).

Figure 1.

The formula for the total attenuation factor is:

TAF = 1 / [fConstant + fLinear * distance + fQuadratic * (distance^2)]

Example

This example creates an instance of the Frame object, adds a light to that frame, and then invokes the GetLightAttenuation method to retrieve the attenuation factors. The example then displays each return value in the Command Window.

Dim myFrame
Set myFrame = World.CreateFrame
myFrame.CreateLight

'Retrieve the attenuation factors. 

myFrame.GetLightAttenuation 0, myConst, myLinear, myQuadratic

'Display the results in the Command Window. 

Trace "The Constant value is: " & myConst
Trace "The Linear value is: " & myLinear
Trace "The Quadratic value is: " & myQuadratic

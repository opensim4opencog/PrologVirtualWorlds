CreateLight

 Method of Frame

Creates and adds a light to a frame. CreateLight is a member of the Frame object.  

Syntax

object.CreateLight [bstrName]

The CreateLight method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
[bstrName] Variant Optional The name of the light.

Return Value

This method does not return a value.

Remarks

The light that is created and added is a white Point light. After you have created the light, you can adjust its properties using the SetLightColor and SetLightType methods.

For more information on the types of Lights available, see the GetLightType topic.

Example

MyFrame.CreateLight "Spotlight1"

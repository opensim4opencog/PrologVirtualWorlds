Copy

 Method of Frame

Duplicate the specified Frame object. Copy is a member of the Frame object.  

Syntax

object.Copy pObjProp

The Copy method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pObjProp Frame Required The destination for the copy of the frame.

Return Value

This method does not return a value.

Remarks

The pObjProp parameter should be set to Nothing when you use this method.  

Example

Dim myFrame
Dim myCopyFrame
Set myFrame = User.Container.GetFrame
myFrame.Copy myCopyFrame

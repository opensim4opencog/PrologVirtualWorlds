GetScene

 Method of Frame

Gets the D3D scene from the frame. GetScene is a member of the Frame object.  

Syntax

object.GetScene()

The GetScene method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.

Return Value

This method returns a Frame.

Remarks

This method returns the top level frame in the hierarchy. A "scene" has no parent frame.

Example

Set Scene = MyFrame.GetScene

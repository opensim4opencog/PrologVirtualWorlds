LookAt

 Method of Frame

Rotates an object to face the given point. LookAt is a member of the Frame object.  

Syntax

object.LookAt fPosX, fPosY, fPosZ

The LookAt method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
fPosX Float Required The x-coordinate of the point in world coordinates.
fPosY Float Required The y-coordinate of the point in world coordinates.
fPosZ Float Required The z-coordinate of the point in world coordinates.

Return Value

This method does not return a value.

Example

MyFrame.LookAt 2.0, 0.0, 1.0

GetLightRange

 Method of Frame

Gets a light's range. GetLightRange is a member of the Frame object.  

Syntax

object.GetLightRange(nIndex)

The GetLightRange method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.

Return Value

This method returns a Float that is the range of the light.

Example

TheRange = MyFrame.GetLightRange (17)

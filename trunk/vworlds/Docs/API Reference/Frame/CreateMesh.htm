CreateMesh

 Method of Frame

Creates and adds an empty mesh to the frame. CreateMesh is a member of the Frame object.  

Syntax

object.CreateMesh 

The CreateMesh method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.

Return Value

This method does not return a value.

Example

This example creates an instance of the Frame object, associates a new mesh with that frame using the CreateMesh property, and defines a face for that mesh.

Dim myFrame
Set myFrame = World.CreateFrame
myFrame.CreateMesh
myFrame.CreateMeshFace 0, Array(0, 0, 0, 1, 1, 1, 2, 2, 2)

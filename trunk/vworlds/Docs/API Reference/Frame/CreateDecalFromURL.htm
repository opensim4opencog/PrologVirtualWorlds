CreateDecalFromURL

 Method of Frame

Adds a decal to a frame from a file at a specified URL. CreateDecalFromURL is a member of the Frame object.

Syntax

object.CreateDecalFromURL bstrTexture

The CreateDecalFromURL method syntax has these parts:
Parameters Type Use Description
object Frame Required An object reference that accesses an object with a Frame object.
bstrTexture String Required A valid URL for the decal. The path for the URL can be relative or explicit.

Return Value

This method does not return a value. 

Remarks

Although decals are still functional for backward compatibility in Microsoft Virtual Worlds 1.5, the Label object is the preferred method of creating text that you can display within your world.

The file type must be a bitmap that is recognized by Virtual Worlds. This method displays an image file as a decal in a virtual world. You can also use the CreateDecalFromText method to create a decal from text.

Example

This example creates a decal and displays it in a virtual world.

Sub CreateKioskDecal()
Dim myFrame
Dim myDecal
Dim myFI

myFI = _
  "file://C:/Program Files/Microsoft Virtual Worlds/Local Content/SDK/Library/models/other/Kiosk.x"

'Set a reference to the frame where you want to place the decal. 

Set myFrame = User.Container.GetFrame
myDecal = myFI

'Create the decal. 

myFrame.CreateDecalFromURL myDecal

End Sub

GetMeshFaceUV

 Method of Frame

Gets the texture parameters U and V for a vertex of a mesh. GetMeshFaceUV is a member of the Frame object.  

Syntax

object.GetMeshFaceUV nMeshIndex, nFaceIndex, index, fU, fV

The GetMeshFaceUV method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nFaceIndex Integer Required The index of the face.
index Integer Required The index of the vertex.
fU Variant Required The U value at the vertex.
fV Variant Required The V value at the vertex.

Return Value

This method does not return a value.

Remarks

The texture parameters fU and fV are used to establish a mapping between pixels in a texture map and points (vertices) on a surface. These parameters are normalized (0.0 to 1.0) floating point values. For example, consider a 256 X 256 texture map that has been wrapped around an object. A vertex at the point (128, 32) on the 256 X 256 texture map would be paired with a fU value of 0.5 and a fV value of 0.125.

The fourth and fifth parameters of GetMeshFaceUV are output parameters. After this method executes, these parameters will be overwritten with new values. Specifically, after a call to GetMeshFaceUV, the fourth and fifth parameters will be set to the fU and fV values for the specified mesh and vertex.  

Example

MyFrame.GetMeshFaceUV 4, 12, 1, UParam, VParam

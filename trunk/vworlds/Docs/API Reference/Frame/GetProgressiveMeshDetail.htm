GetProgressiveMeshDetail

 Method of Frame

Gets the level of detail for a progressive mesh in the current frame. GetProgressiveMeshDetail is a member of the Frame object.  

Syntax

object.GetProgressiveMeshDetail(nMeshIndex)

The GetProgressiveMeshDetail method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.

Return Value

This method returns a Float.

Remarks

A mesh is a visual object that is made up of a set of polygonal faces. A mesh defines a set of vertices and a set of faces. A progressive mesh is a mesh that is stored as a base mesh (a coarse version) and a set of records that are used to increasingly refine the mesh. This allows you to set the level of detail rendered for a mesh, and also allows progressive download from remote sources.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.  

Example

FrameDetail = MyFrame.GetProgressiveMeshDetail (2)

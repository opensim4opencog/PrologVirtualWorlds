GetProgressiveMeshQuality

 Method of Frame

Gets the quality of rendering for a mesh in the current frame. GetProgressiveMeshQuality is a member of the Frame object.  

Syntax

object.GetProgressiveMeshQuality(nMeshIndex)

The GetProgressiveMeshQuality method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.

Return Value

This method returns an Integer. The return values are defined in the following table.
Value Meaning Comment
0 Wireframe No shading
1 Solid shading Not affected by lights
2 Solid shading Affected by lights; each triangle will be a solid color
3 Gouraud shading Affected by lights; smoothly shaded
4 Phong shading Like Gouraud shading, but better quality

Remarks

nMeshIndex is set to the index of a particular mesh that has been added to a Frame.  

Example

MeshQuality = MyFrame.GetProgressiveMeshQuality 8

GetBoundingBox

 Method of Frame

Gets the bounding box of a frame and its children. GetBoundingBox is a member of the Frame object.  

Syntax

object.GetBoundingBox fMinX, fMinY, fMinZ, fMaxX, fMaxY, fMaxZ

The AutoGenerateFromBox method syntax has these parts:
Parts Type Use Description
object Frame Required An object reference that accesses a Frame object.
fMinX Float Required Variable that is populated by the GetBoundingBox method that corresponds to the closer x-coordinate of a bounding box.
fMinY Float Required Variable that is populated by the GetBoundingBox method that corresponds to the closer y-coordinate of a bounding box.
fMinZ Float Required Variable that is populated by the GetBoundingBox method that corresponds to the closer z-coordinate of a bounding box.
fMaxX Float Required Variable that is populated by the GetBoundingBox method that corresponds to the farther x-coordinate of a bounding box.
fMaxY Float Required Variable that is populated by the GetBoundingBox method that corresponds to the farther y-coordinate of a bounding box.
fMaxZ Float Required Variable that is populated by the GetBoundingBox method that corresponds to the farther z-coordinate of a bounding box.

Return Value

This method does not return a value.

Remarks

A bounding box is the smallest possible bounding volume cube that all of the polygons of an object will fit within. For example, imagine a glass cube that surrounds an entire building. The cube cannot touch the building itself, but it must be as close as possible to all sides of the building. This cube illustrates the bounding box of the building.

In the case of a Frame object, a frame can also have child objects that are associated with it. The parent frame establishes the origin of the bounding box. So, the bounding box of a Frame object includes the total area that the parent and any children occupy. Continuing with the illustration above, if the building were a Frame object, the bounding box of the building would also include objects that are not part of the building itself, but are associated with it (like a parking lot or sidewalk). The cube encloses the entire area including the child objects of that building.

The six parameters can be thought of as two sets of Vector coordinates that establish the size of the bounding box in relation to a center point. The parameters are populated by the GetBoundingBoxmethod, and are expressed in local coordinates (relative to the object). The coordinates (fMinX, fMinY, fMinZ) and (fMaxX, fMaxY, fMaxZ) are vertices at opposite corners of the bounding box. The first of these points is the one closer to the origin; the second is a point that is farther away from the origin.

Note that all parameters of this method are output parameters. After a call to this method, each of the six parameters will be overwritten with a new value.  

Example

MyFrame.GetBoundingBox fMinX, fMinY, fMinZ, fMaxX, fMaxY, fMaxZ

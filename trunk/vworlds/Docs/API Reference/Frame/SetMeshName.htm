SetMeshName

 Method of Frame

Sets the name of a mesh. SetMeshName is a member of the Frame object.  

Syntax

object.SetMeshName nMeshIndex, bstrMeshName

The SetMeshName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
bstrMeshName String Required The name of the mesh.

Return Value

This method does not return a value.

Remarks

If you don't set a name for a mesh, you must retrieve the mesh using its index number. When you add a name to the mesh, you do not have to remember the index number for each mesh; you can retrieve the index number of the mesh by using the GetMeshIndexByName method.  

Example

myFrame.SetMeshName 23, "BackMesh"

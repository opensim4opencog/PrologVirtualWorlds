WrapMesh

 Method of Frame

Generates mapping coordinates to wrap a mesh. WrapMesh is a member of the Frame object.  

Syntax

object.WrapMesh nMeshIndex, type, scaleu, scalev

The WrapMesh method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
type Integer Required The type of wrap.
scaleu Float Required The scale in the Up direction.
scalev Float Required The scale in the direction of the mesh orientation.

Return Value

This method does not return a value.

Remarks

Use this method to add automatic texture mapping coordinates in a general way to meshes that do not already have texture coordinates.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

Possible values for type (and their corresponding qualities) are shown in the following table. Of the types shown, Flat is the most common type of wrap.

Wrap type Description
0 Flat
1 Cylinder
2 Sphere
3 Chrome

The scaleu parameter contains the scale in the Up (property of Thing) direction for the frame.  

Example

MyFrame.WrapMesh 4, 0, 1.0, 0.5

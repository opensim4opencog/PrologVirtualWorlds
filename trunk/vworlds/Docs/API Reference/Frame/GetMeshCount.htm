GetMeshCount

 Method of Frame

Returns the number of meshes in the current frame. GetMeshCount is a member of the Frame object.  

Syntax

object.GetMeshCount()

The GetMeshCount method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.

Return Value

This method returns a Variant.

Remarks

In HTML, you can use a frameset to organize multiple frames and nested framesets. A Frame object is an individual frame within a frameset. A mesh is a visual object that is made up of a set of polygonal faces; it defines a set of vertices and a set of faces. You can use the methods for progressive meshes to set the level of detail rendered for a mesh and to allow progressive downloads from remote sources. A progressive mesh is a mesh that is stored as a base mesh (a coarse version) and a set of records that are used to increasingly refine the mesh.

Some of the methods you can use to refine the details of a mesh are:

  • SetMeshColor
  • SetMeshTextureFromURL

    Example

    This example assigns the number of meshes to "myCount".

    Dim myFrame
Dim myCount

Set myFrame = User.Container.GetFrame
myCount = myFrame.GetMeshCount

SetLightRange

 Method of Frame

Sets a light's range. SetLightRange is a member of the Frame object.  

Syntax

object.SetLightRange nIndex, fRange

The SetLightRange method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
fRange Float Required The range of the light.

Return Value

This method does not return a value.

Remarks

A light's range is the distance (in meters) at which an object can be affected by the light.  

Example

MyFrame.SetLightRange 99, 50.0

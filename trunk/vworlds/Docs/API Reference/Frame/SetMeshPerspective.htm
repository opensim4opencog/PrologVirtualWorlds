SetMeshPerspective

 Method of Frame

Sets the perspective correction for textures on a mesh. SetMeshPerspective is a member of the Frame object.  

Syntax

object.SetMeshPerspective nMeshIndex, vbPersp

The SetMeshPerspective method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
vbPersp Boolean Required A flag that determines whether or not perspective correction is on.

Return Value

This method does not return a value.

Remarks

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

Setting vbPersp to True will enable perspective correction; setting it to False will disable perspective correction. However, disabling perspective correction speeds rendering at the cost causing some textures to render incorrectly. 

Example

MyFrame.SetMeshPerspective 4, True

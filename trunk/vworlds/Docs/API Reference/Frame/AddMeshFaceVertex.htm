AddMeshFaceVertex

 Method of Frame

Adds a vertex to the given face on the given mesh. AddMeshFaceVertex is a member of the Frame object.  

Syntax

object.AddMeshFaceVertex nMeshIndex, nFaceIndex, fX, fY, fZ

The AddMeshFaceVertex method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to which to add the vertex.
nFaceIndex Integer Required The index of the face to which to add the vertex.
fX Float Required The x-coordinate of the new vertex.
fY Float Required The y-coordinate of the new vertex.
fZ Float Required The z-coordinate of the new vertex.

Return Value

This method does not return a value.

Remarks

To to add a vertex using this method, the mesh and face must already exist. Meshes and faces can be created by using CreateMesh and CreateMeshFace, respectively.

nMeshIndex is generally set to the index of a particular mesh.  

Example

MyFrame.AddMeshFaceVertex 23, 9, 11.2, 12.9, 8.3

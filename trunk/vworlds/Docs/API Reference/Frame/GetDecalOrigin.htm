GetDecalOrigin

 Method of Frame

Returns the origin (central point) of the decal. GetDecalOrigin is a member of the Frame object.

Syntax

object.GetDecalOrigin nMeshIndex, lx, ly

The GetDecalOrigin method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses an object with a Frame object.
nMeshIndex Long Required The index number of the mesh.
lx Variant Required The value for the x-coordinate can be float or integer.
ly Variant Required The value for the y-coordinate can be float or integer.

Return Value

This method does not return a value. 

Remarks

A Frame contains a list of visuals (visual objects) and a list of lights. A visual can be a mesh, a visual object that is made up of a set of polygonal faces. The text that appears above an avatar is an example of a mesh. Another kind of visual is a decal, a texture that can be rendered directly. The name in text above an avatar's head is an example of a decal. A third kind of visual is a progressive mesh, a mesh that is stored as a base mesh (a coarse version) and a set of records that are used to increasingly refine the mesh. A visual can be referenced by its index in the visual list.

The nMeshIndex parameter is generally set to the index of a particular mesh, but you can set it to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame. A value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The default origin of a decal is its center, expressed in the coordinates of the pixel space for the generated image. For example, if a decal is a 256-by-256 texture, its origin is at (128, 128).

The lx and ly parameters will return the x- and y-coordinates for the origin of the specified decal. After this method executes, these parameters will be overwritten with the current values for the x- and y-coordinates.

You can set the origin by using the SetDecalOrigin method. 

Example

This example changes the size and position of a decal.

Sub ChangeDecalSizeAndPosition()
Dim myFrame
Dim myMeshIndex
Dim myHeight
Dim myWidth
Dim myScale
Dim myX
Dim myY

'Set a reference to the frame where you want to place the decal. 

Set myFrame = User.Container.GetFrame

'Create a decal. 

myFrame.CreateDecalFromText "This is the new Kiosk", "Kiosk"

'Get the values for the variables "myHeight", "myWidth", and "myScale". 

myFrame.GetDecalSize 0, myHeight, myWidth, myScale

'Change the values. 

myHeight = myHeight + .5
myWidth = myWidth - .15

'Set the decal to the new size. 

myFrame.SetDecalSize 0, myHeight, myWidth, myScale

'Get the origin of the decal. 

myFrame.GetDecalOrigin 0, myX, myY

End Sub

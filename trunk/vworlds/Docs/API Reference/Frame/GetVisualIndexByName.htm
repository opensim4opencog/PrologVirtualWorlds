GetVisualIndexByName

 Method of Frame

Gets the index of a visual object contained in a Frame. GetVisualIndexByName is a member of the Frame object.  

Syntax

object.GetVisualIndexByName (bstrName)

The GetVisualIndexByName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
bstrName String Required The name of the visual object.

Return Value

This method returns a Long that corresponds to the index of a visual object within a Frame.

Example

Index = MyFrame.GetVisualIndexByName ("Light 2")

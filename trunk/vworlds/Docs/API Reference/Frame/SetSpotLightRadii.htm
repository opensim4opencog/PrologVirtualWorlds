SetSpotLightRadii

 Method of Frame

Sets spotlight parameters. SetSpotLightRadii is a member of the Frame object.  

Syntax

object.SetSpotLightRadii nIndex, fUmbra, fPenumbra

The SetSpotLightRadii method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
fUmbra Float Required The angle of the umbra cone in radians.
fPenumbra Float Required The angle of the penumbra cone in radians.

Return Value

This method does not return a value.

Remarks

A spotlight emits a cone of light, illuminating only those objects that lie within the cone. The cone produces light of two degrees of intensity; a central brightly-lit section (the fUmbra) acts as a point source, and an outer dimly lit section (the fPenumbra) that merges with the surrounding deep shadow.

The fPenumbra cone angle must be greater than or equal to the fUmbra cone angle. If the fPenumbra cone angle is less than the fUmbra cone angle, the fUmbra cone angle will be reset to be equal to the fPenumbra cone angle. Furthermore, if the fUmbra cone angle is greater than the fPenumbra cone angle, the fPenumbra cone angle will be reset to be equal to the fUmbra cone angle. Both angle measures are in radians.  

Example

MyFrame.SetSpotLightRadii 99, .5, .4

GetMeshFaceVertexColor

 Method of Frame

Gets the color of a vertex for a particular mesh and face. GetMeshFaceVertexColor is a member of the Frame object.  

Syntax

object.GetMeshFaceVertexColor nMeshIndex, nFaceIndex, index, fRed, fGreen, fBlue

The GetMeshFaceVertexColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nFaceIndex Integer Required The index of the face to use.
index Integer Required The index of the vertex to use.
fRed Variant Required The fractional amount of red.
fGreen Variant Required The fractional amount of green.
fBlue Variant Required The fractional amount of blue.

Return Value

This method does not return a value.

Remarks

The fourth, fifth, and sixth parameters of GetMeshFaceVertexColor are output parameters. After this method is executed, these parameters will be overwritten with new values. After a call to GetMeshFaceVertexColor, the fourth, fifth, and sixth parameters will be set to the red, green, and blue color values for the specified mesh and face.

Each color value is a decimal fraction between 0.0 and 1.0.

When you set a color value with SetMeshFaceVertexColor and retrieve it with GetMeshFaceVertexColor, there may be some loss of accuracy. For example, a value of 0.5 may be translated to a number like 0.498999 by the color algorithms.

Vertex colors can be seen only when the render quality is Gouraud or Phong.  

Example

MyFrame.GetMeshFaceVertexColor 23, 17, 5, Red, Green, Blue

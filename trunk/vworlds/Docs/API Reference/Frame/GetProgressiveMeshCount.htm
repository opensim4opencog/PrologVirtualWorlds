GetProgressiveMeshCount

 Method of Frame

Returns the number of progressive meshes contained by a frame. GetProgressiveMeshCount is a member of the Frame object.  

Syntax

object.GetProgressiveMeshCount()

The GetProgressiveMeshCount method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.

Return Value

This method returns a Variant that represents the number of progressive meshes contained by a frame.

Remarks

A Frame contains a list of visuals (visual objects) and a list of lights. A mesh is a visual object that is made up of a set of polygonal faces; it defines a set of vertices and a set of faces. A progressive mesh is a mesh that is stored as a base mesh (a coarse version) and a set of records that are used to increasingly refine the mesh. This allows you to set the level of detail rendered for a mesh if the model that you are using contains progressive meshes.  

Example

This example assigns the number of progressive meshes to "myMeshCount".

Dim myFrame
Dim myMeshCount

Set myFrame = User.Container.GetFrame
myMeshCount = myFrame.GetProgressiveMeshCount

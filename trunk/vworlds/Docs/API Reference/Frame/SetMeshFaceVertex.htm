SetMeshFaceVertex

 Method of Frame

Sets the vertex coordinates for a particular vertex in a given face of a given mesh. SetMeshFaceVertex is a member of the Frame object.  

Syntax

object.SetMeshFaceVertex nMeshIndex, nFaceIndex, index, fX, fY, fZ

The SetMeshFaceVertex method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nFaceIndex Integer Required The index of the face to use.
index Integer Required The index of the vertex to use.
fX Float Required The x-coordinate of the position.
fY Float Required The y-coordinate of the position.
fZ Float Required The z-coordinate of the position.

Return Value

This method does not return a value.

Remarks

The vertex, face, and mesh must already exist when this method is used. To create a vertex, face, or mesh, use AddMeshFaceVertex, CreateMeshFace, or CreateMesh, respectively.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The position parameters fX, fY, and fZ are expressed in local coordinates.  

Example

MyFrame.SetMeshFaceVertex 4, 12, 1, 3.5, 1.0, 0.0

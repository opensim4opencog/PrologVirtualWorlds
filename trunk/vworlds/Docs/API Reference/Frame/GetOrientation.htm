GetOrientation

 Method of Frame

Gets the orientation of the frame relative to the reference position. GetOrientation is a member of the Frame object.  

Syntax

object.GetOrientation pvwfRef, pfDirX, pfDirY, pfDirZ, pfUpX, pfUpY, pfUpZ

The GetOrientation method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object. Usually a parent frame object.
pvwfRef Frame Required The frame for which you want to retrieve the orientation.
pfDirX Variant Required The x-component of the Direction vector.
pfDirY Variant Required The y-component of the Direction vector.
pfDirZ Variant Required The z-component of the Direction vector.
pfUpX Variant Required The x-component of the Up vector.
pfUpY Variant Required The y-component of the Up vector.
pfUpZ Variant Required The z-component of the Up vector.

Return Value

This method does not return a value.

Remarks

The frame's orientation is determined by two vectors: the Direction vector and the Up vector. The Direction vector points away from the front surface of the frame and the Up vector points in the direction of the top of the frame. Both of these are properties of the Thing exemplar.

Set the Frame parameter to Nothing if you want the output values in coordinates relative to the world's origin. Set the Frame parameter to the parent of the calling frame to produce output values relative to the parent frame's origin.

The last six parmeters of GetOrientation are output parameters. After this method executes, these parameters will be overwritten with new values.  

Example

MyFrame.GetOrientation RefFrame, xDir, yDir, zDir, xUp, yUp, zUp

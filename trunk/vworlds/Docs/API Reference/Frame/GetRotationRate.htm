GetRotationRate

 Method of Frame

Gets the rotation rate of the frame relative to the reference frame. GetRotationRate is a member of the Frame object.  

Syntax

object.GetRotationRate pvwfRef, prvX, prvY, prvZ, prvTheta

The GetRotationRate method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwfRef Frame Required The reference frame.
prvX Variant Required The x-component of the rotation axis vector.
prvY Variant Required The y-component of the rotation axis vector.
prvZ Variant Required The z-component of the rotation axis vector.
prvTheta Variant Required The angular rotation rate.

Return Value

This method does not return a value.

Remarks

The rotation rate, prvTheta, expresses the fraction of a full rotation that is rendered in each successive frame. Reasonable values are typically between 0.01 and 0.5; these numbers represent 100 frames per complete rotation and two frames per complete rotation, respectively.  

Example

MyFrame.GetRotationRate RefFrame, xRot, yRot, zRot, RotRate

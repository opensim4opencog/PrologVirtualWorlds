GetPosition

 Method of Frame

Gets the position of the frame relative to the reference frame. GetPosition is a member of the Frame object.  

Syntax

object.GetPosition pvwfRef, pfPosX, pfPosY, pfPosZ

The GetPosition method syntax has these parts:
Part Type Use Description
object Frame Required An object reference to a Frame object.
pvwfRef Frame Required The reference frame. You want to get the position relative to this frame. 
pfPosX Variant Required Receives the x-coordinate.
pfPosY Variant Required Receives the y-coordinate.
pfPosZ Variant Required Receives the z-coordinate.

Return Value

This method does not return a value.

Remarks

The three coordinates represent distances (or offsets) between the corresponding coordinates of the invoking frame and the reference frame.

Set the Frame parameter to Nothing if you want the output values in coordinates relative to the world's origin. Set the Frame parameter to the parent of the calling frame to produce output values relative to the parent frame's origin.

The second, third, and fourth parameters of GetPosition are output parameters. After this method executes, these parameters will be overwritten with new values.  

Example

myFrame.GetPosition RefFrame, x, y, z

SetLightName

 Method of Frame

Sets the name of a light. SetLightName is a member of the Frame object.  

Syntax

object.SetLightName nIndex, bstrName

The SetLightName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
bstrName String Required The name of the light.

Return Value

This method does not return a value.

Example

MyFrame.SetLightName 0, "Main light"

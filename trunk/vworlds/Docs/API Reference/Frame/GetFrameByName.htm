GetFrameByName

 Method of Frame

Returns the Frame object specified by the bstrName parameter.  GetFrameByName is a member of the Frame object.  

Syntax

object.GetFrameByName(bstrName)

The GetFrameByName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
bstrName String Required The name of the frame to retrieve.

Return Value

This method returns the Frame corresponding to the name used in the bstrName parameter. Microsoft Virtual Worlds searches the hierarchy starting from the current frame.

Remarks

In HTML, you can use a frameset to organize multiple frames and nested framesets. A Frame object is an individual frame within a frameset. If you know the name of the frame, you can use the GetFrameByName method to return the Frame object. 

A container in Virtual Worlds is an object, such as a room, that can hold other objects, such as a frame. If you only know in which container a frame resides, you can use the GetFrame method to return a reference to the specified frame. To return the name of a frame for later use with the GetFrameByName method, use the following code. 

myFrame.GetName

Example

This example returns the specified frame by using the value of the bstrName parameter to find the correct frame.

Sub GetDecalsFrame()
Dim myFrame
Dim myDecalFrame

Set myFrame = User.Container.GetFrame
Set myDecalFrame = myFrame.GetFrameByName("Decals")

End Sub

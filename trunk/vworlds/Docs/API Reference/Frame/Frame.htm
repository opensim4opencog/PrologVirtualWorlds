Frame

Object

A frame is a section of the screen or visible user interface that you can use as a container to hold other objects. In HTML, you can use a frameset to organize multiple frames and nested framesets. A Frame object is an individual frame within a frameset. You can use the GetFrame method to retrieve an existing frame, the CreateFrame method to create a new frame, and Copy method to copy an existing frame to another frame. Each thing in a room has a frame organized under the room's name. 

Using the Object

A Frame contains:

  • A list of visuals (visual objects) and a list of lights. A visual can be a mesh, a visual object that is made up of a set of polygonal faces. A sprite is an example of a mesh. Another kind of visual is a decal, a texture that can be rendered directly. The name in text above an avatar's head is an example of a decal. A third kind of visual is a progressive mesh , a mesh that is stored as a base mesh (a coarse version) and a set of records that are used to increasingly refine the mesh. You can reference a visual by its index in the visual list, or by name. Use the GetVisualCount method to determine the number of visuals in a frame. To determine the index of a named visual, use the GetVisualIndexByName method.
  • A transformation, made up of the frame's position, orientation, and scale. You can set and retrieve these attributes by using the transformation methods GetPosition, SetPosition, GetOrientation, SetOrientation, GetScale, and SetScale. In addition, the methods SetVelocity, GetVelocity, SetRotationRate, and GetRotationRate modify the frame's transformation each time the scene is rendered (approximately 15 times per second). For thing's in a room, these values are set from the objects Position, Direction, Up, and Scale properties.
  • A record of the frame hierarchy. At the top level in this hierarchy is the scene or root frame. Beneath this top level there may be child frames , which, in turn, may be parents to other child frames. The AddChild, DeleteChild, IsChildOf, GetParent, GetChildren, and GetScene methods all relate to this hierarchy. Each frame in the hierarchy has its own local transformation, which is combined with those of the frames above it in the hierarchy to produce the world-coordinate transformation.

To create an empty mesh, use the CreateMesh method. You can name the mesh after you create it by using the SetMeshName method. Each frame can have one or more meshes associated with it.

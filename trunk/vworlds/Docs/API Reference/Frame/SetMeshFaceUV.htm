SetMeshFaceUV

 Method of Frame

Sets the texture parameters U and V for a vertex of a mesh. SetMeshFaceUV is a member of the Frame object.  

Syntax

object.SetMeshFaceUV nMeshIndex, nFaceIndex, index, fU, fV

The SetMeshFaceUV method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index number for the mesh that you want to use for the frame.
nFaceIndex Integer Required The index number for the face that you want to use for the frame.
index Integer Required The index number for the vertex that you want to use for the frame.
fU Float Required The U value at the vertex
fV Float Required The V value at the vertex

Return Value

This method does not return a value.

Remarks

The values for the fU and fV  parameters are used to establish mapping between pixels (in a texture map) and points or vertices (on a surface). These parameters are floating point values in the normalized range (0.0 to 1.0). For example, consider a 256 X 256 texture map that has been wrapped around an object. A vertex at the point (128, 32) on the 256 X 256 texture map would be paired with a U value of 0.5 and a V value of 0.125.

The MeshIndex parameter is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

Example

This example sets the mesh face for the specified object.

Sub PolishAFrame()
Dim myFrameToPolish

'Get the frame. 

Set myFrameToPolish = User.Container.GetFrame

'Set the mesh. 

myFrameToPolish.SetMeshFaceUV 4, 12, 1, 0.5, 0.125
End Sub

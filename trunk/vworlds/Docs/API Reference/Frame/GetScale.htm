GetScale

 Method of Frame

Gets the scale of the frame. GetScale is a member of the Frame object.  

Syntax

object.GetScale pvwfRef, pfScaleX, pfScaleY, pfScaleZ

The GetScale method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwfRef Frame Required The frame for which you want to retrieve the scale.
pfScaleX Variant Required The x-component of the scale factor.
pfScaleY Variant Required The y-component of the scale factor.
pfScaleZ Variant Required The z-component of the scale factor.

Return Value

This method does not return a value.

Remarks

Set the Frame parameter to Nothing if you want the output values in coordinates relative to the world's origin. Set the Frame parameter to the parent of the calling frame to produce output values relative to the parent frame's origin.

The second, third, and fourth parameters of GetScale are output parameters. After this method executes, these parameters will be overwritten with new values. Specifically, after this method has been called, the second, third, and fourth parameters will be set to the components of the scale for the invoking frame (on the x, y, and z parameters).  

Example

MyFrame.GetScale ParentFrame, xScale, yScale, zScale

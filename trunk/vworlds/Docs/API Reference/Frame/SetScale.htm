SetScale

 Method of Frame

Sets the scale of the frame. SetScale is a member of the Frame object.  

Syntax

object.SetScale fX, fY

The SetScale method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
fX Float Required The x-component of the scale factor.
fY Float Required The y-component of the scale factor.

Return Value

This method does not return a value.

Remarks

Set the Frame parameter to Nothing if you want the other input values in coordinates relative to the world's origin. Set the Frame parameter to the parent of the calling frame to set the other input values relative to the parent frame's origin.

Important:   Do not set the scale to zero in any coordinate or you may get undesireable results.  

Example

MyFrame.SetScale ParentFrame, 0.5, 0.5, 0.5

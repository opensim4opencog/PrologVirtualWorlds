GetProgressiveMeshFaceDetailRange

 Method of Frame

Gets the range of possible values for the number of faces in a mesh. GetProgressiveMeshFaceDetailRange is a member of the Frame object.  

Syntax

object.GetProgressiveMeshFaceDetailRange nMeshIndex, min, max

The GetProgressiveMeshFaceDetailRange method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
min Variant Required The minimum possible value for the number of faces in the mesh.
max Variant Required The maximum possible value for the number of faces in the mesh.

Return Value

This method does not return a value.

Remarks

Gets the minimum and maximum possible values for the number of faces in a mesh with Min and Max. A number within this range must be used in any call to SetProgressiveMeshFaceDetail.

The second and third parameters of GetProgressiveMeshFaceDetailRange are output parameters. After this method executes, these parameters will be overwritten with new values.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.  

Example

MyFrame.GetProgressiveMeshFaceDetailRange 15, Min, Max

GetLightColor

 Method of Frame

Get a light's color (expressed as a mixture of red, green, and blue). GetLightColor is a member of the Frame object.  

Syntax

object.GetLightColor nIndex, fRed, fGreen, fBlue

The GetLightColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
fRed Variant Required A variant that is populated by the GetLightColor method, which evaluates to the fractional amount of red.
fGreen Variant Required A variant that is populated by the GetLightColor method, which evaluates to the fractional amount of green.
fBlue Variant Required A variant that is populated by the GetLightColor method, which evaluates to the fractional amount of blue.

Return Value

This method does not return a value.

Remarks

Each of the three colors is a number between 0.0 and 1.0, representing the decimal fractional amount of red, green, and blue, respectively.

The second, third, and fourth parameters of GetLightColor are output parameters. After this method executes, these parameters will be overwritten with new values.  

Example

MyFrame.GetLightColor 17, Red, Green, Blue

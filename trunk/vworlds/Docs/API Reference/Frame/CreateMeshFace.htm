CreateMeshFace

 Method of Frame

Adds a face or faces to a mesh. CreateMeshFace is a member of the Frame object.  

Syntax

object.CreateMeshFace nMeshIndex, varArgs

The CreateMeshFace method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index for the mesh to use. The index is zero-based.
varArgs Variant Required The array of faces to add.

Return Value

This method does not return a value.

Remarks

VarArray can be an array of at least three elements, each of which is a vertex represented by a Vector. It can also be an array of at least nine elements, each of which is a vertex coordinate represented by a Float.

nMeshIndex is set to the index of a particular mesh. For example, if you create a new frame and mesh, the mesh will have an index of 0, since it is the first mesh listed for that frame.

Example

This example creates an instance of the Frame object, associates a new mesh with that frame, and defines a face for that mesh. In this example, the varArgs parameter is an array of nine Floats. However, you could also substitute an array of three Vectors for this parameter. 

Dim myFrame
Set myFrame = World.CreateFrame
myFrame.CreateMesh
myFrame.CreateMeshFace 0, Array(0, 0, 0, 1, 1, 1, 2, 2, 2)

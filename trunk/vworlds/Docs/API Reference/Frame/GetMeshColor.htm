GetMeshColor

 Method of Frame

Gets the color of a mesh in the current frame. GetMeshColor is a member of the Frame object.  

Syntax

object.GetMeshColor nMeshIndex, fRed, fGreen, fBlue

The GetMeshColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
fRed Variant Required The fractional amount of red.
fGreen Variant Required The fractional amount of green.
fBlue Variant Required The fractional amount of blue.

Return Value

This method does not return a value.

Remarks

The second, third, and fourth parameters of GetMeshColor are output parameters. After this method executes, these parameters will be overwritten with new values. After a call to GetMeshColor, the second, third, and fourth parameters will be set to the Red, Green, and Blue color values for a specified mesh.

Each color value is a decimal fraction between 0.0 and 1.0.

When you set a color value with SetMeshColor and retrieve it with GetMeshColor, there may be some loss of accuracy. For example, a value of 0.5 may be translated to a number such as 0.498999 by the color algorithms.

MeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes below the current frame in the hierarchy.  

Example

MyFrame.GetMeshColor 5, Red, Green, Blue

SetMeshFaceVertexColor

 Method of Frame

Sets the color of a vertex in a mesh face. SetMeshFaceVertexColor is a member of the Frame object.  

Syntax

object.SetMeshFaceVertexColor nMeshIndex, nFaceIndex, index, fRed, fGreen, fBlue

The SetMeshFaceVertexColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nFaceIndex Integer Required The index of the face within the mesh.
index Integer Required The index of the vertex in the mesh face.
fRed Float Required The fractional amount of red.
fGreen Float Required The fractional amount of green.
fBlue Float Required The fractional amount of blue.

Return Value

This method does not return a value.

Remarks

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The values for fRed, fGreen, and fBlue are Floats in the normalized range 0.0 to 1.0.

Vertex colors can be seen only when the render quality is Gouraud or Phong.  

Example

The following example will set a vertex to pure blue (0% red, 0% green, and 100% blue). The vertex is in mesh 0, face 2, at index 11.

MyFrame.SetMeshFaceVertexColor 0, 2, 11, 0.0, 0.0, 1.0

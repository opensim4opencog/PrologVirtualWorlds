DeleteLight

 Method of Frame

Deletes a light from a frame. DeleteLight is a member of the Frame object.  

Syntax

object.DeleteLight nIndex

The DeleteLight method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light to be deleted.

Return Value

This method does not return a value.

Example

MyFrame.DeleteLight 23

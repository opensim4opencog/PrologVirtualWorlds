GetProgressiveMeshFaceDetail

 Method of Frame

Gets the number of faces in a progressive mesh. GetProgressiveMeshFaceDetail is a member of the Frame object.  

Syntax

object.GetProgressiveMeshFaceDetail(nMeshIndex)

The GetProgressiveMeshFaceDetail method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.

Return Value

This method returns a Long.

Remarks

This method is used for progressive meshes only. To find the number of faces in a regular mesh, use the GetMeshFaceCount method of Frame.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.  

Example

Result = MyFrame.GetProgressiveMeshFaceDetail (3)

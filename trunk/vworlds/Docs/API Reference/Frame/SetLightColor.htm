SetLightColor

 Method of Frame

Sets a light's color by altering its mix of red, green, and blue. SetLightColor is a member of the Frame object.  

Syntax

object.SetLightColor nIndex, fRed, fGreen, fBlue

The SetLightColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
fRed Float Required The fractional amount of red.
fGreen Float Required The fractional amount of green.
fBlue Float Required The fractional amount of blue.

Return Value

This method does not return a value.

Remarks

Each color is a real number between 0.0 and 1.0.  

Example

MyFrame.SetLightColor 5, .25, .5, .8

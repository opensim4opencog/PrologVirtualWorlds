AddChild

 Method of Frame

Adds a child frame to the specified frame. AddChild is a member of the Frame object.  

Syntax

object.AddChild pChildFrame

The AddChild method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pChildFrame Frame Required The frame to be added as a child to the specified frame.

Return Value

This method does not return a value.

Remarks

Frames can exist on multiple levels in Microsoft Virtual Worlds, just as they can on a Web page. To get a reference to a frame within a virtual world use the GetFrame method. The following statements retrieve the frame for the user's container. 

Dim myFrame
Set myFrame = User.Container.GetFrame

To add a child frame to the "myFrame" object, you must first create a child frame using the CreateFrame method. 

Dim myNewFrame
Set myNewFrame = CreateFrame

At this point, the new frame does not have a reference point in the world. You use the AddChild method to add the new frame to an existing one. The object specified is the frame that will be the parent object of the new frame. 

myFrame.AddChild myNewFrame

The child displays relative to its parent.  

Example

This example adds a decal to a child frame.

Sub AddDecalToFrame()
Dim myDecal
Dim myFrame
Dim myDecalFrame

Set myFrame = User.Container.GetFrame
Set myDecalFrame = CreateFrame

myFrame.AddChild myDecalFrame

Set myDecal = _
  myDecalFrame.CreateDecalFromText "CDs and DVDs for sale!", "myDecalFrame"

End Sub

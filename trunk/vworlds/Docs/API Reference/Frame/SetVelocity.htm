SetVelocity

 Method of Frame

Sets the velocity of the calling frame relative to the reference frame. SetVelocity is a member of the Frame object.  

Syntax

object.SetVelocity pvwfRef, vX, vY, vZ

The SetVelocity method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwfRef Frame Required The frame for which you want to retrieve the velocity.
vX Float Required The x-component of the relative velocity.
vY Float Required The y-component of the relative velocity.
vZ Float Required The z-component of the relative velocity.

Return Value

This method does not return a value.

Remarks

The parameters determine the amount of movement per frame for the specified object; dependent on the speed of the graphics card in the client's computer.  

Example

myFrame.SetVelocity RefFrame, 10.0, 0.0, 0.0

SetMeshQuality

 Method of Frame

Sets the quality of rendering for a mesh in the current frame. SetMeshQuality is a member of the Frame object.  

Syntax

object.SetMeshQuality nMeshIndex, nQuality

The SetMeshQuality method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nQuality Integer Required The quality of rendering for the mesh.

Return Value

This method does not return a value.

Remarks

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

Possible values of nQuality and their corresponding meanings are shown in the following table:

Value Meaning
0 Wireframe - fastest
1 Unlit Flat
2 Flat
3 Gouraud - best quality
4 Phong
 

Example

MyFrame.SetMeshQuality 4, 2

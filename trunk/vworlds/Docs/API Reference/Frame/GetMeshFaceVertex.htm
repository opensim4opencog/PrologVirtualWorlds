GetMeshFaceVertex

 Method of Frame

Gets the position of a vertex for a mesh. GetMeshFaceVertex is a member of the Frame object.  

Syntax

object.GetMeshFaceVertex nMeshIndex, nFaceIndex, index, fX, fY, fZ

The GetMeshFaceVertex method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nFaceIndex Integer Required The index of the face.
index Integer Required The index of the vertex.
fX Variant Required The x-coordinate of the vertex.
fY Variant Required The y-coordinate of the vertex.
fZ Variant Required The z-coordinate of the vertex.

Return Value

This method does not return a value.

Remarks

The fourth, fifth, and sixth parameters of GetMeshFaceVertex are output parameters. After this method executes, these parameters will be overwritten with new values. Specifically, after a call to GetMeshFaceVertex, the fourth, fifth, and sixth parameters will be set to the coordinates of the specified vertex of the mesh (on the x, y, and z axes).  

Example

MyFrame.GetMeshFaceVertex 4, 12, 1, x, y, z

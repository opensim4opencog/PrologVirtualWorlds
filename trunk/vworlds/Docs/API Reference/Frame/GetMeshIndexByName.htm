GetMeshIndexByName

 Method of Frame

Get the index of a mesh, given its name. GetMeshIndexByName is a member of the Frame object.  

Syntax

object.GetMeshIndexByName(bstrMeshName)

The GetMeshIndexByName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
bstrMeshName String Required The name of the mesh.

Return Value

This method returns a Long.

Example

Index = MyFrame.GetMeshIndexByName ("Mesh1")

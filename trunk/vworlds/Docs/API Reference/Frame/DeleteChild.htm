DeleteChild

 Method of Frame

Deletes a child frame from a parent frame. DeleteChild is a member of the Frame object.  

Syntax

object.DeleteChild pChildFrame

The DeleteChild method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pChildFrame Frame Required The child frame to delete.

Return Value

This method does not return a value.

Remarks

To delete a child frame, you must first get a reference to the child frame's parent, as shown in the following statements. 

Dim myParentFrame
Set myParentFrame = User.Container.GetFrame

If you don't know which container the parent frame is in, you can use the GetParent method to retrieve a reference to the parent frame, as shown in the following statement. 

Dim myChildFrame
Set myParentFrame = myChildFrame.GetFrame

You can now delete the frame. 

Dim myChildFrame
myParentFrame.DeleteChild myChildFrame

Example

This example deletes an existing frame within the user's container.

Dim 
Set myFrame = User.Container.GetFrame

myFrame.DeleteChild myDecalFrame

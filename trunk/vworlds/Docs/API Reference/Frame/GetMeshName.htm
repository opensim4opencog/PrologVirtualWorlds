GetMeshName

 Method of Frame

Gets the name of a mesh. GetMeshName is a member of the Frame object.  

Syntax

object.GetMeshName(nMeshIndex)

The GetMeshName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.

Return Value

This method returns a String.

Remarks

Graphics-creation programs generally allow artists using them to assign a name to a mesh. The mesh name is then inserted into the graphics file. A mesh can be referred to by either its name or its index within a particular frame. Many people, however, prefer to work with easier-to-remember names than with numbers, which is why this method is provided.

Mesh names are sometimes used to refer to meshes after they have been loaded into Virtual Worlds, although many Frame methods require the index of the mesh. Once you have gotten the name of the mesh (using GetMeshName), you can find its index by calling GetMeshIndexByName.  

Example

TheName = MyFrame.GetMeshName( 2 )

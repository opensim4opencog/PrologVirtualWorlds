SetMeshTextureFromText

 Method of Frame

Sets the texture for a mesh from a text string. SetMeshTextureFromText is a member of the Frame object.  

Syntax

object.SetMeshTextureFromText nMeshIndex, bstrText, bShadowed, nWidth, nHeight, nPointHeight

The SetMeshTextureFromText method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
bstrText String Required The text to be used for the texture.
bShadowed Boolean Required A flag that determines whether the text will be shadowed.
nWidth Long Required The width of the texture in pixels.
nHeight Long Required The height of the texture in pixels.
nPointHeight Long Required The point height of the text within the texture.

Return Value

This method does not return a value.

Remarks

Use this method to create a texture that includes text. In this release, the maximum texture size is 256 by 256.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

A value of True for bShadowed specifies that the text will be shadowed.  

Example

MyFrame.SetMeshTextureFromText 0, "Text to be displayed", True, 128, 128, 48

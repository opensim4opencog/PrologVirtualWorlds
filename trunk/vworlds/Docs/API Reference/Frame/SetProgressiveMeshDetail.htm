SetProgressiveMeshDetail

 Method of Frame

Sets the level of detail of a progressive mesh. SetProgressiveMeshDetail is a member of the Frame object.  

Syntax

object.SetProgressiveMeshDetail nMeshIndex, detail

The SetProgressiveMeshDetail method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
detail Float Required The level of detail of the mesh.

Return Value

This method does not return a value.

Remarks

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The level of detail is expressed on a normalized scale from 0.0 to 1.0, where 0.0 represents the minimum detail allowed by the progressive mesh file, and 1.0 represents the maximum allowed detail.  

Example

MyFrame.SetProgressiveMeshDetail 3, .8

GetVisualCount

 Method of Frame

Gets the number of visual objects contained in the frame. GetVisualCount is a member of the Frame object.  

Syntax

object.GetVisualCount()

The GetVisualCount method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.

Return Value

This method returns a Variant containing the number of visual objects contained in the calling instance of the Frame object.

Example

ObjectCount = MyFrame.GetVisualCount

SetRotationRate

 Method of Frame

Sets the rotation rate of the frame relative to the reference frame. SetRotationRate is a member of the Frame object.  

Syntax

object.SetRotationRate pvwfRef, rvX, rvY, rvZ, rvTheta

The SetRotationRate method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwfRef Frame Required The reference frame.
rvX Float Required The x-component of the rotation axis vector.
rvY Float Required The y-component of the rotation axis vector.
rvZ Float Required The z-component of the rotation axis vector.
rvTheta Float Required The angular rotation rate. That is, the amount to rotate in radians per rendered frame.

Return Value

This method does not return a value.

Remarks

The invoking frame is set rotating around a direction vector (x, y, z) defined in the reference frame, at a rate of rvTheta radians per minute.

The rotation rate, rvTheta, represents the fraction of a complete rotation that should be rendered in each frame. Good values are typically between 0.01 and 0.5; these numbers represent 100 frames for a complete rotation and two frames for a complete rotation, respectively.  

Example

MyFrame.SetRotationRate RefFrame, 2.0, 0.0, 0.0, .45

IsChildOf

 Method of Frame

Determines whether or not the invoking object is a child of the specified frame. IsChildOf is a member of the Frame object.  

Syntax

object.IsChildOf(pvwf)

The IsChildOf method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwf Frame Required The frame to query.

Return Value

This method returns a Boolean.

Remarks

Returns a Boolean flag that will be True if the invoking frame is a child of the frame that is passed to the method, and False otherwise.  

Example

IsChild = MyFrame.IsChildOf (TheFrame)

GetMeshFaceColor

 Method of Frame

Gets the color of a face for a mesh. GetMeshFaceColor is a member of the Frame object.  

Syntax

object.GetMeshFaceColor nMeshIndex, index, fRed, fGreen, fBlue

The GetMeshFaceColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
index Integer Required The index of the face.
fRed Variant Required The fractional amount of red.
fGreen Variant Required The fractional amount of green.
fBlue Variant Required The fractional amount of blue.

Return Value

This method does not return a value.

Remarks

The third, fourth, and fifth parameters of GetMeshFaceColor are output parameters. After this method executes, these parameters will be overwritten with new values.

After a call to GetMeshFaceColor, the third, fourth, and fifth parameters will be set to the red, green, and blue color values for the specified mesh and face. Each color value is a decimal fraction between 0.0 and 1.0.

When you set a color value with SetMeshFaceColor and retrieve it with GetMeshFaceColor, there may be some loss of accuracy. For example, a value of 0.5 may be translated to a number such as 0.498999 by the color algorithms.  

Example

MyFrame.GetMeshFaceColor 4, 5, Red, Green, Blue

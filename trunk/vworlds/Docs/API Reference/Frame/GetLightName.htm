GetLightName

 Method of Frame

Gets the name of a light, given its index. GetLightName is a member of the Frame object.  

Syntax

object.GetLightName(nIndex)

The GetLightName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.

Return Value

This method returns a String that is the name of the light.

Example

TheName = MyFrame.GetLightName (17)

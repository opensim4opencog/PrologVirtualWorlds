GetMeshTextureName

 Method of Frame

Gets the filename for the texture of a mesh. GetMeshTextureName is a member of the Frame object.  

Syntax

object.GetMeshTextureName(nMeshIndex)

The GetMeshTextureName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.

Return Value

This method returns a String.

Remarks

The GetMeshTextureName method assumes only one texture per mesh. If a mesh has more than one texture, then this method returns the texture that is attached to face number zero. 

Example

Result = MyFrame.GetMeshTextureName (1)

SetLightAttenuation

 Method of Frame

Sets a light's attenuation. SetLightAttenuation is a member of the Frame object.  

Syntax

object.SetLightAttenuation nIndex, fConst, fLinear, fQuadratic

The SetLightAttenuation method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
fConst Float Required The constant attenuation factor.
fLinear Float Required The linear attenuation factor.
fQuadratic Float Required The quadratic attenuation factor.

Return Value

This method does not return a value.

Remarks

Lights that have a location (that is, are not infinitely far away) can have attenuation factors for calculating the attenuation of the light based on an object's distance from the light. The formula for the total attenuation factor is:

TAF = 1 / [fConst + fLinear * distance + fQuadratic * (distance^2)]

 

Example

MyFrame.SetLightAttenuation 23, 2.0, 1.0, 0.0

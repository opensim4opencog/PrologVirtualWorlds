SetLightType

 Method of Frame

Sets a light's type. SetLightType is a member of the Frame data type.  

Syntax

object.SetLightType nIndex, iType

The SetLightType method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
iType Integer Required The type of the light.

Return Value

This method does not return a value.

Remarks

The Type parameter can be one of the following values:

Value Type
0 Ambient
1 Point
2 Spot
3 Directional
4 Parallel point

Ambient: A light source that illuminates everything in a scene, regardless of the orientation, position, and surface characteristics of the objects in the scene. Because this type of light illuminates a scene with equal strength everywhere, the position and orientation of the frame to which it is attached are inconsequential. Multiple ambient light sources are combined within a scene.

Point: A light source that radiates equally in all directions from its origin.

Spotlight: A light source that emits a cone of light. Only objects within the cone are illuminated. The cone produces light of two degrees of intensity; the central, brightly lit section (the umbra) acts as a point source, and an outer, dimly lit section (the penumbra) merges with the surrounding deep shadow.

Directional: A light source that is attached to a frame, but appears to illuminate all objects with equal intensity, as if it were at an infinite distance from the objects. Directional light has orientation but no position; it is commonly used to simulate distant light sources, such as the sun.

Parallel point: A light source that illuminates objects with parallel light. The orientation of the light is taken from the position of the parallel point light source. For example, two meshes on either side of a parallel point light source are lit on the side that faces the position of the source.  

Example

MyFrame.SetLightType 8, 2

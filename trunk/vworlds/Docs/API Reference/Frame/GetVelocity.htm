GetVelocity

 Method of Frame

Gets the velocity of the calling frame relative to a reference frame. GetVelocity is a member of the Frame object.  

Syntax

object.GetVelocity pvwfRef, pvX, pvY, pvZ

The GetVelocity method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
pvwfRef Frame Required The reference frame.
pvX Variant Required The x-component of the velocity of the frame.
pvY Variant Required The y-component of the velocity of the frame.
pvZ Variant Required The z-component of the velocity of the frame.

Return Value

This method does not return a value.

Remarks

The second, third, and fourth parameters of GetVelocity are output parameters. After this method executes, these parameters will be overwritten with new values.

Each of the three output parameters represents a particular component of distance per render cycle, which is typically 15 frames per second. This means that a value of .01 for xVelocity moves the object .01 * 15 ( = .15) units per second in the x-axis direction.  

Example

MyFrame.GetVelocity RefFrame, xVel, yVel, zVel

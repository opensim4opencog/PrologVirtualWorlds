SetMeshColor

 Method of Frame

Sets the color of a mesh in the current frame. SetMeshColor is a member of the Frame object.  

Syntax

object.SetMeshColor nMeshIndex, fRed, fGreen, fBlue

The SetMeshColor method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
fRed Float Required The fractional amount of red.
fGreen Float Required The fractional amount of green.
fBlue Float Required The fractional amount of blue.

Return Value

This method does not return a value.

Remarks

Each color component is represented by a Float between 0.0 and 1.0.

When you set a color value with SetMeshColor and retrieve it with GetMeshColor, there may be some loss of accuracy. For example, a value of 0.5 may be translated to a number such as 0.498999 by the color algorithms.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.  

Example

MyFrame.SetMeshColor 4, .25, .50, 0.0

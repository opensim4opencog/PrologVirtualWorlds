GetProgressiveMeshVertexDetailRange

 Method of Frame

Gets the range of possible vertex counts. GetProgressiveMeshVertexDetailRange is a member of the Frame object.  

Syntax

object.GetProgressiveMeshVertexDetailRange nMeshIndex, min, max

The GetProgressiveMeshVertexDetailRange method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
min Variant Required The minimum vertex count.
max Variant Required The maximum vertex count.

Return Value

This method does not return a value.

Remarks

The range of possible vertex counts represents the lowest and highest values for the number of vertices that can be displayed for a progressive mesh.

The second and third parameters of GetProgressiveMeshVertexDetailRange are output parameters. After this method is executed, these parameters will be overwritten with new values.

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.  

Example

MyFrame.GetProgressiveMeshVertexDetailRange 1, Min, Max

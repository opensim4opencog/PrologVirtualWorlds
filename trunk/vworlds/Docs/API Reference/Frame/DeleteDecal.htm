DeleteDecal

 Method of Frame

Deletes the specified decal from a Frame object. DeleteDecal is a member of the Frame object. 

Syntax

object.DeleteDecal nMeshIndex

The DeleteDecal method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses an object with a Frame object.
nMeshIndex Long Required The index number of the mesh to delete.

Return Value

This method does not return a value. 

Remarks

The nMeshIndex parameter is generally set to any positive Long. such as 12. that represents the index of a particular mesh. However, you can also set the parameter to . 1 or . 2 as shown in the following table.

nMeshIndex Value Description

Any positive Long.

 Apply to the specified mesh index number.

-1

 Apply to all meshes in the specified frame.

-2

 Apply to all meshes in all frames of the hierarchy.

Example

This example deletes a decal from a frame in a virtual world.

To run this example: You must have a decal attached to an object in your world. 

Sub DeleteLastDecal()
Dim myFrame

'Set a reference to the frame where you want to delete the decal. 

Set myFrame = User.Container.GetFrame

'Delete the last decal you created. 

myFrame.DeleteDecal myFrame.GetDecalCount - 1

'Trace the count of decals. 

Trace myFrame.GetDecalCount

End Sub

CreateDecalFromText

 Method of Frame

Creates a new decal from the specified parameters and places it in the current Frame object. CreateDecalFromText is a member of the Frame object.

Syntax

object.CreateDecalFromText bstrText [, bstrName]

The CreateDecalFromText method syntax has these parts:
Parameters Type Use Description
object Frame Required An object reference to a Frame object.
bstrText String Required The text for the decal.
[bstrName] String Optional The name of the decal.

Return Value

This method does not return a value. 

Remarks

There are two ways to create graphical text in Microsoft Virtual Worlds: By creating a decal, or by using a Label object.

Virtual Worlds takes the text in the bstrText parameter and converts it into a bitmap that appears in your world. You can change the position and size of the decal by using the SetDecalOrigin and SetDecalSize methods.

Example

This example creates a decal and displays it in a virtual world.

Sub CreateKioskDecal()
Dim myFrame

'Set a reference to the frame where you want to place the decal. 

Set myFrame = User.Container.GetFrame

'Create the decal. 

myFrame.CreateDecalFromText "This is the new Kiosk", "Kiosk"

End Sub

SetMeshTextureFromURL

 Method of Frame

Sets the texture for a mesh from a file. SetMeshTextureFromURL is a member of the Frame object.  

Syntax

object.SetMeshTextureFromURL nMeshIndex, bstrTexture

The SetMeshTextureFromURL method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
bstrTexture String Required The URL of the file containing the texture data.

Return Value

This method does not return a value.

Remarks

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The bstrTexture parameter must be a URL and can be in any of the following file formats: .bmp, .gif, or .jpg.  

Example

MyFrame.SetMeshTextureFromURL 4, "http://www.microsoft.com/mytexture.dat"

SetProgressiveMeshFaceDetail

 Method of Frame

Sets the number of faces to display in a progressive mesh. SetProgressiveMeshFaceDetail is a member of the Frame object.  

Syntax

object.SetProgressiveMeshFaceDetail nMeshIndex, detail

The SetProgressiveMeshFaceDetail method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
detail Long Required The number of faces to display.

Return Value

This method does not return a value.

Remarks

nMeshIndex is generally set to the index of a particular mesh, but can be set to two additional values. A value of -1 indicates that this method should be applied to all meshes in the frame, and a value of -2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The detail parameter must fall within the range specified by the GetProgressiveMeshFaceDetailRange method of VWFrame. The request is satisfied to within one face.  

Example

MyFrame.SetProgressiveMeshFaceDetail MeshIndex, Detail

GetSpotLightRadii

 Method of Frame

Gets parameters for the umbra and penumbra of a spotlight. GetSpotLightRadii is a member of the Frame object.  

Syntax

object.GetSpotLightRadii nIndex, fUmbra, fPenumbra

The GetSpotLightRadii method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nIndex Long Required The index of the light.
fUmbra Variant Required The umbra angle in radians.
fPenumbra Variant Required The penumbra angle in radians.

Return Value

This method does not return a value.

Remarks

A spotlight emits a cone of light that only illuminates objects that lie within the cone. The cone produces light of two degrees of intensity: a central, brightly-lit section (the umbra) acts as a point source, and a outer, dimly lit section (the penumbra) merges with the surrounding deep shadow.

The second and third parameters of GetSpotLightRadii are output parameters. After this method executes, these parameters will be overwritten with new values.  

Example

MyFrame.GetSpotLightRadii 1, Umbra, Penumbra

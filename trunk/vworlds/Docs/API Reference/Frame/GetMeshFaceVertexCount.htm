GetMeshFaceVertexCount

 Method of Frame

Gets the number of vertices in a given face of a mesh. GetMeshFaceVertexCount is a member of the Frame object.  

Syntax

object.GetMeshFaceVertexCount(nMeshIndex, nFaceIndex)

The GetMeshFaceVertexCount method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
nMeshIndex Long Required The index of the mesh to use.
nFaceIndex Long Required The index of the face to use.

Return Value

This method returns a Long.

Example

VertCount = MyFrame.GetMeshFaceVertexCount (23, 17)

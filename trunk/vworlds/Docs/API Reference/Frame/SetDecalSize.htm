SetDecalSize

 Method of Frame

Sets the size of the specified decal. SetDecalSize is a member of the Frame object.

Syntax

origin.SetDecalSize nMeshIndex, fHeight, fWidth, bScale

The SetDecalSize method syntax has these parts:
Part Type Use Description
origin Frame Required An object reference to a Frame object.
nMeshIndex Long Required The index number of the mesh.
fHeight Float Required The height of the decal, in pixels.
fWidth Float Required The width of the decal, in pixels.
bScale Boolean Required True if you want the depth of the decal taken into account when the decal is rendered.

Return Value

This method does not return a value. 

Remarks

The bScale parameter is set to True if you want the depth taken into account when Microsoft Virtual Worlds renders the decal. This parameter is also set to True if the decal is a texture on a frame in a scene and is rendered smaller when it is farther from the viewer and larger when it is closer to the viewer. If you want to ignore the depth of the decal and render the decal the same size whether it is near or far from the viewer, set the bScale parameter to False.  

Example

This example changes the size and position of a previously created decal.

Sub ChangeDecalSizeAndPosition()
Dim myFrame
Dim myMeshIndex
Dim myHeight
Dim myWidth
Dim myScale
Dim myX
Dim myY

'Set a reference to the frame where you want to place the decal. 

Set myFrame = User.Container.GetFrame

'Create a decal. 

myFrame.CreateDecalFromText "This is the new Kiosk", "Kiosk"

'Get the values for the variables, "myHeight", "myWidth", "myScale". 

myFrame.GetDecalSize 0, myHeight, myWidth, myScale

'Trace the values. 

Trace "Decal height is: " & myHeight
Trace "Decal width is: " & myWidth
Trace "Decal height is: " & myScale

'Change the values. 

myHeight = myHeight + .5
myWidth = myWidth - .15

'Set the decal to the new size. 

myFrame.SetDecalSize 0, myHeight, myWidth, myScale

'Trace the values. 

Trace "Decal height is: " & myHeight
Trace "Decal width is: " & myWidth
Trace "Decal height is: " & myScale

'Get the origin of the decal. 

myFrame.GetDecalOrigin 0, myX, myY

End Sub

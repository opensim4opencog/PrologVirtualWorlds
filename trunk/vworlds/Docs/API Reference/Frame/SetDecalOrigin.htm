SetDecalOrigin

 Method of Frame

Returns the origin (central point) of the decal. SetDecalOrigin is a member of the Frame data type.

Syntax

object.SetDecalOrigin nMeshIndex, lx, ly

The SetDecalOrigin method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses an object with a Frame data type.
nMeshIndex Long Required The index number of the mesh.
lx Variant Required The value for the x-coordinate can be float or integer.
ly Variant Required The value for the y-coordinate can be float or integer.

Return Value

This method does not return a value. 

Remarks

The nMeshIndex parameter is generally set to the index of a particular mesh, but you can set it to two additional values. A value of . 1 indicates that this method should be applied to all meshes in the frame. A value of . 2 indicates that this method should be applied to all meshes in all frames in the hierarchy.

The default origin of a decal is its center, expressed in the coordinates of the pixel space for the generated image. For example, if a decal is a 256-by-256 texture, its origin is at (128, 128).

After the decal origin is set, you can return the origin by using the GetDecalOrigin method.

Example

This example creates a decal and changes the size and position of the decal.

Sub ChangeDecalSizeAndPosition()
Dim myFrame
Dim myMeshIndex
Dim myHeight
Dim myWidth
Dim myScale
Dim myX
Dim myY

'Set a reference to the frame where you want to place the decal. 

Set myFrame = User.Container.GetFrame

'Create a decal. 

myFrame.CreateDecalFromText "This is the new Kiosk", "Kiosk"

'Get the values for the variables "myHeight", "myWidth", and "myScale". 

myFrame.GetDecalSize 0, myHeight, myWidth, myScale

'Change the values. 

myHeight = myHeight + .5
myWidth = myWidth - .15

'Set the decal to the new size. 

myFrame.SetDecalSize 0, myHeight, myWidth, myScale

'Get the origin of the decal. 

myFrame.GetDecalOrigin 0, myX, myY

'Set the value for the y-coordinate to a minimum number of 30,
'relative to the original value of the y-coordinate. 

If Not myY => 30 Then
    myY = myY + 30
    myFrame.SetDecalOrigin 0, myX, myY
End If

End Sub

CreateUnitMesh

 Method of Frame

Creates a default (unit) mesh with 1 face. CreateUnitMesh is a member of the Frame object.  

Syntax

object.CreateUnitMesh 

The CreateUnitMesh method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.

Return Value

This method does not return a value.

Remarks

A unit mesh is a square of size 1 unit by 1 unit. The CreateUnitMesh method creates a default mesh that is centered on the origin of the frame (0, 0, 0), on the x-y plane.

Since the mesh size is 1 unit by 1 unit centered on the origin of the Frame object, the vertex coordinates that define the mesh are (0.5, 0.5, 0), (0.5, -0.5, 0), (-0.5, -0.5, 0), and (-0.5, 0.5, 0).  

Example

MyFrame.CreateUnitMesh

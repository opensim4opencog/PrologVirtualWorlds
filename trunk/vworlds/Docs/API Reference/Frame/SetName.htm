SetName

 Method of Frame

Sets the D3DFrame of the calling Frame. SetName is a member of the Frame object.  

Syntax

object.SetName bstrName

The SetName method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses a Frame object.
bstrName String Required The name of the Frame.

Return Value

This method does not return a value.

Example

TheFrame.SetName "Frame2"

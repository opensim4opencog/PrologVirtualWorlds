GetDecalCount

 Method of Frame

Returns the number of decals in a specified Frame object as a Long. GetDecalCount is a member of the Frame object. 

Syntax

object.GetDecalCount()

The GetDecalCount method syntax has these parts:
Part Type Use Description
object Frame Required An object reference that accesses an object with a Frame object.

Return Value

This method returns a count of the decals within a specified frame. 

Remarks

This method is similar to a count property. When you execute this method, it searches the specified frame and counts the number of decals within the frame. Common usage for this method is safely iterating through the list of decals to alter them.

Examples

The first example creates a decal and displays it in a virtual world.

Sub CreateKioskDecal(strDecalText)
Dim myFrame
Dim myDecalCount

'Set a reference to the frame where you want to place the decal. 

Set myFrame = User.Container.GetFrame

'Get the number of decals. 

myDecalCount = myFrame.GetDecalCount
Print myDecalCount

'Check the number of decals in the frame. 

If myDecalCount = 5 Then
    Print "You have reached the limit of the decals for this frame."
    Exit Sub
Else
    'Okay to create the decal.
    myFrame.CreateDecalFromText strDecalText, "Kiosk"
End If
End Sub

Example 2

The second example iterates through the decals in a frame and changes their size.

For i = 0 To myFrame.GetDecalCount - 1 
    myFrame.SetDecalSize 0, .5, .15, 2
Next

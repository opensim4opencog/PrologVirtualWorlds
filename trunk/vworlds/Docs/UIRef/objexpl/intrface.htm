Object Explorer Interface

In the Object Explorer window, you can add, modify, and delete properties and methods on any object for which you have write access. (If you are a Wizard, you have write access for every object in the world.)

Object Explorer Options

Objects Pane

The Objects Pane lists World properties and methods, modules (if any) that have been added to the world, and the room containing the user.

World

Select World to view properties and methods of the world. This icon has no right-click menu.

Modules

Any modules that have been added to the world are listed beneath Modules. If you want to add a module to the world, select Modules, right-click, and then click New module.... This action opens the New Module Dialog Box.

After a module has been added to the world, there are two additional right-click menu items that become available when you select it: Reload Module, and Delete Module. Use Reload Module if you have made changes to the code of the module and you want to make these changes available to your world. Use Delete Module if you have no further use for that module.

The User's Room

Icons for all objects in the User's room appear beneath this icon. Click an object in this pane to select it for editing. The object's properties and methods appear in the appropriate panes to the right of the Objects pane.

You can add new objects, delete objects, and edit their properties from this pane too.

  • To add an object, right-click anywhere in this pane and choose New Object... from the pop-up menu.
  • To delete an object, right-click it and choose Delete from the pop-up menu.
  • To edit an object's properties, right-click it and choose Properties... from the pop-up menu. The Property Page dialog boxes will appear.
  • To add a module, right-click on Modules, then choose New Module.

Properties Pane

You can add a new property in this pane, or click on a property to edit it. You can also select which sets of properties to display.

  • To add a property, right-click anywhere in this pane and choose New... from the pop-up menu.
  • To edit a property's name, right-click it and choose Rename from the pop-up menu.
  • To edit a property's value, double-click it, or right-click it and choose Edit Value from the pop-up menu.
  • To delete a property, right-click it and choose Delete from the pop-up menu.
  • To view properties on the instance of the object (as opposed to properties shared by other objects with the same exemplar),  right-click anywhere in this pane and select Show Instance Properties from the pop-up menu. (This option is turned on by default.)
  • To view properties on the object's exemplar (properties shared by other objects which share the same exemplar), right-click anywhere in this pane and select Show Exemplar Properties from the pop-up menu. (This option is turned on by default.)

Name

You can click on the title of this field in order to sort the properties displayed by their names. (This is the default behavior.)

Type

You can click on the title of this field in order to sort the properties displayed alphabetically by their types. (Boolean will come before Float, which will come before String, and so on.)

Value

You can click on the title of this field in order to sort the properties displayed alphabetically by their values. This feature is useful if you are looking for a property with a particular value.

Methods Pane

You can add a new method in this pane, or click on a method to edit it.

  • To add a method, right-click anywhere in this pane and choose Add New Method from the pop-up menu.
  • To edit a method's name, right-click it and choose Edit Name from the pop-up menu.
  • To change a method from being in a COM module to being in an inline module, right-click it and choose Change Module from the pop-up menu.
  • To edit a method's value, double-click it, or right-click it and choose Edit Sub Name from the pop-up menu.
  • To delete a method, right-click it and choose Delete Method from the pop-up menu.
  • To view methods on the instance of the object (as opposed to methods shared by other objects with the same exemplar), right-click anywhere in this pane and select Show Instance Methods from the pop-up menu. (This option is turned on by default.)
  • To view methods on the object's exemplar (methods shared by other objects with the same exemplar), right-click anywhere in this pane and select Show Exemplar Methods from the pop-up menu. (This option is turned on by default.)

Name

You can click on the title of this field in order to sort the properties displayed by their names. (This is the default behavior.)

Module

You can click on the title of this field in order to sort the properties displayed alphabetically by their modules. Methods in inline modules will be displayed first.

Subroutine

You can click on the title of this field in order to sort the properties displayed alphabetically by their subroutines.

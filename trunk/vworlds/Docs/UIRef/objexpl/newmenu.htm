New Menu Item Dialog Box (Inline Method Editor)

Use this dialog box to create a new menu item and attach it to a particular object. The menu is the shortcut menu for an object that you see if you right-click on that object in the virtual world. You can set menu items to appear to certain levels of visibility in Virtual Worlds.

This dialog box appears when you click the Add Menu Item button on the Inline Method Editor dialog box. The menu item you add invokes the method you are editing.

Options

Add item to the object's Menu

Adds the menu item you specified to the shortcut menu of an object. This menu item appears when anyone right-clicks on the object.

The object is selected in the Object text field of the Inline Method Editor dialog box.

Add item to the object's PeerMenu

Adds a menu item to the shortcut menu of all the object's peers. A peer is defined as another object that has the ability to right-click on other objects (at the moment, that means objects that are avatars).

PeerMenu Example

You set a PeerMenu item on an artifact and name the menu item Pull Lever. All other peers (avatars) in the room can right-click on the artifact and see the Pull Lever menu item.

Note: This only works if PeerMenu is defined on the right-clicking object (avatar).

Add item to the object's ContainerMenu

Adds a menu item to the objects contained within that object. Each object inside the container object gets the menu item added to their shortcut menus.

ContainerMenu Example

Adding ContainerMenu to a room causes that menu item to be displayed when you right-click on any objects on any objects in that room.

Add item to the object's ContentMenu

Adds a menu item to the object's container.

ContentMenu Example

Adding ContentMenu to an artifact held in the inventory of an avatar adds the menu item to that avatar's shortcut menu.

Note: Rooms do not have shortcut menus, so the object must be contained by something else to see any effects.

Add item to the object's SelfMenu

Adds a menu item to yourself (avatars only, since avatars are the only objects that can right-click on themselves). This menu item is visible only to you, other avatars will not see it if they right-click on your avatar.

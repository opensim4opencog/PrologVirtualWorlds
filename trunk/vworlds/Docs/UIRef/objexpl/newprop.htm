New Property Dialog Box (Object Explorer)

Use this dialog box to point users to your world's web page. The web page URL is displayed next to the world's name in its Directory listing.

Options

Object

This option displays the name of the object to which you are adding the property.

Property Name

Enter the name of the property you are adding into this text box.

Property Type

You can choose an appropriate variable type to give your new property with this menu. Possible types include the following: String, Boolean, Double, Float, Long, Integer, Vector, Menu, Property List, Property Map, and Thing. The default type is String.

Initial Value

Enter the value your new property should contain into this text box. This is useful both in setting permanent values for properties that will never change, and also in setting a default value that may be changed later.

Property Flags

Copy On Get

Clicking this box to add a check mark indicates that the property will be accessed on an instance, rather than on the exemplar of the instance.

Not Logged

Clicking this box to add a check mark indicates that changes to the property are not saved to the Virtual Worlds checkpoint file.

Not Remoted

Clicking this box to add a check mark indicates that changes to the property are saved to the Virtual Worlds checkpoint file, but not sent to other clients.

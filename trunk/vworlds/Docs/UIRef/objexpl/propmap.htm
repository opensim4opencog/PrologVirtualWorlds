Property Map Editor Dialog Box

Use this dialog box to add or delete properties of a property map, and to change their names or the values they contain.

You might edit a property map to add named items to a list with corresponding values.

Options

  • To add a new property to the property list, right-click anywhere in the pane listing the existing properties and select Add New Property from the menu that appears, or just click the Add button.
  • To edit the name of a property in the property list, triple-click the name in the pane containing the list of properties, or right-click the property and select Edit Name from the menu that appears.
  • To edit the value of a property in the property list, double-click the value listed for the property, or right-click the property and select Edit Value from the menu that appears. You can also select a property, then click the gray rectangle that appears to its right in order to bring up the Advanced Text Editor dialog box.
  • To delete a property from the property list, right-click it and select Delete Property from the menu that appears, or just select the property and click the Remove button.

Object

This text displays the name of the object on which the current property map appears. This object is always the one currently selected in the Object Explorer.

Property

This text displays the name of the property map currently being edited.

Name

You can click the title of this field to sort the displayed properties by their names. (This is the default behavior.)

Type

You can click the title of this field to sort the displayed properties alphabetically by their types. (Boolean will come before Float, which will come before String, and so on.)

Value

You can click the title of this field to sort the displayed properties alphabetically by their values. This feature is useful if you are looking for a property with a particular value.

Add

Click this button to add a property to the currently-selected property list

Remove

Click this button to remove the selected property from the property list.

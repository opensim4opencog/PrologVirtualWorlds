New Property List Item Dialog Box

Use this dialog box to add a new property to a property list.

This dialog box appears when you click the Add button on the Property List Editor Dialog Box or when you right-click anywhere in the Properties pane of that dialog box and then select Add New Property from the menu that appears.

Options

Object

This text displays the name of the object on which the property list being edited resides.

Property List Name

This text displays the name of the property list being edited, which is always the property list that was selected in the Properties pane of the Object Explorer.

Property Type

You can choose an appropriate variable type to give your new property with this menu. Possible types include the following: String, Boolean, Double, Float, Long, Integer, Vector, Menu, Property List, Property Map, and Thing. The default type is String.

Initial Value

Enter the value your new property should contain into this text box. This is useful both in setting permanent values for properties that will never change, and also in setting a default value that may be changed later.

Menu Editor

Use this dialog box to change the menu items in a menu, or to give new behavior to the menu items on a menu that already exists.

Options

  • To add a new menu item to the current menu, click the Add button or right-click anywhere in the pane and select Add New Menu Item from the menu that appears.
  • To change the name of a menu item, right-click the item and select Edit Menu Item Text from the menu that appears, or triple-click the menu item's name.
  • To delete a menu item, select it with the mouse and then click Remove, or right-click it and select Delete Menu Item from the menu that appears.
  • To change the position of a menu item in the current menu, select it and click the Up or Down button to move the item up or down in the list, or right-click it and select Move Menu Item Up or Move Menu Item Down from the menu that appears.
  • To change the method attached to the menu item, triple-click the menu item's current method and select a new method from the list that appears. Only the currently-available methods will be listed. (Legal methods are listed in the Methods pane, immediately below the Properties pane in the Object Explorer.)

Add

Click this button to create a new menu item for the current menu.

Remove

Click this button to remove the selected menu item from the current menu.

Up

Click this button to move the selected menu item up in the current menu. It will appear in the client at the same position in which it appears in the Menu Editor.

Down

Click this button to move the selected menu item down in the current menu. It will appear in the client at the same position in which it appears in the Menu Editor.

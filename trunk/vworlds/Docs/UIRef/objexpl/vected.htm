Vector Editor

Use this dialog box to edit the X, Y, and Z coordinates of a Vector property.

A Virtual Worlds Vector property can represent either a mathematical vector or a mathematical point.

Options

Object

This text displays the name of the object on which the current property appears. This object is always the one currently selected in the Object Explorer.

Property

This text displays the name of the Vector property currently being edited.

Values

You can enter new values for the X, Y, and Z coordinates of the Vector object into the X, Y, and Z text boxes under this heading. You can also enter a Float, or floating-point number (one that contains a decimal point), into any of the boxes.

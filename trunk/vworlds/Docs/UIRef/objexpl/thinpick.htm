Thing Selector

Use this dialog box to set a Thing property in the Object Explorer to one of the Thing objects in the current virtual world.

For example, if you have an avatar named Bob in the same world as an artifact called BigBox, you can use the Thing Selector to set the Container property of Bob to be BigBox.

Options

Object

This text displays the name of the object on which the current property appears. This object is always the one currently selected in the Object Explorer.

Property

This text displays the name of the Thing property currently being edited.

Things

This pane displays a list of all objects in the world to which you can set the edited property. Click an object, and then click the Select button to set the Thing property to one of the objects listed. The Thing Selector will list not only Thing objects in this pane, but also objects in the world that inherit from Thing, such as WebPage, Avatar, and Artifact objects.

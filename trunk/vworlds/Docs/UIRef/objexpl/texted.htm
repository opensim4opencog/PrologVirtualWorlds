Advanced Text Editor (Object Explorer)

Use this dialog box to enter substantial quantities of text into a Virtual Worlds String property.

This dialog box appears when you click the gray rectangle to the right of a String property in the Properties pane of the Object Explorer, the Property List Editor, or the Property Map Editor. Normally, when you edit the value of a String property, you can enter only a relatively small number of characters. The Advanced Text Editor, however, allows you to enter an unlimited number of text characters into a String.

Options

Object

This text displays the name of the object where the String property that is being edited resides.

Property

This text displays the name of the String property that is being edited, which is always the property to the left of the gray rectangle that was clicked in the Properties pane of the Object Explorer, the Property List Editor, or the Property Map Editor.

Value

Enter the text you are adding to the String property into this text box.

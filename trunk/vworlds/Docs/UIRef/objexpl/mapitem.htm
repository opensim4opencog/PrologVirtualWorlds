New Property Map Item Dialog Box

Use this dialog box to add a new property to a property map.

This dialog box appears when you click the Add button on the Property Map Editor Dialog Box, or when you right-click anywhere in the Properties pane of that dialog box and select Add New Property from the pop-up menu.

Options

Object

This text displays the name of the object on which the property map being edited resides.

Property Map Name

This text displays the name of the property map being edited, which is always the property map that was selected in the Properties pane of the Object Explorer.

Property Name

Enter the name of the new property you are adding into this text box.

Property Type

You can choose an appropriate variable type to give your new property with this menu. Possible types include the following: String, Boolean, Double, Float, Long, Integer, Vector, Menu, Property List, Property Map, and Thing. The default type is String.

Initial Value

Enter the value your new property should contain into this text box. This is useful both in setting permanent values for properties that will never change, and also in setting a default value that may be changed later.

Object Wizard

You can use this dialog box to create several different types of object interactively within a virtual world, without having to resort to Visual Basic Script commands.

Options

Artifact

Click this option, then click the Create button to create an Artifact object, a type of object used to represent ordinary objects in Virtual Worlds (usually the sort that can be carried from one place to another). The Artifact exemplar is the parent exemplar for WebPage and StateMachine, as described below.

Interworld Portal

Click this option, then click the Create button to create an IWPortal object, a type of portal that can transport avatars from one virtual world to another. (A Room Portal is an object that can transport an avatar from one room in a world to another in the same world, as described below.)

Room Portal

Click this option, then click the Create button to create a Room Portal object, a type of object that can transport avatars from one room in a virtual world to another room.

State Machine

Click this option, then click the Create button to create a StateMachine object, a type of object that can be programmed to perform routine actions in a virtual world. (Virtual Worlds bots, or objects that can interact with and sometimes give the appearance of conversing with avatars, are usually state machines.)

Web Page

Click this option, then click the Create button to create a WebPage object, a type of object that can be made to display an HTML page on the Internet or the server when clicked.  (The message to be displayed can also be simple text that contains no HTML tags.)

Web page objects can be used for a variety of purposes, depending on the graphic that represents the message in the world. They can be used to create notes, newspapers, instructions, newbie information, links to external web pages, and so forth.

Create

Clicking this button finalizes your decision to create the type of object you have chosen. It also displays a new dialog box in which you can edit the properties of the object you are creating.

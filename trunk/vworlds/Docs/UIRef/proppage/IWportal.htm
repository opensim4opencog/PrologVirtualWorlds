Interworld Portal Property Page

Interworld portals are one-way doorways between worlds. These objects can transport avatars between this world and another. You can specify the world to connect to and the look and description of this portal. You can also use the Directory dialog box to move between worlds, or to get the server and world name for these settings.

This object is also known as an IWPortal. The default appearance of the interworld portal is an open doorway with starry space showing beyond the door.

Property Page Options

Portal connect to

Server

Enter the name of the server of the target world. You can find this information in the Directory dialog box. For example, if the Directory dialog box showed a world link as MAINSERVER/Big_House, MAINSERVER would be the name of a computer that is running the Virtual Worlds Server application.

World

Enter the name of the world you want to connect to. You can find this information in the Directory dialog box. For example, if the Directory dialog box showed a world link as MAINSERVER/Big_House, Big_House would be the name of a world on that server.

Advanced

Click this to go directly to the Object Explorer with this object.

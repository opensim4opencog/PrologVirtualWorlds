Web Page Property Page

Web page objects are objects that display either a message or the contents of a web page. You can type in a message to display or specify an URL to a web page. Web pages are displayed as pop-up windows without a browser. You can set the size and position of the web page object's display window. Examples of web page objects include greetings, logos, newspapers, instructions, or hints.

The default appearance of the web page object is a poster with a URL texture. The web page object is activated by clicking it in the world view or selecting Show from the shortcut menu for the web page object.

Property Page Options

Show URL

The default setting for the web page object. Enter the URL of the web page that you want to display. You can click the links on the displayed URL to navigate through the web pages.

Show HTML

Displays a message instead of a web page. Enter the HTML code and text into the edit box.

Window

Position (Top and Left)

Enter the position of the upper left corner of the web page object window.

Size (Width and Height)

Enter the width and height of the web page object window.

Advanced

Click this to go directly to the Object Explorer with this object.

General Property Page

You can set the basic information about your object in this page. These are the first things other avatars may notice about this object, such as the name and description, and whether they can move the object or not. The name of the object's owner appears in the lower right corner of the page.

Property Page Options

Object Name

Enter the name of the object. The name is shown in the room and avatar inventory lists and also for object descriptions displayed to other avatars.

Description

Enter the description of the object. This description is shown when other avatars click More About... on this object.

Is solid

Select to make the object appear solid to avatars who collide with it. If this check box is cleared, avatars are able to pass through the object. Objects are solid by default.

Is visible

Select to make your object invisible in the world, but still have it appear in the room inventory. This is useful for light or sound objects. Objects are visible by default.

Can be taken

When selected, your object can be taken by other avatars and stored in their inventory. The default is that objects cannot be taken.

Can be moved

Select to make the object's position adjustable by the owner or a wizard. When this check box is cleared, the object is "anchored" in place. This is especially useful for pillars, walls, and permanent landscape or interior features. The default is that objects can be moved.

Is user interface accessible

Select to make the object disappear from inventory lists, and to prevent any access through the world view. The only way to modify objects that are not accessible in the user interface is to use the Object Explorer and change the properties there. The default is that objects are user interface accessible.

Advanced

Click this to go directly to the Object Explorer with this object selected.

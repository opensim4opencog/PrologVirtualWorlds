Command Window Interface

Use this dialog box to execute Virtual Worlds commands in Microsoft Visual Basic Scripting Edition©  immediately.

Options

Main Pane

This pane displays the commands you enter into the text entry pane below it, as well as the results of those commands. See the illustration above for some examples.

Text Entry Pane

Enter the commands you want Virtual Worlds to execute into this text box. The commands you type are executed immediately, with the following exception: if you type the first line of a Visual Basic Script subroutine (the line containing the keyword Sub), all subsequent lines are interpreted as part of this subroutine until you enter the line containing End Sub. Similarly, you can enter a multi-line subroutine or function into the command window by entering all of the code for the subroutine on one line, with each statement separated from the next by a colon (:).  After such a subroutine or function has been entered into the command window, it can be called by name at any time during the same command window session.

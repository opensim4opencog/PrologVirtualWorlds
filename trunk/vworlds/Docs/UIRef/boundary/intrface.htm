Boundary Editor Interface

Boundaries are objects that are composed of a set of walls that enclose an area (sometimes with an object inside the area). Boundaries are two-dimensional only, lying on the X and Z plane. Boundaries around objects move when the object is moved.

Boundaries can make objects appear to be solid, by preventing the avatar from passing through the object. Boundaries can also act as tripwires, triggering events when the avatar passes through the boundary. In the Virtual Worlds environment, boundaries are invisible. Each object in the room can have a boundary. Boundaries can also be automatically generated around objects.

With the Boundary Editor, you can place boundaries around objects, shape the boundaries, and establish the nature of the boundaries. To see the Boundary Editor tab and the Tools tab in the Virtual Worlds window, you must have author or wizard status, to have permission to build within the world.

Boundaries are automatically generated around new objects in the world. If an object is invisible (objects whose  IsVisible property is set to False), it may not have a boundary.

You cannot add a boundary to a spinning object. If an object is spinning, use the Motion property page to stop it first, add any boundaries to it, and then you can set it spinning again. Boundaries do not spin with their objects.

The toolbar appears beneath the Virtual Worlds Menu Bar. This toolbar contains specific shortcuts for the Boundary Editor; the Geometry Editor uses a different toolbar.

Selected Object

Pick the object in the current world that you want to associate with a boundary (or edit the boundary already associated with that object). You can enter the first letter of the object to scroll the list faster.

Lock Selection

Freezes the current object or the current object's boundary so that you are only editing that selection. This is effective when a boundary is surrounded by many close or overlapping boundaries. If an object or area has multiple boundaries associated with it, you can use the Boundary Browser to select the specific boundary to edit.

Center View on Selection

Moves the view of the world so that the selected object is in the middle.

Delete Selection

Removes a boundary or a vertex point in a boundary. You must select a boundary, and you cannot use the Navigation tool at the same time. To delete the entire boundary, click to select it, and then click this button. To delete a vertex point, click the unwanted vertex point until it appears with a small square around it, and then click this button.

Boundary Browser

Click for the Boundary Browser, a tool that lists all the possible boundaries associated with an object or area (for selecting or deleting).

Pan View

Changes your mouse pointer to the Navigation tool. When you are using this tool, you can move around the room, but you cannot select or change objects in it. You can use this tool to move quickly through the room and to see an overview of the boundaries.

Move Boundary or Vertex

Changes your mouse pointer to the Move tool. This is the tool you can use to change boundaries: you can select a boundary, a boundary vertex, or an object by clicking in the main view. You can also move or stretch boundaries by dragging them with the mouse, to create custom shapes for your boundaries.

When you select a boundary object with this tool, four input boxes appear on the right side of the menu bars. The boxes marked X and Y indicate the size and position of the chosen boundary. The boxes marked F and C indicate the position of the boundary floor and ceiling, respectively.

Rotate Boundary Tool

Changes your mouse pointer to the Rotate Boundary tool. You can use this tool to select a particular boundary and then rotate it as you like, by clicking and dragging.

When you select a boundary object with this tool, three input boxes appear on the right side of the menu bars. The Degrees box shows the total number of degrees you have rotated the boundary in each drag of the mouse. When you drop the boundary, Degrees are reset to 0.0. The boxes marked F and C indicate the position of the boundary floor and ceiling, respectively.

Scale Boundary Tool

Changes your mouse pointer to the Scale Boundary tool. With this tool, you can select a boundary (by clicking it), then drag it to resize it and rotate it, keeping the boundary's original shape intact. This can be useful if you create a boundary and then want to shrink it, so that it fits closely around the object, or expand it to stand away from the object.

When you select a boundary with this tool, four input boxes appear on the right side of the menu bars. The boxes marked X and Y indicate the size and position of the chosen boundary. The boxes marked F and C indicate the position of the boundary floor and ceiling, respectively.

Zoom In Tool

Changes your mouse pointer to the Zoom In tool. When you are using this tool, you can click on the main view to produce a larger image of the current room. You can enlarge the graphics to see small objects.

Zoom Out Tool

Changes your mouse pointer to the Zoom Out tool. When you are using this tool, you can click on the main view to produce a smaller image of the current room. This is useful when you wish to see an overview of the room, the objects, and their boundaries.

Zoom Level

Shows the proportion between the actual size of the world graphics and the current view. You can type a number to scroll the list faster, or choose from a list of preset possible views that appears when you click the down-arrow.

Draw Boundary and Insert Vertex Tool

Changes your mouse pointer to the Create Boundary tool, which uses two shapes: a pencil and a cross. When you move over an area that can include a boundary, the pencil appears. You can use it to "sketch" a boundary by clicking to create a corner, then moving the pointer to another corner and clicking, around the perimeter of the new boundary. New boundaries are attached to the selected object. If there is no selected object, new boundaries are attached to the room.

When you pass the pointer over a boundary, the cross appears. You can create a new vertex by moving the tool to the location and clicking. If you wish to move and stretch the new vertex, however, you must switch to the Translate Boundary tool.

When you select a boundary object with this tool, four input boxes appear on the right side of the menu bars. The boxes marked X and Y indicate the size and position of the chosen boundary. The boxes marked F and C indicate the position of the boundary floor and ceiling, respectively.

Add Automatic Boundary to Selected Object

Creates a boundary that has a standard shape and size around the current object. This selection is not available if the current object is a boundary.

Toggle Boundary Passable

Makes the selected boundary passable or impassable, depending on its current status. When the button is up, the boundary is impassable, which makes the object appear solid. When the button is clicked into the down position, the boundary is passable, and avatars can pass through the object.

X Axis Lock

Click to lock the selected boundary to movement only on the X axis, or "up" and "down" on the Boundary Editor's plan view of the world.

Z Axis Lock

Click to lock the selected boundary to movement only on the Z axis, or "left" and "right" on the Boundary Editor's plan view of the world.

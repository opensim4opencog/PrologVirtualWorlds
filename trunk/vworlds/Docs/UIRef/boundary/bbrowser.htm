Boundary Browser Dialog Box

Options

Objects

Displays a list of objects in the world that have boundaries. Only objects that have boundaries are displayed.

Boundaries

Displays a list of all boundaries possessed by the object whose name is highlighted in the Objects pane. The boundaries are numbered starting from 0, so if an object has four boundaries, they will be listed as 0, 1, 2, and 3.

Select

Selects the boundary whose number is highlighted in the Boundaries pane within the Boundary Editor.

Delete

Permanently deletes the boundary whose number is highlighted in the Boundaries pane from the virtual world.

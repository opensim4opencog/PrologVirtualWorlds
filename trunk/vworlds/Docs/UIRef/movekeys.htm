Shortcut Keys in Microsoft Virtual Worlds

In Microsoft Virtual Worlds, there are many shortcut keys for moving in a three-dimensional space. You can use these instead of using the mouse.

Three-Dimensional Navigation Keys

Action Keyboard Notes
Walk forward Up arrow Combine with SHIFT for speed.
Walk backward Down arrow Combine with SHIFT for speed.
Turn left Left arrow Combine with SHIFT for speed.
Turn right Right arrow Combine with SHIFT for speed.
Run SHIFT Accelerates movement and turning.
Look up Page Up Changes viewpoint without moving the avatar.
Look down Page Down Changes viewpoint without moving the avatar.
Re-center view Home Changes viewpoint without moving the avatar.
Sidestep left CTRL + Left arrow Moves horizontally left without turning.
Sidestep right CTRL + Right arrow Moves horizontally right without turning.
Fly up, down, left or right CTRL + Up, down, left or right arrow Fly in the direction of the chosen arrow key.

Geometry Editor and Boundary Editor Keys

In the Boundary Editor and Geometry Editor, the camera can move about instead of the avatar, while in Navigation mode. Cameras can pass through boundaries while moving.

Shortcut Keys For All Modes of Operation (Navigation, Translation, Rotation, and Scale)

Action Keyboard Notes
Navigation Mode 1 In this mode, move the camera.
Translation Mode 2 In this mode, move a selected object.
Rotation Mode 3 In this mode, rotate a selected object.
Scale Mode 4 In this mode, scale/resize a selected object.
Fly up, down, left or right CTRL + Up, down, left or right arrow Fly in the direction of the chosen arrow key.
Look up Page Up Tilts view up, without moving the avatar.
Look down Page Down Tilts view down, without moving the avatar.
Re-center view Home Centers view on horizon, without moving the avatar.
Delete object DELETE Deletes the selected object.
Speed up movement SHIFT Accelerates movement and turning.
Slow motion S Slows down movement, rotation and scaling of objects.
Lock Y and Z axes X Constrains movement, rotation, and scale to the X axis.
Lock X and Z axes Y Constrains movement, rotation, and scale to the Y axis.
Lock X and Y axes Z Constrains movement, rotation, and scale to the Z axis.

Shortcut Keys Specific to Navigation Mode (Geometry and Boundary Editors)

Action Keyboard Notes
Mouse drag forward Up arrow Moves the camera forward.
Mouse drag backward Down arrow Moves the camera backwards.
Mouse drag left Left arrow Turns the camera to the left.
Mouse drag right Right arrow Turns the camera to the right.

Shortcut Keys Specific to Translate Mode (Geometry and Boundary Editors)

Action Keyboard Notes
Mouse drag forward Up arrow Moves the selected object forward, on the Z axis (away from the camera).
Mouse drag backward Down arrow Moves the selected object backwards, on the Z axis (towards the camera).
Mouse drag left Left arrow Moves the selected object to the left, on the X axis.
Mouse drag right Right arrow Moves the selected object to the right, on the X axis.
Orbit object O Orbits around the selected object while "O" is pressed (only in Translation mode).

Shortcut Keys Specific to Rotation Mode (Geometry and Boundary Editors)

Action Keyboard Notes
Mouse drag forward Up arrow Rotates the selected object around the X axis (away from the camera).
Mouse drag backward Down arrow Rotates the selected object around the X axis (towards the camera).
Mouse drag left Left arrow Rotates the selected object to the left, around the Y axis.
Mouse drag right Right arrow Rotates the selected object to the right, around the Y axis.

Shortcut Keys Specific to Scale Mode (Geometry and Boundary Editors)

Action Keyboard Notes
Mouse drag forward Up arrow Scales the selected object larger on the Z axis.
Mouse drag backward Down arrow Scales the selected object smaller on the Z axis.
Mouse drag left Left arrow Scales the selected object smaller on the X axis.
Mouse drag right Right arrow Scales the selected object larger on the X axis.
Uniform scaling SHIFT Scales the object in all directions at once.

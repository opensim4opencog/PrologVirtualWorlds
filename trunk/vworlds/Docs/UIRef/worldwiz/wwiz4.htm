Create a World Wizard, Step 4 of 4

With this page you can choose from a list of predefined world types to use for your new world, or load your own world from script.

Window Options

World Types

Select the type of world you want to create from this menu. The types of world available for you to use will depend on your installation of Virtual Worlds.

If you select the Virtual Worlds Tutorial, a tutorial world is constructed for you that will help you learn about creating and programming Virtual Worlds objects.

Styles

Select the style of world from the list that appears. For example, if you selected the Landscape world type (as depicted above) you might be offered Valley, Hill Top, or other styles of outdoors world. Depending on your installation of Virtual Worlds, world styles available may include Empty, Gallery, Home, Landscape, Office, Tutorial, and others.

View of...

This pane previews an image of what someone might see on a visit to your world.

Description of...

This line of text describes any distinguishing features of your world style.

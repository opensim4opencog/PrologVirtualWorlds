Create a World Wizard, Step 1 of 4

You can name your world, describe it, and set the location of the world's server with this dialog box.

You can open the Create a World Wizard by selecting New World... from the File menu in the Virtual Worlds Server application, by typing Ctrl+N within the server, or by clicking the Create a World Wizard link on the Virtual Worlds Start Screen.

Wizard Options

Name of the world

Type the name of your new world into this text box. This is the name that is displayed in the Directory dialog box whenever people choose a world to visit on the server (and the "Show World in Directory Services" option is turned on for your world). The name can be up to 255 characters long.

Brief Description

Type a short description of your new world into this text box, to give potential visitors some information about it.

Server location

Type the location of the Virtual Worlds server you plan to use into this text box. If the server you plan to use is the computer on which you are currently creating the new world, type localhost.

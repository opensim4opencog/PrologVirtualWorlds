Create a World Wizard, Step 3 of 4

You can specify the Internet address of your virtual world with this dialog box.

Wizard Options

Content server

Type the location from which your Virtual Worlds content will be served into this box. Content means the graphics, geometry, and special avatars for your world, as well as HTML web pages, scripts and code modules, and other objects unique to your virtual world. The location may be on the Internet or in your local network.

If your content server is going to be the computer on which you are currently working, you can type localhost.

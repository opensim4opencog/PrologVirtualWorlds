Things in Room Tab

With the menu on this tab in the Virtual Worlds client window, you can get information about the objects in the world with you, and modify and interact with them in various ways.

Options

Shortcut Menu

If you click an object's name, a shortcut menu will appear for that object, as depicted above. With the shortcut menu, you can get information about the objects your avatar is carrying; you can also drop them, view and edit their properties, and perform various actions on them (for example, you can display a Web Page object by selecting Show... from its shortcut menu).

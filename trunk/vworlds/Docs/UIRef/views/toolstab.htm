Authoring Tools Tab

This tab of the Virtual Worlds client contains options that will transport you instantly to another room, allow you to add an object or a room to the world, or to perform advanced scripting-related tasks like entering VBScript commands or modifying methods and properties of objects in the world.

Options

Teleport to room:

When you select a room from this menu, you will teleport to that room when you click "go." Teleporting means that your avatar will be transported instantly to the room selected, without traversing any of the terrain or passing through a portal between your current position and the room you have chosen.

Go

Click this button to teleport to the selected room.

Add an Object

Click this button to bring up the Object Wizard, with which you can create a new object in the virtual world.

Add a Room

Click this button to bring up the Room Wizard, with which you can create a new room and add it to the virtual world.

Command Window

Click this button to bring up the Command Window, with which you can enter scripting commands directly in VBScript. This feature is of interest to advanced users only.

Object Explorer

Click this button to bring up the Object Explorer, with which you can modify properties and methods attached to objects in the world.

Client Interface

The client interface is your primary means of interacting with the Virtual Worlds client, and with created virtual worlds. Nearly all views and dialog boxes in the Virtual Worlds client can be reached directly or indirectly from this screen.

Options

Log Off

Click this menu item to disconnect your avatar from the virtual world you are presently in.

Directory

Click this menu item to display a dialog box that lists the other Virtual Worlds that are online. From this dialog box, you can go to the other worlds and their web pages. For more information, see the Directory dialog box topic.

Camera

Click this menu item to display a dialog box you can use to change the vantage point from which you see the virtual world you are in, including whether you see your own avatar and what parts of it you can see. For more information, see the Camera Views Dialog Box topic.

Options

Click this menu item to select several options, including how you want Virtual Worlds to display graphics, whether it should play sound, and what should be included in the text history pane. For more information, see the Options Dialog Box topic.

Help

Click this menu item to display Virtual Worlds Help. (You are reading a page in Virtual Worlds Help right now.) For more information, see Getting Additional Help.

  Avatars Tab

This tab appears in the upper right-hand corner of the client window and looks like a group of people standing together. Click it to display a menu whose options contain information about the avatars in the room with you. The menu also provides ways for you to modify and interact with the avatars, and displays their status (whether they are on your Friends or Ignored lists, and whether they belong to you). For more information, see the Avatars Tab topic.

  Room Contents Tab

This tab appears in the upper right-hand corner of the client window and looks like a house. Click it to display a menu whose options contain information about the objects in the room with you. The menu also provides ways for you to modify and interact with the objects. For more information, see the Room Contents Tab topic.

  My Things Tab

This tab appears in the upper right-hand corner of the client window and looks like a suitcase. Click it to display a menu whose options contain information about the objects you are carrying. The menu also provides ways for you to modify and interact with the objects. For more information, see the My Things Tab topic.

  Authoring Tools Tab

This tab appears in the upper right-hand corner of the client window and looks like a monkey wrench. Click it to display a dialog box with which you can teleport instantly to another room, add an object or a room to the world, or perform advanced scripting-related tasks such as entering VBScript commands or modifying methods and properties of objects in the world. This tab is only available to Wizards and Authors who have the full Software Development Kit installed. For more information, see the Authoring Tools Tab topic.

Normal View

Click this tab to display the screen from which you can navigate around a virtual world and interact with objects and avatars in it. This view is depicted above.

This tab is only available to Wizards and Authors who have the full Software Development Kit installed.

Geometry Editor

Click this tab to display a screen from which you can change some properties of objects as well as their position and orientation within the room. This tab is only available to Wizards and Authors who have the full Software Development Kit installed. For more information, see the Geometry Editor Interface page.

Boundary Editor

Click this tab to display a screen from which you can place boundaries around objects, shape the boundaries as you like, and establish the nature of the boundaries. This tab is only available to Wizards and Authors who have the full Software Development Kit installed. For more information, see the Boundary Editor Interface topic.

Self View

This view displays your avatar as it appears to others in the virtual world with you, in real time.

Gesture Buttons

With these buttons, shaped like small faces, you can make your avatar perform certain preset gestures, such as smiling, blushing, flirting, and so on. For more information on what these buttons do, see the Appearance Dialog Box topic.

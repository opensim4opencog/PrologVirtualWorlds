Avatars Tab

This tab in the Virtual Worlds client window presents a menu that helps you to get information about the avatars in the world with you, as well as interact with them in various ways. It also displays the status of the avatars (whether a particular avatar is on your Friends or Ignored lists, and whether it belongs to you).

Options

Avatar Name Highlighting

The name of every avatar in the room is listed in the tab next to a thumbnail graphic of the avatar. (A gavel will appear instead of a thumbnail for avatars who are Hosts.)

  • Blue: If an avatar's name is highlighted in blue, it means the avatar is the one you are currently using.
  • Green: If an avatar's name is highlighted in green, it means the avatar has been added to your Friends list.
  • Gray Italic: If an avatar's name is listed in gray italic letters, it means you are currently ignoring that avatar.
  • Black: If an avatar's name is listed in plain black, that means it falls into none of the other categories.

Avatar Shortcut Menu

If you click on an avatar's name, an avatar shortcut menu will drop down for that avatar, as depicted above. The shortcut menu allows you to get information about the avatars in the world with you, as well as whisper to them, ignore them, add them to your Friends list, approach them, and view various properties of those avatars.

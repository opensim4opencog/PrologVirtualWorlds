Tools Menu (Virtual Worlds Server)

The Tools menu of the Virtual Worlds Server application contains operations that set properties on the world such as its Directory Service listing and server security.

Tools Menu Options

Directory Server Address…

Opens the Directory Server Address dialog box, where you can set the location of your Directory Services server, a server that displays a list of all Virtual Worlds that have been published to it. Users can log on to any world listed on the Directory Services server just by clicking the world's name.

The Directory Server Address must be a fully qualified URL (Uniform Resource Locator).

Clear Performance Averages

This resets the statistics displayed in the Server Log's bottom status window.

User Authentication…

Note: Only available when using Microsoft NT. Opens the User Authentication dialog box, where you can turn on NTLM Authentication and specify a local group to further restrict access to a server within an authenticated domain.

For a client connecting to a server who hasn't been authenticated against the domain that the server machine belongs to, a username/password dialog box will be presented on the client machine. Most users within an intranet, will have been authenticated upon logging into their machine.

Local groups are groups greated on the local domain of the server. If this option is activated, only users who are members of the specified local group can access worlds on the server.

Set Max User Limit…

Note: Only available when using Microsoft NT. Sets the maximum number of users for the server. The minimum valid number is 1.

Show World in Directory Services

Displays the world's name in the Directory Services. Note that even if your world is not listed on the Directory Server, people can still log on to the world, as long as they know its URL (Uniform Resource Locator).

Take World Offline

Disconnects all avatars from the selected world, and closes it. Use this option when you want to make repairs to the world or other authoring changes. You can also use this option if you want to limit access to your world to certain hours of the day, or certain days of the week.

Web Address...

Opens the Web Address dialog box, where you can set the URL (Uniform Resource Locator) for a web page that contains information about your world. This is used by the Directory Services page.

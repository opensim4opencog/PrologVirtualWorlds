Window Menu (Virtual Worlds Server)

The Window menu of the Virtual Worlds Server application manages the display of multiple windows in the server application. You can rearrange the layout of the windows, switch to the Server Log and Statistics window, or switch to a specific world window.

Window Menu Options

Cascade

Displays all windows in stacked, offset format.

Tile

Displays all windows side by side so that they fill all the space in the Virtual Worlds Server window.

Arrange Icons

Tidies up the display of any minimized world windows, arranging them in a row along the bottom of the server window.

1 <World name> (Public): # Users

If checked, displays the World window on the top level.

2 Server Log and Statistics

If checked, displays the Server Log and Statistics window on the top level.

Note: If the server is hosting more than one world, each additional world has its own menu option, similar to the fourth menu item.

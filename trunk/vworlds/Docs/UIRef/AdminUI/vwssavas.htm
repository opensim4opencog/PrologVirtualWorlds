Save As... Dialog Box (Virtual Worlds Server)

You can save the world in a checkpoint file (extension .VWC) for the world, using a new filename. After you do this, all activity is switched to the world with the new name you specify: the registry is changed to point to the newly created checkpoint file, and any avatars in the world are sent a warning message, asking whether they want to switch to the newly named world.

To save the world with a new name:

  1. In the File name entry box, type in the new checkpoint filename.
  2. Click Save.

After a world has been saved to a checkpoint file, it can be restored by using the Open menu item of the File menu.

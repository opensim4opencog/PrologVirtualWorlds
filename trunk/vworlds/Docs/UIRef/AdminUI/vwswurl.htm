Web Address Dialog Box (Virtual Worlds Server)

Use this dialog box to set the URL (Uniform Resource Locator) of a web site that lists information about your world. For example, this web site can describe entry requirements to your world, the download size of the files needed, screen shots of the environment, help files, and any membership guidelines. This information is displayed in the Directory of worlds so that people can find out more about a world before entering it.

To use the Web Address dialog box:

  1. Type the URL for the web site that has information about your world. For example, if your world information page is MyWorldInfo.HTM at the web site www.goatworld.com, you would type http://www.goatworld.com/MyWorldInfo.htm.
  2. Click Set.

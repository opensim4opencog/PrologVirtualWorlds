View Menu (Virtual Worlds Server)

The View menu of the Virtual Worlds Server application enables or disables the display of status messages. When this setting is enabled (a check mark is shown), status messages appear along the bottom edge of the server window.

View Menu Options

Status Bar

Turns the display of status messages at the window bottom on or off. The status messages provide brief explanations about the purpose of each menu option on all menus. When none of the menu options is selected, the status bar message is, "For Help, press F1".

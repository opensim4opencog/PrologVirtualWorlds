Log Off Host Control Dialog Box

You can use this dialog box to log off another avatar (since you are a host avatar) according to the rules of conduct established by your world. Enter your reasons for logging off the avatar. These reasons are displayed to the avatar as they are logged out of the world.

An avatar who has host privileges cannot log off an avatar with wizard privileges.

Options

Reason for logging off...

Enter your reasons for logging off the avatar here. This text is displayed to the avatar as they are logged off. The following message is displayed to the avatar being logged off (and to everyone in the world):

Host (Name) is logging you off for the following reason: (reason here)

Log Off

Click this button to log off the selected avatar.

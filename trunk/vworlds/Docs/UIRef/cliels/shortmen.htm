Client Shortcut Menus

Client shortcut menus contain actions you can use on objects to change them or get information on them. If you right-click an object, a menu appears, and you can drag to select an option.

Client Shortcut Menu Options

You can only click objects that are in the same room as your avatar.

More about...

Displays the current description of the object.

Go Through

Moves your avatar through the portal. If the portal is a Room Portal, you will be taken to another room in the same world. If the portal is an Interworld Portal, you will be transported to another virtual world. (This menu item is only available on Room Portal and Interworld Portal objects.)

Show...

Displays the text and graphics of a web page. (This menu item is only available on Web Page objects.)

Take

Adds the object to your avatar's inventory. (This menu item is only available on certain objects. If you own an object and want to make it takeable, use the Properties... menu item.)

Approach ->>

Selecting this option makes your avatar approach the selected object. This will usually bring your avatar facing the front of the selected object, although it may bring you to a slightly different position, depending on the object's settings.

Properties...

Displays the property pages for the object, with which you can set certain attributes of the object. (This menu item is only available on objects you own, unless you are a Wizard.)

Avatar Profile Dialog Box

Use this dialog box to change the appearance or description of your avatar, to edit public and private information about yourself, to make other avatars your friends, and to ignore or unignore other avatars.

To find more information about other Dialog boxes that are connected to the Avatar Profile Dialog Box (such as the Details Dialog Box), follow the links provided in the following sections of this topic.

Options

Appearance

Use the Appearance option to decide which avatar to use, by previewing the appearance of various avatars, and seeing them perform a variety of gestures.

Description

Use the Description option to view or change the description of you that others see when they click your name in the Avatars pane of the Virtual Worlds client.

Details

Use the Details option to view or change information about yourself that Virtual Worlds will make available to others, and to make this information public, visible only to friends, or private.

Friends

Use the Friends option to view your Friends list or delete avatars from it.

Ignored

Click the Ignored option to see a list of avatars you are ignoring, or to remove an avatar from that list.

Description & Details (Other Avatar's Profiles) Dialog Box

Use this dialog box to view information about other avatars in Microsoft Virtual Worlds. Their description is always visible to other avatars. The detail information can be set to be viewed by everyone (public), just that avatar's friend list (friends), or just by that avatar (private).

Description

This is the text description that describes more about the avatar.

Details

The following details may be visible (depending on your viewing privileges set with that avatar):

  • Where from (place of origination)
  • Home page (the URL for their home page on the World Wide Web)
  • Gender
  • Real name (their real name used outside of Virtual Worlds)
  • E-Mail alias (their e-mail address for further contact)
  • Age

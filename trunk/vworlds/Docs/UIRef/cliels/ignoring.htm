Ignored (Avatar Profile) Dialog Box

Use this dialog box to list avatars you are ignoring, or to remove an avatar from that list.

Options

You're ignoring

This text field displays all the avatars you are currently ignoring in the world. Click an avatar's name to select it.

Remove

Click this button to remove the selected avatar from your Ignored list.

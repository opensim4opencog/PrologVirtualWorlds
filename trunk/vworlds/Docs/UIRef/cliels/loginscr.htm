Virtual Worlds Start Screen

You can select your avatar's name, set your avatar's profile, and choose the world you want to visit from this screen.

Options

Avatar Name

Select the name of the avatar, or select (new avatar) from the menu to create a new avatar. Your name appears over your avatar's head in three-dimensional worlds. If your avatar's name duplicates one in the world you are visiting, then a number will be appended to it to differentiate the avatars (for example, Ted3).

Delete

Deletes the selected avatar permanently from your computer.

Destination

Select a world you have already visited from this drop-down menu, or enter a new world. The syntax is servername/worldname. For example, the following are possible valid server/world combinations:

  • microsoft11/corpworld
  • tedsserver/tedsworld
  • homeworlds/virtualia
  • localhost/myworld

Localhost is your own computer. To use localhost as the server name, you must have Virtual Worlds Server running and a world created.

Directory

Displays the Directory dialog box, which lists worlds available for you to visit, their descriptions, and (if available) a URL to a related web page.

Options

Displays the Options dialog box, which allows you to change certain settings of Virtual Worlds.

Create a new world using the Create a World Wizard

Clicking this link launches the Create a World Wizard, with which you can create your own world from several different types of pre-built kits, including houses, offices, outdoor scenes, and a tutorial world.

Create a custom avatar using the Avatar Construction Kit

Clicking this link launches the Avatar Construction Kit, with which you can create custom avatars that have their own animated gestures and thumbnails.

Visit the Virtual Worlds external web site

Clicking this link takes you to the Virtual Worlds external web site, where you can find more help, or contact a community of Virtual Worlds wizards, authors, scripters, and artists.

Details (Avatar Profile) Dialog Box

Use this dialog box to view or change information about yourself that Virtual Worlds makes available to others, and to make these details public, private, or visible only to friends.

Options

You can click the menu next to any of the detail options listed below to change the access permissions for that detail. You can choose from the following three options:

  • Public: Everyone in the virtual world can read this information.
  • Friend: Only the people on your Friends list can read this information.
  • Private: Only you can read this information.

LastRemoteSpriteURL

This Profile key tracks and allows users to utilize avatar graphics located on remote file shares and web servers. In the past, there was no convenient way for users to utilize avatar graphics that weren't already installed into their /Local Content/Avatar Graphics local directory.

Where from

Type the town, city, or region where you live into this text box.

Home page

Type a URL for your home page on the World Wide Web into this text box.

Gender

Type your gender into this text box. This is a text box, so you can type whatever you like, including, "male," "female," "M," "F," "n/a," or "none of your business."

Real name

Type your real name into this text box. This should probably be the name you use in real life, not the name of your avatar, which is already available to everyone. Remember, if you do not want your name to be available to everyone, you can make the access on this detail Friends or Private (see above).

E-Mail alias


Type an e-mail address into this text box.  Remember, if you do not want your e-mail address to be available to everyone, you can make the access on this detail Friends or Private (see above).

Age

Type your age into this text box.

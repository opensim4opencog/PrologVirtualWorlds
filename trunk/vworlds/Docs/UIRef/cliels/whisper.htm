Whisper Dialog Box

You can whisper to one or more avatars in your world with this dialog box. Whispering is a type of chatting in which only those people you have specifically selected can read your messages.

Options

Avatars

Displays all the avatars to whom you could whisper, but are not currently whispering.

Whisper To

Displays all the avatars to whom you are currently whispering.

Add

Moves the selected avatars from the Avatars field to the Whisper To field, so that you can whisper to them.

Remove

Moves the selected avatars from the Whisper To field to the Add field, so that you are no longer whispering to them.

Show avatars in all rooms in this world

Displays the names of avatars in other rooms in the world. If this box is not checked, you only see the names of avatars in the room you are in.

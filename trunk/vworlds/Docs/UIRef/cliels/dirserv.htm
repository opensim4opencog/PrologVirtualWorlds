Directory Dialog Box

The Directory shows you which other Virtual Worlds are online, links to those worlds' web pages, if available, and how many people are logged into that world. You can also use the Directory to enter any world listed.

Options

Directory Main Pane

The Virtual Worlds Directory is displayed in this pane, with the world, the number of avatars, the maximum occupancy of the world, and the web page for the world, if available. Click the name of the world you want to enter.

Clicking on the name of a world in this pane logs you off from your current world before you enter the new one. If you want to be in more than one world at the same time, open another session of Virtual Worlds.

Refresh

Click this button to update the list of worlds available.

Starting and Running the Server

Starting the server:

  1. On the Start menu, click Programs, and then click Microsoft Virtual Worlds.
  2. Click Virtual Worlds Server.

Running the server:

Once you have started the server, you do not need to perform any special actions to keep it running. The server is aware of all worlds that have been created on the server in the past and will show existing ones in the server window. Worlds will not be loaded into memory until they are connected to. Worlds created on the server after the server was started will not show up in the server window unless opened from the File->Open menu item, or when the first user explicitly enters the new world directly (i.e. not via Directory Services).

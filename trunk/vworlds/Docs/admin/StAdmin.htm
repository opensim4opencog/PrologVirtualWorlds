Server Reference Start Page

Purpose

The Virtual Worlds Server application controls the worlds created and loaded for Virtual Worlds client programs.

The Virtual Worlds Server must be running on a computer you can access before you can create a new world or access a world with the Virtual Worlds client program from a different machine.

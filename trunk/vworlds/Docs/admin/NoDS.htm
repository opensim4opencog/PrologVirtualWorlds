Adding Directory Services

The Directory shows you which other Virtual Worlds are online, as well as links to those worlds' web pages, if available, and how many people are logged into that world. You can also use the directory to enter any linked world by clicking on an entry.

Directory Services is not active right after installing Microsoft Virtual Worlds. To get the Directory entries for all the worlds, you or your system administrator can set up a server computer to manage the Directory.

After you install Virtual Worlds, no entries will appear in the Directory dialog box. In the administration facility, Publish to Directory Services will give you an error message.

Your system administrator can set up an active Directory Services web server by contacting Virtual Worlds (vworlds@microsoft.com) for specific instructions and files. To support a Directory Service for your users, your system administrator will need two additional files (an .ASP and a .HTML file) from Microsoft. These files should be installed on a server computer running Windows NT Option Pack, IIS, and Site Server. Your administrator also needs to set two registry keys on every client computer.
